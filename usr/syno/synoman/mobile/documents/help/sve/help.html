DSM mobile ger DSM admin (eller en användare som hör till gruppen administrators) möjlighet att hantera DSM-inställningar och kontrollera DSM-informationen med en iOS® (iPhone®, iPad®, eller iPod touch®) eller Android®-enhet. Detta hjälpavsnitt om DSM mobile innehåller följande ämnen och resurser för att hjälpa dig att komma igång.

Logga in och Logga ut

Efter att du anslutit till DSM med webbläsaren på din iOS- eller Android-enhet kommer du att se inloggningsskärmen för inmatning av dina DSM-kontouppgifter.

Så här gör du för att logga in på DSM mobile:

Skriv in dina DSM-användaruppgifter och tryck på Logga in. Om du vill spara din inloggningsinformation och logga in automatiskt nästa gång, sätt på Kom ihåg mig innan du loggar in.

Så här gör du för att logga ut från DSM mobile:

  1. Tryck på alternativknappen högst upp till höger på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet).
  2. Tryck på Inställningar och tryck på Logga ut.

Hantera DSM-användare

Tryck på Användare & grupper på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att se listan över användare.

Skapa en användare:

  1. Tryck på + och skriv sedan in användarnamnet och lösenordet och e-postadressen för den nya användaren.
  2. Aktivera Skicka meddelande för att skicka ett meddelande till användarens e-postadress när hans eller hennes DSM-användarkonton har skapats.
    Obs: Innan meddelandet skickas, se till att meddelandeinställningarna har slutförts på din Synology NAS. Logga in på skrivbordsversionen av DSM med din dators webbläsare och gå till Huvudmeny > Kontrollpanel > Meddelande > E-post för att ändra inställningarna.
  3. Tryck på Klart för att bekräfta eller tryck den vänstra pilen för att avbryta.

Så här gör du för att snabbt hitta en användare:

Skriv in användarens namn i sökfältet.

Så här gör du för att ändra en användares uppgifter:

  1. Tryck på användarens namn och gör något av följande:
    • Ändra användarens namn eller lösenord i fältet Namn eller Lösenord.
    • Välj Avaktivera för att avaktivera användaren. Du kan välja att aktivera användaren vid ett senare tillfälle utan att förlora hans eller hennes inställningar.
    • Aktivera Administrator för att lägga till användaren till gruppen admin. Användare som tillhör gruppen admin tilldelas administrativa privilegier för att ändra DSM-inställningarna med DSM mobile eller datorns webbläsare.
  2. Gör något av följande under Privilegier och tryck sedan på Bakåt:
    • Tryck på Delad mapp och tryck på den delade mappens namn för att ändra användarens åtkomstprivilegier: Läsa/skriva: Användaren kan göra ändringar i filerna och undermapparna i den delade mappen; Endast läsbar: Användaren kan komma åt filerna och undermapparna i den delade mappen, men kan inte göra ändringar i dem; Ingen åtkomst: Användaren kan inte komma åt filerna och undermapparna i de delade mapparna.
      Obs: Om det uppstår en konflikt mellan privilegier kommer prioriteten att vara som följande: Ingen åtkomst > Läsa/skriva > Endast läsbar.
    • Tryck på Applikation för att ändra användarens åtkomstprivilegier till DSM-applikationer.
  3. Tryck på Klart för att bekräfta eller tryck den vänstra pilen för att avbryta.

Hantera delade mappar

Tryck på Delad mapp på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att se listan över delade mappar på din Synology NAS.

Så här gör du för att skapa en delad mapp:

  1. Tryck på + och skriv sedan in namnet för den nya mappen.
  2. Tryck på Klart för att bekräfta eller tryck den vänstra pilen för att avbryta.

Så här gör du för att ändra en delad mapp:

Tryck på den delade mappens namn och gör något av följande:

  • Ändra den delade mappens namn i fältet Namn.
  • Tryck på Användarbehörigheter och tryck på en användare för att ändra hans eller hennes åtkomstprivilegier till den delade mappen: Läsa/skriva: Användaren kan göra ändringar i filerna och undermapparna i den delade mappen; Endast läsbar: Användaren kan komma åt filerna och undermapparna i den delade mappen, men kan inte göra ändringar i dem; Ingen åtkomst: Användaren kan inte komma åt filerna och undermapparna i de delade mapparna.
    Obs: Om det uppstår en konflikt mellan privilegier kommer prioriteten att vara som följande: Ingen åtkomst > Läsa/skriva > Endast läsbar.
  • Du kan även trycka på Gruppbehörigheter för att konfigurera privilegier för grupper.
    Obs: Gruppbehörigheter har högre prioritet än användarbehörigheter.

Aktivera tjänster

Tryck på Tjänst på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-datorplatta) för att aktivera eller avaktivera Synology NAS-tjänster.

Att hantera LAN-tjänster i Synology NAS:

  • Aktivera eller avaktivera något av följande alternativ:
    • Windows-filservice låter din Synology NAS bli en Windows-filserver (CIFS) inom LAN.
    • MAC-filservice låter din Synology NAS bli en Mac-filserver (AFP) inom LAN.
    • NFS-tjänst låter Linux-klienter inom LAN komma åt Synology NAS-data.
    • Bonjour-skrivare låter Mac-klienter skriva ut på USB-skrivare som anslutna till din Synology NAS.
    • Media Server låter DLNA/UPnP-hemmaenheter inom LAN att bläddra bland och spela upp multimediainnehåll på din Synology NAS.
    • iTunes låter iTunes-klienter inom LAN bläddra bland och spela upp musik eller videor på din Synology NAS.

Att hantera Internettjänster i Synology NAS:

  • Aktivera eller avaktivera något av följande alternativ:
    • FTP aktiverar din Synology NAS så att den blir en FTP-server.
    • Webbtjänster låter dig vara värd för webbsidor (med PHP+MySQL-stöd) på din Synology NAS.
    • WebDAV låter klientprogram hantera och redigera filer på Synology NAS via HTTP-protokollet.
    • WebDAV (HTTPS) låter klientprogram hantera och redigera filer på Synology NAS via HTTPS-protokollet.
    • File Station är en webbaserad applikation som gör att du kan hantera filer på Synology NAS genom en anpassad port utan att behöva logga in på DSM.
    • File Station (HTTPS) gör att du kan ansluta till File Station via HTTPS-protokollet.
    • Surveillance Station är en webbaserad applikation som kan hantera IP-kameror för att skydda din hem- eller kontorsmiljö.
    • Audio Station är en webbaserad applikation som ger dig åtkomst till musikbiblioteket på din Synology NAS.
    • Photo Station är ett fotoalbum online som är integrerad med en blogg som sparas på din Synology NAS.
    • Download Station (BT/HTTP/FTP/NZB/RSS) är en webbaserad applikation som hjälper dig att ladda ner filer från Internet till din Synology NAS.
    • eMule aktiverar eMule-stödet för Download Station.

Obs:

För mer information om de ovannämnda tjänsterna hänvisas du till följande:

  • DSM hjälp (tillgänglig när du loggar in på skrivbordsutgåvan av DSM med din dators webbläsare och går till Huvudmeny > DSM hjälp)
  • Bruksanvisning för Synology DSM (finns på Synology:s Nerladdningscenter)

Övervaka systemresurser

Tryck på Resursskärm på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att övervaka processor- och minnesanvändning på Synology NAS.

Visa systeminformation

Tryck på Systeminfo på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att se en statusöversikt för din Synology NAS.

Så här gör du för att se systeminfo för din Synology NAS:

  1. Gör något av följande:
    • Tryck på Allmänt för att se den grundläggande informationen om din Synology NAS, inklusive serienummer, modellnamn, mängden fysiskt minne, DSM-versionen, samt tidinformation och temperaturstatus.
    • Tryck på Nätverk för att nätverkskonfigurationen för din Synology NAS samt statusen för varje nätverksgränssnitt.
    • Tryck på Lagring för att se volymanvändning för Synology NAS.
  2. Tryck på knappen Uppdatera högst upp till höger för att hålla systeminformation uppdaterad.

Fast programvara

Tryck på Fast programvara för att visa och uppdatera din DSM-version.

Att konfigurera inställningar för uppgradering av DSM:

  1. Tryck på DSM-uppgradering.
  2. Välj valfritt av följande alternativ:
    • Endast viktiga uppdateringar
    • Nyaste DSM och alla uppdateringar
  3. Tryck på Klart.

Att ladda ner tillgängliga uppdateringar:

  1. Kontrollera din DSM-version i Status.
  2. Om din DSM-version inte är uppdaterad, tryck på knappen Ladda ner som finns under avsnittet Status för att uppgradera din DSM.

Hantera IP-blockering

Tryck på IP-blockering på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-datorplatta) för att förhindra oauktoriserad inloggning.

Att hantera IP-blockering:

  1. Tryck på knappen för konfiguration i det övre högra hörnet.
  2. Sätt på Aktivera auto block för att aktivera IP-blockering.
  3. Ange en siffra för Inloggningsförsök och Inom (minuter) för att blockera en IP-adress efter det specificerade antalet inloggningsförsök inom angivet antal minuter.
  4. Du kan sätta på Aktivera förfallotid för blockering och specificera Ta bort blockering efter (dagar) för att ta bort blockeringen av en IP-adress efter det specificerade antalet dagar. Om du anger ”0” här kommer IP-adressen att blockeras på obestämt tid till dess att du tar bort den från blocklistan.
  5. Tryck på Klart.

Att lägga till IP-adresser till tillåten lista/blockeringslista:

  1. Om du vill lägga till den tillåtna/blockerade IP-adressen till listan, tryck på knappen för att lägga till i det övre högra hörnet och tryck på Tillåten lista eller Blockeringslista.
  2. Efter att IP-adressen angivits, tryck på Klart.

Att hantera Paketcenter:

Att köra eller stoppa installerade paket:

Tryck på väljaren bredvid ett filpaket för att köra eller stoppa det.

Att visa filpaket:

Tryck på ett filpaket för att se dess information.

Att uppdatera filpaket:

  1. Tryck på fliken Uppdatera för att visa vilka versioner som för närvarande är tillgängliga för uppdatering.
  2. Tryck på knappen på den högra sidan av ett filpaket för att uppdatera till den senaste versionen.

Kör mobilapplikationer

DSM mobile ger dig ett praktiskt utrymme för att köra Synology-mobilapplikationer som finns installerade på din iOS- eller Android-enhet.

Så här gör du för att köra Synology-mobilapplikationer med DSM mobile:

Tryck på en av följande applikationer i den nedre delen av startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-datorplatta):

Välj en åtgärd för Synology NAS
(starta om, stäng av, eller pip)

Med DSM mobile kan du starta om eller stänga av din Synology NAS med din iOS- eller Android-enhet. Om det finns flera Synology NAS i närheten kan du dessutom snabbt identifiera din Synology NAS genom att be den spela upp pipljud.

Att välja en åtgärd för Synology NAS:

  1. Tryck på alternativknappen högst upp till höger på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet).
  2. Gör något av följande:
    • Tryck på Starta om och tryck sedan på Ja för att starta om din Synology NAS.
    • Tryck på Stäng av och tryck sedan på Ja för att stänga av din Synology NAS.
    • Tryck på Hitta mig för att be Synology NAS avge ett pipljud. Tryck på OK för att stoppa pip-ljudet.

Använda Lagringshanterare

Lagringshanterare låter dig snabbt se användningen och hälsan för systemets volymer och diskar.

Att visa volyminformation:

  1. Tryck på Lagringshanterare på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-datorplatta).
  2. Tryck på fliken Volym.
  3. Tryck på en volym i listan för att visa dess detaljerade information.

Att visa diskinformation:

  1. Tryck på Lagringshanterare på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-datorplatta).
  2. Tryck på fliken HDD/SSD.
  3. Tryck på en disk i listan för att visa dess information om hälsa och S.M.A.R.T.-information.

Att starta eller stoppa ett S.M.A.R.T.-test:

  1. Tryck på Lagringshanterare på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-datorplatta).
  2. Tryck på fliken HDD/SSD.
  3. Tryck på en disk i listan för att visa dess sida med information.
  4. Tryck på S.M.A.R.T. Test för att starta ett S.M.A.R.T.-test eller Stopp för att stoppa det.
