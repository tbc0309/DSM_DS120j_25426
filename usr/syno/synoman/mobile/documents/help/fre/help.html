Avec DSM mobile, l'admin (ou un utilisateur appartenant au groupe administrators) peut gérer les paramètres DSM et vérifier les informations DSM à l'aide d'un périphérique iOS® (iPhone®, iPad® ou iPod touch®) ou Android®. Cette aide de DSM mobile met à votre disposition les rubriques et ressources suivantes permettant de vous assister au début de votre utilisation.

Remarque :

Vous trouverez ci-dessous les périphériques mobiles et les navigateurs actuellement pris en charge :

  • Périphériques mobiles : iPhone, iPad, et iPod sous iOS 10 ou version ultérieure et périphériques sous Android 6.0 ou version ultérieure
  • Navigateurs : Chrome, Safari

Connexion et déconnexion

Une fois connecté au DSM à l'aide de votre navigateur web sur votre périphérique iOS ou Android, vous verrez l'écran de connexion dans lequel vous pouvez saisir les informations de votre compte DSM.

Pour se connecter à DSM mobile :

Saisissez vos identifiants d'utilisateur DSM et appuyez sur Connexion. Si vous souhaitez enregistrer vos informations de connexion et vous connecter automatiquement la prochaine fois, activez l'option Se souvenir de moi avant de vous connecter.

Pour vous déconnecter de DSM mobile :

  1. Appuyez sur le bouton d'option situé dans l'angle supérieur droit de la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android).
  2. Appuyez sur Paramètres, puis sur Déconnexion.

Gérer les utilisateurs DSM

Appuyez sur Utilisateur et groupe dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android) pour afficher la liste des utilisateurs.

Pour créer un utilisateur :

  1. Appuyez sur + puis saisissez le nom de l'utilisateur, son adresse électronique et le mot de passe du nouvel utilisateur.
  2. Activez l'option Envoyer notification pour envoyer un e-mail de notification à l'adresse e-mail de l'utilisateur lorsque son compte utilisateur DSM est créé.
    Remarque : Avant d'envoyer une notification, assurez-vous que les paramètres de notification ont été configurés sur votre Synology NAS. Connectez-vous à l'édition desktop de DSM à l'aide du navigateur Web de votre ordinateur, puis accédez à Menu principal > Panneau de configuration > Notification > E-mail pour modifier les paramètres.
  3. Appuyez sur Terminé pour confirmer ou appuyez sur la flèche gauche pour annuler.

Pour détecter rapidement un utilisateur :

Entrez le nom de l'utilisateur dans le champ Recherche.

Pour modifier les informations d'un utilisateur :

  1. Appuyez sur le nom de l'utilisateur et procédez comme suit :
    • Modifiez le nom de l'utilisateur ou le mot de passe dans le champ Nom ou Mot de passe.
    • Appuyez sur Désactiver pour désactiver l'utilisateur. Vous pouvez choisir d'activer le compte de cet utilisateur plus tard sans perdre ses paramètres.
    • Activez l'option Administrateur pour ajouter l'utilisateur au groupe admin. Tous les utilisateurs appartenant au groupe admin bénéficient de privilèges administratifs leur permettant de modifier les paramètres de DSM sur DSM mobile ou le navigateur Web d'un ordinateur.
  2. Sous Privilèges, effectuez l'une des opérations suivantes puis appuyez sur Retour :
    • Appuyez sur Dossier partagé, puis sur le nom d'un dossier partagé pour modifier les privilèges d'accès de l'utilisateur : Lecture/écriture : L'utilisateur peut apporter des modifications aux fichiers et sous-dossiers du dossier partagé ; Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers du dossier partagé, mais il ne peut pas y apporter de modifications. Aucun accès : L'utilisateur ne peut pas accéder aux fichiers et sous-dossiers du dossier partagé.
      Remarque : Si des conflits de privilèges se produisent, la priorité du privilège sera : Aucun accès > Lecture/écriture > Lecture seule.
    • Appuyez sur Application pour modifier les privilèges d'accès de l'utilisateur aux applications DSM.
  3. Appuyez sur Terminé pour confirmer ou appuyez sur la flèche gauche pour annuler.

Gérer des dossiers partagés

Appuyez sur Dossier partagé dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android) pour afficher la liste des dossiers partagés sur votre Synology NAS.

Pour créer un dossier partagé :

  1. Appuyez sur + puis saisissez le nom du nouveau dossier.
  2. Appuyez sur Terminé pour confirmer ou appuyez sur la flèche gauche pour annuler.

Pour modifier un dossier partagé :

Appuyez sur le nom du dossier partagé et procédez comme suit :

  • Modifiez le nom du dossier partagé dans le champ Nom.
  • Appuyez sur Autorisations de l'utilisateur, puis sur un utilisateur pour modifier ses privilèges d'accès au dossier partagé : Lecture/écriture : L'utilisateur peut apporter des modifications aux fichiers et sous-dossiers du dossier partagé ; Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers du dossier partagé, mais il ne peut pas y apporter de modifications. Aucun accès : L'utilisateur ne peut pas accéder aux fichiers et sous-dossiers du dossier partagé.
    Remarque : Si des conflits de privilèges se produisent, la priorité du privilège sera : Aucun accès > Lecture/écriture > Lecture seule.
  • Vous pouvez également appuyer sur Autorisations des groupes pour configurer des privilèges pour les groupes.
    Remarque : Les permissions des groupes ont une priorité supérieure aux permissions des utilisateurs.

Activer les services

Appuyez sur Service dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android) pour activer ou désactiver les services du serveur Synology NAS.

Pour gérer les services LAN du serveur Synology NAS :

  • Activez ou désactiver l'un des services suivants :
    • AFP : votre Synology NAS peut devenir un serveur de fichiers Mac sur le réseau LAN.
    • Diffusion imprimante Bonjour : les clients Mac peuvent imprimer vers la ou les imprimantes USB connectées à votre Synology NAS.
    • Service NFS : les clients Linux du réseau LAN peuvent accéder aux données du Synology NAS.
    • SMB : les clients Windows, Mac et Linux prenant en charge le protocole SMB/CIFS peuvent accéder aux données stockées sur votre Synology NAS au sein du réseau LAN.
    • SSH/Telnet : votre Synology NAS peut prendre en charge les services de l'interface de ligne de commande Telnet et SSH.

Pour gérer les services Internet du serveur Synology NAS :

  • Activez ou désactiver l'un des services suivants :
    • FTP/FTPS : votre Synology NAS peut devenir un serveur FTP.
    • Service NTP : votre Synology NAS peut devenir un serveur de temps réseau pour synchroniser l'heure entre différents périphériques du réseau et le Synology NAS via le réseau.
    • rsync : vous pouvez sauvegarder des données via rsync à partir d'un Synology NAS ou de clients compatibles rsync sur votre Synology NAS.
    • SFTP : le transfert de fichiers est sécurisé pour les utilisateurs de DSM sans besoin de mots de passe, et le risque de fuite des identifiants de l'utilisateur est réduit.

Remarque :

Pour plus d'informations sur les services mentionnés ci-dessus, consultez l'une des méthodes suivantes :

  • Aide DSM (disponible en vous connectant à l'édition desktop de DSM à l'aide du navigateur Web de votre ordinateur et en accédant à Menu principal > Aide DSM)
  • Guide de l'utilisateur de Synology DSM (disponible dans le Centre de téléchargement de Synology)

Afficher les infos système

Appuyez sur Informations système dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android) pour obtenir une vue d'ensemble de l'état de votre serveur Synology NAS.

Pour afficher les informations système de votre serveur Synology NAS :

  1. Procédez de l'une des manières suivantes :

Progiciel

Appuyez sur Progiciel pour afficher votre version de DSM et la mettre à jour.

Pour configurer les paramètres de mise à jour de DSM :

  1. Appuyez sur Mise à niveau de DSM.
  2. Choisissez l'une des options suivantes :
    • Mises à jour importantes uniquement
    • DSM le plus récent et toutes les mises à jour
  3. Appuyez sur Terminé.

Pour télécharger les mises à jour disponibles :

  1. Vérifiez votre version de DSM dans État.
  2. Si votre version de DSM n'est pas à jour, appuyez sur le bouton Télécharger qui s'affiche en dessous de la section État pour mettre à jour DSM.

Gestion du blocage des IP

Appuyez sur Blocage d'une adresse IP dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android) pour empêcher une connexion non autorisée.

Pour gérer le blocage des IP :

  1. Appuyez sur le bouton de configuration, dans le coin supérieur droit.
  2. Activez l'option Activer le blocage automatique pour activer le blocage des IP.
  3. Saisissez un nombre pour Tentatives de connexion et Dans les (minutes) pour bloquer une adresse IP après un nombre spécifique d'échecs de tentatives de connexion sous le nombre de minutes spécifié.
  4. Vous pouvez activer l'option Activer l'expiration des blocages et spécifier Débloquer après (jours) pour débloquer une adresse IP après le nombre de jours indiqué. Si vous entrez « 0 » ici, l'Adresse IP bloquée restera bloquée indéfiniment jusqu'à ce que vous la retiriez de la liste des blocages.
  5. Appuyez sur Terminé.

Pour ajouter des adresses IP à la liste des autorisations/des blocages :

  1. Si vous souhaitez ajouter l'IP autorisée/bloquée à la liste, appuyez sur le bouton d'ajout dans l'angle supérieur droit, puis sur Liste des autorisations ou Liste des blocages.
  2. Après avoir saisi l'adresse IP, appuyez sur Terminé.

Configurer le serveur Synology NAS dans le menu action
(redémarrer, éteindre ou émettre des bips)

DSM mobile vous permet de redémarrer ou d'éteindre votre serveur Synology NAS à l'aide d'un périphérique iOS ou Android. De plus, lorsque plusieurs Synology NAS sont disponibles, vous pouvez rapidement identifier votre serveur Synology NAS en lui demandant d'émettre des bips.

Pour configurer votre serveur Synology NAS avec le menu action :

  1. Appuyez sur le bouton d'option situé dans l'angle supérieur droit de la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android).
  2. Procédez de l'une des manières suivantes :
    • Appuyez sur Redémarrer, puis sur Oui pour redémarrer votre serveur Synology NAS.
    • Appuyez sur Éteindre, puis sur Oui pour éteindre votre serveur Synology NAS.
    • Appuyez sur Me trouver pour demander à votre serveur Synology NAS d'émettre un bip. Appuyez sur OK pour arrêter le signal sonore.

Utiliser le gestionnaire de stockage

Le Gestionnaire de stockage vous permet d'afficher rapidement l'utilisation et l'état de santé des volumes et des disques du système.

Pour afficher les informations des volumes :

  1. Appuyez sur Gestionnaire de stockage dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android).
  2. Appuyez sur l'onglet Volume.
  3. Appuyez sur un volume dans la liste pour afficher ses informations détaillées.

Pour afficher les informations sur le disque :

  1. Appuyez sur Gestionnaire de stockage dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android).
  2. Appuyez sur l'onglet HDD/SDD.
  3. Appuyez sur un disque dans la liste pour afficher son état de santé et les informations S.M.A.R.T.

Pour démarrer ou arrêter le test S.M.A.R.T. :

  1. Appuyez sur Gestionnaire de stockage dans la page d'accueil (iPhone/iPod touch, téléphone Android) ou dans le panneau de gauche (iPad, tablette Android).
  2. Appuyez sur l'onglet HDD/SDD.
  3. Appuyez sur un disque dans la liste pour afficher sa page d'information.
  4. Appuyez sur S.M.A.R.T. Appuyez sur Test pour lancer un test S.M.A.R.T. ou sur Arrêter pour l'arrêter.
