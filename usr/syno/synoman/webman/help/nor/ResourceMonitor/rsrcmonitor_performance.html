Ytelse

Klikk på kategorien Ytelse for å vise data for ressursutnyttelse. Detaljert dato, tidspunkt og informasjon i tabellen vises når du beveger musen over.

CPU

Du kan se det samlede CPU-forbruket for Bruker, System og I/O Vent. CPU-belastningen kan være høy når ressursovervåkeren først startes, siden systemet må samle inn sine ressursdata og laste inn grensesnittsiden samtidig.

Bruker viser prosentandel av CPU som brukes for å kjøre programmer. System viser prosentandel av CPU som brukes i systemets kjerne, maskinvare-/programvare-IRQ. I/O Vent viser prosentandel av CPU som brukes på I/O Vent.

Utnyttelse er summen av CPU-forbruk for Bruker og System.

Belastningsgjennomsnitt innenfor 1, 5 og 15 minutter vises med kjørekølengde (dvs. summen av antall prosesser som kjører pluss antallet som venter på å kjøre).

Begrepsforklaring

  • I/O Vent: I/O Vent indikerer at CPU er inaktiv og ikke brukes av noen programmer.

    • I/O Vent er normalt. Siden hastigheten til CPU er raskere enn hastigheten til I/O, er det normalt at CPU må vente på at overføringen av disker eller nettverk skal fullføres. Men hvis prosentandelen av I/O Vent er ekstremt høy, betyr det at CPU bruker mesteparten av tiden sin på I/O Vent og er muligens ikke godt utnyttet. Du kan bruke mer effektive diskgruppekonfigurasjoner (for eksempel SHR eller RAID 5) eller nettverkskonfigurasjoner (for eksempel Bonding) for å oppnå bedre systemytelse.

Minne

Det samlede minneforbruket for det fysiske minnet og veksleplass vil bli vist. Minebruken holder seg høy fordi systemet lagrer data som det bruker ofte i bufferen, slik at det får rask tilgang til dataene uten å bruke harddisken. Bufferminnet frigis hvis det generelle minnet er utilstrekkelig.

Høyt forbruk av veksleplass indikerer utilstrekkelig systemminne og vil også påvirke systemytelsen. Du kan vise hastigheten for bytt inn og bytt ut ved å velge Bytt fra rullegardinmenyen øverst.

Nettverk

Nettverksflyttabellen viser hastighet for sendt og mottatt data i KB per sekund. Hvis PPPoE er tilkoblet, blir også overføringshastigheten vist i tabellen.

Disk

Du kan velge å vise den samlede diskutnyttelsen, overføringshastighet eller IOPS ved å klikke på rullegardinmenyen øverst i tabellen. Du kan klikke på Vis alle nederst for å velge flere disker ved å merke av for å legge til deres data i tabellen.

Volum

Denne siden viser volumenes generelle status. Klikk i rullegardinmenyen ovenfor tabellen for å vise overføringsbruk, overføringshastighet eller IOPS. Klikk på knappen Vis alle for å velge enkeltvolum og legge dem til i tabellen.

iSCSI

Denne siden viser kjørende iSCSI LUN-ers generelle status. Klikk på rullegardinmenyen ovenfor tabellen for å velge et LUN og se statusen. Tre tabeller vises som standard, nemlig Gjennomstrømming, IOPS og Ventetid. For mer detaljert informasjon, kan du klikke på Velg tabeller-knappen for å velge andre tabeller.
