Oppgavebehandling*

I Oppgavebehandling-fanen kan du vise CPU- og minneforbruk for alle tjenester og prosesser.

Tjenester

Klikk på Tjenester-fanen for å overvåke ulike verdier for alle tjenester, inkludert CPU-forbruk, CPU-tid, minneforbruk og diskens lese/skrivestatus.

Prosesser

Klikk på Prosesser-fanen for å vise CPU- og minneforbruk for hver prosess i rekkefølge etter minkende CPU-belastning.

Prosessen Status inkluderer Kjører, Sover og Stoppet. Linux-ekvivalenter av prosesstatus er kjører, stoppet/sporer, sover.

Sover kan etterfølges av en bokstav for detaljert informasjon, for eksempel 'D' for disksøvn, 'Z' for zombie, 'X' for død.

____
* Bare tilgjengelig på bestemte modeller.
