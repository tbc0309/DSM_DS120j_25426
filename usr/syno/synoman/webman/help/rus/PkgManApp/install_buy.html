Установка и приобретение пакетов

Центр пакетов предоставляет разнообразные веб-приложения, которые могут расширить функциональные возможности Synology NAS. С помощью Центра пакетов можно скачивать, устанавливать и запускать пакеты несколькими щелчками мыши.

В приведенной ниже статье описана установка пакетов на Synology NAS с помощью Центра пакетов. Информацию об отдельных пакетах см. здесь.

Установка и приобретение пакетов:

  1. Откройте Центр пакетов.
  2. Перейдите на страницу Все пакеты для просмотра всех доступных пакетов.

    3. Примечание.

    • Если вы не можете найти нужный пакет, возможно, для этого пакета требуется более поздняя версия DSM, либо он доступен только на моделях с определенными требованиями к оборудованию. Для получения дополнительной информации свяжитесь с разработчиком этого пакета.
  3. Найдите пакет, который необходимо установить, и нажмите Установить. (Для платных пакетов нажмите Купить или Пробная версия, чтобы подключиться к учетной записи Synology и оплатить покупку с помощью кредитной карты или воспользоваться пробной версией в целях ознакомления.)
  4. После успешной установки пакет должен появиться в Главном меню.

Приобретение пакетов:

Перед приобретением пакетов обратите внимание на следующие примечания.

  • Зарегистрируйте учетную запись Synology для доступа к таким дополнительным функциям, как предварительная платежная информация и журнал покупок. Информация, которую вы отправляете, будет использоваться исключительно для повышения качества услуг в будущем.
  • По умолчанию цены на пакеты указаны в долларах США. После входа в учетную запись Synology цены будут указаны в валюте, соответствующей настройкам учетной записи.

Отказ от гарантий относительно конвертации валют:

Банк, выпустивший ваши кредитную/дебетовую карту, может взимать дополнительные сборы за операции, включающие конвертацию валюты. Такие сборы не отражены в указанной цене. Банк, выпустивший вашу карту, может взимать сборы за любые покупки, включающие обмен иностранных валют и международные операции. Банк, выпустивший вашу карту, проведет оценку всех применяемых сборов, за которые компания Synology не может выплатить компенсацию. Для получения дополнительной информации обратитесь в банк, выпустивший кредитную/дебетовую карту.

Установка пакетов вручную:

  1. Чтобы установить пакет с помощью выгрузки файла .spk (доступен в Центре загрузок Synology или на веб-сайтах сторонних производителей), нажмите Установка вручную и следуйте инструкциям мастера для установки нового пакета.

Поиск пакета:

  1. Введите ключевое слово в строке поиска в левом верхнем углу и нажмите Enter на клавиатуре, чтобы начать поиск.

Выбор способа сортировки пакетов:

Вы можете изменить порядок отображения пакетов в Центре пакетов. Нажмите раскрывающееся меню под баннером и выберите один из вариантов сортировки пакетов:

  • Сортировать по названию: нажмите, чтобы отсортировать пакеты в алфавитном порядке.
  • Сортировать по популярности: нажмите, чтобы отсортировать пакеты по количеству установок, начиная с самых популярных.

Скрытие установленных пакетов:

выберите любую категорию, нажмите раскрывающееся меню под баннером и выберите Скрыть установленные пакеты. В результате уже установленные пакеты будут скрыты, а пакеты, которые еще не установлены, не обновлены или не восстановлены, будут отображаться.
