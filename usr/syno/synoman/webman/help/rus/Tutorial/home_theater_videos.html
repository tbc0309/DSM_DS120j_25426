Определение категорий и воспроизведение видео с помощью Video Station

Обзор

С помощью Video Station можно с легкостью упорядочивать видеоколлекции и распределять их по библиотекам — системным или созданным пользователями. Это позволяет быстро находить нужные видеофайлы. Кроме того, воспроизведение и обмен видеофайлами можно выполнить несколькими щелчками мыши.

1. Установка Video Station

Чтобы установить Video Station, перейдите в Центр пакетов, найдите Video Station, затем нажмите Установить.

2. Хранение и распределение видеофайлов в библиотеках по умолчанию

Video Station поддерживает три библиотеки по умолчанию (см. панель слева). Фильмы, ТВ-передача и Домашнее видео. Чтобы правильно индексировать и упорядочить видеофайлы, необходимо добавить папки видео для каждой библиотеки, затем в первую очередь сохранить видеофайлы в соответствующей папке. По умолчанию три папки видео создаются в общей папке видео для трех библиотек по умолчанию: фильмы, ТВ-передачи и домашнее видео. Однако в дополнение к этим трем папкам можно выполнить следующие действия для добавления папок видео для библиотек.

Примечание.

Система автоматически находит в Интернете соответствующие метаданные и другие сведения (например, постер видео) для двух типов видео (фильмы и ТВ-передачи). Таким образом, чтобы для видео отображалась правильная информация, настоятельно рекомендуется указать для видео тип, соответствующий библиотеке.

Добавление папок видео для библиотек по умолчанию

  1. Перейдите в Настройки > Библиотека.
  2. Нажмите значок Добавить папку, чтобы выбрать папку в системе Synology NAS, затем выберите библиотеку назначения. Для одной библиотеки можно добавить до 100 папок видео, однако после выбора папки эта папка и ее подпапки не могут быть выбраны для другой библиотеки.
  3. Сохраняйте видеофайлы в папках видео, чтобы распределять их по соответствующим библиотекам.
  4. Теперь вы можете находить и просматривать видео из библиотек!

3. Хранение и распределение видеофайлов в пользовательских библиотеках

Помимо трех библиотек по умолчанию, доступ к которым может получить по умолчанию любой пользователь, вы в качестве администратора DSM admin или пользователь, принадлежащий к группе administrators, можете создавать пользовательские библиотеки и назначать права доступа пользователям.

Пользовательские библиотеки

  1. Перейдите в Настройки > Библиотека и нажмите значок плюса для создания библиотеки.
  2. Укажите имя новой библиотеки, выберите соответствующий тип видеофайлов, затем нажмите Публичный доступ (библиотека будет доступна всем пользователям) или Частный доступ (доступ будет предоставлен только определенным пользователям, имеющим соответствующее разрешение). Обратите внимание, что размещенные в данной папке видеофайлы должны соответствовать типу, выбранному в этом разделе. Это обеспечит точность информации, получаемой о видео из Интернета.
  3. Если выбран параметр Частный доступ, нажмите Назначение разрешений, затем установите соответствующие флажки для предоставления доступа пользователям.

4. Воспроизведение видео

После распределения видеофайлов по библиотекам выполните указанные ниже действия, чтобы воспроизвести видео в Video Station. Настоятельно рекомендуется сначала установить VLC media player (ОС Windows, Linux) на компьютере. Помимо воспроизведения видео на локальном компьютере вы также можете выполнять потоковую передачу видео на устройства DMA, совместимые с DLNA/UPnP, или совместимые с AirPlay устройства для воспроизведения видео.

Воспроизведение видео с помощью встроенного видеопроигрывателя

  1. Выберите видеоклип, который требуется воспроизвести, затем нажмите значок воспроизведения на видеоизображении.
  2. Воспроизведение должно начаться сразу же. Если этого не произошло, убедитесь, что формат видео поддерживается веб-браузером и на компьютере установлен плагин VLC.
  3. При необходимости используйте экранные кнопки и шкалу выполнения в нижней части видеопроигрывателя. Можно также изменить громкость, выбрать качество воспроизведения видео и аудиодорожку, включить субтитры, а также воспроизвести потоковую передачу видео на другом устройстве (если применимо).
Примечание.

Для получения подробных сведений о плагине VLC и поддерживаемых устройствах выберите Справка DSM > Video Station > Воспроизведение видео.

5. Публичный общий доступ к видео

С помощью функции публичного общего доступа можно с легкостью обмениваться видео с другими пользователями через Video Station — все общие видео будут доступны для всех пользователей по ссылке.

Предоставление доступа к видео для других пользователей

  1. Перейдите в любую библиотеку видео, затем нажмите Все в левом верхнем углу для просмотра всех видео в этой библиотеке.
  2. Нажмите значок Режим выбора или нажмите и удерживайте клавишу Ctrl или Shift для выбора нескольких видео. Затем нажмите значок Публичный общий доступ.
    • Если выбрано только одно видео, он добавляется в общую коллекцию с именем Общие видео. Здесь можно управлять всеми видео, которые были сделаны общедоступными поодиночке.
    • Если для предоставления общего доступа выбрано несколько видео, они будут сгруппированы в новую общую коллекцию.
  3. При необходимости укажите срок действия.
  4. Можно также нажать значок Дополнительно на соответствующем видеоизображении, затем нажать Публичный общий доступ, чтобы добавить выбранное видео в список Общие видео.
  5. Просто скопируйте и вставьте ссылку в сообщения электронной почты, мгновенные сообщения и веб-страницы, чтобы поделиться видео со всеми.

Для подключения к Video Station через мобильные устройства скачайте DS video на устройства iOS/Android с помощью QR-кодов (см. ниже).
