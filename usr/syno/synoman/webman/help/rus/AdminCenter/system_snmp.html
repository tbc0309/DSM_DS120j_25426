SNMP

Служба SNMP (в меню Панель управления > Терминал и SNMP > SNMP) позволяет пользователям отслеживать Synology NAS с помощью программы сетевого управления. В настоящий момент поддерживаются протоколы SNMPv1, SNMPv2c и SNMPv3.

Включение службы SNMP

Установите флажок в поле Включить службу SNMP и выполните следующие действия.

Включение службы SNMPv1, SNMPv2c

  1. Установите флажок Служба SNMPv1, SNMPv2c.
  2. Введите имя Сообщества.
  3. Нажмите Применить.

Включение службы SNMPv3

  1. Установите флажок Служба SNMPv3.
  2. Введите Имя пользователя и Пароль. Выберите Протокол, который необходимо использовать.
  3. Нажмите Применить.

Включение конфиденциальности SNMP

  1. Установите флажок Включить конфиденциальность SNMP.
  2. Выберите Протокол, который необходимо использовать. После этого введите Пароль.
  3. Нажмите Применить.

При указании параметров Имя, Местоположение и Контакт можно определить Synology NAS. Данные поля не являются обязательными.

Примечание

  • Службой SNMP поддерживаются IPv4 и IPv6.
  • Служба SNMP поддерживает SNMPv1, SNMPv2c и SNMPv3 одновременно.

Ограничение для сообщества SNMPv1, SNMPv2c

Имя сообщества должно содержать от 1 до 64 поддерживаемых символов. Запрещается использовать перечисленные ниже символы:
" ' \ и пробел.

Ограничение для имени пользователя SNMPv3

Имя пользователя должно содержать от 1 до 64 поддерживаемых символов. Запрещается использовать перечисленные ниже символы:
" ' \ и пробел.

Ограничения для пароля SNMPv3

Пароль вводится с учетом регистра и должен быть длиной от 8 до 127 отображаемых символов, включая буквы, цифры и знаки. Запрещается использовать перечисленные ниже символы:
" ' \ и пробел.

Скачивание MIB-файлов Synology

Synology предоставляет возможность отслеживания Synology NAS посредством систем сетевого управления, включая систему, диск и состояние томов RAID.

Примечание

  • Использование RAID не поддерживается моделями без дисков, например EDS14. Для получения дополнительных сведений посетите веб-сайт Synology.

Просмотреть дополнительную информацию и скачать необходимые файлы вы можете на веб-сайте Synology.

Примечание

  • Системы управления сетью используют MIB-файлы для определения отслеживаемых устройств. Дополнительные сведения см. здесь.
