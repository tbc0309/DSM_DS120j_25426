Управление ключами доступа к зашифрованным общим папкам

С помощью Диспетчер ключей можно управлять ключами общих папок и расшифровывать несколько общих папок одновременно.

Объяснение терминов:

  • Хранилище ключей. Хранилище ключей — это любое внешнее устройство или системный раздел, поддерживаемый Synology NAS, и используемый в качестве физического ключа для расшифровки общих папок.
  • Шифр. Шифр — это способ шифрования ключей общих папок. Диспетчер ключей поддерживает два типа шифров,
    • Пароль. Ключи, зашифрованные паролем, может расшифровать любой пользователь, знающий пароль.
    • Машинный ключ. Ключи, зашифрованные машинным ключом, можно расшифровать с помощью Synology NAS.

Инициализация хранилища ключей

  1. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  2. Выберите внешнее устройство или системный раздел в качестве хранилища ключей в разделе Местоположение хранилища ключей.
  3. Введите пароль от хранилища ключей в поле Пароль.

Рекомендации по безопасности

  • В целях безопасности и управления зашифрованный файл и соответствующий ключ следует хранить на разных устройствах.
  • Чтобы повысить уровень безопасности данных, в качестве Местоположения хранилища ключей рекомендуется выбрать внешнее устройство.

Добавление нового ключа в хранилище ключей

  1. Убедитесь, что хранилище ключей инициализировано.
  2. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  3. Нажмите Добавить.
  4. Выберите зашифрованную общую папку.
  5. Выберите шифр для ключа шифрования. Можно выбрать либо Пароль, либо Машинный ключ.
  6. Введите или импортируйте ключ защиты.
  7. Нажмите кнопку OK, чтобы сохранить изменения.

Расшифровка нескольких зашифрованных общих папок вручную

  1. Убедитесь, что ключи добавлены в хранилище ключей.
  2. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  3. Выберите зашифрованные общие папки, которые необходимо расшифровать.
  4. Нажмите Подключить.

Автоматическая расшифровка нескольких зашифрованных общих папок при запуске

  1. Убедитесь, что ключи добавлены в хранилище ключей.
  2. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  3. Установите флажок Подключать при загрузке рядом с зашифрованными общими папками, которые необходимо расшифровывать при загрузке.
  4. Нажмите кнопку OK, чтобы сохранить изменения.

Извлечение хранилища ключей после загрузки

  1. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  2. Нажмите Настройка.
  3. Установите флажок Извлекать устройство после загрузки.
  4. Нажмите кнопку OK, чтобы сохранить изменения.

Перенос существующих ключей шифрования в хранилище ключей для общих папок, которые подключаются автоматически

Перенос ключей во время инициализации хранилища ключей

  1. При инициализации хранилища ключей установите флажок Перенести ключи шифрования в данное хранилище ключей для всех существующих общих папок, которые подключаются автоматически (рекомендуется).

Перенос ключей вручную

  1. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  2. Нажмите Настройка.
  3. Нажмите Перенести сейчас.
  4. При наличии повторяющихся ключей можно выбрать Перезаписывать повторяющиеся ключи.
  5. Нажмите кнопку OK, чтобы сохранить изменения.

Клонирование хранилища ключей в режиме High Availability

  1. Выберите Панель управления > Общая папка > Действие > Диспетчер ключей.
  2. Нажмите Клонировать.
  3. Выберите исходное хранилище ключей на активном сервере в разделе Исходное хранилище ключей.
  4. Выберите целевое хранилище ключей на пассивном сервере в разделе Целевое хранилище ключей.
  5. Нажмите Применить, чтобы сохранить настройки.
