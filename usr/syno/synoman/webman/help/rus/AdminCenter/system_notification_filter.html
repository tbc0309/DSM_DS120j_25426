Дополнительно

Вы можете выбрать типы событий, при возникновении которых будут отправляться уведомления, а также способ их получения (электронная почта, SMS и мобильные устройства).

Каждое событие имеет собственное уведомление, установленное по умолчанию, но вы также можете настроить его по собственному желанию, отредактировав текст сообщения. Вы также сможете восстановить сообщение по умолчанию, если захотите сбросить собственные настройки.

Поиск событий уведомлений

  1. Введите ключевое слово в поле поиска, расположенном в правом верхнем углу.
  2. Отобразятся события, связанные с введенным ключевым словом.

Выбор способов получения уведомлений для событий уведомлений

  1. Установите флажки для каждого события, чтобы выбрать необходимый способ получения уведомления.
  2. Нажмите кнопку Сохранить, чтобы применить настройки.

Редактирование сообщений уведомления

  1. В списке событий выберите событие и затем нажмите Редактировать сообщение.
  2. В отобразившемся окне отредактируйте тему и основной текст для настройки уведомления.
  3. Также можно отредактировать следующие переменные. Обновленные переменные будут применяться ко всем уведомлениям:
    • %COMPANY_NAME%: подпись, которая отображается в конце уведомления. В зависимости от используемой модели значением по умолчанию будет Synology DiskStation, Synology RackStation, Synology FlashStation или Synology Unified Controller.
    • %HTTP_URL%: гиперссылка на Synology NAS. Если этот параметр не будет задан, система будет приводить в уведомлении все возможные ссылки.
  4. Нажмите кнопку Сохранить, чтобы изменения вступили в силу.
  5. Восстановление уведомления по умолчанию
    1. В списке событий выберите событие и затем нажмите Редактировать сообщение.
    2. В отобразившемся окне нажмите Восстановить значения по умолчанию и выберите Да для подтверждения.

Примечание.

  • Рекомендуется сохранить сегменты, содержащие «%», в теме и тексте сообщения. Они будут заменены фактическими системными данными при отправке уведомления.
