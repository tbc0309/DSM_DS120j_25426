Дополнительно

На вкладке Дополнительно доступна функция Быстрое клонирование файлов. С ее помощью можно сэкономить емкость хранилища при создании клонов файлов. Кроме того, в настройках можно предоставить пользователям возможность обнаруживать Synology NAS в сети через службы Bonjour и SSDP (Simple Service Discovery Protocol).

Быстрое клонирование файлов на Synology NAS

С помощью функции быстрого клонирования файлов блоки физических данных будут копироваться только в случае изменения файлов-клонов, и это поможет сэкономить место для хранения.

  1. В разделе Быстрое клонирование файлов установите флажок Включить быстрое клонирование файлов.
  2. Нажмите Применить, чтобы сохранить настройки.

Включение обнаружения Synology NAS через Bonjour

Служба Bonjour позволяет пользователям находить Synology NAS и выполнять задачи печати и резервного копирования на компьютере Mac.

  1. В разделе Bonjour установите флажок для параметра Включить службу Bonjour.
  2. Можно выбрать параметры, связанные с Bonjour, в соответствии с вашими потребностями.
    • Включение вещания принтера Bonjour: выберите, чтобы разрешить компьютерам Mac находить USB-принтер, подключенный к Synology NAS для печати документов.
    • Включить передачу Bonjour Time Machine через SMB: выберите, чтобы разрешить Mac Time Machine находить Synology NAS и выполнять резервное копирование файлов в общие папки DSM через SMB.
    • Включить передачу Bonjour Time Machine через AFP: выберите, чтобы разрешить Mac Time Machine находить Synology NAS и выполнять резервное копирование файлов в общие папки DSM через AFP.
  3. При выборе любого параметра, связанного с Time Machine, нажмите Укажите папки Time Machine, чтобы указать папки назначения для резервных копий Time Machine.
  4. Нажмите Применить, чтобы сохранить настройки.

Примечание.

  • При отключении службы SMB/AFP (в разделе Панель управления > Файловые службы > SMB/AFP/NFS) соответствующий параметр Bonjour будет автоматически отключен.
  • При выборе параметра Включить передачу Bonjour Time Machine через SMB будут включены следующие параметры (в разделе Панель управления > Файловые службы > SMB/AFP/NFS > Дополнительные настройки):
    • Включить SMB 3 (для параметра Макс. протокол для файловой службы Windows)
    • Opportunistic Locking
    • Включить аренду SMB2
    • Включить надежные обработчики SMB (Блокировка файла кросс-протокола будет отключена)

Включение обнаружения Synology NAS через SSDP

Служба SSDP позволяет пользователям находить Synology NAS через обнаружение сети Windows.

  1. В разделе SSDP установите флажок для параметра Включить обнаружение сети в Windows, чтобы разрешить доступ из веб-браузеров.
  2. Нажмите Применить, чтобы сохранить настройки.

Включение обнаружения Synology NAS через WS-Discovery

Служба WS-Discovery позволяет пользователям находить службу SMB на Synology NAS через обнаружение сети Windows.

  1. В разделе WS-Discovery установите флажок для параметра Включить обнаружение сети в Windows, чтобы разрешить доступ к файлам по протоколу SMB.
  2. Нажмите Применить, чтобы сохранить настройки.
