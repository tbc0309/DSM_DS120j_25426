Индексирование мультимедийных файлов

Служба Индексирования мультимедийных файлов может автоматически сканировать мультимедийные файлы, сохраненные на Synology NAS. Эти файлы можно скомпилировать в библиотеку мультимедиа для индексации, а также предоставить возможность поиска и воспроизведения устройствам DMA и мультимедийными приложениями Synology (например, сервер мультимедиа, Video Station, Audio Station, Photo Station и iTunes Server).

Управление проиндексированными папками

Чтобы выполнить индексирование существующей папки или создать проиндексированную папку, выполните следующие действия:

  1. Выберите Панель управления > Служба индексирования > Индексирование мультимедийных файлов.
  2. Нажмите Проиндексированная папка > Создать > Выбрать и выполните одно из следующих действий.
    • Индексирование существующей папки Выберите нужную папку из списка и нажмите Выбрать.
    • Создание проиндексированной папки Выберите родительскую папку из списка. Нажмите Создать папку, укажите имя папки и нажмите ОК. Выберите созданную папку и нажмите Выбрать.
  3. Укажите типы файлов (фото, музыка или видео), которые вы планируете проиндексировать. Устройства DMA и мультимедийные приложения Synology могут получать доступ только к файлам индексированных типов.
  4. Нажмите ОК, чтобы начать индексирование.

Примечание.

  • Общие папки по умолчанию (например, Музыка), созданные для мультимедийных приложений Synology, индексируются автоматически, тогда как другие папки можно проиндексировать только вручную.
  • Удаленные папки, созданные с помощью File Station, не могут быть добавлены в качестве проиндексированных папок.
  • Внешние устройства хранения необходимо индексировать повторно при каждом перезапуске Synology NAS.
  • На Synology NAS можно добавить до 100 проиндексированных папок.

Редактирование проиндексированной папки

  1. Выберите Панель управления > Служба индексирования > Индексирование мультимедийных файлов > Проиндексированная папка.
  2. Во всплывающем окне выберите нужную папку.
  3. Нажмите Редактировать и измените параметры.
  4. Нажмите OK, затем нажмите Сохранить.

Удаление папок из списка проиндексированных

Для удаленных папок индексация будет отменена. К содержащимся в них мультимедийным файлам доступ с устройств DMA и мультимедийных приложений Synology будет запрещен.

  1. Выберите Панель управления > Служба индексирования > Индексирование мультимедийных файлов > Проиндексированная папка.
  2. Во всплывающем окне выберите папку, которую нужно удалить. Чтобы выбрать несколько ненужных папок, нажмите и удерживайте клавиши Shift или Ctrl во время выбора.
  3. Нажмите Удалить.
  4. Нажмите Да, затем нажмите Сохранить.

Примечание.

  • Удаленные папки не будут удалены с Synology NAS.

Переиндексирование мультимедийных файлов

При потере структуры индекса воссоздать ее поможет переиндексирование папок.

  1. Выберите Панель управления > Служба индексирования > Индексирование мультимедийных файлов.
  2. Нажмите Переиндексировать для повторного индексирования всех проиндексированных ранее папок.

Примечание.

  • Автоматическое сканирование мультимедийных файлов, добавленных по протоколам NFS или SCP в проиндексированные папки, не выполняется. Чтобы выполнить индексирование таких файлов, нажмите Переиндексировать.

Преобразование проиндексированных мультимедийных файлов

Настройка качества миниатюр

Значки загруженных фотографий и видео для использования на сервере мультимедиа или Photo Station создаются автоматически. Качество значков можно задать здесь.

  1. Выберите Панель управления > Служба индексирования > Индексирование мультимедийных файлов > Настройки значка.
  2. Выберите качество в раскрывающемся меню.
  3. Нажмите Применить.

Примечание.

  • Создание миниатюр высокого качества занимает больше времени.
  • Чтобы загрузить большое количество фотографий/видео в библиотеку Photo Station и быстро создать для них значки, рекомендуется использовать Photo Station Uploader (утилиту Synology для рабочего стола, доступную в Центре загрузок).

Включение преобразования видео для мобильных устройств:

Видео в общей папке для фотографий (выделенной для Photo Station) можно воспроизвести на мобильном устройстве при условии их правильного преобразования.

  1. Выберите Панель управления > Служба индексирования > Индексирование мультимедийных файлов > Настройки видео.
  2. Установите флажок Включить преобразование видео для мобильных устройств.
  3. Нажмите Применить. После этого вы сможете просматривать видео на мобильных устройствах везде, где бы вы ни находились.

Примечание.

  • В формат для мобильных устройств можно преобразовать только видео, загруженные в общую папку photo на Photo Station.
  • Преобразование файлов MP4 не выполняется.

Просмотр хода выполнения преобразования фото/видео

  1. Во время загрузки фотографий или видео на Synology NAS нажмите на значок Ход выполнения преобразования в правом верхнем углу панели задач.
  2. Во всплывающем окне выполните одно из следующих действий.
    • В разделе Ход выполнения преобразования можно следить за процессом создания миниатюр в фоновом режиме.
    • Чтобы приостановить процесс преобразования, выполните следующие действия:
      1. Выберите Задержка на [количество] часов в раскрывающемся меню и нажмите Пауза, чтобы приостановить процесс.
      2. Ход выполнения преобразования будет восстановлен после указанного времени задержки. Кроме того, можно нажать кнопку Продолжить для немедленного возобновления процесса.
      Примечание. Поскольку процесс преобразования требует более интенсивного потребления ресурсов процессора, приостановка этого процесса позволяет использовать ресурсы системы для выполнения других текущих задач.
  3. После завершения процесса преобразования соответствующий значок исчезает с панели задач.
