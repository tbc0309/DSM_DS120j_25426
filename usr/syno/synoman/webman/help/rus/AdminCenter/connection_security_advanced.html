Дополнительно

Для изменения нижеприведенных параметров сеансов просмотра DSM перейдите в Панель управления > Безопасность > Дополнительно:

Сжатие HTTP

Можно включить параметр Сжатие HTTP для Synology NAS. Уменьшение пропускной способности сетевого трафика увеличит скорость загрузки веб-страницы.

Включение/отключение сжатия HTTP

  • Установите/снимите флажок с параметра Сжатие HTTP, перейдя на Панель управления > Безопасность > Дополнительно.

Уровень профиля TLS / SSL

В профиле TLS / SSL выберите уровень безопасности зашифрованных подключений HTTPS. Доступно три следующих уровня:

  • Современная совместимость: поддерживает браузеры и мобильные устройства только последних версий.
  • Промежуточная совместимость: между последними и старыми версиями. Это значение системы установлено по умолчанию.
  • Совместимость со старыми версиями: это небезопасный вариант, но он поддерживает большинство браузеров и мобильных устройств.

Настройка уровня профиля TLS / SSL для всех служб

На главной странице можно выбрать уровень профиля TLS /SSL, который будет применен ко всем службам, использующим Профиль по умолчанию.

  1. Выберите надлежащий уровень профиля TLS / SSL.
  2. Нажмите OK.

Настройка уровня профиля TLS / SSL для конкретной службы

Уровень профиля службы можно изменить в соответствии с требованиями безопасности службы.

  1. Нажмите Пользовательские настройки, чтобы отобразить все службы и соответствующие им уровень профиля.
  2. Нажмите текущий уровень профиля целевой службы.
  3. Выберите нужный уровень профиля в раскрывающемся меню.
  4. Нажмите OK.

Примечание.

  • Уровень профиля по умолчанию будет применен к подключению, не включенному в список служб.
