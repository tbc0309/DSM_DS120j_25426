Link Aggregation

Synology NAS поддерживает несколько локальных сетей и позволяет объединять интерфейсы сетей с помощью технологии Link Aggregation. Link Aggregation увеличивает пропускную способность Synology NAS с помощью объединения нескольких сетевых интерфейсов и обеспечивает аварийное переключение трафика для сохранения сетевого соединения в случае его потери.

После объединения интерфейсов локальных сетей новый интерфейс Bond отобразится в списке Панель управления > Сеть > Сетевой интерфейс, который можно настроить точно так же, как и другие интерфейсы (например, локальная сеть или PPPoE).

Примечание.

  • Link Aggregation поддерживается только на моделях с двумя портами LAN.
  • При активации Link Aggregation сетевой трафик будет автоматически настроен и сбалансирован для каждого подключенного устройства.
  • Если сетевые интерфейсы подключены к нескольким коммутаторам, убедитесь, что порты коммутаторов находятся в одной локальной сети или сети VLAN.

Объединение нескольких локальных сетей с помощью Link Aggregation:

  1. Выберите Панель управления > Сеть > Сетевой интерфейс. Нажмите Создать > Создать Bond.
  2. Выберите необходимый режим.

    Описание режимов

    • Адаптивное распределение нагрузки (Adaptive Load Balancing). Этот режим оптимизирует сетевой трафик, принимаемый и отправляемый Synology NAS, независимо от того, поддерживает ли коммутатор Link Aggregation или нет. Чтобы избежать непредвиденных ошибок, не включайте Link Aggregation на коммутаторе, даже если эта функция поддерживается им.
    • Динамический режим IEEE 802.3ad Link Aggregation. Этот режим оптимизирует сетевой трафик, принимаемый и отправляемый Synology NAS, и требует включения на коммутаторах динамического режима IEEE 802.3ad Link Aggregation (LACP, 802.1AX). Если используется несколько коммутаторов, они должны быть наращиваемыми и правильно настроенными.
    • Распределение XOR. Этот режим позволяет сбалансировать сетевой трафик, принимаемый и отправляемый Synology NAS, и требует включения на коммутаторах статического режима Link Aggregation. Если используется несколько коммутаторов, они должны быть наращиваемыми и правильно настроенными. Если Link Aggregation не поддерживается коммутатором, трафик, отправляемый вашим устройством Synology NAS, все равно можно сбалансировать. С помощью этого режима можно выявить потерю соединений, но не причину (например, отсоединение кабелей Ethernet или неправильная настройка коммутатора).
    • Активный режим/Режим ожидания. Этот режим обеспечивает network fault tolerance для Synology NAS. То есть в случае сбоя активного сетевого интерфейса подключение к сети обеспечит один из других портов локальной сети. Можно выбрать этот режим без настройки коммутатора или при помощи коммутатора, поддерживающего IEEE 802.3ad Link Aggregation.

    Если вы включили Open vSwitch, можно выбрать один из следующих режимов.

    • Распределение SLB. Этот режим позволяет сбалансировать сетевой трафик без необходимости в поддержке определенных сетевых коммутаторов и подключить два коммутатора.
    • Распределение TCP. Этот режим позволяет подключить коммутаторы, настроенные в режиме Dynamic Link Aggregation (IEEE 802.3ad LACP). При использовании нескольких коммутаторов они должны быть наращиваемыми и правильно настроенными.
    • Активный режим/Режим резервного копирования. Этот режим обеспечивает network fault tolerance для Synology NAS с помощью двух сетевых интерфейсов. В случае сбоя активного сетевого интерфейса подключение к сети обеспечит другой сетевой интерфейс. Этот режим можно выбрать без настройки коммутатора.
  3. Выберите интерфейсы для создания Link Aggregation.
  4. Настройте параметры IP. Здесь также можно включить VLAN. Нажмите Применить, чтобы сохранить изменения.
  5. После завершения отобразится новый интерфейс Bond 1 в списке Сетевой интерфейс.

Изменение настроек Link Aggregation

  1. Чтобы изменить режим Link Aggregation или просмотреть информацию о физических устройствах, перейдите в раздел Панель управления > Сеть > Сетевой интерфейс.
  2. Выберите интерфейс Bond 1 и нажмите кнопку Редактировать.
  3. Перейдите на соответствующие вкладки для изменения требуемых настроек.

Удаление Link Aggregation

  1. Выберите Панель управления > Сеть > Сетевой интерфейс.
  2. Выберите интерфейс Bond 1 и нажмите кнопку Удалить.
  3. После завершения отобразятся два отдельных интерфейса локальной сети в списке «Сетевой интерфейс».
