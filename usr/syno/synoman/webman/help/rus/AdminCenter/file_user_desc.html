Пользователь

Благодаря гибким параметрам управления пользователями можно создавать учетные записи для ваших друзей или коллег. Создание пользователей позволяет задавать и управлять настройками каждого пользователя, например правами доступа к общей папке, квотами для хранилища/общей папки и ограничением полосы пропускания.

    Примечание.
  1. Управление папкой общего доступа применимо только к платформам Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake и Denverton. Сведения о наличии поддержки данной функции в вашей модели см. в данной статье.
  2. Управление квотой хранилища/общей папки не применимо к Docker DSM.
