Индексирование файлов

Индексирование общей папки или ее подпапок поможет мгновенно найти нужные файлы и материалы при поиске.

Примечание

Невозможно индексировать удаленные папки, облачные папки и папки .iso, подключенные к вашему устройству Synology NAS.

Индексирование содержимого папок и файлов

File Station выполняет индексирование папок и вложенных в них подпапок/файлов по имени. Также можно выполнять поиск элементов по имени файла. В случае индексирования по типу файла File Station выполняет индексирование соответствующих файлов не только по имени, но и по содержимому. Это значит, что поиск элементов можно выполнять и по имени файла, и по его содержимому.

  1. Выберите Панель управления > Служба индексирования > Индексирование файлов и нажмите Список проиндексированных папок.
  2. Во всплывающем окне нажмите Создать > Выбрать для отображения папок, готовых к индексированию.
  3. Выберите нужную папку и нажмите Выбрать. Чтобы создать новую папку для индексирования, нажмите Создать папку и продолжите.
  4. Выберите папку, которую нужно проиндексировать, и нажмите Выбрать.
  5. Для индексирования определенных типов файлов откройте раздел Тип файла и выберите Изображение, Аудио, Видео или Документ. Если некоторые типы файлов не выбраны (например, Аудио), поиск файлов такого типа (т. е. аудио) можно выполнять только по имени файлов.
  6. Для завершения индексирования нажмите кнопку OK.

Примечание.

  • Для индексирования можно выбрать до 1000 папок без учета вложенных в них подпапок.
  • Если в File Station выбрать Включить поиск по содержимому файлов, вы сможете выполнить поиск содержимого файлов в проиндексированных папках.
  • Том, в котором размещена индексированная папка, должен иметь не менее 10 МБ свободного пространства. В противном случае индексирование будет приостановлено. Чтобы возобновить индексирование, расширьте том и нажмите Переиндексировать.
  • Если папка и ее подпапки были проиндексированы одновременно, система проиндексирует только подпапки, чтобы сэкономить системные ресурсы.
  • После завершения индексирования папок и их подключения к компьютеру Mac по протоколу SMB или AFP для поиска содержимого файла можно использовать Finder.

Удаление проиндексированных папок:

  1. Откройте Панель управления > Служба индексирования > Индексирование файлов > Список проиндексированных папок и выберите нужные проиндексированные папки.
  2. Нажмите Удалить, чтобы остановить индексирование папок.

Переиндексирование файлов

Переиндексирование файлов не требуется в нормальных условиях работы. Однако если файлы в папке соответствуют определенным критериям, но не находятся при поиске файлов, необходимо выполнить переиндексирование файлов для изменения структуры индекса.

Индексирование файлов в общей папке:

  1. Выберите Панель управления > Служба индексирования > Индексирование файлов > Список проиндексированных папок.
  2. Выберите указанные в списке папки для переиндексирования.
  3. Нажмите Переиндексировать.

Примечание

File Station не поддерживает индексирование файлов в удаленных папках и внешних устройствах хранения (например, USB-диски/карты SD).

Управление индексированием файлов на рабочем столе

При запуске индексирования файлов на панели задач отобразится Ход выполнения индексирования (плавающий значок лупы). Поскольку при индексировании файлов затрачивается больше ресурсов процессора, управление индексированием поможет сэкономить ресурсы.

  • Чтобы приостановить/возобновить индексирование файлов, нажмите кнопку воспроизведения.
  • Чтобы отложить индексирование до подходящего времени, выберите нужную опцию из раскрывающегося списка.
  • Для простора того, какие папки индексируются, щелкните Отобразить список индексированных папок.

Примечание

  • Приостановленное действие индексирования может быть возобновлено только нажатием кнопки воспроизведения на панели задач рабочего стола.
