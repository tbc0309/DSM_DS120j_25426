Копирование пользователя

Настройки существующего пользователя можно скопировать в учетную запись нового пользователя. Это позволит быстро создать нового пользователя без необходимости повторной настройки тех же параметров.

  1. Выберите пользователя, которого необходимо копировать.
  2. Нажмите Создать > Копировать пользователя. Запустится окно Мастер создания пользователей.
  3. Настройки выбранного пользователя будут применены к новому пользователю по умолчанию (кроме личных сведений, включая имя пользователя и адрес электронной почты).
