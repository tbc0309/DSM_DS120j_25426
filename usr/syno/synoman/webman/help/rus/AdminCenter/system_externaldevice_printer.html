Принтеры

Synology NAS можно настроить как сервер печати, чтобы клиентские компьютеры или мобильные устройства могли получать доступ к принтерам, подключенным к Synology NAS. Кроме того, поддержка беспроводной печати Apple позволяет удаленно выполнять печать с помощью устройств iOS.

Для просмотра совместимых моделей принтеров посетите веб-сайт www.synology.com.

Настройка принтеров

См. инструкции ниже по настройке и подключению сетевого и USB-принтера с помощью Synology NAS.

Настройка USB-принтера

  1. Включите принтер и подключите его к порту USB на устройстве Synology NAS.
  2. Выберите Панель управления > Внешние устройства > Принтер.
  3. Выберите принтер, который необходимо настроить. Обратите внимание, что Synology NAS не распознает принтеры, подключенные с помощью USB-концентраторов.
  4. Выберите Диспетчер принтеров и нажмите Настроить принтер.
  5. В диалоговом окне выберите один из следующих режимов.
    • Сетевое МФУ: позволяет клиентам в локальной сети использовать функции принтера, сканера и факса вашего многофункционального принтера. Сканирование и отправка факсов доступны только на компьютерах с Windows.
    • Сетевой принтер: выберите данный режим для принтеров, которые поддерживают только печать, или если вы хотите разрешить клиентам использовать только функцию печати.
  6. Настройка параметров беспроводной печати Apple (необязательно).
  7. Нажмите Сохранить.

Примечание.

  • К сети МФУ может одновременно осуществлять доступ только один клиент.

Настройка сетевого принтера

  1. Включите сетевой принтер и подключите его к той же локальной сети, к которой подключено устройство Synology NAS.
  2. Выберите Панель управления > Внешние устройства > Принтер.
  3. Нажмите Добавить сетевой принтер и следуйте инструкциям мастера установки.

Примечание.

  • Имя принтера: имя принтера, которое следует указать во время установки, может включать до 32 символов, в том числе буквы, цифры и нижние подчеркивания.
  • Протокол: как правило, сетевые принтеры HP используют Socket; сетевые принтеры Canon — BJNP, а другие принтеры — LPR.
  • Имя очереди: имя очереди требуется для сетевых принтеров LPR и IPP, но не применяется для принтеров Socket и BJNP.
  • Нет сетевого МФУ: сетевые принтеры, подключенные к Synology NAS, в данный момент не используют режим сетевого МФУ.
  • Нет принтеров USB: USB-принтеры, к которым осуществляется общий доступ с продукта Synology, не могут использоваться в качестве сетевых принтеров.

Настройка USB-принтера PostScript для Mac

Для Mac OS 10.12.X

  1. Выполните действия, описанные в разделе Настройка USB-принтера.
  2. В Mac OS выберите Свойства системы > Принтер и сканеры и затем нажмите +, чтобы добавить новый принтер.
  3. Выберите тип принтера: IP.
    • Введите IP-адрес или имя хоста принтера.
    • Выберите Line Printer Daemon - LPD для параметра Протокол.
    • Введите «usbprinter» в поле Очередь. Если не удается найти имя очереди, выберите Панель управления > Внешние устройства > Принтер в DSM.
  4. Выберите для параметра Использование значение Общий принтер PostScript.
  5. Нажмите кнопку Добавить, чтобы завершить настройку.

Для Mac OS 10.6.X

  1. Выполните действия, описанные в разделе Настройка USB-принтера.
  2. В ОС Mac выберите Свойства системы > Принтер и факс и затем нажмите +, чтобы добавить новый принтер.
  3. Выберите IP-адрес.
    • Выберите Line Printer Daemon - LPD для параметра Протокол.
    • Введите IP-адрес принтера.
    • Введите «usbprinter» в поле Очередь.
    • Выберите для параметра Использование принтера значение Общий принтер PostScript.
    • Нажмите кнопку Добавить, чтобы завершить настройку.

Примечание.

  • AppleTalk больше не поддерживается Mac OS 10.6.X. Если вы хотите использовать принтер AppleTalk, ознакомьтесь с его документацией для поиска возможностей подключения к компьютеру Mac или локальной сети. Если такой возможности нет, подключите принтер к компьютеру Mac с операционной системой Mac OS 10.5.X или более ранней и настройте общий доступ.

Для Mac OS 10.5.X

  1. Выполните действия, описанные в разделе Настройка USB-принтера.
  2. В ОС Mac выберите Свойства системы > Принтер и факс и затем нажмите +, чтобы добавить новый принтер.
  3. Выберите метод.
    • Метод 1. AppleTalk
      • Выберите для параметра Использование принтера значение Общий принтер PostScript.
      • Нажмите кнопку Добавить, чтобы завершить настройку.
    • Метод 2. IP-адрес
      • Выберите значение Line Printer Daemon - LPD для параметра «Протокол».
      • Введите IP-адрес принтера.
      • Введите «usbprinter» в поле Очередь.
      • Выберите для параметра Использование принтера значение Общий принтер PostScript.
      • Нажмите кнопку Добавить, чтобы завершить настройку.

Управление принтерами

На вкладке принтера доступны следующие функции управления сетевыми или USB-принтерами, подключенными к Synology NAS.

  • Вывести информацию. Отображает информацию о выбранном принтере, например имя устройства, производителя, состояние и др.
  • Удалить. Удаление и отключение принтера от Synology NAS.
  • Диспетчер принтеров. Включает следующие параметры:
    • Настройка принтера. Изменение настроек принтера. Инструкции по настройке беспроводной печати Apple см. ниже.
    • Очистить спул-пространство. Отмена задач печати на всех принтерах.
    • Печать пробной страницы. Печать пробной страницы для проверки правильности конфигураций.
    • Отключение принтера. Отключение принтера и клиента (только в режиме сетевого МФУ). При подключении клиента к многофункциональному принтеру для получения факса или сканирования файлов выполнение задачи может занять длительное время. При необходимости подключение может быть остановлено для предоставления следующему клиенту доступа к принтеру.

Включение беспроводной печати Apple

  1. Выберите Панель управления > Внешние устройства > Принтер.
  2. Выберите принтер в списке устройств.
  3. Выберите Диспетчер принтеров > Настройка принтера.
  4. Установите флажок Включить беспроводную печать Apple.
  5. Выберите марку и драйвер принтера.
  6. Нажмите Сохранить.

После завершения настройки вы можете напечатать пробную страницу, нажав Диспетчер принтеров > Печать пробной страницы.

Примечание.

  • Беспроводная печать Apple также включает Вещание принтера Bonjour для DSM.
  • Беспроводная печать Apple доступна только для iOS 4.2 и более новых версий.

Доступ к принтерам

См. раздел ниже для доступа к принтерам с помощью ПК или Mac.

Доступ к принтерам с компьютеров Windows

Удобнее всего обращаться к службе печати устройства Synology NAS, настроив подключение к принтеру с помощью программы Synology Assistant. Подробные инструкции см. в Справке этой программы.

Если программа Synology Assistant не установлена на вашем компьютере, можно использовать другие способы обращения к службе печати устройства Synology NAS.

Тип 1. Общий сетевой принтер

  1. На ПК откройте Сеть в Проводнике и дважды нажмите Synology NAS.
  2. Дважды нажмите usbprinter или имя сетевого принтера; система предложит выполнить установку необходимого драйвера, после чего нажмите Да.
  3. Драйвер к принтеру доступен на компакт-диске или на веб-сайте производителя. Выберите производителя и модель принтера или нажмите Установить с диска для выбора местоположения драйвера. Нажмите ОК.
  4. Система начнет установку сетевого принтера. После этого вы можете начать пользоваться принтером.

Тип 2. Порт LPR (рекомендуется)

  1. На ПК нажмите Пуск, затем выберите Устройства и принтеры.
  2. Нажмите Добавить принтер.
  3. Выберите Добавить сетевой, беспроводной или Bluetooth-принтер.
  4. Выберите Нужного принтера нет в списке.
  5. Выберите Добавить принтер с помощью TCP/IP-адреса или имени хоста. Чтобы продолжить, нажмите Далее.
  6. Выберите тип устройства, которое будет Устройством с портом TCP/IP.
  7. Введите IP-адрес устройства Synology NAS. Чтобы продолжить, нажмите Далее.
  8. Выберите Тип устройства: Пользовательский > Настройки > …. Чтобы продолжить, нажмите Далее.
  9. Для протокола выберите LPR; введите usbprinter или имя сетевого принтера для параметра «Имя очереди» и установите флажок в поле Разрешен подсчет байтов в LPR. Нажмите ОК для продолжения.
  10. Драйвер к принтеру доступен на компакт-диске или на веб-сайте производителя. Выберите производителя и модель принтера или нажмите Установить с диска для выбора местоположения драйвера. Нажмите ОК.
  11. Система начнет установку сетевого принтера. После этого вы можете начать пользоваться принтером.

Доступ к принтерам с компьютеров Mac

Пользователи Mac могут обращаться к службе печати устройства Synology NAS, включив реализованную в DSM функцию «Вещание принтера Bonjour» и добавив принтер в «Свойства системы».

Перед тем как продолжить, убедитесь, что Synology NAS использует принтер Postscript.

Шаг 1. Включение функции «Вещание принтера Bonjour» в DSM

  1. Выполните вход в DSM, используя системную запись администратора admin или пользователя, принадлежащего к группе administrators.
  2. Выберите Панель управления > Файловые службы > Win/Mac/NFS.
  3. Установите флажок Включить вещание принтера Bonjour.

Шаг 2. Доступ к принтеру в ОС Mac

  1. В ОС Mac выберите Свойства системы > Печать и сканирование и затем нажмите +.
  2. В раскрывающемся меню выберите usbprinter[номер] или имя сетевого принтера. Система Mac автоматически настроит принтер.

Примечание.

  • RC18015xs+ не поддерживает использование USB-принтеров.
