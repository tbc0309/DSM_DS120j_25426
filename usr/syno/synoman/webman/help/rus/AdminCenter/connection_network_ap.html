Зона доступа*

Ваше устройство Synology NAS позволяет создать точку доступа, благодаря которой другие устройства могут подключаться к Интернету по беспроводной сети. Для этого необходимо установить USB-адаптер для Wi-Fi или активировать встроенный Wi-Fi-адаптер на устройстве Synology NAS. Synology NAS можно использовать в качестве беспроводной точки доступа или беспроводного маршрутизатора в зависимости от того, как вы планируете использовать подключение к Интернету.

Режим зоны доступа Описание
Беспроводная точка доступа
  • Обеспечивает взаимодействие локальной сети и сети Wi-Fi.
  • Беспроводные клиенты и клиенты локальной сети будут находиться в одной локальной сети.
  • Это применимо, если у вас уже есть маршрутизатор, и вы планируете использовать Synology NAS в качестве дополнительной точки доступа в существующей сети.
  • Разрешает общий доступ к сетевым ресурсам, например устройствам DLNA, принтерам (не применимо к платформам Grantley; чтобы узнать, поддерживает ли модель данную функцию, см. эту статью), Synology Web Assistant или другие службы полагаются на широковещание по сети.
  • На устройствах Synology NAS, которые используются в качестве беспроводной точки доступа, DHCP Server будет отключен. Чтобы повторно активировать DHCP Server на устройстве Synology NAS повторно, удалите аппаратные ключи Wi-Fi и перезапустите Synology NAS.
  • В случае моделей с несколькими портами для подключения к локальной сети для связи с беспроводной точкой доступа можно использовать только порт LAN 1
Беспроводной маршрутизатор
  • Используется для создания беспроводной сети, если устройства находятся в частной подсети.

Примечание.

  1. Некоторые модели поддерживают создание зон доступа только при условии подключения беспроводных аппаратных ключей. Для получения полного списка поддерживаемых аппаратных ключей посетите веб-сайт www.synology.com.
  2. Параметр «Беспроводная зона доступа» в DSM 4.3 переименован в «Беспроводной маршрутизатор» в DSM 5.0 и более поздних версиях (кроме модели DS213air).

Управление настройками зоны доступа

В этом разделе описаны способы включения настроек зоны доступа для вашего Synology NAS и управления ими.

Преобразование Synology NAS в точку доступа:

  1. Выберите Панель управления > Беспроводная сеть > Карта сети. Нажмите Беспроводная точка доступа или Беспроводной маршрутизатор.
  2. Перейдите на вкладку Wi-Fi и нажмите Редактировать для управления следующими параметрами:
    • Имя (SSID): введите имя гостевой сети.
    • Показать/скрыть: укажите режим отображения беспроводной сети для других устройств.
    • Режим беспроводного подключения: выберите стандарт беспроводного подключения. Каждый стандарт поддерживает разный набор частот и разную пропускную способность.
    • Ширина пропускания канала: выберите ширину пропускания канала (доступно только при работе в режиме 802.11n).
    • Канал: выберите частоту для беспроводной сети. Для лучшей производительности сети выберите одну из частот, используемых расположенными рядом сетями.
    • Уровень мощности передачи: выберите уровень мощности сигнала беспроводной сети.
    • Макс. кол-во подключений: укажите, сколько клиентских устройств можно подключить к сети. Максимально возможное количество подключений — 256.
    • Включить WMM QoS: Wi-Fi Multimedia (WMM) QoS (Quality of Service) по сети Wi-Fi. Этот параметр позволяет назначать приоритеты для трафика и оптимизировать обмен ресурсами между приложениями.

Настройка расписания беспроводной сети*:

Можно включить расписание беспроводной сети вашего Synology NAS, что позволит контролировать ее активность и доступность.

  1. Для этого откройте вкладку Wi-Fi и нажмите Редактировать.
  2. Установите флажок Включить расписание.
  3. Нажмите Настроить расписание.
  4. На экране расписания укажите время активности беспроводной сети.

Настройки безопасности

В разделе Панель управления > Беспроводная сеть > Wi-Fi можно задать настройки безопасности для беспроводной сети.

Настройка параметров безопасности беспроводной сети:

  1. Нажмите Редактировать.
  2. Выберите один из следующих протоколов безопасности в раскрывающемся меню Настройки безопасности:
    • WEP: базовый протокол безопасности, при использовании которого пользователю необходимо ввести пароль для доступа к беспроводной сети. Этот протокол обеспечивает минимальный уровень безопасности.
    • Личный WPA/WPA2: рекомендуется. Пользователям необходимо ввести пароль для доступа к беспроводной сети. Этот протокол считается более безопасным, чем WEP.
    • WPA/WPA2 Enterprise*: протокол безопасности, созданный для предприятий и использующий аутентификацию пользователей с помощью внешнего сервера аутентификации.

Включение Wi-Fi Protected Setup*:

WPS (Wi-Fi Protected Setup) — это стандарт Wi-Fi Alliance, позволяющий упростить настройки безопасности беспроводного подключения для пользователей. Можно использовать кнопку нажатия или ПИН-код для безопасного подключения по сети Wi-Fi. Чтобы включить стандарт Wi-Fi Protected Setup, нажмите WPS.

Гостевая сеть*

Помимо возможности создания основной беспроводной сети устройство Synology NAS также поддерживает создание гостевой беспроводной сети. В настройках гостевой сети можно ограничить доступ пользователей, что удобно для предприятий и других учреждений, предоставляющих доступ по беспроводной сети для посетителей, но желающих ограничить доступ к основной сети.

Включение гостевой сети:

  1. Перейдите в раздел Панель управления > Беспроводная сеть > Wi-Fi.
  2. Нажмите Редактировать и на вкладке Гостевая сеть измените по своему усмотрению следующие параметры:
    • Имя (SSID): введите имя гостевой сети.
    • Показать/скрыть: укажите режим отображения беспроводной сети для других устройств.
    • Макс. кол-во подключений: укажите, сколько клиентских устройств можно подключить к сети. Максимально возможное количество подключений — 256.

Настройка параметров безопасности гостевой сети:

Выберите один из следующих протоколов безопасности в раскрывающемся меню Настройки безопасности:

  • WEP: базовый протокол безопасности, при использовании которого пользователю необходимо ввести пароль для доступа к беспроводной сети. Этот протокол обеспечивает минимальный уровень безопасности.
  • Личный WPA/WPA2: рекомендуется. Пользователям необходимо ввести пароль для доступа к беспроводной сети. Этот протокол считается более безопасным, чем WEP.

Настройка расписания гостевой сети:

Можно включить расписание гостевой беспроводной сети вашего Synology NAS, что позволит контролировать ее активность и доступность.

  1. Установите флажок Включить расписание гостевой сети.
  2. Нажмите Настроить гостевое расписание.
  3. На экране расписания укажите время активности гостевой беспроводной сети.

Примечание.

  • Эта функция доступна только в DS213air.
  • Активность беспроводной гостевой сети ограничивается периодом активности основной беспроводной сети.

Настройка фильтра MAC*

С помощью фильтра MAC-адресов (Панель управления > Беспроводная сеть) можно указать MAC-адреса, которым разрешен доступ к беспроводной сети, предоставляя или блокируя доступ к сети определенным устройствам. Эта функция позволяет контролировать только доступ к сети, не включая другие параметры шифрования.

Создание правил фильтра MAC-адресов:

  1. Введите описание и MAC-адрес устройства.
  2. Выбор политики доступа:
    • Запретить: ограничивает доступ к беспроводной сети для указанного MAC-адреса.
    • Разрешить: разрешает доступ к беспроводной сети для указанного MAC-адреса.

Родительский контроль*

Функция родительского контроля позволяет определить, когда каждое клиентское устройство, подключенное к локальной сети Synology NAS, может получить доступ к Интернету. Например, если компьютеры или мобильные устройства детей подключаются к Интернету через Synology NAS, родители могут задать определенные настройки родительского контроля и ограничить использование Интернета для конкретных устройств и в определенное время.

Включение родительского контроля:

  1. Выберите Панель управления > Беспроводная сеть > Настройки родительского контроля.
  2. Нажмите Добавить, чтобы добавить клиентское устройство.
    • Имя клиента: укажите устройство, подключенное к Synology NAS, или введите имя устройства вручную (например, «Машин ноутбук»).
    • MAC-адрес: введите MAC-адрес устройства, для которого требуется ограничить использование Интернета
    • Настроить расписание: укажите периоды, в течение которых данному устройству будет разрешен доступ к Интернету с помощью Synology NAS.

Список подключений*

В списке подключений (Панель управления > Беспроводная сеть) отображаются сведения об устройствах, подключенных к Synology NAS. Из этого списка можно добавлять устройства в фильтр MAC напрямую.

Добавление устройств в список фильтра MAC:

  1. Установите флажок рядом с устройством, которое нужно добавить в фильтр MAC.
  2. Нажмите Добавить в список фильтров MAC.
