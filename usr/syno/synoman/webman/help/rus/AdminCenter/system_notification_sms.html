SMS

Вы можете активировать уведомления по SMS для получения уведомлений от устройства Synology NAS по SMS при изменении состояния или возникновения ошибки в системе, например сбоя подключения, изменений в работе вентилятора, сбоя питания, отсутствия свободного места в разделе, сбоя резервного копирования или восстановления и др.

Примечание

  • Изменение поведения вентилятора на модели без вентилятора недоступно, например EDS14. Для получения дополнительных сведений посетите веб-сайт Synology.

Включение уведомлений по SMS

Установите флажок Включить уведомления по SMS и выполните следующие настройки:

  1. Выберите провайдера SMS в раскрывающемся списке Провайдер службы SMS. Провайдеры службы по умолчанию: «clickatell», «clickatell-2017» и «SendinBlue-v3».
  2. Нажмите Добавить провайдера службы SMS, чтобы добавить новую конфигурацию провайдера службы SMS, или нажмите Удалить, чтобы удалить выбранную конфигурацию провайдера SMS.
  3. Если провайдер службы SMS использует аутентификацию по имени пользователя и паролю, выберите соответствующие категории. В полях вкладки Имя пользователя введите имя пользователя учетной записи службы SMS. В полях Пароль и Подтвердите пароль введите и подтвердите пароль учетной записи службы SMS.
  4. Если провайдер службы SMS использует аутентификацию по ключу API, выберите категорию Ключ API. В поле Ключ API введите ключ API, предоставленный провайдером службы SMS.
  5. Если провайдер службы SMS поддерживает возможность изменения имени отправителя, выберите категорию Отправитель. В поле Отправитель введите имя отправителя в соответствии с правилами, заданными провайдером службы SMS.
  6. Введите Основной номер телефона или Второй номер телефона. Если ввести два этих номера, SMS-уведомления будут отправляться на оба номера.
  7. Если вы не хотите получать много сообщений, установите флажок Включить минимальный интервал SMS и ограничение на количество уведомлений, равное одному сообщению за указанный временной интервал.
  8. Нажмите Отправить тестовое сообщение SMS, чтобы отправить тестовое сообщение SMS на указанный номер телефона. Если сообщение не получено, значит указана неверная конфигурация.
  9. Нажмите Применить, чтобы сохранить конфигурацию, или нажмите Сброс, чтобы отменить конфигурацию.

Примечание

  • Язык уведомлений SMS установлен в разделе Язык уведомлений. Чтобы изменить язык уведомлений SMS, перейдите в Главное меню > Панель управления > Региональные параметры, выберите вкладку Язык и из
    раскрывающегося меню выберите необходимый язык в разделе Язык уведомлений.
  • Все провайдеры службы SMS отправляют уведомления на английском языке, кроме провайдеров службы SMS по умолчанию: «clickatell» и «clickatell-2017».

Ограничения:

  • Имя пользователя: вводится с учетом регистра и должно содержать от 1 до 32 символов Unicode.
  • Пароль: вводится с учетом регистра, допускается от 0 до 128 отображаемых символов, включая буквы, цифры, знаки и пробелы.
  • Ключ API: вводится с учетом регистра и должен содержать от 1 до 256 символов Unicode.
  • Отправитель: вводится с учетом регистра и должен содержать от 1 до 64 символов Unicode.
  • Минимальный интервал SMS должен находиться в пределах от 1 до 10000 минут.

Создание конфигурации провайдера SMS

  1. Нажмите Добавить провайдера службы SMS.
  2. В поле Имя провайдера введите имя провайдера службы SMS.
  3. В поле URL SMS введите URL SMS. Получить URL SMS можно у провайдера службы SMS.
  4. Выберите GET или POST в поле Метод HTTP. Эти сведения можно узнать в документации по API провайдера службы SMS.
  5. Нажмите Далее.
  6. В поле Изменить заголовок запроса HTTP необходимо ввести параметры, указанные в документации по API провайдера службы SMS.
  7. Нажмите Далее.
  8. В поле Изменить текст запроса HTTP (если выбран POST в качестве Метод HTTP) необходимо ввести параметры, указанные в документации по API провайдера службы SMS.
  9. Нажмите Далее.
  10. Система разделяет URL SMS, заголовок запроса HTTP и текст запроса HTTP на несколько параметров. Задайте каждый параметр, выбрав соответствующую категорию. Каждую категорию параметра можно использовать только один раз, за исключением категории Другое.

Ограничения:

  1. Имя провайдера: вводится с учетом регистра и должно содержать от 1 до 32 символов Unicode.
  2. URL SMS: вводится с учетом регистра и должен содержать от 1 до 500 символов Unicode.Данный URL SMS предназначен только для настройки. После завершения настройки сообщения SMS отправляться не будут. Формат данного URL-адреса предоставляется провайдером службы SMS.В примере с Clickatell вариантом URL-адреса будет следующее:
    https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world
  3. Параметр: вводится с учетом регистра и должен содержать от 1 до 32 символов Unicode.
  4. Значение: вводится с учетом регистра и должно содержать от 1 до 256 символов Unicode.
  5. Можно добавить не более 10 параметров.
  6. Обратите внимание, что в параметре «содержимое сообщения» пробел между «Hello» и «world» будет заменен на символ, установленный провайдером службы.
    В примере с Clickatell значение параметра «содержимое сообщения» будет иметь вид: «Hello+world», символ «+» заменяет пробел в Clickatell.
