Защита учетной записи

Функция защиты учетной записи повышает уровень безопасности Synology NAS, защищая учетные записи от ненадежных клиентов, с которых выполнено слишком много неудачных попыток входа. Это позволяет уменьшить риск взлома учетных записей в результате атак путем полного перебора.

Примечания

  • Поддерживаемые службы и пакеты: DSM, File Station, Audio Station, Video Station, Download Station, Mail Station, Cloud Station и мобильные приложения Synology.

Включение защиты учетной записи

  1. Выберите Панель управления > Безопасность > Защита учетной записи.
  2. Установите флажок Включить защиту учетной записи.
  3. Укажите число в поле Попытки входа и в поле В течение (минут). Ненадежный клиент будет заблокирован, если в течение предварительно заданного периода времени будет зарегистрировано слишком большое количество неуспешных попыток входа.
  4. Укажите число в поле Разблокировать (спустя минут). Защита учетной записи будет отменена по истечении предварительно заданного периода времени.
  5. Нажмите Применить, чтобы сохранить настройки.

Отмена защиты выбранных учетных записей

  1. Нажмите Управление защищенными учетными записями.
  2. Выберите учетные записи, защиту которых требуется отменить, и нажмите Отменить защиту.

Разблокирование доверенных клиентов

  1. Нажмите Управление доверенными устройствами.
  2. Выберите клиентские устройства, которые требуется разблокировать, и нажмите Разблокировать.
