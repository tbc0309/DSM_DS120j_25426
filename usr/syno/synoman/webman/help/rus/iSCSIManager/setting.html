iSNS

Служба iSNS (Internet Storage Name Service) обеспечивает механизм централизованного управления. Вы можете зарегистрировать на сервере iSNS все iSCSI Target.

Регистрация объектов iSCSI Target на сервере iSNS

  1. Нажмите iSNS и установите флажок Включить iSNS.
  2. Введите IP-адрес сервера iSNS. Нажмите кнопку OK, чтобы применить настройку.

Глубина очереди операций ввода-вывода

По умолчанию глубина очереди операций ввода-вывода — 16, однако это значение можно изменить для всех Target.

Запись при недостатке свободного пространства

Если на томе осталось менее 1 ГБ свободного пространства, LUN на томе будут защищены системой для предотвращения сбоя файловой системы, и пользователи не смогут записывать данные на них. Функция записи при недостатке свободного пространства позволяет пользователям продолжать запись данных на LUN, когда в томе заканчивается свободное пространство. Однако рекомендуется отключить этот параметр после устранения проблемы нехватки места.

Примечание.

  • Даже если эта функция включена, не следует выполнять запись данных на LUN.

Режим реорганизации пространства

Ниже приведены три режима освобождения пространства. Регулируя скорость, можно контролировать влияние освобождения пространства на производительность LUN.

  • Приоритизация согласованности операций ввода-вывода. Приоритизация согласованности операций ввода-вывода сводит к минимуму влияние на производительность при освобождении пространства, которое занимал устаревший расширенный LUN. Скорость снижается при непрерывном входящем потоке операций ввода-вывода. При получении последовательных команд освобождения пространства этот процесс будет отложен до тех пор, пока не будут выполнены другие команды освобождения пространства.
  • Сбалансированный. Режим освобождения по умолчанию. Этот режим обеспечивает баланс между скоростью выполнения операций ввода-вывода и скоростью освобождения пространства.
  • Приоритизация освобождения пространства. Приоритизация освобождения пространства подразумевает, что входящий поток операций ввода-вывода оказывает незначительное влияние на скорость освобождения пространства. Освобождение пространства будет выполняться на полной скорости без ущерба для последовательности ввода-вывода.
