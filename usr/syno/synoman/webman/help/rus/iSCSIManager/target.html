Target

В зависимости от модели Synology NAS поддерживает создание нескольких Target, с которыми можно сопоставить номера LUN.

Примечание. Количество поддерживаемых Target зависит от моделей Synology NAS.

Создание Target

  1. Нажмите кнопку Создать.
  2. Введите информацию о Target, включая Название, IQN; кроме того, вы можете включить аутентификацию CHAP.
  3. Нажмите Далее.
  4. Настройте сопоставление iSCSI LUN. Выберите Создать новый iSCSI LUN или Сопоставить существующие LUN.
  5. Нажмите Далее.
  6. Нажмите Назад, чтобы изменить, или Применить, чтобы подтвердить настройки.

Терминология

  • IQN. IQN (iSCSI Qualified Name) представляет собой набор уникальных имен для Target. Он не должен конфликтовать с Target IQN на других серверах. Имя IQN не должно превышать 128 символов и может содержать буквы (нечувствительны к регистру), числа, двоеточия, тире и точки. В качестве формата имени рекомендуется использовать iqn.[дата].[домен]:[идентификатор устройства], например: iqn.2000-04.com.synology:storage.diskarray.1.
  • Двусторонняя проверка подлинности CHAP: параметр Двусторонний CHAP позволит клиенту и серверу выполнять двустороннюю проверку подлинности. Любой пароль CHAP, используемый для выполнения проверки подлинности клиента, не должен быть настроен для выполнения проверки подлинности любого сервера, а любой пароль CHAP, используемый для серверной проверки подлинности, не должен быть настроен для выполнения проверки подлинности любого клиента.

    • Примечание. Процедура обнаружения на инициаторе iSCSI может привести к ошибке, если аутентификация CHAP в системе NAS включена. Вначале необходимо выполнить обнаружение, затем включить аутентификацию CHAP (до подключения к целевому объекту). В противном случае можно подключиться к целевому объекту с помощью IP-адреса и IQN.

Удаление Target

Вы можете удалить Target и любые сопоставленные LUN. При удалении iSCSI LUN все данные на iSCSI LUN будут удалены.

  1. Выберите Target, который хотите удалить.
  2. Нажмите Удалить.
  3. Чтобы удалить сопоставленные LUN, выберите Удалить сопоставленные LUN и выберите элементы для удаления.
  4. Нажмите OK.

Редактирование Target

  1. Выберите Target, который хотите отредактировать.
  2. В раскрывающемся меню Действие выберите Редактировать.
  3. Отредактируйте настройки необходимым образом. Нажмите OK, чтобы сохранить изменения.

Терминология

  • Контрольная сумма CRC: включите контрольную сумму CRC для проверки наличия ошибок при передаче данных.
  • Многосессионный режим: включите многосессионный режим с нескольких инициаторов для одновременного подключения к одной Target. Для защиты файлов от повреждения или потери данных настоятельно рекомендуется использовать файловую систему общего доступа или кластерную систему, например VMware Virtual Machine File System (VMFS) и Oracle Cluster File System (OCFS).
  • Маска: маска позволяет предоставлять доступ к Target только определенным IQN инициатора iSCSI.

Включение/отключение Target

  1. Выберите Target, который требуется включить или отключить.
  2. Выберите Включить или Отключить в раскрывающемся меню Действие.
