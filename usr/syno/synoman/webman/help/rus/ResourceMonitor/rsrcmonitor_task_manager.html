Диспетчер задач*

На вкладке Диспетчер задач можно просмотреть сведения об использовании процессора и памяти каждой службой и процессом.

Службы

Щелкните вкладку Службы для отслеживания различных значений каждой службы, включая использование процессора, время работы процессора, использование памяти и состояние чтения/записи диска.

Процессы

Нажмите вкладку Процессы, чтобы просмотреть использование процессора и памяти по каждому процессу для снижения нагрузки на процессор.

Состояние процесса: Выполняется, Спящий режим и Остановлено. Статусы процесса в Linux: выполнятся, остановлено/трассировка, спящий режим.

Состояние Спящий режим может сопровождаться буквой. Например, буква «D» означает спящий режим диска, «Z» — зависание, «X» — процесс не работает.

Общая память отображает объем используемой процессом физической памяти, которым можно поделиться с другими процессами. Частная память отображает объем используемой процессом физической памяти, который не могут использовать другие процессы.
