Журналы

На странице Центр журналов > Журналы можно просматривать и экспортировать журналы, а также выполнять поиск по журналам.

Поиск по журналам:

  1. В раскрывающемся меню поиска можно задать следующие критерии.
    • Ключевое слово: введите ключевое слово для поиска соответствующих сообщений в журналах.

      Примечания

      Для расширенного поиска можно использовать следующие специальные символы:

      • [*]: Этот специальный символ представляет собой символы, не являющиеся пробелами, или их отсутствие. Например, поисковой запрос [п*о] обозначает любое слово, начинающееся на [п] и заканчивающееся на [о], например [перо], [пианино] или [по].
      • [?]: Этот специальный символ представляет собой один символ, не являющийся пробелом. Например, [со?] обозначает любое слово, начинающееся на [со] и заканчивающееся одним символом, например, [сок], [сон] или [сом].
      • [AND]: Этот оператор будет показывать результаты, содержащие все поисковые запросы. Например: [администратор AND root] покажет результаты, содержащие слова [администратор] и [root].
      • [OR]: Этот оператор будет показывать результаты, содержащие один из поисковых запросов. Например, [администратор OR root] покажет результаты, содержащие слово [администратор] или [root].
      • [NOT]: Этот оператор исключает слова из поиска. Например, [NOT администратор] покажет результаты, не содержащие слово [администратор].
      • [()]: Скобки определяют порядок, в котором интерпретируются поисковые запросы, если используется и оператор AND, и оператор OR. Например, [(администратор OR root) AND NOT вход] покажет результаты, содержащие слово [администратор] или [root] и не содержащие слово [вход].
    • Диапазон дат: поиск журналов по заданному диапазону даты.
    • Уровень: поиск журналов по заданному уровню.
    • Имя хоста: поиск журналов по заданному имени хоста.
    • Программа: поиск журналов по заданному названию программы.
    • Категория: поиск журналов по заданной категории программы.
  2. Нажмите Поиск, чтобы искать журналы, удовлетворяющие указанным критериям.

Экспорт журналов

  1. Нажмите стрелку рядом с кнопкой Экспорт.
  2. Выберите HTML или CSV, чтобы скачать журналы в нужном формате.

Примечания

  • Кодировка символов в экспортированных файлах CSV — UTF-8. Чтобы открыть такой файл, используйте программу, поддерживающую кодировку UTF-8.
