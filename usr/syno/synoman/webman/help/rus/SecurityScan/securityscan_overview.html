Обзор

Вы можете проверить текущее состояние DSM, выполнить сканирование системы, а также проверить ход выполнения сканирования и сводную информацию о результатах. Security Advisor сканирует DSM и Synology NAS, проверяя количество правил безопасности. Количество и типы правил зависят от того, как настроена Базовая безопасность. Базовая безопасность определяется вашими потребностями. Вы используете Synology NAS для работы или для личного использования? При первом запуске сканирования вам будет предложено выбрать Базовую безопасность. Позднее этот параметр можно изменить на странице Дополнительно.

Состояние системы

Security Advisor имеет семь состояний. После завершения сканирования появится сводная информация о результатах с разделением на пять категорий. Различные состояния:

  • Готов к сканированию: это состояние отображается, когда сканирование Synology NAS еще не началось. Нажмите Сканировать для начала сканирования.
  • Сканирование: сканирование выполняется. Можно посмотреть ход выполнения сканирования. Нажмите Остановить, чтобы остановить сканирование.
  • Хорошее: текущее сканирование завершено. Приложение Security Advisor не обнаружило уязвимостей или рисков безопасности. В настоящее время Synology NAS в безопасности.
  • Предупреждение. текущее сканирование завершено. Из-за некоторых уязвимых настроек безопасности приложение Security Advisor сообщило о низких/умеренных рисках безопасности. Для повышения безопасности Synology NAS выполните рекомендуемые действия.
  • В зоне риска: текущее сканирование завершено. Из-за некоторых уязвимых настроек безопасности приложение Security Advisor сообщило о высоких рисках безопасности. Для повышения безопасности Synology NAS обязательно выполните рекомендуемые действия.
  • Опасность: текущее сканирование завершено. Приложение Security Advisor обнаружило вредоносное ПО на устройстве Synology NAS. Необходимо выполнить рекомендуемые действия, чтобы предотвратить дальнейшие нежелательные последствия для Synology NAS.
  • Устаревший: ваша версия DSM устарела. Для удобства работы и обеспечения безопасности Synology NAS рекомендуется обновить DSM для получения новых функций, улучшений и исправлений безопасности.

Security Advisor делит проверки на пять категорий:

  • Вредоносное ПО: определяет, есть ли вредоносное ПО на Synology NAS и подвергается ли Synology NAS атаке
  • Система: определяет настройки безопасности системы.
  • Учетная запись: определяет настройки безопасности учетной записи и пароля.
  • Сеть: определяет настройки безопасности сети.
  • Обновить: определяет, используется ли последняя версия DSM и других установленных пакетов, а также проверяет настройки обновлений.

После завершения сканирования можно посмотреть подробные результаты в форме отчета, нажав Просмотреть результаты.

Отчеты

Просмотр отчета по текущему анализу входа и проверка безопасности

  1. Нажмите Создать отчеты сейчас.
  2. Откроется всплывающее окно с сообщением о том, что отчет успешно создан. Нажмите кнопку ОК, чтобы просмотреть отчет.

Просмотр отчетов из истории:

  1. Нажмите перевернутый треугольник рядом с пунктом Создать отчеты сейчас.
  2. Нажмите Просмотреть отчеты из истории.
  3. Нажмите отчет, чтобы открыть его.
