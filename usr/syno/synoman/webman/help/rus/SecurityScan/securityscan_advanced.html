Дополнительно

Вы можете просматривать и изменять дополнительные параметры Security Advisor.

Базовая безопасность

Выбрав Базовую безопасность, можно определить, какие правила безопасности будет проверять Security Advisor. Количество и типы правил отличаются для каждой Базовой безопасности. Сделайте выбор в зависимости от ваших целей. Вы используете Synology NAS для работы или для личного использования? Можно выбрать следующие варианты:

  • Для дома и личного использования: Security Advisor выполнит основные проверки для защиты устройства Synology NAS от основных кибер-атак.
  • Для работы и бизнеса: Security Advisor выполнит комплексные проверки в соответствии с требованиями безопасности для организаций.
  • Пользовательский. определите, какие правила безопасности будет проверять Security Advisor.

График сканирования

Можно задать график регулярного сканирования, нажав Включить график регулярного сканирования и выбрав время выполнения проверки Security Advisor. Security Advisor будет проверять правила в соответствии с выбранной Базовой безопасностью.

Настройки отчета

Получение отчетов можно настроить в Security Advisor. Выберите общую папку в качестве места для хранения отчетов. Затем установите флажок Включить ежемесячный отчет или Включить ежедневный отчет. Чтобы получать отчеты по электронной почте, перейдите в Панель управления > Уведомления > Электронная почта и включите уведомления по электронной почте.
