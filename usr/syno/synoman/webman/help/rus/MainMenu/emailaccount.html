Учетная запись электронной почты

Учетные записи электронной почты можно добавить в Личная > Учетная запись электронной почты, а затем, используя добавленные сюда адреса электронной почты, отправлять файлы, хранящиеся в File Station, в виде вложений. Кроме того, можно добавить учетную запись электронной почты в разделе Настройки > Учетная запись электронной почты в Drive и Calendar. После добавления учетной записи можно рассылать приглашения на события по электронной почте с помощью Calendar и уведомлений электронной почты (при предоставлении доступа к файлам в Drive другим пользователям).

К поддерживаемым поставщикам услуг электронной почты относятся Gmail, Outlook и пользовательский SMTP-сервер.

Чтобы добавить учетную запись электронной почты, выполните следующие действия.

  1. Нажмите Добавить и выберите поставщика услуг для учетной записи электронной почты, которую необходимо настроить.
  2. Введите свой адрес электронной почты, пароль и имя отправителя.
  3. Если установить флажок Использовать данную учетную запись электронной почты как запись по умолчанию, File Station, Calendar и Drive будут использовать указанную учетную запись электронной почты по умолчанию при настройке нескольких учетных записей.
  4. Выберите Синхронизировать контакты, чтобы синхронизировать контакты электронной почты с Synology NAS, если используемый поставщик услуг (Gmail и Outlook) поддерживает эту функцию. Обратите внимание, что контакты будут синхронизированы с учетной записи электронной почты, указанной в открывшемся окне аутентификации (не с учетной записи, указанной на предыдущем этапе). Таким образом, если в окне аутентификации отображается другая или нежелательная учетная запись, вручную переключитесь на нужную учетную запись электронной почты или на учетную запись, которая использовалась ранее.
  5. Нажмите Применить, чтобы сохранить настройки.

Чтобы изменить учетную запись электронной почты, выполните следующие действия.

Выберите нужную учетную запись из списка и затем нажмите Редактировать.

Чтобы удалить учетную запись электронной почты, выполните следующие действия.

Выберите нужную учетную запись из списка и затем нажмите Удалить.

Примечание.

  • Учетная запись по умолчанию не может быть удалена при отсутствии других учетных записей. Если вы хотите удалить учетную запись электронной почты, которая в настоящее время используется по умолчанию, перед продолжением измените учетную запись по умолчанию на другую учетную запись.

Чтобы изменить учетную запись по умолчанию, выполните следующие действия.

Выберите из списка учетную запись, которую необходимо использовать по умолчанию, и затем выберите Установить по умолчанию.

Чтобы отправить файлы в виде вложений, выполните следующие действия.

  1. Щелкните правой кнопкой мыши любые файлы в File Station и выберите Отправить как вложение электронной почты.
  2. При добавлении нескольких учетных записей электронной почты вы можете выбрать нужный адрес электронной почты отправителя в выпадающем списке, чтобы отправить файлы указанному получателю.
