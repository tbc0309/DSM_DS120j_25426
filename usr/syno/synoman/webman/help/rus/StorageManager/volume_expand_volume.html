Расширение тома

Размер тома можно изменить, если том создан в пуле ресурсов хранения, который поддерживает несколько томов. Однако если том создан в пуле ресурсов хранения, который поддерживает только один том, вы можете расширить размер тома, добавив диски в пул ресурсов хранения и сначала увеличив емкость пула ресурсов хранения.

Для пула ресурсов хранения с несколькими томами

Для расширения размера тома выполните следующие действия.

  1. Перейдите на страницу Том и выберите том, который требуется расширить.
    Примечание. При необходимости можно добавить диски в соответствующий пул ресурсов хранения, чтобы сначала расширить емкость пула ресурсов хранения. Подробнее см. в разделе Добавление дисков для расширения емкости пула ресурсов хранения.
  2. В раскрывающемся меню Действие выберите Настроить.
  3. Перейдите на вкладку Общие и введите новый размер тома в поле Изменить распределенный размер (ГБ) или нажмите кнопку Макс., чтобы автоматически указать максимальный размер тома.
  4. Нажмите OK для завершения.

Для пула ресурсов хранения с одним томом

Для расширения размера тома выполните следующие действия.

  1. Добавьте диски в пул ресурсов хранения для расширения емкости пула ресурсов хранения.
    Примечание. Подробнее см. в разделе Добавление дисков для расширения емкости пула ресурсов хранения.
  2. Размер соответствующего тома будет автоматически расширен после расширения емкости пула ресурсов хранения.
