Hot Spare

Диски горячего резерва Hot Spare — это резервные диски, которые могут использоваться для восстановления неисправного пула ресурсов хранения путем автоматической замены отказавшего диска. Диски Hot Spare глобально назначаются для защиты любого пула ресурсов хранения в Synology NAS, если этот пул ресурсов хранения удовлетворяет следующим критериям.

  • Тип RAID пула ресурсов хранения поддерживает защиту данных (т. е. RAID 1, RAID 5, RAID 6, RAID 10, RAID F1).
  • Размер диска Hot Spare должен быть не меньше, чем размер самого малого диска в пуле ресурсов хранения.

Настройка дисков Hot Spare

  1. Нажмите Управление.
  2. Откроется диалоговое окно, в котором отображается состояние дисков Hot Spare в каждом пуле ресурсов хранения, а также доступные диски. В сведения о состоянии пула ресурсов хранения входят следующие данные:
    • Назначенные: Пул ресурсов хранения защищен как минимум одним назначенным диском Hot Spare.
    • Доступные: Имеются доступные диски, которые могут обеспечить защиту пула ресурсов хранения, но диски Hot Spare еще не назначены.
    • Недоступные: Нет доступных дисков Hot Spare, которые могут обеспечить защиту пула ресурсов хранения.
  3. В разделе Доступные диски показаны диски, которые можно назначить как диски Hot Spare. Чтобы назначить диск Hot Spare, установите флажок рядом с диском, который будет выполнять функцию Hot Spare.
  4. Нажмите Применить, чтобы сохранить настройки.

Примечание.

  • Если диск назначен в качестве диска Hot Spare, все данные на этом диске будут удалены. Перед назначением диска Hot Spare убедитесь, что на диске нет важных данных.
  • Массив RAID, созданный на основе жестких дисков, не будет автоматически восстанавливаться твердотельными дисками Hot Spare, и наоборот.
  • Рекомендуется выбрать совместимые диски для использования в режиме Hot Spare для предотвращения проблем с Synology NAS.

Настройка параметров Hot Spare для пула ресурсов хранения

  1. Нажмите Настройка.
  2. На вкладке Пул ресурсов хранения отображается список пулов ресурсов хранения, которые могут быть защищены дисками Hot Spare. Если снять флажки, поврежденные данные соответствующих пулов ресурсов хранения не будут автоматически восстанавливаться дисками Hot Spare.
  3. На вкладке Hot Spare для разных устройств можно разрешить дискам Hot Spare одного устройства восстанавливать поврежденные RAID на другом устройстве.
  4. Нажмите кнопку OK, чтобы сохранить изменения.
