Пул ресурсов хранения

Один или несколько дисков можно объединить в один блок хранения, называемый пулом ресурсов хранения, который можно защитить с помощью RAID. Разные типы RAID обеспечивают разные уровни защиты данных. Общий список типов RAID, совместимых с Synology NAS, см. в разделе Общие. После создания пула ресурсов хранения можно создать на нем один или несколько томов. Том является единицей хранения данных и может быть расширен, если соответствующий пул ресурсов хранения имеет распределяемую емкость. Прежде чем начать сохранение данных на Synology NAS, необходимо создать хотя бы один том.

Примечание.

  • Рекомендуется использовать диски, указанные в Списке совместимых продуктов Synology, чтобы избежать ошибок совместимости. Использование несовместимых дисков может снизить производительность системы или даже привести к потере данных.
  • Пул ресурсов хранения должен состоять из дисков одного типа. Это означает, что пул ресурсов хранения не может быть создан с использованием сочетания дисков SATA и SAS.
  • Для моделей Synology NAS, поддерживающих RAID Group, пул ресурсов хранения RAID 5, RAID 6 или RAID F1 с поддержкой нескольких томов может содержать несколько массивов RAID. Все массивы RAID в одном пуле ресурсов хранения будут относиться к одному и тому же типу RAID (например, RAID 5, RAID 6 или RAID F1). Максимальное количество дисков на массив RAID может составлять 6, 12 или 24. Задать максимальное количество дисков можно при создании пула ресурсов хранения. Изменить это количество впоследствии будет невозможно. После назначения максимального количества дисков массиву RAID следующий диск, добавляемый в пул ресурсов хранения, можно назначить новому массиву RAID. Новый массив RAID должен содержать минимальное количество дисков, необходимых для определенного типа RAID (т. е. три диска для RAID 5 или RAID F1 и четыре диска для RAID 6).
  • Если вы хотите использовать определенный диск для создания пула ресурсов хранения, но этот диск не отображается в Мастере создания пула ресурсов хранения, перейдите на страницу HDD/SSD для проверки состояния диска.
