Преобразование пула ресурсов хранения с одним томом в пул ресурсов хранения с несколькими томами

Можно добавить диски в пул ресурсов хранения и превратить этот пул ресурсов хранения в пул, поддерживающий несколько томов. Исходные данные на одиночном томе не затрагиваются.

Преобразование пула ресурсов хранения с одним томом в пул ресурсов хранения с несколькими томами:

  1. Откройте Диспетчер хранения.
  2. Перейдите в раздел Пул ресурсов хранения.
  3. Нажмите Действие > Преобразование пула ресурсов хранения с одним томом в пул ресурсов хранения с несколькими томами.

Примечание.

  • Эта функция поддерживается только пулами ресурсов хранения SHR. SHR (Synology Hybrid RAID) поддерживается только в некоторых моделях. Для получения дополнительных сведений посетите веб-сайт Synology.
  • Количество добавленных дисков должно увеличить размер пула ресурсов хранения; в противном случае преобразование не будет выполнено.
  • Рекомендуется использовать диски, указанные в Списке совместимых продуктов Synology, чтобы избежать ошибок. Использование несовместимых дисков может снизить производительность системы или даже привести к потере данных на устройстве Synology NAS.
