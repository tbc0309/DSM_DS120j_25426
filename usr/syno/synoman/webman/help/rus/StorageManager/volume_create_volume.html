Создание тома

Поскольку том располагается в пуле ресурсов хранения, до создания тома необходимо создать пул ресурсов хранения. Однако если модель устройства поддерживает режим быстрого создания, можно создать том и пул ресурсов хранения одновременно; при этом мастер создания томов автоматически оптимизирует емкость и производительность хранилища (заботиться о технических аспектах не требуется). Режим быстрого создания доступен только в моделях, поддерживающих SHR (Synology Hybrid RAID). Для получения дополнительных сведений посетите веб-сайт Synology.

Создание тома на модели устройства с поддержкой режима быстрого создания

Можно одновременно создать том и пул ресурсов хранения. Выберите один из способов создания нового тома.

  • Быстрый. Лучший вариант для новичков. Выберите этот режим, чтобы автоматически оптимизировать емкость и производительность хранилища, не заботясь о технических аспектах, например типах RAID.
  • Пользовательский. Этот способ подходит пользователям, которые хотят настроить дополнительные параметры создания тома, например типы RAID и распределение емкости. (Сведения о настройке пула ресурсов хранения см. здесь.)

Чтобы создать том в быстром режиме, выполните указанные ниже действия.

  1. Откройте Диспетчер хранения.
  2. Выберите Том и нажмите Создать.
  3. Отобразится мастер создания томов. Нажмите Быстрый или Пользовательский на свой выбор. Следуйте инструкциям мастера для завершения создания тома.

Создание тома на других моделях

  1. Убедитесь, что на устройстве Synology NAS существует пул ресурсов хранения с емкостью, доступной для выделения.
  2. Нажмите кнопку Создать.
  3. Выберите пул ресурсов хранения, в котором будет создан новый том.
  4. Выберите емкость, которая будет выделена для тома.
  5. Нажмите Применить для подтверждения настроек.

Примечание.

  • Если при создании пула ресурсов хранения был выбран тип Улучшенная производительность, вы не сможете задать выделяемую емкость для томов, созданных в этом пуле ресурсов хранения.
  • Максимальный размер тома варьируется в зависимости от модели. Подробные сведения см. в технических характеристиках модели.
  • Минимальная выделяемая емкость для тома должна составлять не менее 10 ГБ.
  • Том Btrfs резервирует 5% пространства для метаданных.
