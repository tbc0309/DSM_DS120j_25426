Создание нового пула ресурсов хранения

Внимание!

  • Перед началом работы убедитесь, что на дисках, из которых будет создан пул ресурсов хранения, нет важных данных. При создании все имеющиеся данные будут удалены.
  • Откройте страницу HDD/SSD и убедитесь, что для каждого диска установлен статус инициализирован или Не инициализирован.
  • Рекомендуется использовать диски, указанные в Списке совместимых продуктов Synology, чтобы избежать ошибок. Использование несовместимых дисков может снизить производительность системы или даже привести к потере данных на устройстве Synology NAS.

Создание пула ресурсов хранения

  1. Откройте Диспетчер хранения.
  2. Выберите Пул ресурсов хранения и нажмите кнопку Создать.
  3. Выберите тип пула ресурсов хранения.
    • Улучшенная производительность. Данный тип пула хранения поддерживает только один том, но обеспечивает более высокую производительность.
    • Дополнительная гибкость. Этот тип пула ресурсов хранения поддерживает несколько томов, Synology Hybrid RAID (SHR) и функцию RAID Group.
  4. Настройка свойств пула ресурсов хранения
    • Описание пула ресурсов хранения Описание данного пула ресурсов хранения.
    • Тип RAID Разные типы RAID обеспечивают разные уровни защиты данных. Дополнительная информация представлена в разделе Общее.
    • Минимальное число дисков в массиве RAID Минимальное число дисков, необходимое для выбранного типа RAID.
    • Максимальное число дисков в массиве RAID Увеличение количества дисков в массиве RAID повышает вероятность отказа RAID. Рекомендуется изменить предельное число дисков в одном массиве RAID и использовать RAID Group для обеспечения нужного баланса между безопасностью данных и емкостью пула ресурсов хранения.
    • Функция RAID Group Указывает, поддерживается ли функция RAID Group выбранным типом RAID. Один массив RAID может иметь ограниченное количество дисков, а функция RAID Group позволяет сформировать блок хранения из нескольких массивов RAID. Это дает возможность увеличить емкость и избыточность дисков. RAID Group поддерживается конфигурациями RAID 5, RAID 6 и RAID F1.
  5. Завершите создание пула ресурсов хранения, следуя инструкциям мастера.

Примечание.

  • SHR (Synology Hybrid RAID) поддерживается только в некоторых моделях. Для получения дополнительных сведений посетите веб-сайт Synology.
  • Максимальное число дисков в каждом массиве RAID и функция RAID Group доступны только в некоторых моделях. Для получения дополнительных сведений посетите веб-сайт Synology.
  • Максимальное число дисков в каждом массиве RAID можно изменить, только если выбранный тип RAID поддерживает функцию RAID Group.
  • Если пул ресурсов хранения использует конфигурацию RAID 5, RAID 6, RAID F1, SHR (указанные выше три диска) или SHR-2, система выполнит повторную синхронизацию RAID с пулом ресурсов хранения, чтобы улучшить скорость доступа и повысить отказоустойчивость данных.
