Настройка ограничений скорости повторной синхронизации

Повторная синхронизация — это глобальная настройка, регулирующая предел скорости для следующих операций: исправление ошибок данных, восстановление пулов ресурсов хранения, расширение пулов ресурсов хранения и изменение типа RAID для пулов ресурсов хранения.

Регулировка скорости повторной синхронизации

  1. Откройте Диспетчер хранения.
  2. Перейдите в раздел Пул ресурсов хранения > Конфигурации.
  3. Выберите один из следующих параметров ограничения скорости:
    • Снизить воздействие на общую производительность системы (рекомендуется) Выберите этот параметр, чтобы избежать снижения производительности из-за повторной синхронизации.
    • Выполнить повторную синхронизацию RAID быстрее
    • Настроить
      • Макс. Максимальна скорость во время бездействия системы.
      • Мин. Максимальная скорость при выполнении операций ввода-вывода.
        Примечание. Фактическая скорость может варьироваться в зависимости от моделей и дисков.
    • Нажмите Применить.

Примечание.

  • Эта функция доступна только в некоторых моделях.
