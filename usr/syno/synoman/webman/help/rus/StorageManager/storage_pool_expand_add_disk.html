Добавление дисков для расширения емкости пула ресурсов хранения

При наличии пустых слотов для дисков на Synology NAS можно установить дополнительные диски и добавить их в существующий пул ресурсов хранения. Эта статья содержит инструкции по расширению существующего пула ресурсов хранения путем установки дополнительных дисков.

Диски могут быть добавлены в пулы ресурсов хранения со следующими конфигурациями RAID.

  • JBOD
  • RAID 5
  • RAID 6
  • RAID F1
  • Synology Hybrid RAID (SHR), состоящий как минимум из 2 дисков

Примечание.

  • Состояние новых дисков должно быть Инициализировано или Не инициализировано.

Требования к дискам

Убедитесь, что диски, которые вы хотите установить в Synology NAS, соответствуют следующим требованиям.

  • Для конфигурации RAID или SHR требуются диски одного типа. Использование дисков разного типа влияет на надежность и производительность системы. Использование дисков разного типа, как показано ниже, не поддерживается для конфигурации RAID или SHR.
    • Диски SATA и SAS
    • Диски, выполненные по технологии 4K, и диски, не выполненные по этой технологии
  • Для SHR. Емкость добавляемого диска должна быть не меньше самого большого диска в пуле ресурсов хранения или быть равной любому диску в пуле ресурсов хранения.
    Пример. Если пул ресурсов хранения SHR состоит из трех дисков (2 ТБ, 1,5 ТБ и 1 ТБ), рекомендуется использовать диск объемом не менее 2 ТБ для более эффективного использования емкости. Можно добавить диски емкостью 1,5 ТБ и 1 ТБ, но обратите внимание, что часть емкости диска 2 ТБ останется неиспользуемой.
  • Для RAID 5, RAID 6 или RAID F1. Емкость добавляемого диска должна быть не меньше, чем у диска с самой малой емкостью в пуле ресурсов хранения.
    Пример. Если пул ресурсов хранения RAID 5, RAID 6 или RAID F1 состоит из трех дисков емкостью 2 ТБ, 1,5 ТБ и 1 ТБ, то емкость нового диска должна быть не меньше 1 ТБ.

Примечание.

  • SHR (Synology Hybrid RAID) поддерживается только в некоторых моделях. Перейдите по этой ссылке, чтобы узнать, обеспечивает ли ваша модель ограниченную поддержку SHR.
  • RAID F1 поддерживается только в некоторых моделях. Перейдите по этой ссылке, чтобы узнать, поддерживает ли ваша модель RAID F1.
  • Рекомендуется использовать диски, указанные в Списке совместимых продуктов Synology, чтобы избежать ошибок. Использование несовместимых дисков может снизить производительность системы или даже привести к потере данных.

Добавление диска в пул ресурсов хранения

Перед началом убедитесь, что пул ресурсов хранения, который требуется расширить, находится в состоянии Исправно. Если состояние пула ресурсов хранения неисправно, выполните рекомендуемые действия, указанные под состоянием, чтобы устранить проблемы с диском. Затем выполните следующие действия, чтобы добавить диск в пул ресурсов хранения.

  1. Выключите Synology NAS.
    Примечание. Если Synology NAS поддерживает горячую замену, пропустите этот шаг.
  2. Вставьте новый диск в пустой отсек для дисков.
    Примечание. Во избежание получения травм и повреждения устройства Synology NAS перейдите в Центр загрузок и выберите свой продукт Synology. В разделе Документы > Руководство по установке оборудования можно скачать руководство по установке оборудования на выбранном языке, чтобы ознакомиться с инструкциями по установке диска.
  3. Включите Synology NAS и запустите Диспетчер хранения.
  4. Перейдите на страницу Пул ресурсов хранения и выберите пул ресурсов хранения, который требуется расширить. В раскрывающемся меню Действие выберите Добавить диск.
  5. Выберите в списке диск, который требуется добавить. Будут перечислены только диски, отвечающие требованиям к типу и емкости.
  6. Следуйте инструкциям мастера для завершения процесса.

Расширение тома

Если пул ресурсов хранения имеет достаточную емкость, размер соответствующего тома можно расширить автоматически или вручную (в зависимости от типа пула ресурсов хранения).

  • Пул ресурсов хранения с одним томом Для пула ресурсов хранения, который поддерживает один том: размер тома будет автоматически расширен после расширения емкости пула ресурсов хранения. Подробнее см. в разделе Добавление диска в пул ресурсов хранения.
  • Пул ресурсов хранения с несколькими томами Для пула ресурсов хранения, который поддерживает несколько томов: размер тома можно расширить вручную, если пул ресурсов хранения имеет достаточную емкость. Подробнее см. в разделе Расширение тома.
