Kopiera en användare

Du kan kopiera inställningarna för en befintlig användare till en ny användare. Det här gör att du snabbt kan skapa en ny användare utan att behov av att konfigurera samma inställningar igen.

  1. Välj den användare du vill kopiera.
  2. Klicka på Skapa > Kopiera användare. Fönstret för guiden Skapa användare öppnas.
  3. Inställningarna för den valda användare tillämpas som standard på den nya användaren (exklusive personlig information som användarnamn och e-postadress).
