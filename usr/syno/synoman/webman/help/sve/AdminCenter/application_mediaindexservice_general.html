Mediaindexering

Mediaindexering kan automatiskt skanna multimediafiler som lagras på din Synology NAS. Dessa filer kompileras till mediabiblioteket för indexering så att de kan bläddras bland och spelas upp med DMA-enheter och Synology:s multimedia-applikationer (t.ex. Media Server, Video Station, Audio Station, Photo Station och iTunes-server).

Hantera indexerade mappar

Att indexera en befintlig mapp eller skapa en indexerad mapp:

  1. Gå till Kontrollpanel > Indexeringstjänst > Mediaindexering.
  2. Klicka på Indexerad mapp > Skapa > Välj och gör något av nedanstående:
    • Att indexera en befintlig mapp: Markera de önskade mapparna i listan och klicka på Välj.
    • Att skapa en indexerad mapp: Välj föräldramappen från listan. Klicka på Skapa mapp för att specificera mappnamnet och klicka på OK. Markera sedan den mapp som skapats och klicka på Välj.
  3. Välj filtyperna (dvs. Bild, Musik och Video) som ska indexeras. Endast filer av den indexerade typen tillåts åtkomst från DMA-enheter och Synology:s multimedia-applikationer.
  4. Klicka på OK för att börja indexera.

Obs:

  • Mappar som är skapade för Synology:s multimedia-applikationer är som standard delade mappar (t.ex. music) och indexeras automatiskt, medan andra mappar bara kan indexeras manuellt.
  • Fjärrmappar som monterats med hjälp av File Station kan inte läggas till som indexerade mappar.
  • Du kan lägga till upp till 100 användare på din Synology NAS.

Redigera en indexerad mapp:

  1. Gå till Kontrollpanel > Indexeringstjänst > Mediaindexering > Indexerad mapp.
  2. I popup-fönstret, markera den önskade mappen.
  3. Klicka på Redigera och gör ändringar i inställningarna.
  4. Klicka på OK och sedan på Spara.

Ta bort mappar från listan över indexerade mappar:

Borttagna mappar blir avindexerade. De multimediafiler de innehåller blir inte åtkomliga för DMA-enheter och Synology:s multimedia-applikationer.

  1. Gå till Kontrollpanel > Indexeringstjänst > Mediaindexering > Indexerad mapp.
  2. I popup-fönstret, markera den oönskade mappen. För att välja flera oönskade mappar, tryck och håll kvar på Shift eller Ctrl vid markering.
  3. Klicka på Radera.
  4. Klicka på Ja och klicka sedan på Spara.

Obs:

Borttagna mappar raderas inte från din Synology NAS.

Att skapa nytt index för multimediafiler:

Om indexstrukturen förloras hjälper det att skapa nytt index för de indexerade mapparna för att återuppbygga strukturen.

  1. Gå till Kontrollpanel > Indexeringstjänst > Mediaindexering.
  2. Klicka på Skapa nytt index för att indexera alla tidigare indexerade mappar igen.

Obs:

Multimediafiler som läggs till indexerade mappar via protokollen NFS eller SCP kommer inte att skannas automatiskt. För att indexera sådana fler, klicka på Skapa nytt index.

Konvertera indexerade multimediafiler

Ställa in miniatyrbildens kvalitet:

Miniatyrbilder skapas automatiskt för bilder/videor som laddas upp för användning med Media Server eller Photo Station. Här kan du ställa in kvaliteten för miniatyrbilderna.

  1. Gå till Kontrollpanel > Indexeringstjänst > Mediaindexering > Inställningar för miniatyrbilder.
  2. Välj en kvalitet från rullgardinsmenyn.
  3. Klicka på Tillämpa.

Obs:

  • Det tar längre tid att skapa miniatyrbilder av hög kvalitet.
  • För att ladda upp ett stort antal bilder/videor till ditt bibliotek i Photo Station rekommenderar vi att Photo Station Uploader används (ett Synology-skrivbordsverktyg som finns i Nerladdningscenter) för snabbt skapande av miniatyrbilder.

Aktivera videokonvertering för mobila enheter:

Videor i den delade mappen photo (dedikerad till Photo Station) kan spelas upp på din mobila enhet om det är ordentligt konverterade.

  1. Gå till Kontrollpanel > Indexeringstjänst > Mediaindexering > Videoinställningar.
  2. Markera Aktivera videokonvertering för mobila enheter.
  3. Klicka på Tillämpa så kommer du även att kunna se dina videor på mobila enheter i en handvändning.

Obs:

  • Endast de videor som laddats upp till den delade mappen photo i Photo Station kan konverteras till det format som stöds av mobila enheter.
  • MP4-filer kommer inte att konverteras.

Visa bild-/videokonverteringsförloppet:

  1. När bilder och videor laddas upp till Synology NAS, klicka på ikonen Konverteringsförlopp som finns i det övre högra hörnet av aktivitetsfältet.
  2. Gör något av nedanstående i popup-fönstret:
    • I avsnittet Konverteringsförlopp, övervaka förlopp och status för miniatyrbilder som skapas i bakgrunden.
    • För att pausa konvertering, följ nedanstående steg:
      1. Välj Fördröjning för [antal] timme (timmar) från rullgardinsmenyn och klicka på Paus för att pausa förloppet.
      2. Konvertingsförloppen återupptas efter den specificerade fördröjningstiden. Alternativt kan du klicka på Fortsätt för att återuppta processen omedelbart.

      Obs:

      Eftersom konvertering kräver mer processorresurser kan en paus i processen spara systemresurser för andra pågående uppgifter.

  3. När konverteringsprocessen är slutförd kommer ikonen på aktivitetsfältet att försvinna.
