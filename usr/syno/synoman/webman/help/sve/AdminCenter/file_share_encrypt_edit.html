Redigera en krypterad mapp

Efter att en mapp har krypterats kan du använda rullgardinsmenyn Kryptering för att redigera mappen.

Posterna som visas i menyn beror på status av vald delad mapp:

  • Om du väljer en okrypterad delad mapp kommer rullgardinsmenyn Kryptering att vara inaktiverad.
  • Om du väljer en krypterad delad mapp som är monterad, kan du välja Avmontera från rullgardinsmenyn Kryptering för att avmontera mappen, eller Exportera nyckel för att spara krypteringsnyckeln.
  • Om du väljer en krypterad delad mapp som är omonterad kan du välja Montera från rullgardinsmenyn Kryptering och sedan ange eller importera krypteringsnyckeln för att montera mappen för tillträde.
  • Om du väljer flera krypterade delade mappar som inte är monterade och deras krypteringsnycklar hanteras av Nyckelhanterare, kan du välja Montera från rullgardinsmenyn Kryptera och sedan ange lösenfrasen för Nyckelhanterare för att montera dessa mappar för åtkomst.
    Obs:
  1. Det är mycket viktigt att komma ihåg krypteringsnyckeln. Det rekommenderas starkt att du exporterar och sparar krypteringsnyckeln. Om du glömmer eller tappar bort nyckeln finns det inget sätt att rädda den krypterade datan.
  2. När en krypterad delad mapp monteras kan du döpa om den eller ändra dess volymplats. Du kan endast utföra ändringar efter att du avmonterat den.
