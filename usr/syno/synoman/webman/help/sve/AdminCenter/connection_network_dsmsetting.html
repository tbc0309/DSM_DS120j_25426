DSM-Inställningar

I Kontrollpanel > Nätverk > DSM-inställningar kan du ändra portnumret som klienter använder för att få åtkomst till din Synology NAS.

Ändra standardportnumret:

Både HTTP- och HTTPS-anslutningar aktiveras alltid i DSM. HTTP-standardport är 5000 och HTTPS-standardport är 5001.

  1. Ange portnumret i fältet HTTP eller HTTPS.
  2. Klicka på Tillämpa.

Att omdirigera HTTP- till HTTPS-anslutningar:

Du kan omdirigera klienter till HTTPS-porten när de försöker få åtkomst till din Synology NAS genom HTTP-porten.

  1. Välj Omdirigera automatiskt HTTP-anslutningar till HTTPS (Web Station och Photo Station uteslutna).
  2. Klicka på Tillämpa.

Portar som används för HTTPS-anslutning:

  • DSM: Standardportnumret är 5001 så att du kan använda säker åtkomst till DSM genom att ange följande adress: https://server-hostname:5001/
  • Web Station och Photo Station: Standardportnumret är 443 så du kan använda säker åtkomst till åt Web Station och Photo Station genom att ange följande adress: https://server-hostname/

Att aktivera HTTP/2:

Du kan snabba upp HTTPS-anslutningar genom att aktivera HTTP/2.

  1. Bocka för Aktivera HTTP/2.
  2. Klicka på Tillämpa.

Att anpassa det maximala antalet samtidiga HTTP-anslutningar:

Du kan aktivera Anpassa maximalt antal samtidiga HTTP-anslutningar för att öka HTTP-anslutningarna på din Synology NAS.

Obs:

  • Ju högre antal maximalt samtidiga HTTP-anslutningar, desto mer systemminne krävs.

Att anpassa ”Server”-huvudet i HTTP-svar:

  1. Bocka för Aktivera ”Server”-huvud i HTTP-svar.
  2. Ange ett huvud för din webbserver (t.ex. nginx).
  3. Klicka på Tillämpa.

Du kan avaktivera ”Server”-huvudet genom att bocka för kryssrutan.

Domän

Under Domän kan du specificera ett skräddarsytt domännamn för åtkomst till Synology NAS.

Att aktivera anpassad domän:

  1. Bocka för Aktivera anpassad domän.
  2. Specificera ett domännamn som används för din Synology NAS.
  3. Du kan även bocka i Aktivera HSTS för att framtvinga att anslutningarna krypteras med HTTPS på den här domänen.
  4. Klicka på Tillämpa.

Obs:

  • Varje domännamn kan bara användas för en särskild applikation på din Synology NAS.
  • Om domännamnet inte är längre än 15 tecken och överensstämmer med NetBIOS-namngivningskonventioner blir domännamnet (t.ex. MyDSM) automatiskt registrerat och utsänt av systemet. Windows-användare kan få åtkomst till skrivbordet i DSM på samma lokala nätverk med en liknande webbadress: http://MyDSM eller https://MyDSM
  • När du får åtkomst till applikationer via webbadresser som använder domännamn går du genom HTTP-standardporten (80) eller HTTPS-standardporten (443).
