Tjänstupptäckt

Tjänstupptäckt gör att användare kan hitta sin Synology NAS på nätverket via SSDP (Simple Service Discovery Protocol) och Bonjour-tjänster.

Att göra din Synology NAS möjlig att upptäcka via SSDP:

SSDP-tjänsten gör att användare kan hitta Synology NAS via Windows nätverksidentifiering.

  1. Gå till Kontrollpanel > Filtjänster > Tjänstupptäckt > SSDP.
  2. Välj Aktivera upptäckt av Windowsnätverk.
  3. Klicka på Tillämpa för att spara inställningarna.

Att göra så att det går att upptäcka din Synology NAS via Bonjour-tjänsten:

Bonjour-tjänsten gör det möjligt för användare att hitta Synology NAS och utföra utskrifts- och säkerhetskopieringsuppgifter från Mac-datorn.

  1. Gå till Kontrollpanel > Filtjänster > Tjänstupptäckt > Bonjour.
  2. Välj Aktivera upptäckt av Bonjour-tjänsten.
  3. Du kan välj Bonjour-relaterade alternativ som passar dina behov:
    • Aktivera Bonjour Skrivarsändning: Välj att tillåta att Mac-datorer hittar USB-skrivaren som är ansluten till din Synology NAS, för utskrift av dokument.
    • Aktivera utsändning av Bonjour Time Machine via SMB: Välj att tillåta att Mac Time Machine hittar din Synology NAS och säkerhetskopierar filer till delade mappar i DSM via SMB.
    • Aktivera utsändning av Bonjour Time Machine via AFP: Välj att tillåta att Mac Time Machine hittar din Synology NAS och säkerhetskopierar filer till delade mappar i DSM via AFP.
  4. Om du har valt något alternativ för Time Machine, klicka på Ställ in mappar för Time Machine för att ange destinationsmappar för Time Machine-säkerhetskopieringar.
  5. Klicka på Tillämpa för att spara inställningarna.

Obs:

  • Om du avaktiverar tjänsten SMB/AFP (I Kontrollpanel > Filtjänster > SMB/AFP/NFS) blir tillhörande alternativ för Bonjour automatiskt avaktiverat.
  • När Aktivera utsändning av Bonjour Time Machine via SMB är valt blir följande alternativ aktiverade (i Kontrollpanel > Filtjänster > SMB/AFP/NFS > Avancerade inställningar):
    • Aktivera SMB 3 (för Maximalt antal protokoll för Windows-filservice)
    • Opportunistic Locking
    • Aktivera SMB2-lån
    • Aktivera SMB-varaktiga referenser (fillåsning mellan protokoll avaktiveras)
