Uppgiftshanterare*

På fliken Uppgiftshanterare kan du visa CPU och minnesanvändning för alla tjänster och processer.

Tjänster

Klicka på fliken Tjänster för att övervaka olika värden för alla tjänster, inklusive CPU-användning, CPU-tid, minnesanvändning och status för läsning av/skrivning till disk.

Processer

Klicka på fliken Processer för att visa CPU- och minnesanvändning för varje process i sjunkande ordning baserat på CPU-belastning.

Processtatus omfattar Körs, Sover och Stoppad. Motsvarande processtatus för Linux är körs, stoppad/spårning och sover.

Sover kan följas av en bokstav med detaljerad information, t.ex. 'D' för hårddisk i viloläge, 'Z' för zombie och 'X' för död.

____
* Endast tillgängligt för vissa modeller.
