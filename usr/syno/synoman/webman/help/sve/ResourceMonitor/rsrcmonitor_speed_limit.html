Hastighetsbegränsning

Under fliken Hastighetsbegränsning kan du se och hantera listan över filöverföringsprocesser som du har tillämpat hastighetsbegränsningen på.

Gör så här för att uppdatera listan:

Klicka på Uppdatera för att ladda om sidan och få de senaste posterna.

Gör så här för att stoppa en filöverföringsprocess:

Klicka på Avsluta anslutning för att stoppa filöverföringen. Du måste starta om filöverföringen för att framgångsrikt avsluta processen.
