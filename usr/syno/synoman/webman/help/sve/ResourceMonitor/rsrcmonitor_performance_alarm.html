Prestandalarm*

Att skapa en regel för prestandalarm:

  1. Gå till Inställningar > Inställningar och kontrollera att du har bockat för Aktivera prestandalarm.
  2. Gå till Inställningar > Prestandalarmregel.
  3. Klicka på knappen Skapa.
  4. Följande regelinställningar finns tillgängliga:
    • Resurs: Välj den resurs du vill tillämpa regeln för.
    • Typ: Välj den typ av regel du vill skapa.
    • Nivå: Välj larmnivån.
    • Större än: Ställ in den maximala procenten för din regeltyp där systemet ska meddela dig om den överskrids.
  5. Bocka för Aktiverad om du vill aktivera den här regeln omedelbart.
  6. Klicka på OK för att lägga till regeln.

Mekanism för prestandalarm:

Prestandalarm fungerar med följande mekanism för att minimera falsklarm och för att förhindra överdriven konsumtion av systemresurser inom en kort tidsperiod.

  1. Tjänsten prestandalarm kontrollerar alla regler var femte minut. Ingen åtgärd utlöses om alla de övervakade värdena befinner sig under det tröskelvärde som du har ställt in.
  2. När ett övervakat värde överskrider sitt tröskelvärde utlöser det inte ett larm omedelbart. I stället kontrollerar systemet regeln ytterligare fem gånger i intervaller om 30 sekunder.
  3. Efter att längden på intervallet ändrats och det övervakade värdet överskrider tröskelvärdet fem gånger i följd blir ett larm utlöst; skulle det övervakade värdet dock falla under tröskelvärdet vid någon av dessa fem kontroller ändras längden på intervallet tillbaka till fem minuter.

Av den anledningen blir ett larm för en enskild händelse utlöst senast 450 sekunder efter att händelsen först upptäcktes.

Larmnivåer:

Prestandalarm utlöser olika åtgärder beroende på larmnivåerna.

  • Varning: Skapar ett varningsmeddelande i systemloggen men utlöser inte något systemmeddelande.
  • Kritisk: Skapar inte bara ett meddelande i systemloggen utan utlöser även systemmeddelande.

Loggfiler för prestandalarm kan visas på fliken Prestandalarm eller DSM:s Loggcentral.

Att visa prestandalarm:

  • Gå till Prestandalarm för att visa alla utlösta prestandalarm.
  • Klicka på Uppdatera för att få de mest uppdaterade larmen.

Att exportera prestandalarm:

  1. Gå till Prestandalarm.
  2. Klicka på pilen bredvid knappen Exportera.
  3. Du kan välja att exportera dina larm som antingen HTML eller CSV.

Obs:

  • Stöd för Prestandalarm skiljer sig åt mellan modeller av Synology NAS. För mer information, vänligen se www.synology.com.
