Loggfiler

På sidan Loggcentral > Loggfiler kan du titta på, söka efter och exportera loggfiler.

För att söka loggfiler:

  1. I rullgardinsmenyn för sökning kan du specificera följande situationsbundna kriteria:
    • Nyckelord: Ange nyckelord för att söka efter loggmeddelanden som innehåller överensstämmande nyckelord.

      Obs:

      Du kan använda följande jokertecken och operatorer för avancerad sökning:

      • [*]: Jokertecknet motsvarar inga eller flera tecken som inte är mellanslag. [a*n] kan exempelvis representera valfritt ord som börjar med [a] och slutar med [n], såsom [action] [attention] eller [an].
      • [?]: Jokertecknet motsvarar exakt ett tecken som inte är mellanslag. [ca?] kan exempelvis representera valfritt ord som börjar med [ca] och slutar med ett enda tecken, såsom [car], [cat] eller [cap].
      • [AND]: Operatorn visar resultat som tillsammans innehåller sökertermerna. Till exempel: [admin AND root] visar resultat i loggfilen som innehåller både [admin] och [root].
      • [OR]: Operatorn visar resultat som innehåller en av sökertermerna. Till exempel: [admin OR root] visar resultat i loggfilen som innehåller antingen [admin] eller [root].
      • [NOT]: Operatorn utesluter ord från din sökning. Till exempel: [NOT admin] visar resultat som inte innehåller [admin].
      • [()]: Parenteser innesluter sökord för att specificera ordningen i vilken de tolkas om både AND och OR används. Till exempel: [(admin OR root) AND NOT logged] visar resultat som innehåller [admin] eller [root], och sedan utesluter [logged].
    • Datumintervall: Sök efter loggfiler efter det specificerade datumintervallet.
    • Nivå: Sök efter loggfiler efter den specificerade nivån.
    • Värdnamn: Sök efter loggfiler efter det specificerade värdnamnet.
    • Program: Sök efter loggfiler efter det specificerade programnamnet.
    • Kategori: Sök efter loggfiler efter den specificerade programkategorin.
  2. Klicka på Sök för att söka efter loggfiler som överensstämmer med specificerat kriteria.

För att exportera loggfiler:

  1. Klicka på pilen bredvid knappen Exportera.
  2. Välj HTML eller CSV för att ladda ned loggfilerna i ditt önskade format.

Obs:

  • Teckentabell för exporterade loggfiler i CSV är UTF-8. Du kommer att behöva öppna filen med ett program som har stöd för teckentabell UTF-8.
