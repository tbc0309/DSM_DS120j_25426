高级

您可以在控制面板 > 安全性 > 高级中为 DSM 浏览会话修改以下高级设置：

HTTP 压缩

您可为 Synology NAS 启用 HTTP 压缩。通过降低网络流量的带宽，进而加快网页加载速度。

若要启用/禁用 HTTP 压缩：

  • 勾选 / 取消勾选 HTTP 压缩 (在控制面板 > 安全性 > 高级)。

TLS / SSL 配置文件等级

在 TLS/SSL 配置文件中，您可选择 HTTPS 加密连接的安全级别。提供以下三种等级：

  • 现代兼容性 - 仅与最新浏览器和移动设备兼容。
  • 中等兼容性 - 介于现代兼容性和回溯兼容性之间。此选项为系统默认值。
  • 回溯兼容性 - 此选项不安全，但支持大多数浏览器和移动设备。

若要配置所有服务的 TLS/SSL 配置文件等级：

您可在主页上选择一种 TLS/SSL 配置文件等级，该等级将应用到使用默认配置文件的所有服务。

  1. 选择适当的 TLS/SSL 配置文件等级。
  2. 单击确定。

若要配置特定服务的 TLS/SSL 配置文件等级：

您可根据服务的安全要求更改其配置文件等级。

  1. 单击自定义设置可显示所有服务及其对应的配置文件等级。
  2. 单击目标服务的当前配置文件等级。
  3. 从下拉菜单中选择适当的配置文件等级。
  4. 单击确定。

注：

  • 系统默认配置文件等级会应用于不在服务列表中的连接。
