短信

您可以启用短信通知，以便在出现连接丢失、风扇运转改变、电源故障、存储卷空间不足或备份和还原失败等系统状态更改或系统错误时，让 Synology NAS 通过短信发送通知信息。

注：

  • 风扇运转改变的事件在无风扇型号（如 EDS14）上不可用。要了解更多信息，请访问 Synology 网站。

若要启用短信通知：

请勾选启用短信通知并完成下列设置：

  1. 从短信服务供应商下拉列表选择短信供应商。默认服务供应商为“clickatell”、“clickatell-2017”和“SendinBlue-v3”。
  2. 单击新增短信服务提供商来创建新的短信供应商配置，或单击删除来删除所选的短信供应商配置。
  3. 如果短信服务供应商使用用户帐号和密码进行验证，请为参数选择对应类别。在用户帐号字段中，输入您的短信服务帐户的用户名。在密码和确认密码栏中，输入并确认短信服务帐户的密码。
  4. 如果短信服务供应商使用 API 密钥进行验证，请为对于参数选择类别 API 密钥。在 API 密钥栏中，输入短信服务供应商提供的 API 密钥。
  5. 如果短信服务供应商提供了用于修改发件人名称的参数，请为对应参数选择类别发件人。在发件人栏中，输入发件人名称，该名称必须遵循短信服务供应商的规则。
  6. 输入您的主要电话号码或次要电话号码。如果您已输入两组号码，则短信通知信息将发送到两组号码。
  7. 如果您不想收到太多信息，请勾选设置短信间隔限制，并限制在您指定的间隔时间内只能收到一个通知信息。
  8. 单击寄送测试短信将测试短信信息发送到您所指定的电话号码。若您未收到信息，表示配置不正确。
  9. 单击应用来保存配置，或单击重置来取消该设置。

注：

  • 短信通知的语言在通知语言中设置。若要更改短信通知语言，请进入主菜单 > 控制面板 > 区域选项，单击语言选项卡，从
    通知语言区域的下拉菜单中选择偏好的语言。
  • 所有短信服务供应商都会采用英文发送通知信息，除了默认短信服务供应商：“clickatell”和“clickatell-2017”。

限制：

  • 用户帐号：有大小写之分，可设置 1 到32 个 Unicode 字符。
  • 密码：有大小写之分，最多可设置 128 个 Unicode 字符，包含字母、数字、符号，和空格。
  • API 密钥：区分大小写，必须介于 1 到 256 个 Unicode 字符之间。
  • 发件人：区分大小写，必须介于 1 到 64 个 Unicode 字符之间。
  • 短信间隔限制必须介于 1 至 10000 分钟之间。

若要创建短信供应商配置：

  1. 请单击新增短信服务提供商。
  2. 在短信服务提供商名称栏输入短信服务供应商的名称。
  3. 在测试网址栏中输入测试网址。您可以从短信供应商处取得测试网址。
  4. 在 HTTP 方法中选择 GET 或 POST。您可以从短信服务供应商的 API 文档中获取此信息。
  5. 单击下一步。
  6. 在编辑 HTTP 请求标题中，必须添加在短信服务供应商的 API 文档中列出的参数。
  7. 单击下一步。
  8. 在编辑 HTTP 请求正文中，如果在 HTTP 方法中选择了 POST，则必须添加在短信服务供应商的 API 文档中列出的参数。
  9. 单击下一步。
  10. 系统会将测试网址、HTTP 请求标题和 HTTP 请求正文拆分成几个参数。选择相应类别来定义每一个参数。除了其他参数以外，每个参数类别只能使用一次。

限制：

  1. 短信服务提供商名称：区分大小写，须介于 1 到32 个 Unicode 字符间。
  2. 测试网址：区分大小写，必须介于 1 到 500 个 Unicode 字符之间。此测试网址仅供设置之用。设置完成后不会发送任何短信通知。网址的格式由短信服务供应商提供。以 Clickatell 为例，可能的网址为：https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world
  3. 参数：区分大小写，必须介于 1 到 32 个 Unicode 字符之间。
  4. 值：区分大小写，必须介于 1 到 256 个 Unicode 字符之间。
  5. 您最多可以添加 10 个参数。
  6. 请注意“短信内容”参数中，“Hello”与“world”之间的空格会被服务供应商定义的特定字符替换。
    以 Clickatell 为例，“短信内容”参数的值会是“Hello+world”，字符“+”在 Clickatell 中用作空格字符。
