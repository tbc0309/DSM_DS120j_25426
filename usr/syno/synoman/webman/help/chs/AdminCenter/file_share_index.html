文件索引

对共享文件夹或其子文件夹进行建索可帮助您在文件搜索过程中快速查找所含的文件和内容。

注：

装载到 Synology NAS 的远程文件夹、云文件夹或 .iso 文件夹无法建索。

若要对文件夹和文件内容建立索引：

对文件夹建索后，File Station 将按名称对它或其子文件夹/文件进行建索，并让您按文件名搜索所需的项目。如果您按文件类型进一步建索，File Station 将不仅按名称，并且也会按照内容对相应的文件进行建索；在此方式下，您可按文件名和文件内容搜索所需的项目。

  1. 请进入控制面板 > 索引服务 > 文件索引，然后单击索引文件夹列表。
  2. 在弹出窗口中，单击创建 > 选择以显示要建立索引的文件夹。
  3. 选择所需的文件夹，然后单击选择。若要为索引创建新文件夹，单击创建文件夹并继续。
  4. 单击要索引的文件夹，并单击选择。
  5. 若要对特定文件类型进行建索，请进入文件类型并选择图像、音频、视频或文档。如果某些文件类型（如音频）未被选取，您可以在类型下只按文件名搜索文件（如音频文件）。
  6. 单击确定以完成建索。

注：

  • 您只能为建索选择最多 1000 个文件夹；所选文件夹中的子文件夹不包括在此数中。
  • 如果您在 File Station 选择启用文件内容搜索，可搜索索引文件夹中的文件内容。
  • 存放索引文件夹的卷必须至少有 10 MB 的可用空间。否则，索引将暂停。若要恢复建索，请扩充卷然后单击重建索引。
  • 如果您同时对文件夹及其子文件夹建立索引，系统只会索引子文件夹以节省系统资源。
  • 文件夹全面建索并通过 SMB 或 AFP 协议装载到 Mac 计算机后，您可用 Finder 来搜索文件内容。

若要删除索引文件夹：

  1. 请进入控制面板 > 索引服务 > 文件索引 > 索引文件夹列表，然后选择所需的索引文件夹。
  2. 单击删除可停止对文件夹的索引。

为文件重建索引

在一般情况下，您不需要为文件重建索引。然而，如果文件夹中的文件符合特定搜索条件，却没有出现在文件的搜索结果里，您也许需要重建索引来重新创建文件的索引结构。

若要为共享文件夹中的文件重建索引：

  1. 请进入控制面板 > 索引服务 > 文件索引 > 索引文件夹列表。
  2. 选择列示的文件夹以重建索引。
  3. 单击重建索引。

注：

File Station 不支持远程文件夹或外接存储设备 (如 U 盘或 SD 卡）的文件索引。

通过桌面管理文件检索

当文件检索开始后，您可在任务栏上找到索引进度 (浮动式放大镜图标)。由于文件索引需要更多的 CPU 资源，此功能可让您节省 CPU 资源。

  • 若要暂停/恢复文件索引，单击播放按钮。
  • 若要延迟文件检索至适当时间，请从下拉菜单中选择所需的选项。
  • 若要查看检索的文件夹，请单击查看索引文件夹列表。

注：

  • 如果您暂停索引操作，它只能单击桌面任务栏上的播放按钮才能恢复。
