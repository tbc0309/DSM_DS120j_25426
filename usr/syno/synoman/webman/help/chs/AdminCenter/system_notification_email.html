电子邮件

通过启用电子邮件通知，可在发生系统错误、网络连接丢失、风扇运转改变、电源故障或备份任务失败等指定事件时，让 Synology NAS 向您的电子邮件地址发送消息。

注：

  • 风扇运转改变的事件在无风扇型号（如 EDS14）上不可用。要了解更多信息，请访问 Synology 网站。

若要启用电子邮件通知：

  1. 请进入控制面板 > 通知 > 电子邮件。
  2. 勾选启动电子邮件通知。
  3. 在收信人电子邮件地址字段内输入一个或多个电子邮件地址。系统通知将发送给此处输入的电子邮件地址。允许输入两个地址。您可使用分号 (;) 将多个电子邮件地址分开。
  4. 必要时，可在主题前缀字段中添加主题前缀。此前缀将添加到 Synology NAS 发送的每条消息的主题中，帮助您识别并过滤消息。
  5. 如果您要使用第三方电子邮件服务器（如 Gmail 或 Outlook）来发送消息，请参见以下“第三方电子邮件服务”章节。如果您要使用自定义服务器，请参见“自定义 SMTP 服务器”章节。

若要使用第三方电子邮件服务：

  1. 请进入服务提供商菜单，选择一个第三方电子邮件服务，如 Gmail 或 Outlook。
  2. 为所选电子邮件服务输入用户名和密码。
  3. 单击应用来保存设置。
  4. 单击发送测试邮件来查看设置是否正确。

若要使用自定义 SMTP 服务器：

  1. 在服务供应器菜单，选择自定义 SMTP 服务器。
  2. 在 SMTP 服务器栏内输入 SMTP 服务器名称或 IP 地址。
  3. 在 SMTP 端口栏中输入端口号 SMTP 端口好。（如果您在设置端口转发规则时已指定另一个 SMTP 端口，请输入此号码。）
  4. 如果您的 SMTP 服务器需要验证，请勾选需要验证，并输入您的电子邮件服务器的用户名及密码。
  5. 如果您的 SMTP 服务器需要 SSL/TLS 连接，请勾选需要安全连接 (SSL/TLS)。
  6. 如果您要自定义 Synology NAS 发送的电子邮件信息中出现的发件人名称，请在发件人名称字段内输入该名称。如果您要自定义发件人的电子邮件地址，请在发件人电子邮件字段内输入电子邮件地址。如果该字段为空，将使用收件人电子邮件地址字段的第一个电子邮件地址。
  7. 单击应用来保存设置。
  8. 单击发送测试邮件来查看设置是否正确。

注：

  • 如果您要向每个新创建的用户帐户发送欢迎信息，请勾选向新用户发送欢迎信息，每位用户在创建帐户时都会收到一条欢迎信息。
  • 电子邮件通知的语言可在通知语言中设置。若要更改电子邮件通知语言，请进入主菜单 > 控制面板 > 区域选项，单击语言选项卡，从通知语言区域的下拉菜单中选择偏好的语言。
  • 如果您正在使用 Gmail 的 SMTP 服务器并已启用两步骤验证，请使用 Google 为 SMTP 验证提供的“应用专用密码”。

SMTP 服务器域名：

SMTP 服务器可以作为 IP 地址或域名。如果作为域名，请确认您已在主菜单 > 控制面板 > 网络中输入有效的 DNS 服务器 IP，并确认该服务器连接到 Internet。

注：

由于 Google Gmail API 策略发生更改，用户必须从服务提供商下拉菜单中选择自定义 SMTP 服务器设置，以便针对以下三个目的发送电子邮件。有关如何进行设置的更多信息，请参阅本教程。

  • 使用 ssmtp 二进制文件发送电子邮件，例如在任务计划程序中运行用户定义的脚本。
  • 使用 Web Station 构建网站并使用 PHP 发送电子邮件。这还包括从第三方网站套件（如 Wordpress、MantisBT 等）发送电子邮件。
  • 当有人在 Photo Station 中对照片或文章留下评论时发送邮件通知。
