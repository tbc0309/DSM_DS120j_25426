高级

在高级选项卡中，您可启用文件快速克隆功能，以在克隆时节省存储空间。此外，您可配置设置以让用户通过 Bonjour 和 Bonjour（简单服务发现协议）在网络上找到此 Synology NAS。

若要在Synology NAS上执行文件快速克隆：

通过文件快速克隆功能，物理数据区块将仅在克隆文件修改时复制，此功能将有助于节省存储空间。

  1. 在文件快速克隆区域下，勾选启用文件快速克隆复选框。
  2. 单击应用以保存设置。

若要通过 Bonjour 服务将 Synology NAS 变成可发现：

Bonjour 服务可让用户找到 Synology NAS，并执行来自 Mac 计算机的打印和备份任务。

  1. 在 Bonjour 区域下，勾选 启用 Bonjour 服务发现。
  2. 您可选择 Bonjour 相关选项以满足您的需要：
    • 启动 Bonjour 打印机共享：选择此选项可让 Mac 计算机找到连接 Synology NAS 的 USB 打印机以进行文档打印。
    • 启用通过 SMB 进行 Bonjour Time Machine 播送：选择此选项可让 Mac Time Machine 找到 Synology NAS 并通过 SMB 将文件备份到 DSM 共享文件夹。
    • 启用通过 AFP 进行 Bonjour Time Machine 播送：选择此选项可让 Mac Time Machine 找到 Synology NAS 并通过 AFP 将文件备份到 DSM 共享文件夹。
  3. 如果您已选择了任何与 Time Machine 相关的选项，请单击设置 Time Machine 文件夹为 Time Machine 备份指定目的地文件夹。
  4. 单击应用以保存设置。

注：

  • 如果您禁用 SMB/AFP 服务（在控制面板 > 文件服务 > SMB/AFP/NFS），则相关联的 Bonjour 选项将被自动禁用。
  • 当选择启用通过 SMB 进行 Bonjour Time Machine 播送后，以下选项将被启用（在控制面板 > 文件服务 > SMB/AFP/NFS > 高级设置）：
    • 启用 SMB 3（为 Windows 文件服务的最大协议数）
    • Opportunistic Locking
    • 启用 SMB2 租用
    • 启用 SMB durable handles（跨协议文件锁定将被禁用）

若要通过 SSDP 将 Synology NAS 变成可发现：

SSDP 服务可让用户通过 Windows 网络发现找到 Synology NAS。

  1. 在 SSDP 区域下，勾选启用 Windows 网络发现以允许通过网络浏览器访问复选框。
  2. 单击应用以保存设置。

若要通过 WS-Discovery 将 Synology NAS 变成可发现：

WS-Discovery 服务可让用户通过 Windows 网络发现找到 Synology NAS 上的 SMB 服务。

  1. 在 WS-Discovery 区域下，勾选启用 Windows 网络发现以允许通过 SMB 访问文件复选框。
  2. 单击应用以保存设置。
