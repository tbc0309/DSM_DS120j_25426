SNMP

SNMP 服务（在控制面板 > 终端机和 SNMP > SNMP中）可让用户通过网络管理软件来监控 Synology NAS。目前支持的 SNMP 协议为 SNMPv1、SNMPv2c 以及 SNMPv3。

启用 SNMP 服务

勾选启用 SNMP 服务并按以下步骤操作。

若要启用 SNMPv1、SNMPv2c 服务：

  1. 请勾选 SNMPv1、SNMPv2c 服务复选框。
  2. 输入社区名称。
  3. 单击应用。

若要启用 SNMPv3 服务：

  1. 请勾选 SNMPv3 服务复选框。
  2. 输入用户名和密码。选择您要使用的协议。
  3. 单击应用。

若要启用 SNMP 隐私：

  1. 勾选启用 SNMP 隐私。
  2. 选择您要使用的协议。然后输入密码。
  3. 单击应用。

此外，输入名称、位置和联系人将有助您识别 Synology NAS。这些字段为可选字段。

注：

  • SNMP 服务支持 IPv4 和 IPv6。
  • SNMP 服务同时支持 SNMPv1、SNMPv2c 和 SNMPv3 协议。

SNMPv1、SNMPv2c 社区限制：

社区名称必须为 1 至 64 个可显示的字符。不允许使用以下字符：
" ' \ 和空格。

SNMPv3 用户名限制：

用户名必须为 1 至 64 个可显示的字符。不允许使用以下字符：
" ' \ 和空格。

SNMPv3 密码限制：

密码区分大小写，长度为 8 至 127 个可显示的字符，包含字母、数字和符号。不允许使用以下字符：
" ' \ 和空格。

SNMP 隐私密码限制：

密码区分大小写，长度为 8 至 127 个可显示的字符，包含字母、数字和符号。不允许使用以下字符：
" ' \ 和空格。
