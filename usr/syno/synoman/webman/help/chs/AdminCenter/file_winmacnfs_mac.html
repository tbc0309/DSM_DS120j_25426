AFP

在控制面板 > 文件服务 > SMB/AFP/NFS 中，您可以启用 AFP 服务，将 Synology NAS 变成一台 Mac 文件服务器 (AFP 服务器)。

若要启用 Mac 文件服务：

  1. 请勾选启用 AFP 服务，将 Synology NAS 变成一台 Mac 文件服务器（AFP 服务器）。
  2. 单击应用。

启用传输日志

勾选这个选项，系统将会记录通过 AFP（所谓的苹果文件协议）访问文件的记录，包括对文件及文件夹的创建、删除、读取、写入、移动及重命名。

高级设置

应用默认的 UNIX 权限：

启动此选项在上传或创建文件和文件夹时应用默认的 UNIX 权限。所应用的权限将与 UNIX umask 命令所应用的权限相同。启用此选项后，文件的 UNIX 权限为 644，而文件夹为 755。禁用此选项后，文件的 UNIX 权限为 666，而文件夹为 777。默认 umask 值为 022。

注：

启动此选项可能会导致不同协议之间出现权限不一致的问题。为避免不一致问题，建议您保持禁用此选项。

在断开连接后立即释放资源：

立即启用此选项可在 AFP 服务器连接意外断开连接时释放资源。如果您在使用 Time Machine 时收到“卷在使用”的错误信息，请启用此选项。

若要通过 Finder 搜索文件内容：

文件夹全面建索并装载到 Mac 计算机后，您可用 Finder 来搜索所含文件的内容。

  1. 请进入 Universal Search > 偏好设置 > 文件索引 > 索引文件夹列表 > 创建来按文件类型对特定文件夹及其所含文件进行索引。
  2. 单击确定以执行索引。
  3. 将索引文件夹装载到您的 Mac 计算机。
  4. 在您的 Mac 计算机上，单击 Finder 左侧面板上需装载的文件夹，使用 Finder 可搜索所含文件的内容。

注：

  • Network video recorder (NVR) 型号不支持通过 Mac Finder 搜索文件内容。
  • 使用 Finder 可在装载的文件夹内进行搜索。Mac 桌面上的 Spotlight 仅可用来搜索本地文件。
  • 在 Mac OS X 10.9 及以上版本，Finder 支持按标签名称和类别在装载的文件夹中进行搜索。
  • 通过 AFP 访问共享文件夹时，若共享文件夹数量超过 255，仅会显示 255 个共享文件夹（按字母顺序排列）。
