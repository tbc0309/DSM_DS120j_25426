Instantané

Pour prendre l'instantané d'un LUN :

L'instantané d'un LUN est la copie en lecture seule à un instant donné d'un LUN.

  1. Sélectionnez le LUN duquel vous souhaitez prendre des instantanés.
  2. Cliquez sur Instantané et choisissez Prendre un instantané.
  3. Saisissez la description de cet instantané.
  4. Choisissez l'un des types d'instantanés suivants selon vos besoins :
    • Cohérent avec l'application : Ce type d'instantané ne peut être pris que lorsque Synology Storage Console ou Synology Snapshot Manager est installé sur votre serveur Windows ou sur VMware vCenter Server selon votre environnement pour communiquer avec DSM. Lorsqu'un instantané cohérent avec l'application est déclenché sur DSM, les applications sont notifiées et transfèrent toutes les données depuis la mémoire vers le LUN sur lequel l'instantané est pris pour garantir la cohérence des données.
    • Cohérent avec le plantage : Si aucun Storage Console ou Snapshot Manager n'est installé sur l'un de vos clients, vous ne pouvez capturer que des instantanés cohérents avec le plantage des LUN. Lorsqu'ils sont déclenchés, les applications ne sont pas notifiées de transférer les données depuis la mémoire sur les LUN, par conséquent cela ne garantit aucune cohérence des données.
  5. Cliquez sur OK.

Remarque :

  • Il est conseillé d'installer Synology Storage Console ou Snapshot Manager afin d'éviter toute incohérence de données.

Terminologie :

  • Synology Storage Console : Synology Storage Console est un plug-in de gestion du stockage sur VMware vSphere Client ou serveur Windows qui simplifie la gestion et la surveillance de systèmes de stockage Synology multiples. Il vous permet également de prendre des instantanés cohérents avec l'application directement à partir de DSM.
    • Pour les environnements VMware, le plug-in est uniquement compatible avec VMware vCenter Server Appliance 6.5 U2 ou versions ultérieures.
    • Pour plus d'informations, téléchargez le plug-in dans le Centre de téléchargements et reportez-vous à la section Synology Storage Console for VMware et Synology Storage Console for Windows.
  • Synology Snapshot Manager : Synology Snapshot Manager est un plug-in logiciel qui vous permet de prendre des instantanés cohérents avec l'application directement à partir de DSM.
    • Pour les environnements VMware, ce plug-in est uniquement compatible avec VMware vCenter Server 6.0 ou versions antérieures.

Pour supprimer un instantané :

  1. Sélectionnez le LUN duquel vous avez pris des instantanés.
  2. Cliquez sur Instantané > Liste d'instantanés.
  3. Sélectionnez l'instantané que vous souhaitez supprimer et cliquez sur Supprimer.

Remarque :

  • Le traitement de la récupération d'espace prend du temps après la suppression des instantanés.

Pour restaurer un LUN :

Sélectionnez l'instantané d'un point dans le temps pour restaurer les données sur un LUN.

  1. Sélectionnez le LUN que vous souhaitez restaurer.
  2. Cliquez sur Instantané et choisissez Liste d'instantanés.
  3. Sélectionnez l'instantané que vous souhaitez restaurer et cliquez sur Restaurer.

Pour créer un clone de LUN à partir d'instantanés :

Effectuez une copie accessible en écriture d'un LUN ou de l'instantané d'un LUN.

  • Sélectionnez un instantané LUN dans la Liste d'instantanés et cliquez sur Cloner.

Remarque :

  • Les fonctions instantané LUN, restauration et clonage ne sont disponibles que sur les LUN dotés de commandes d'accélération de stockage.
  • Si vous avez pris un instantané ou créé un clone et que l'état est devenu Anormal/En panne, veuillez supprimer cet instantané/clone et en créer un autre.
  • Toutes les modifications apportées à un LUN après la création d'un instantané seront perdues si vous restaurez le LUN à partir de cet instantané.
  • Les fonctions de restauration et de clonage LUN ainsi que la prise d'instantanés et le clonage d'instantanés LUN ne sont pas disponibles lorsqu'un volume dispose de moins de 1 Go d'espace libre.

Politique de conservation

Cette option offre une solution idéale pour vous aider à spécifier le nombre maximal de versions d'instantanés pour enregistrer votre espace de volume. Vous pourrez néanmoins avoir besoin de conserver vos instantanés pendant une période plus longue. Il y a plusieurs choses à savoir sur la politique de conservation :

  • Synology utilise la politique de conservation GFS, ou Grandfather-Father-Son (grand-père-père-fils). Vous pouvez configurer le nombre maximal de versions d'instantanés à conserver pour les périodes suivantes, respectivement : horaire, quotidienne, hebdomadaire, mensuelle et annuelle.
  • Un instantané peut respecter plusieurs règles de conservation en même temps. Par exemple, un instantané peut respecter les règles de conservation quotidienne, hebdomadaire et mensuelle.
  • Les instantanés les plus anciens sont conservés si vous prenez plusieurs instantanés au cours d'une certaine période. Par exemple, si vous définissez une règle de conservation des instantanés de 10 semaines et que votre calendrier d'instantanés est défini pour prendre 1 instantané par jour, Snapshot Replication conservera l'instantané le plus ancien de chaque semaine pendant les 10 dernières semaines.
  • Si vous n'avez pas pris d'instantanés pendant une certaine période, le système conserve les instantanés les plus récents. Par exemple, si la règle de conservation quotidienne d'instantané est définie sur 15 et qu'aucun instantané n'a été pris au cours des 30 derniers jours, le système conserve les instantanés pris il y a 30 à 45 jours.
  • Toutes les versions des instantanés prises en une heure, depuis la prise de l'instantané le plus récent, sont conservées par défaut, pour vous aider à rechercher et restaurer de manière pratique parmi les versions récentes des instantanés. Vous pouvez également verrouiller un instantané afin d'empêcher sa suppression automatique selon vos règles de conservation.

Exemple 1 :
Si vous souhaitez conserver des instantanés quotidiens pour les 3 derniers mois, des instantanés hebdomadaires pour les 4 à 6 derniers mois et des instantanés mensuels pour les 7 à 9 derniers mois, vous devez saisir 0 pour les instantanés horaires, 90 pour les instantanés quotidiens, 24 pour les instantanés hebdomadaires, 9 pour les instantanés mensuels et 0 pour les instantanés annuels.

Exemple 2 :
Si vous souhaitez conserver des instantanés horaires pour les 2 derniers jours, des instantanés quotidiens pour la semaine passée, des instantanés hebdomadaires pour le mois dernier et des instantanés mensuels pour l'année passée, vous devez saisir 48 pour les instantanés horaires, 7 pour les instantanés quotidiens, 4 pour les instantanés hebdomadaires, 12 pour les instantanés mensuels et 0 pour les instantanés annuels.
