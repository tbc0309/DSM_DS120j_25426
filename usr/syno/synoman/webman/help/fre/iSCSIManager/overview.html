Vue d'ensemble

La page Vue d'ensemble dans Gestionnaire iSCSI est un tableau de bord de vue d'ensemble qui vous permet de surveiller rapidement les trois éléments les plus importants du service iSCSI.

  • Cible
  • LUN
  • Événements : Les événements d'informations non lues, d'avertissement et d'erreur s'affichent ici.
    • 99 événements non lus peuvent être affichés au maximum.
    • Les événements d'avertissement et d'erreur s'affichent en premier.
    • Pour afficher tous les événements, accédez à la page Journal.
