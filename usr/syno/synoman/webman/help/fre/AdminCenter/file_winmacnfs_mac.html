AFP

Dans Panneau de configuration > Services de fichiers > SMB/AFP/NFS, vous pouvez activer le service AFP pour que votre Synology NAS devienne un serveur de fichiers Mac (serveur AFP).

Pour activer le service de fichier Mac :

  1. Cochez la case Activer le service AFP pour que Synology NAS devienne un serveur de fichiers Mac (serveur AFP).
  2. Cliquez sur Appliquer.

Activation de l'accès au transfert

Cette option permet d'activer l'enregistrement des opérations d'accès aux fichiers, y compris créer, supprimer, lire, écrire, déplacer et renommer les fichiers ou des dossiers par le biais du protocole AFP, également appelé Apple Filing Protocol.

Paramètres avancés

Application des permissions UNIX par défaut :

L'activation de cette option applique les permissions UNIX par défaut lors du chargement ou de la création de fichiers et dossiers. Les autorisations appliquées sont les mêmes que les autorisations appliquées par la commande UNIX umask. Lorsque cette option est activée, les permissions UNIX sont de 644 pour les fichiers et de 755 pour les dossiers. Lorsque cette option est désactivée, les permissions UNIX sont de 666 pour les fichiers et de 777 pour les dossiers. La valeur umask par défaut est 022.

Remarque :

L'activation de cette option pourrait provoquer des problèmes d'incohérence de permission entre différents protocoles. Afin d'éviter les incohérences, nous suggérons de laisser cette option désactivée.

Libérer les ressources aussitôt après la déconnexion :

L'activation de cette option permet de libérer immédiatement les ressources lorsque la connexion au serveur AFP est brusquement interrompue. Si vous recevez un message d'erreur « Volume utilisé » lors de l'utilisation de Time Machine, désactivez cette option.

Pour effectuer des recherches sur le contenu de fichiers via Finder :

Une fois les dossiers complètement indexés et montés sur votre ordinateur Mac, vous pouvez utiliser Finder pour rechercher les fichiers contenus.

  1. Accédez à Universal Search > Préférences > Indexation de fichier > Liste des dossiers indexés > Créer pour indexer des dossiers spécifiques et des fichiers contenus par type de fichier.
  2. Cliquez sur OK pour procéder à l'indexation.
  3. Montez les dossiers indexés sur votre ordinateur Mac.
  4. Sur votre ordinateur Mac, cliquez sur le dossier monté souhaité dans le volet de gauche du Finder et utilisez Finder pour rechercher du contenu dans les fichiers qui s'y trouvent.

Remarque :

  • Des modèles de Network video recorder (NVR) ne prennent pas en charge la recherche du contenu des fichiers via Mac Finder.
  • Utilisez Finder pour effectuer des recherches dans des dossiers montés. Spotlight sur le bureau du Mac ne peut être utilisé que pour rechercher des fichiers locaux.
  • Sous Mac OS X 10.9 et les versions ultérieures, Finder prend en charge la recherche sur plusieurs dossiers montés par nom les catégorie de tag.
  • Lors de l'accès à des dossiers partagés via AFP, seuls 255 dossiers partagés s'affichent (par ordre alphabétique) si leur nombre excède 255.
