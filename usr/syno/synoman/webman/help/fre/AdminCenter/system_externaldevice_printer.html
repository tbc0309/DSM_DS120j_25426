Imprimantes

Synology NAS peut se transformer en serveur d'impression, permettant ainsi aux ordinateurs clients ou périphériques mobiles d'accéder aux imprimantes connectées à Synology NAS. En outre, la prise en charge de l'impression sans fil Apple vous permet d'imprimer à distance à l'aide de périphériques iOS.

Rendez-vous sur www.synology.com pour consulter la liste des imprimantes compatibles.

Configuration des imprimantes

Consultez les instructions ci-dessous pour configurer et connecter une imprimante USB ou réseau à votre Synology NAS.

Pour configurer une imprimante USB :

  1. Allumez et connectez l'imprimante à un port USB sur le Synology NAS.
  2. Accédez à Panneau de configuration > Périphériques externes > Imprimante.
  3. Sélectionnez l'imprimante que vous voulez configurer. Veuillez noter que Synology NAS ne reconnaît pas les imprimantes connectées via des concentrateurs USB.
  4. Cliquez sur Gestionnaire d'imprimante et sélectionnez Configurer l'imprimante.
  5. Dans la boîte de dialogue, sélectionnez l'un des modes suivants :
    • Multifonctions réseau : Permet aux clients sur le réseau local d'utiliser les fonctions imprimer, numériser et télécopie de votre imprimante multifonctions. Les fonctions de numérisation et de télécopie sont disponibles uniquement pour les ordinateurs Windows.
    • Imprimante réseau : Choisissez ce mode pour les imprimantes qui ne prennent en charge que l'impression ou si vous souhaitez autoriser les clients à utiliser la fonction d'impression uniquement.
  6. Configurez les paramètres d'impression sans fil Apple (facultatif).
  7. Cliquez sur Enregistrer.

Remarque :

  • Une imprimante multifonctions réseau n'est accessible que par un seul client à la fois.

Pour configurer une imprimante réseau :

  1. Allumez et connectez l'imprimante réseau au même réseau local que votre Synology NAS.
  2. Accédez à Panneau de configuration > Périphériques externes > Imprimante.
  3. Cliquez sur Ajouter une imprimante réseau et suivez les instructions de l'assistant de configuration.

Remarque :

  • Nom de l'imprimante : lors de la configuration, vous devez saisir un nom d'imprimante qui peut comporter jusqu'à 32 caractères, comprenant des lettres, des chiffres et des caractères de soulignement.
  • Protocole : En général, les imprimantes réseau HP utilisent Socket ; les imprimantes réseau Canon utilisent BJNP ; et d'autres utilisent le plus souvent LPR.
  • Nom de la file d'attente : l'entrée d'un nom de file d'attente est nécessaire pour les imprimantes réseau LPR et IPP, mais ne s'applique pas aux imprimantes Socket ou BJNP.
  • Pas de MFP réseau : les imprimantes réseau connectées à Synology NAS ne peuvent pas actuellement utiliser le mode Multifonctions réseau.
  • Pas d'imprimante USB : Les imprimantes USB partagées par un produit Synology ne peuvent pas être définies comme imprimantes réseau.

Pour configurer une imprimante PostScript USB avec un Mac :

Pour Mac OS 10.12.X :

  1. Suivez les étapes indiquées dans la section Pour configurer une imprimante USB.
  2. Sur votre Mac, accédez à Préférences système > Imprimante et scanners, puis cliquez sur + pour ajouter une nouvelle imprimante.
  3. Choisissez IP comme type d'imprimante.
    • Entrez l'adresse IP ou le nom d'hôte de l'imprimante.
    • Choisissez Line Printer Daemon - LPD comme Protocole.
    • Entrez « usbprinter » dans le champ File d'attente. Si vous ne trouvez pas le nom de votre file d'attente, accédez à Panneau de configuration > Périphériques externes > Imprimante dans votre DSM.
  4. Sélectionnez Imprimante Generic PostScript pour Utilisation.
  5. Cliquez sur Ajouter pour terminer.

Pour Mac OS 10.6.X :

  1. Suivez les étapes indiquées dans la section Pour configurer une imprimante USB.
  2. Sur votre Mac, accédez à Préférences système > Imprimante et fax, puis cliquez sur + pour ajouter une nouvelle imprimante.
  3. Choisissez IP.
    • Choisissez Line Printer Daemon - LPD comme Protocole.
    • Entrez l'adresse IP de l'imprimante.
    • Entrez « usbprinter » dans le champ File d'attente.
    • Sélectionnez Imprimante Generic PostScript pour l'option Imprimer avec.
    • Cliquez sur Ajouter pour terminer.

Remarque :

  • AppleTalk n'est plus pris en charge par Mac OS 10.6.X. Si vous devez utiliser une imprimante AppleTalk, lisez sa documentation pour savoir comment la connecter à votre ordinateur Mac ou à votre réseau local. S'il n'y a pas moyen de connecter l'imprimante, connectez-la à un ordinateur Mac avec Mac OS 10.5.x ou une version antérieure, et configurez-la pour être partagée.

Pour Mac OS 10.5.X :

  1. Suivez les étapes indiquées dans la section Pour configurer une imprimante USB.
  2. Sur votre Mac, accédez à Préférences système > Imprimante et fax, puis cliquez sur + pour ajouter une nouvelle imprimante.
  3. Choisissez une méthode.
    • Méthode 1 : AppleTalk
      • Sélectionnez Imprimante Generic PostScript pour l'option Imprimer avec.
      • Cliquez sur Ajouter pour terminer.
    • Méthode 2 : IP
      • Choisissez Line Printer Daemon - LPD comme Protocole.
      • Entrez l'adresse IP de l'imprimante.
      • Entrez « usbprinter » dans le champ File d'attente.
      • Sélectionnez Imprimante Generic PostScript pour l'option Imprimer avec.
      • Cliquez sur Ajouter pour terminer.

Gestion des imprimantes

L'onglet dédié aux imprimantes comprend les options suivantes qui permettent de gérer les imprimantes USB ou réseau connectées au Synology NAS.

  • Obtenir des informations : affiche des informations relatives à l'imprimante sélectionnée, telles que le nom du périphérique, le fabricant, l'état, etc.
  • Supprimer : supprime et déconnecte une imprimante du Synology NAS.
  • Gestionnaire d'imprimante : comprend les options suivantes :
    • Configurer l'imprimante : pour modifier les paramètres de l'imprimante. Pour configurer l'impression sans fil Apple, reportez-vous aux instructions ci-dessous.
    • Nettoyer l'espace de la file d'attente : annule toutes les tâches d'impression sur toutes les imprimantes.
    • Imprimer une page test : imprime une page de test pour s'assurer que les configurations sont correctes.
    • Libérer l'imprimante : déconnecte l'imprimante et le client (pour le mode Multifonctions réseau uniquement). Lorsqu'un client se connecte à une imprimante multifonctions pour recevoir des télécopies ou pour scanner des fichiers, la tâche peut prendre un certain temps. Si nécessaire, la connexion peut être arrêtée afin de permettre au client suivant dans la file d'attente d'accéder à l'imprimante.

Pour activer l'impression sans fil Apple :

  1. Accédez à Panneau de configuration > Périphériques externes > Imprimante.
  2. Sélectionnez une imprimante dans la liste des périphériques.
  3. Cliquez sur Gestionnaire d'imprimante > Configurer l'imprimante.
  4. Cochez la case Activer l'impression sans fil Apple.
  5. Sélectionnez la marque et le pilote de votre imprimante.
  6. Cliquez sur Enregistrer.

Une fois la configuration terminée, vous pouvez imprimer une page de test en cliquant sur Gestionnaire d'imprimante > Imprimer une page test.

Remarque :

  • L'activation de l'impression sans fil Apple active également la fonction Diffusion imprimante Bonjour pour DSM.
  • L'impression sans fil Apple est disponible pour iOS 4.2 ou les versions ultérieures.

Accès aux imprimantes

Voyez la section ci-dessous pour accéder aux imprimantes à l'aide d'un PC ou d'un Mac.

Pour accéder aux imprimantes à l'aide d'un PC Windows :

La meilleure façon d'accéder au service d'impression d'un périphérique Synology NAS est de configurer la connexion à l'imprimante à l'aide de Synology Assistant. Consultez son Aide pour des instructions détaillées.

Si Synology Assistant n'est pas installé sur votre ordinateur, veuillez consulter ci-dessous les autres manières d'accéder au service d'impression du périphérique Synology NAS.

Type 1. Partagé une imprimante réseau

  1. Sur votre PC, ouvrez Réseau dans l'Explorateur de fichiers et double-cliquez sur le Synology NAS.
  2. Double-cliquez sur usbprinter ou nom de l'imprimante réseau ; le système vous invite à installer le pilote approprié, puis cliquez sur Oui.
  3. Veuillez obtenir le pilote de l'imprimante sur le CD ou le site Web du fabricant. Sélectionnez le fabricant et le modèle de l'imprimante, ou cliquez sur Disque fourni pour sélectionner l'emplacement du pilote. Cliquez ensuite sur OK.
  4. Le système commence la configuration de l'imprimante réseau. Une fois la configuration terminée, vous pouvez commencer à utiliser l'imprimante.

Type 2. Port LPR (Recommandé)

  1. Sur votre PC, cliquez sur Démarrer et accédez à Périphériques et imprimantes.
  2. Cliquez sur Ajouter une imprimante.
  3. Choisissez l'option Ajouter une imprimante réseau, sans fil ou Bluetooth.
  4. Choisissez l'option Mon imprimante ne figure pas dans la liste.
  5. Sélectionnez Ajouter une imprimante à l'aide d'une adresse TCP/IP ou d'un nom d'hôte. Cliquez sur Suivant pour continuer.
  6. Définissez le type de périphérique sur Périphérique de port TCP/IP.
  7. Saisissez l'adresse IP du Synology NAS. Cliquez sur Suivant pour continuer.
  8. Sélectionnez Type de périphérique : Personnalisé > Paramètres > …. Cliquez sur Suivant pour continuer.
  9. Sélectionnez le protocole LPR, saisissez usbprinter ou nom d'imprimante réseau comme nom de file d'attente, puis cochez la case Décompte des octets LPR activé. Cliquez sur OK pour continuer.
  10. Veuillez obtenir le pilote de l'imprimante sur le CD ou le site Web du fabricant. Sélectionnez le fabricant et le modèle de l'imprimante, ou cliquez sur Disque fourni pour sélectionner l'emplacement du pilote. Cliquez ensuite sur OK.
  11. Le système commence la configuration de l'imprimante réseau. Une fois la configuration terminée, vous pouvez commencer à utiliser l'imprimante.

Pour accéder aux imprimantes à l'aide d'un Mac :

Les utilisateurs de Mac peuvent accéder au service d'impression du périphérique Synology NAS en activant la diffusion imprimante Bonjour de DSM et en ajoutant l'imprimante dans les Préférences système.

Vérifiez que votre Synology NAS utilise bien une imprimante Postscript avant de poursuivre.

Étape 1. Activer la diffusion imprimante Bonjour de DSM

  1. Connectez-vous à DSM en tant qu'admin ou utilisateur appartenant au groupe administrators.
  2. Accédez à Panneau de configuration > Services de fichiers > Win/Mac/NFS.
  3. Cochez la case Activer la diffusion imprimante Bonjour.

Étape 2. Accéder à l'imprimante en utilisant un Mac

  1. Sur votre Mac, accédez à Préférences système > Imprimer et numériser, puis cliquez sur +.
  2. Sélectionnez usbprinter[numéro] ou nom de l'imprimante réseau dans le menu déroulant. Votre Mac configurera l'imprimante automatiquement.

Remarque :

  • RC18015xs+ ne prend pas en charge les imprimantes USB.
