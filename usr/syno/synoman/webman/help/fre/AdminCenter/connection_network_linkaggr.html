Link Aggregation

Synology NAS prend en charge les réseaux locaux multiples, ce qui vous permet de combiner ces interfaces LAN en utilisant la technologie Link Aggregation. Link Aggregation augmente la bande passante de votre Synology NAS en regroupant plusieurs interfaces réseau et offre un basculement du trafic pour maintenir la connexion réseau en cas de panne de celle-ci.

Une fois que vos interfaces LAN sont combinées, vous pouvez voir une nouvelle interface nommée Bond dans Panneau de configuration > Réseau > Interface réseau, pouvant être configurée comme toutes les autres interfaces (LAN ou PPPoE, par exemple).

Remarque :

  • Link Aggregation n'est disponible que sur les modèles dotés d'au moins deux ports LAN.
  • Lorsque Link Aggregation est activé, le trafic réseau est automatiquement ajusté et équilibré pour chaque périphérique connecté.
  • Si les interfaces réseau sont connectées à plusieurs commutateurs, assurez-vous que les ports des commutateurs se trouvent sur le même LAN ou VLAN.

Pour combiner plusieurs réseaux locaux avec Link Aggregation :

  1. Accédez à Panneau de configuration > Réseau > Interface réseau. Cliquez sur Créer > Créer Bond.
  2. Sélectionnez le mode souhaité.

    Descriptions des modes :

    • Adaptive Load Balancing : ce mode permet d'optimiser le trafic réseau reçu et envoyé par votre Synology NAS, que le commutateur prenne en charge Link Aggregation ou non. Pour éviter des erreurs inattendues, n'activez pas Link Aggregation sur le commutateur, même lorsqu'il est pris en charge.
    • IEEE 802.3ad Dynamic Link Aggregation : ce mode permet d'optimiser le trafic réseau reçu et envoyé par votre Synology NAS et nécessite l'activation de (Dynamic) Link Aggregation (LACP, 802.1AX) IEEE 802.3ad sur les commutateurs. Si vous utilisez plusieurs commutateurs, ceux-ci doivent être empilables et configurés de manière appropriée.
    • Balance XOR : ce mode permet d'équilibrer le trafic réseau reçu et envoyé par votre Synology NAS et nécessite l'activation de Link Aggregation statique sur les commutateurs. Si vous utilisez plusieurs commutateurs, ceux-ci doivent être empilables et configurés de manière appropriée. Si Link Aggregation n'est pas pris en charge par le commutateur, il est encore possible d'équilibrer le trafic envoyé par votre Synology NAS. Grâce à ce mode, il est possible de détecter les liens perdus, mais pas la cause (par exemple, lorsque que des câbles Ethernet sont débranchés ou que le commutateur n'est pas correctement configuré).
    • Actif/Veille : ce mode offre une fonctionnalité Network Fault Tolerance à votre Synology NAS. Cela signifie que lorsque l'interface réseau active tombe en panne, l'un des autres ports LAN assure la prise en charge afin de maintenir la connexion réseau. Vous pouvez sélectionner ce mode sans avoir à configurer le commutateur ou avec un commutateur qui prend en charge IEEE 802.3ad Link Aggregation.

    Si vous avez activé Open vSwitch, vous pouvez sélectionner l'un des modes suivants :

    • Balance-SLB : ce mode équilibre le trafic réseau sans nécessiter la prise en charge de commutateurs réseau spécifiques et permet de connecter deux commutateurs.
    • Balance-TCP : ce mode permet de connecter les commutateurs configurés en tant que Dynamic Link Aggregation (LACP IEEE 802.3ad). Si vous utilisez plusieurs commutateurs, ceux-ci doivent être empilables et configurés de manière appropriée.
    • Mode actif/sauvegarde : grâce à ce mode, votre Synology NAS bénéficie d'une tolérance aux pannes réseau à l'aide de deux interfaces réseau. Lorsque l'interface réseau active est défaillante, l'autre interface réseau prend le relais afin de maintenir la connexion réseau. Vous pouvez choisir ce mode sans configurer le commutateur.
  3. Sélectionnez les interfaces pour créer Link Aggregation.
  4. Configurez les paramètres de l'IP. Vous pouvez également activer VLAN, si nécessaire. Cliquez sur Appliquer pour enregistrer les modifications.
  5. Lorsque vous avez terminé, la nouvelle interface Bond 1 s'affiche dans la liste Interface réseau.

Pour modifier les paramètres de Link Aggregation :

  1. Pour modifier le mode Link Aggregation ou afficher les informations des périphériques physiques, accédez à Panneau de configuration > Réseau > Interface réseau.
  2. Sélectionnez l'interface Bond 1, puis cliquez sur le bouton Modifier.
  3. Accédez aux onglets correspondants pour modifier les paramètres souhaités.

Pour supprimer Link Aggregation :

  1. Accédez à Panneau de configuration > Réseau > Interface réseau.
  2. Sélectionnez l'interface Bond 1, puis cliquez sur le bouton Supprimer.
  3. Lorsque vous avez terminé, deux interfaces LAN distinctes s'affichent dans la liste Interface réseau.
