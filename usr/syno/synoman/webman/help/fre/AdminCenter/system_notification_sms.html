SMS

Vous pouvez activer les notifications par SMS pour que votre Synology NAS vous envoie des messages de notification via SMS quand des changements d’état ou des erreurs se produisent dans le système, comme une perte de connexion, un changement de comportement du ventilateur, une panne d’électricité, un manque d’espace libre sur un volume de stockage ou un échec de sauvegarde et restauration, etc.

Remarque :

  • L'événement de changement de comportement du ventilateur n'est pas disponible sur un modèle sans ventilateur, tel que l'EDS14. Pour plus d'informations, accédez au site Web Synology.

Pour activer les notifications par SMS :

Cochez Activer les notifications par SMS et configurez les paramètres suivants :

  1. Sélectionnez un fournisseur SMS dans la liste déroulante Fournisseur de service SMS. Les fournisseurs de services par défaut « clickatell », « clickatell-2017 » et « SendinBlue-v3 ».
  2. Cliquez sur Ajouter un fournisseur de service SMS pour créer une nouvelle configuration de fournisseurs de services SMS ou cliquez sur Supprimer pour supprimer la configuration de fournisseur SMS sélectionnée.
  3. Si votre fournisseur de services SMS utilise Nom d’utilisateur et Mot de passe pour l'authentification, sélectionnez les catégories correspondantes pour les paramètres. Dans le champ Nom d'utilisateur, saisissez le nom d'utilisateur de votre compte SMS. Dans les champs Mot de passe et Confirmer le mot de passe, entrez et confirmez le mot de passe de votre compte SMS.
  4. Si votre fournisseur de services SMS utilise Clé API pour l'authentification, sélectionnez la catégorie Clé API pour le paramètre correspondant. Dans le champ Clé API, saisissez la clé API délivrée par votre fournisseur de services SMS.
  5. Si le fournisseur de services SMS vous fournit un paramètre pour modifier le nom d'utilisateur, sélectionnez la catégorie Expéditeur pour le paramètre correspondant. Dans le champ Expéditeur, saisissez le nom d'expéditeur qui doit suivre la règle du fournisseur de services SMS.
  6. Entrez votre Numéro de téléphone principal ou Numéro de téléphone secondaire. Si les deux numéros sont entrés, les messages SMS de notification seront envoyés aux deux numéros.
  7. Si vous ne voulez pas recevoir trop de messages, cochez la case Activer la limite d'intervalle SMS et limitez le nombre de messages de notification à un message par intervalle spécifié.
  8. Cliquez sur Envoyer un message SMS de test pour envoyer un message SMS de test au numéro de téléphone spécifié. Si vous ne recevez pas de message, la configuration est incorrecte.
  9. Cliquez sur Appliquer pour enregistrer la configuration ou sur Réinitialiser pour annuler le démarrage.

Remarque :

  • La langue de notification par SMS est définie dans Langue de notification. Pour changer votre langue de notification par SMS, accédez à Menu principal > Panneau de configuration > Options régionales, cliquez sur l'onglet Langue, et choisissez votre langue préférée dans le
    menu déroulant de la section Langue de notification.
  • Tous les fournisseurs de services SMS envoient des messages de notification en anglais, à l'exception des fournisseurs de services SMS par défaut: « clickatell » et « clickatell-2017 ».

Limites :

  • Nom d'utilisateur : doit respecter la casse et être d'une longueur comprise entre 1 et 32 caractères Unicode.
  • Mot de passe : doit respecter la casse et être d'une longueur allant de 0 à 128 caractères affichables, comme des lettres, nombres, symboles et espaces.
  • Clé API : doit respecter la casse et être d'une longueur comprise entre 1 et 256 caractères Unicode.
  • Expéditeur : doit respecter la casse et être d'une longueur comprise entre 1 et 64 caractères Unicode.
  • La limite d'intervalle SMS doit être comprise entre 1 et 10 000 minutes.

Pour créer une configuration de fournisseur SMS :

  1. Cliquez sur Ajouter un fournisseur de service SMS.
  2. Dans le champ Nom du fournisseur, saisissez le nom du fournisseur de service SMS dans le champ.
  3. Dans le champ URL SMS, saisissez une URL SMS. Vous pouvez obtenir cette URL SMS auprès de votre fournisseur de service SMS.
  4. Choisissez GET ou POST dans Méthode HTTP. Vous pouvez obtenir ces informations dans les documents de l'API de votre fournisseur de services SMS.
  5. Cliquez sur Suivant.
  6. Dans Modifier l'en-tête de la demande HTTP, vous devez ajouter des paramètres qui sont répertoriés dans les documents de l'API de votre fournisseur de services SMS.
  7. Cliquez sur Suivant.
  8. Dans Modifier le corps de la demande HTTP, si vous choisissez POST dans méthode HTTP, vous devez ajouter les paramètres qui sont répertoriés dans les documents de l'API de votre fournisseur de services SMS.
  9. Cliquez sur Suivant.
  10. Le système décomposera l'URL du SMS, l'en-tête de la demande HTTP et le corps de la demande HTTP en différents paramètres. Définissez chaque paramètre en sélectionnant la catégorie correspondante. A l'exception d'Autre, les catégories de paramètre ne peuvent être utilisées qu'une fois.

Limites :

  1. Nom du fournisseur : doit respecter la casse et être d'une longueur comprise entre 1 et 32 caractères Unicode.
  2. URL SMS : sensible à la casse et doit comporter entre 1 et 500 caractères Unicode.
    Cette URL de SMS est uniquement destinée à la configuration. Aucun SMS ne sera envoyé une fois la configuration terminée. Le format de cette URL est fourni par le fournisseur de services SMS.
    À titre d'exemple, si nous utilisons Clickatell, une URL possible est : https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world
  3. Paramètre : doit respecter la casse et être d'une longueur comprise entre 1 et 32 caractères Unicode.
  4. Valeur : doit respecter la casse et être d'une longueur comprise entre 1 et 256 caractères Unicode.
  5. Vous pouvez monter 10 paramètres au plus.
  6. Notez que dans le « contenu du message », l'espace entre « Hello » et « world » sera remplacé par un caractère particulier défini par le fournisseur de service.
    Si nous utilisons Clickatell à titre d'exemple, la valeur du paramètre« contenu du message » devient « Hello+world », le caractère « + » est utilisé comme espace chez Clickatell.
