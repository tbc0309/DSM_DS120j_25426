Indexation multimédia

Indexation multimédia peut automatiquement balayer les fichiers multimédias stockés sur votre Synology NAS. Ces fichiers seront compilés en une bibliothèque multimédia en vue d'une indexation, et permettent la navigation et la lecture par des périphériques DMA et des applications multimédias de Synology (par ex.,Media Server, Video Station, Audio Station, Photo Station, et iTunes Server).

Gestion des dossiers indexés

Pour indexer un dossier existant ou créer un dossier indexé :

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation multimédia.
  2. Cliquez sur Dossier indexé > Créer > Sélectionner, et effectuez l'une des actions suivantes :
    • Pour indexer un dossier existant : Sélectionnez le dossier désiré dans la liste et cliquez sur Sélectionner.
    • Pour créer un dossier indexé : Sélectionnez le dossier parent dans la liste. Cliquez sur Créer dossier pour spécifier le nom du dossier et cliquez sur OK. Puis, sélectionnez le dossier créé et cliquez sur Sélectionner.
  3. Sélectionnez les types de fichiers (par ex. Photo, Musiqueet Vidéo) à indexer. Seuls les fichiers des types indexés permettent l'accès à partir de périphériques DMA et d'applications multimédias Synology.
  4. Cliquez sur OK pour commencer l'indexation.

Remarque :

  • Les dossiers partagés par défaut (par ex. music) créés pour des applications multimédias Synology sont automatiquement indexés, alors que d'autres dossiers ne peuvent être indexés que manuellement.
  • Les dossiers partagés distants qui utilisent File Station ne peuvent pas être ajoutés en tant que dossiers indexés.
  • Les appareils de stockage externes doivent être réindexés à chaque redémarrage de votre Synology NAS.
  • Vous pouvez ajouter jusqu'à 100 dossiers indexés sur votre Synology NAS.

Pour modifier un dossier indexé :

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation multimédia > Dossier indexé.
  2. Dans la fenêtre contextuelle, sélectionnez le dossier souhaité.
  3. Cliquez sur Modifier, et modifiez les paramètres.
  4. Cliquez sur OK puis cliquez sur Enregistrer.

Pour supprimer un/des dossier(s) de la liste des dossiers indexés :

Les fichiers supprimés seront désindexés. Les fichiers multimédias contenus ne seront pas accessibles aux périphériques DMA et aux applications multimédias Synology.

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation multimédia > Dossier indexé.
  2. Dans la fenêtre contextuelle, sélectionnez le dossier indésirable. Pour sélectionner plusieurs dossiers indésirables, appuyez de manière prolongée sur Maj ou Ctrl pendant la sélection.
  3. Cliquez sur Supprimer.
  4. Cliquez sur Oui puis cliquez sur Enregistrer.

Remarque :

  • Les dossiers effacés ne sont pas supprimés de votre Synology NAS.

Pour réindexer des fichiers multimédias :

Si la structure de l'index est perdue, la réindexation des dossiers indexés permet de reconstruire la structure.

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation multimédia.
  2. Cliquez sur Ré-indexation pour indexer à nouveau les dossiers précédemment indexés.

Remarque :

  • Les fichiers multimédias ajoutés aux dossiers indexés via le protocole NFS ou SCP ne seront pas automatiquement analyseés. Pour indexer ces fichiers, cliquez sur Ré-indexation.

Conversion des fichiers multimédias indexés

Pour définir la qualité des miniatures :

Des miniatures sont automatiquement générées pour les photos/vidéo transférées pour une utilisation avec Serveur multimédia ou Photo Station. Vous pouvez configurer la qualité de la miniature ici.

  1. Accédez à Panneau de configuration > Services d'indexation > Indexation multimédia > Paramètres des vignettes.
  2. Choisissez une qualité à partir du menu déroulant.
  3. Cliquez sur Appliquer.

Remarque :

  • La création de miniatures de qualité élevée prendra plus de temps.
  • Pour transférer un grand nombre de photos/vidéos vers votre bibliothèque Photo Station, nous vous recommandons d'utiliser Photo Station Uploader (un utilitaire de bureau Synology disponible au Centre de téléchargement) pour une génération rapide des miniatures.

Pour activer la conversion vidéo pour les appareils mobiles :

Les vidéos du dossier partagé photo (dédié à Photo Station) peuvent être lues sur votre périphérique mobile si elles sont correctement converties.

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation multimédia > Réglages vidéo.
  2. Sélectionnez Activer la conversion vidéo pour les appareils mobiles.
  3. Cliquez sur Appliquer, et vous pourrez voir les vidéos sur vos appareils mobiles en déplacement.

Remarque :

  • Seules les vidéos chargées dans le dossier partagé photode Photo Station peuvent être converties au format pris en charge par les appareils mobiles.
  • Les fichiers MP4 ne seront pas convertis.

Pour afficher la progression de conversion photo/vidéo :

  1. Lorsque les photos ou les vidéos ont été téléchargées sur Synology NAS, cliquez sur l'icône Progression de la conversion qui apparaît dans le coin supérieur droit de la barre de tâche.
  2. Dans la fenêtre contextuelle, effectuez l'une des actions suivantes :
    • Dans la section, Progression de la conversion, surveillez la progression et le statut de la création de miniatures effectuée en arrière-plan.
    • Pour mettre la conversion en pause, suivez les étapes ci-dessous :
      1. Sélectionnez Retarder de [nombre] heure(s) dans la liste déroulante et cliquez sur Pause pour mettre en pause la progression.
      2. La progression de la conversion reprend après le délai spécifié. Vous pouvez également cliquer sur Continuer pour reprendre immédiatement le processus.
      Remarque : Étant donné que la progression de la conversion demande plus de ressources processeur, mettre en pause le processus peut économiser les ressources système pour d'autres tâches en cours.
  3. Lorsque la progression de conversion est terminée, l'icône de la barre de tâche disparaît.
