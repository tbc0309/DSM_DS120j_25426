Utilisateur

Grâce aux options flexibles de gestion de l'utilisateur, vous pouvez créer des comptes utilisateur pour les membres individuels de votre famille ou entreprise. La création des utilisateurs vous permet de définir et de gérer les permissions pour chaque personne, telles que les permissions d'accès au dossier partagé, les quotas de stockage/dossier partagé ou les limitations de bande passante.

    Remarque :
  1. La gestion des dossiers partagés n'est applicable qu'aux plateformes Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake, Denverton. Pour savoir si votre modèle prend en charge cette fonction, reportez-vous à cet article.
  2. La gestion des quotas de dossier de stockage/partagé n'est pas applicable à Docker DSM.
