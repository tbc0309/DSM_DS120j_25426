Avancé

Vous pouvez modifier les paramètres suivants dans Panneau de configuration > Sécurité > Avancé pour les sessions de navigation DSM :

Compression HTTP

Vous pouvez activer Compression HTTP pour Synology NAS. En réduisant la bande passante de trafic réseau, vous pouvez augmenter la vitesse de chargement de la page Web.

Pour activer/désactiver la compression HTTP：

  • Cochez/décochez Compression HTTP dans Panneau de configuration > Sécurité > Avancé.

Niveau de profil TLS / SSL

Dans Profil TLS/SSL, vous pouvez choisir le niveau de sécurité des connexions chiffrées HTTPS. Il offre les trois niveaux suivants :

  • Compatibilité moderne - Uniquement compatible avec les derniers navigateurs et périphériques mobiles.
  • Compatibilité intermédiaire - Entre la compatibilité moderne et l'ancienne rétrocompatibilité Il s'agit de la valeur par défaut du système.
  • Rétrocompatibilité ancienne - Cette option n'est pas sécurisée, mais prend en charge la plupart des navigateurs et des périphériques mobiles.

Pour configurer le niveau du profil TLS / SSL pour tous les services :

Vous pouvez sélectionner un niveau de profil TLS /SSL sur la page principale, ce qui s'applique à tous les services qui utilisent le Profil par défaut.

  1. Sélectionnez un niveau de profil TLS / SSL approprié.
  2. Cliquez sur OK.

Pour configurer le niveau du profil TLS/SSL d'un service spécifique :

Vous pouvez modifier le niveau du profil d'un service en fonction des exigences de sécurité du service.

  1. Cliquez sur Paramètres personnalisés pour afficher tous les services ainsi que leur niveau de profil correspondant.
  2. Cliquez sur le niveau du profil actuel du service ciblé.
  3. Sélectionnez le niveau approprié du profil dans le menu déroulant.
  4. Cliquez sur OK.

Remarque :

  • Le profil Défaut système s'applique à la connexion qui ne se trouve pas dans la liste des services.
