Gestion des clés des dossiers partagés chiffrés

Vous pouvez utiliser le gestionnaire de clés pour gérer les clés des dossiers partagés et déchiffrer plusieurs dossiers partagés chiffrés en même temps.

Explication terminologique :

  • Magasin de clé : Un magasin de clés est un périphérique externe, ou une partition système, pris en charge par Synology NAS et utilisé en tant que clé physique pour déchiffrer des dossiers partagés.
  • Chiffre : Un chiffre est une méthode de chiffrement des clés des dossiers partagés. Le gestionnaire de clés offre deux types de chiffres :
    • Phrase de passe : Les clés chiffrées par une phrase de passe peuvent être déchiffrées par toute personne la connaissant.
    • Clé machine : Les clés chiffrées par une clé machine ne peuvent être déchiffrées que par le Synology NAS lié.

Pour initialiser un magasin de clés :

  1. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  2. Sélectionnez un périphérique externe ou une partition système comme magasin de clés à partir de l'Emplacement du magasin de clés.
  3. Saisissez une phrase de passe dans le champ Phrase de passe pour ce magasin de clés.

Suggestions de sécurité :

  • Pour des raisons de sécurité de gestion, il est plus sûr de stocker le fichier chiffré et la clé correspondante sur des périphériques différents.
  • Nous vous recommandons de sélectionner un périphérique externe comme Emplacement de stockage des clés afin d'améliorer la sécurité de vos données.

Pour ajouter une nouvelle clé à un magasin de clés :

  1. Assurez-vous d'avoir initialisé un magasin de clés.
  2. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  3. Cliquez sur Ajouter.
  4. Sélectionnez un dossier partagé chiffré.
  5. Sélectionnez le chiffre de la clé de chiffrement. Vous pouvez sélectionner Phrase de passe ou Clé machine.
  6. Entrez ou importez la clé de chiffrement.
  7. Cliquez sur OK pour enregistrer les paramètres.

Pour déchiffrer manuellement plusieurs dossiers partagés chiffrés :

  1. Assurez-vous d'avoir ajouté les clés au magasin de clés.
  2. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  3. Sélectionnez les dossiers partagés chiffrés que vous souhaitez déchiffrer.
  4. Cliquez sur Monter.

Pour déchiffrer automatiquement plusieurs dossiers partagés chiffrés au démarrage :

  1. Assurez-vous d'avoir ajouté les clés au magasin de clés.
  2. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  3. Cochez Monter au démarrage pour les dossiers partagés chiffrés que vous souhaitez déchiffrer.
  4. Cliquez sur OK pour enregistrer les paramètres.

Pour éjecter un magasin de clés après le démarrage :

  1. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  2. Cliquez sur Configurer.
  3. Cochez Éjecter le périphérique après le démarrage.
  4. Cliquez sur OK pour enregistrer les paramètres.

Pour migrer automatiquement des clés de chiffrement existantes dans un magasin de clés pour des dossiers partagés montés automatiquement :

Migrer les clés pendant l'initialisation du magasin de clés :

  1. Lors de l'initialisation d'un magasin de clés, cochez Migrer automatiquement les clés de chiffrement dans ce magasin de clés pour tous les dossiers partagés montés existants (recommandés).

Migrer les clés manuellement :

  1. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  2. Cliquez sur Configurer.
  3. Cliquez sur Migrer maintenant.
  4. Dans le cas de clés en double, vous pouvez choisir Remplacer les clés dupliquées.
  5. Cliquez sur OK pour enregistrer les paramètres.

Pour cloner un magasin de clés en mode High Availability :

  1. Accédez à Panneau de configuration > Dossier partagé > Action > Gestionnaire de clés.
  2. Cliquez sur Cloner.
  3. Sélectionnez un magasin de clés source sur le serveur actif dans Magasin de clés source.
  4. Sélectionnez un magasin de clés de destination sur le serveur passif dans Magasin de clés de destination.
  5. Cliquez sur Appliquer pour enregistrer les paramètres.
