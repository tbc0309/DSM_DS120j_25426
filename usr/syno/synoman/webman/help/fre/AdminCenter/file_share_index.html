Indexation de fichier

L'indexation d'un dossier partagé ou de ses sous-dossiers peut vous aider à trouver rapidement des fichiers ou dossiers qui s'y trouvent pendant la recherche de fichiers.

Remarque :

Il est impossible d'indexer des dossiers distants, des dossiers d'un cloud ou des dossiers .iso montés sur votre Synology NAS.

Pour indexer des dossiers et le contenu de fichiers :

Lorsque vous indexez un dossier, File Station l'indexe par nonm, ainsi que ses sous-dossiers/fichiers, et vous permet de rechercher les éléments nécessaires par nom de fichier. Si vous poussez l'indexation par type de fichier, File Station indexera les fichiers correspondants non seulement par nom, mais également en fonction de leur contenu ; de cette manière, vous pouvez rechercher les éléments nécessaires par nom et par contenu de fichier.

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation de fichier, et cliquez sur Liste des dossiers indexés.
  2. Dans la fenêtre contextuelle, cliquez sur Créer > Sélectionner pour afficher des dossiers prêts à l'indexation.
  3. Sélectionnez le dossier désiré et cliquez sur Sélectionner. Pour créer un nouveau dossier pour l'indexation, cliquez sur Créer dossier et continuez.
  4. Cliquez sur le dossier que vous souhaitez indexer, puis sur Sélectionner.
  5. Pour indexer des types spécifiques de fichiers, accédez à Type de fichier et sélectionnez Image, Audio, Vidéo ou Document. Si un certain type de fichies (par ex. Audio) n'est pas sélectionné, vous ne pouvez rechercher des fichiers sous le type (par ex. fichiers audio) que par nom de fichier.
  6. Cliquez sur OK pour terminer l'indexation.

Remarque :

  • Vous ne pouvez sélectionner que 1000 dossiers au maximum pour l'indexation ; les sous-dossiers des dossiers sélectionnés ne sont pas inclus dans ce nombre.
  • Si vous sélectionnez Activer la recherche par contenu de fichier dans File Station, vous pouvez rechercher le contenu de fichiers dans des dossiers indexés.
  • Le volume qui héberge le dossier indexé doit disposer de plus de 10 Mo d'espace libre. Sinon, l'indexation est mise en pause. Pour reprendre l'indexation, développez le volume et cliquez sur Ré-indexation.
  • Si vous avez indexé un dossier et ses sous-dossiers en même temps, le système n'indexe que les sous-dossiers pour économiser les ressources du système.
  • Une fois les dossiers complètement indexés, puis montés sur un ordinateur Mac via le protocole SMB ou AFP, vous pouvez utiliser Finder pour rechercher le contenu des fichiers.

Pour supprimer des dossiers indexés :

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation de fichier > Liste des dossiers indexés et sélectionnez le ou les dossiers indexés souhaités.
  2. Cliquez sur Supprimer pour arrêter l'indexation des dossiers.

Ré-indexation des fichiers

La réindexation de fichiers n'est pas requise en temps normal. Cependant, si les fichiers d'un dossier correspondent à des critères spécifiques de recherche, mais qu'ils n'apparaissent finalement pas pendant la recherche de fichiers, il vous faudra peut-être effectuer une réindexation des fichiers pour reconstruire la structure d'index.

Pour réindexer des fichiers dans un dossier partagé :

  1. Accédez à Panneau de configuration > Service d'indexation > Indexation de fichier > Liste des dossiers indexés.
  2. Sélectionnez le ou les dossiers répertoriés à réindexer.
  3. Cliquez sur Ré-indexation.

Remarque :

File Station ne prend pas en charge l'indexation des fichiers sur des dossiers distants ou des périphériques de stockage externes (par ex. des lecteurs USB/cartes SD).

Gérer l'indexation des fichiers via le bureau

Lorsque l'indexation des fichiers commence, vous pouvez trouver le Progression de l'indexation (une icône de loupe flottante) dans la barre de tâches. Comme l'indexation des fichiers nécessite davantage de ressources du processeur, ceci vous permet de les économiser.

  • Pour mettre en pause/reprendre l'indexation des fichiers, cliquez sur le bouton de lecture.
  • Pour retarder l'indexation des fichiers jusqu'à l'heure appropriée, sélectionnez l'option souhaitée dans le menu déroulant.
  • Pour afficher les dossiers en cours d'indexation, cliquez sur Afficher la liste des dossiers indexés.

Remarque :

  • Si vous mettez en pause l'action d'indexation, elle ne peut être reprise qu'en cliquant sur le bouton de lecture de la barre de taches du bureau.
