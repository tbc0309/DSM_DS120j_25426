Avancé

Vous pouvez sélectionner les types d'événements qui déclencheront l'envoi de messages de notification par le système. De même, vous pouvez choisir le support (e-mail, SMS et périphériques mobiles) sur lequel vous recevrez ces messages.

Chacun des événements a son propre message de notification par défaut, mais vous avez également l'option de personnaliser votre propre contenu de notification en modifiant le message de notification. Le message par défaut peut être restauré si vous décidez d'annuler vos paramètres personnalisés plus tard.

Pour rechercher des notifications particulières :

  1. Saisissez un mot-clé dans le champ de recherche, dans le coin supérieur droit.
  2. Les événements relatifs aux mots-clés saisis s'affichent.

Pour sélectionner les moyens de notification des événements de notification :

  1. Cochez les cases pour chaque événement individuel pour sélectionner les moyens de notification préférés.
  2. Cliquez sur Enregistrer pour appliquer les paramètres.

Pour modifier le message de notification :

  1. Sélectionnez un événement dans la liste des événements, puis cliquez sur Modifier le message.
  2. Dans la fenêtre qui s'affiche, modifiez l'objet et le contenu du corps pour personnaliser le message de notification.
  3. Vous pouvez également modifier les variables suivantes. Les variables mises à jour s'appliquent à tous les messages de notification :
    • %COMPANY_NAME%: La signature qui s'affiche à la fin d'un message de notification. La valeur par défaut Synology DiskStation, Synology RackStation, Synology FlashStation, ou Synology Unified Controller, selon le modèle dont vous disposez.
    • %HTTP_URL% : Un hyperlien vers votre Synology NAS. S'il n'est pas spécifié, le système répertorie tous les liens possibles dans le message de notification.
  4. Cliquez sur Enregistrer pour appliquer les modifications.
  5. Pour restaurer le message de notification à sa valeur par défaut :
    1. Sélectionnez un événement dans la liste des événements, puis cliquez sur Modifier le message.
    2. Dans la fenêtre qui s'affiche, cliquez sur Restaurer les paramètres par défaut puis sur Oui dans la boîte de dialogue de confirmation.

Remarque :

  • Il est recommandé de réserver les segments de chaîne contenant « % » dans l'objet et le contenu du corps. Ils seront remplacés par les informations réelles du système quand une notification est envoyée.
