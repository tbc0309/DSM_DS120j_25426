Avancé

Dans l'onglet Avancé, vous pouvez activer la fonction Clonage rapide des fichiers pour économiser de l'espace de stockage lorsque vous effectuez des clonages de fichiers. En outre, vous pouvez configurer les paramètres pour autoriser les utilisateurs à trouver ce Synology NAS sur le réseau via les services Bonjour et SSDP (Simple Service Discovery Protocol).

Pour effectuer un clonage rapide de fichiers sur votre Synology NAS :

Grâce à la fonction de clonage rapide des fichiers, les blocs de données physiques sont copiés uniquement lors de la modification des fichiers clonés, ce qui vous aide à économiser votre espace de stockage.

  1. Dans la section Clonage rapide des fichiers, cochez la case Activer le clonage rapide des fichiers.
  2. Cliquez sur Appliquer pour enregistrer les paramètres.

Pour rendre votre Synology NAS visible via Bonjour :

Le service Bonjour permet aux utilisateurs de rechercher le Synology NAS et de procéder à des tâches d'impression et de sauvegarde à partir du Mac.

  1. Dans la section Bonjour, cochez Activer la découverte du service Bonjour.
  2. Vous pouvez sélectionner des options relatives à Bonjour en fonction de vos besoins :
    • Activer la diffusion imprimante Bonjour : Sélectionnez l'option autorisant les ordinateurs Mac à rechercher l'imprimante USB connectée à votre Synology NAS pour l'impression de documents.
    • Activer la diffusion Bonjour Time Machine via SMB : Sélectionnez l'option autorisant du Mac à rechercher votre Synology NAS et à sauvegarder les fichiers dans des dossiers partagés de DSM via SMB.
    • Activer la diffusion Bonjour Time Machine via AFP : Sélectionnez l'option autorisant du Mac à rechercher votre Synology NAS et à sauvegarder les fichiers dans des dossiers partagés de DSM via AFP.
  3. Si vous avez sélectionné une option relative à Time Machine, cliquez sur Configurer les dossiers de Time Machine pour spécifier les dossiers de destination des sauvegardes de Time Machine.
  4. Cliquez sur Appliquer pour enregistrer les paramètres.

Remarque :

  • Si vous désactivez le service SMB/AFP (dans Panneau de configuration > Services de fichiers > SMB/AFP/NFS), l'option Bonjour associée est automatiquement désactivé.
  • Lorsque Activer la diffusion Bonjour Time Machine via SMB est sélectionné, les options suivantes sont activées (dans Panneau de configuration> Services de fichiers > SMB/AFP/NFS > Paramètres avancés) :
    • Activer SMB 3 (pour Protocole max pour le service de fichiers Windows)
    • Opportunistic Locking
    • Activer le bail SMB2
    • Activer les handles durables de SMB (Le verrouillage inter-protocole des fichiers sera désactivé.)

Pour rendre votre Synology NAS visible via SSDP :

Le service SSDP permet aux utilisateurs de rechercher Synology NAS via la découverte de réseaux Windows.

  1. Dans la section SSDP, cochez la case Activer la découverte des réseaux Windows pour autoriser l'accès via des navigateurs Web.
  2. Cliquez sur Appliquer pour enregistrer les paramètres.

Pour rendre votre Synology NAS visible via WS-Discovery :

Le service WS-Discovery permet aux utilisateurs de rechercher un service SMB sur le Synology NAS via la découverte de réseaux Windows.

  1. Dans la section WS-Discovery, cochez la case Activer la découverte de réseaux Windows pour autoriser l'accès aux fichiers via SMB.
  2. Cliquez sur Appliquer pour enregistrer les paramètres.
