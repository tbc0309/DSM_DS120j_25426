Copier un utilisateur

Vous pouvez copier les paramètres d'un utilisateur existant vers un nouvel utilisateur. Cela vous permet de créer rapidement un nouvel utilisateur sans avoir à configurer à nouveau les mêmes paramètres.

  1. Sélectionnez l'utilisateur que vous souhaitez copier.
  2. Cliquez sur Créer > Copier un utilisateur. La fenêtre de l'Assistant de création d'utilisateur s'ouvre.
  3. Les paramètres de l'utilisateur sélectionné sont appliqués aux nouveaux utilisateurs par défaut (à l'exception des informations personnelles telles que le nom d'utilisateur et l'adresse e-mail).
