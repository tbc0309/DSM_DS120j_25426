Utilisateurs connectés

Sur cet onglet, vous pouvez voir et gérer les clients qui sont actuellement connectés au système et accèdent aux ressources. Cette section affiche les clients qui sont connectés via HTTP, TELNET, SSH, SMB/CIFS, AFP, FTP et NFS.

Pour gérer les connexions actuelles, effectuez l'une des opérations suivantes :

  • Cliquez sur Enregistrer pour exporter la liste actuelle dans un fichier HTML.
  • Cliquez sur Actualiser pour actualiser la liste.
  • Cliquez sur Tuer la connexion pour empêcher l'utilisateur d'accéder à la ressource. L'utilisateur déconnecté devra être autorisé à nouveau pour obtenir 'accès à la ressource.
  • Cliquez sur Désactiver l'utilisateur pour désactiver l'utilisateur. Pour activer l'utilisateur à nouveau, accédez à Menu principal > Panneau de configuration > Utilisateur.

Remarque :

  • L'option Tuer la connexion ne fonctionne pas avec les clients NFS.
  • Si les clients NFS ne démontent pas correctement les dossiers partagés, ils peuvent toujours apparaître dans la liste des utilisateurs connectés.
  • Si le Synology NAS redémarre, les clients NFS qui étaient connectés précédemment n'apparaîtront pas dans la liste des utilisateurs connectés.
