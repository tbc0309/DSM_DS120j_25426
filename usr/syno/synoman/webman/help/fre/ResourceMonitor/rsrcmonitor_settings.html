Paramètres

Cliquez sur Paramètres pour configurer les options suivantes :

  • Choisissez la Durée en temps réel en minutes dans le menu déroulant.
  • Cochez Activer l'historique d'utilisation si vous voulez afficher les données précédentes. Vous pouvez permuter entre Temps réel, 1 jour, 1 semaine, 1 mois et 1 an dans le menu déroulant du coin supérieur droit lorsque vous affichez les tableaux des performances.
  • Cochez Activer l'alarme de performance si vous souhaitez configurer des règles d'alarme de performance.

Cliquez sur Appliquer pour enregistrer les paramètres.

Règles de l'alarme des performances

Pour créer une règle d'alarme de performances :

  1. Accédez à Paramètres > Paramètres et assurez-vous d'avoir coché la case Activer l'alarme des performances.
  2. Accédez à Paramètres > Règles de l'alarme des performances.
  3. Cliquez sur le bouton Créer.
  4. Les paramètres de règle suivants sont disponibles :
    • Resources : Choisissez de ressources auxquelles vous souhaitez appliquer la règle.
    • Type : Choisissez le type règle que vous souhaitez créer.
    • Niveau : Choisissez le niveau d'alarme.
    • Plus grand que : Définissez le pourcentage maximum de votre type de règles lorsque le système vous avertit en cas de dépassement.
  5. Cochez Activer souhaitez activer immédiatement règle.
  6. Cliquez sur OK pour ajouter la règle.

Mécanisme pour les alarmes de performances :

Les alarmes de performance fonctionnent via le mécanisme suivant pour réduire les fausses alarmes et éviter une consommation excessive des ressources système sur une courte période.

  1. Le service des alarmes de performance vérifie toutes les règles toutes les cinq minutes. Aucune action n'est déclenchée si toutes les valeurs surveillées tombent sous les seuils que vous avez définis.
  2. Lorsqu'une valeur surveillée dépasse son seuil, l'alarme n'est pas déclenchée immédiatement. Au lieu de cela, le système vérifie la règle cinq fois de plus à des intervalles de 30 secondes.
  3. Après la modification de la longueur de l'intervalle, si la valeur surveillée dépasse le seuil cinq fois consécutives, une alarme est alors déclenchée ; toutefois, si la valeur surveillée tombe sous le seuil pendant l'une de ces cinq vérifications, la longueur de l'intervalle revient à cinq minutes.

Par conséquent, pour un évènement unique, une alarme sera déclenchée au plus tard 450 secondes après la première détection de l'événement.

Niveaux d'alarme :

Les alarmes de performance déclenchent différentes actions selon les niveau d'alarme.

  • Avertissement : Génère un message d'avertissement dans le journal système, mais ne déclenche aucune notification système.
  • Critique: Génère non seulement un message dans le journal système, mais déclenche également des notifications système.

Les journaux des alarmes de performance sont visibles dans l'onglet Alarme des performances ou le Centre des journaux de DSM.
