Journaux

Sur la page Centre des journaux > Journaux, vous pouvez voir, rechercher et exporter des journaux.

Pour rechercher des journaux :

  1. Dans le menu déroulant de recherche, vous pouvez spécifier les critères suivants selon la situation :
    • Mot-clé : Saisissez un mot clé pour rechercher les messages de journal contenant le mot clé correspondant.

      Remarque :

      Vous pouvez utiliser les caractères de remplacement et les opérateurs suivants pour une recherche avancée :

      • [*] : le caractère de remplacement représente zéro caractère (ou plus), différent de l'espace. Par exemple, [a*n] peut représenter un mot commençant par [a] et se terminant par [n], tel que [action], [attention] ou [an].
      • [?] : le caractère de remplacement représente exactement un caractère différent de l'espace. Par exemple, [ca?] peut représenter un mot commençant par [ca] et se terminant par un seul caractère, tel que [car], [cat] ou [cap].
      • [AND] : L'opérateur affiche les résultats qui comprennent tous les termes de la recherche. Par exemple : [admin AND root] affiche les résultats du journal qui comprennent à la fois [admin] et [root].
      • [OR] : L'opérateur affiche les résultats qui comprennent un des termes de la recherche. Par exemple, [admin OR root] affiche les résultats qui contiennent [admin] ou [root].
      • [NOT] : L'opérateur exclut des mots de votre recherche. Par exemple, [NOT admin] affiche les résultats qui ne comprennent pas [admin].
      • [()] : les parenthèses entourent les termes de recherche afin de spécifier l'ordre dans lequel ils sont interprétés si AND et OR sont tous deux utilisés. Par exemple, [(admin OR root) AND NOT logged] affiche les résultats qui comprennent [admin] ou [root], mais ne comprennent pas [logged].
    • Plage de date : Recherche les journaux selon la période spécifiée.
    • Niveau : Recherche les journaux selon le niveau spécifié.
    • Nom d'hôte : Recherche les journaux selon le nom d'hôte spécifié.
    • Programme : Recherche les journaux selon le nom de programme spécifié.
    • Catégorie : Recherche les journaux selon le nom de catégorie spécifié.
  2. Cliquez sur Recherche pour rechercher les journaux correspondant aux critères spécifiés.

Pour exporter des journaux :

  1. Cliquez sur la flèche à côté du bouton Export.
  2. Choisissez HTML ou CSV pour télécharger les journaux à l'aide du format désiré.

Remarque :

  • Le cryptage des caractères des fichiers journaux exportés CSV est UTF-8. Vous devez ouvrir le fichier avec un programme qui prend en charge le cryptage UTF-8.
