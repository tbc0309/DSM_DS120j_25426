Vue d'ensemble

La vue d'ensemble affiche le statut global de vos espaces de stockage, notamment la santé globale de tous les espaces, l'état des volumes, l'état du groupe de stockage et les informations sur les disques.

Santé du stockage

Il y a trois types de statut :

  • Sain : tous les espaces de stockage (volumes ou groupes de stockage) sont normaux.
  • Attention : un espace de stockage (volume ou groupe de stockage) est dégradé, le cache SSD a planté ou une partition système a échoué.
  • Danger : un espace de stockage (volume ou groupe de stockage) a planté.

Si plusieurs anomalies se produisent en même temps, l'état de santé sera affiché dans l'ordre suivant : 1) espace de stockage (volume ou groupe de stockage) ; 2) cache SSD et 3) partition système. En cas de problème, veuillez suivre les instructions suggérées sous le statut.

Sélecteur de groupe de stockage

Le sélecteur de groupe de stockage répertorie tous les groupes de stockage. Sélectionnez un groupe de stockage pour consulter les détails qui s'y rapportent.

État du volume

Cette section affiche la capacité utilisée des volumes figurant dans le groupe de stockage sélectionné. Si l'utilisation totale atteint la valeur critique du volume, le graphique à secteurs devient rouge ; au-delà de la valeur d'avertissement, il devient orange ; sinon, il reste bleu. Les valeurs critiques et d'avertissement sont modifiables dans Panneau de configuration > Notification > Avancé > Stockage interne > Volume plein.

Groupes de stockage / volumes

Lorsqu'aucun groupe de stockage spécifique n'est sélectionné, les détails de tous les groupes de stockage s'affichent. Lorsqu'un groupe de stockage spécifique est sélectionné, les détails et l'attribution de l'espace de ce groupe de stockage s'affichent.

Informations du disque

Cette section affiche les informations du disque et du matériel. Vous pouvez vérifier combien de disques sont utilisés et par quels espaces de stockage, le nombre de disques de secours et le nombre d'emplacements de disques disponibles.
