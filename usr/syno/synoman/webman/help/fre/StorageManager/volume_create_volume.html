Créer un volume

Comme un volume est constitué d'un groupe de stockage, vous devez créer un groupe de stockage avant de créer un volume. Toutefois, si le mode de création rapide est pris en charge sur votre modèle, vous pouvez simultanément créer un volume et un groupe de stockage, et l'assistant de création de volume optimisera la capacité de stockage et les performances sans que vous ayez à vous soucier des aspects techniques. Le mode de création rapide n'est disponible que sur les modèles qui prennent en charge SHR (Synology Hybrid RAID). Pour plus d'informations, visitez le site Web de Synology.

Pour créer un volume avec des modèles qui prennent en charge le mode de création rapide :

Vous pouvez créer simultanément un volume et un groupe de stockage. Lors de la création d'un nouveau volume, vous pouvez choisir une des méthodes suivantes :

  • Rapide : C'est la meilleure option pour les débutants. Choisissez ce mode si vous voulez optimiser automatiquement la capacité et les performances de stockage sans vous soucier des aspects techniques tels que les types de RAID.
  • Personnalisé : Pour les utilisateurs qui souhaitent configurer les options avancées de la création de volume, telles que les types de RAID et l'allocation de capacité. (Cliquez ici pour connaître la méthode de configuration du groupe de stockage.)

Suivez les étapes ci-dessous pour créer un volume en mode de création rapide.

  1. Ouvrez le Gestionnaire de stockage.
  2. Accédez à Volume et cliquez sur Créer.
  3. Vous verrez apparaître l'assistant de création de volume. Sélectionnez Rapide ou Personnalisé selon vos besoins. Puis suivez les étapes de l'assistant pour terminer la création du volume.

Pour créer un volume avec d'autres modèles :

  1. Assurez-vous que votre Synology NAS dispose d'un groupe de stockage existant avec de l'espace pouvant être alloué.
  2. Cliquez sur le bouton Créer.
  3. Choisissez un groupe de stockage sur lequel le nouveau volume sera créé.
  4. Configurez la taille d'allocation pour le volume.
  5. Cliquez sur Appliquer pour confirmer les paramètres.

Remarque :

  • Si vous sélectionnez Performances améliorées comme type de groupe de stockage lors de la création du groupe de stockage, vous ne pourrez pas configurer la taille d'allocation des volumes créés sur ce groupe de stockage.
  • La taille maximale du volume varie en fonction des modèles. Reportez-vous aux spécifications du modèle pour plus de détails.
  • La taille minimale allouée à un volume doit être d'au moins 10 Go.
  • Un volume Btrfs réserve 5 % de l'espace aux metadata.
