Hot Spare

Les disques Hot Spare sont des disques de secours qui permettent de réparer un groupe de stockage dégradé en remplaçant automatiquement un disque défaillant. Les disques Hot Spare peuvent être affectés globalement pour protéger tout groupe de stockage dans votre Synology NAS, à condition que le groupe de stockage corresponde aux critères suivants :

  • Le stockage doit être un type de RAID avec protection des données (c'est-à-dire RAID 1, RAID 5, RAID 6, RAID 10, RAID F1).
  • La taille du disque Hot Spare doit être égale ou supérieure à la taille du plus petit disque du groupe de stockage.

Pour configurer les disques Hot Spare :

  1. Cliquez sur Gérer.
  2. Une boîte de dialogue apparaît, affichant l'état Hot Spare de chaque groupe de stockage ainsi que les disques disponibles. L'état du groupe de stockage comprend les éléments suivants :
    • Affecté : le groupe de stockage est protégé par au moins un disque Hot Spare affecté.
    • Disponible : des disques sont disponibles pour protéger le groupe de stockage mais aucun Hot Spare n'a encore été affecté.
    • Indisponible : aucun disque Hot Spare n'est disponible pour protéger le groupe de stockage.
  3. La section disques disponibles affiche les disques qui peuvent être affectés comme disques Hot Spare. Pour affecter un disque Hot Spare, cochez la case à côté du disque que vous souhaitez affecter comme Hot Spare.
  4. Cliquez sur Appliquer pour enregistrer les paramètres.

Remarque :

  • Lorsqu'un disque est attribué en tant que disque Hot Spare, toutes les données du disque sont effacées. Avant d'affecter un disque Hot Spare, assurez-vous que le disque ne contient pas de données importantes.
  • Un RAID assemblé par des HDD ne sera pas automatiquement réparé par des disques SSD Hot Spare et vice versa.
  • Nous vous recommandons de choisir des disques compatibles à utiliser comme Hot Spare pour éviter tout problème sur votre Synology NAS.

Pour configurer les paramètres Hot Spare pour un groupe de stockage :

  1. Cliquez sur Configurer.
  2. Les groupes de stockage qui peuvent être protégés par des disques Hot Spare sont répertoriés dans l'onglet Groupe de stockage. Si vous décochez les cases, les groupes de stockage correspondants ne sont plus automatiquement réparés par ces disques Hot Spare en cas de dégradation.
  3. Dans l'onglet Hot Spare inter-périphériques, vous pouvez autoriser des disques Hot Spare sur un périphérique pour réparer un RAID dégradé qui se trouve sur un périphérique différent.
  4. Cliquez sur OK pour enregistrer les paramètres.
