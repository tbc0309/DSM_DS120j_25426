Vue d'ensemble du stockage

La vue d'ensemble du stockage indique le statut des disques installés, les connexions de câbles et voyants d'état matériel.

Voyants d'état matériel

Les voyants d'état matériel sur la droite présentent les deux couleurs suivantes :

  • Vert : l'état matériel (par exemple l'alimentation, le ventilateur, les connexions entre le serveur et les unités d'extension, etc.) est normal.
  • Rouge : une anomalie matérielle s'est produite.

Cliquer sur un voyant d'état ouvrira une liste détaillant l'état de tout le matériel, y compris l'alimentation, les ventilateurs et les connexions.

Afficher/Masquer le nombre de disques

Vous pouvez afficher ou masquer les nombres de chacun des disques avec le bouton Afficher/Masquer le nombre de disques.
