Nettoyage des données

Effectuer le nettoyage de données

Le nettoyage des données est un paramètre de maintenance des données qui inspecte les groupes de stockage. Les deux types suivants de nettoyage de données sont effectués de manière séquentielle :

  • Nettoyage du système de fichiers : cette fonction vérifie les volumes du système de fichiers Btrfs. Si des données sont incohérentes par rapport à la somme de contrôle détectée, le système essaie d'utiliser une sauvegarde pour réparer les données et le chemin du fichier est enregistré dans le Log Center.
  • Nettoyage du RAID : cette fonction peut être utilisée avec les groupes de stockage qui implémentent SHR (comportant trois disques ou plus), RAID 5 ou RAID 6. Elle peut réparer toute incohérence de données détectée.

Nous recommandons d'effectuer régulièrement le nettoyage des données afin d'assurer la cohérence des données et d'éviter de perdre des données en cas de panne de disque.

Remarque :

  • Le nettoyage des données et les fonctions connexes décrites dans cet article sont disponibles sur certains modèles uniquement.
  • Le nettoyage des données ne peut pas être effectué lorsque l'état du groupe de stockage a Planté ou Dégradé.
  • Si la somme de contrôle des données pour l'intégrité avancée n'est pas activée pour un dossier partagé, le nettoyage des fichiers système n'est pas en mesure de vérifier et de réparer les données dans le dossier pour un manque de somme de contrôle.

Configuration de la planification du nettoyage des données

Vous pouvez configurer une planification du nettoyage des données. Le système exécutera le nettoyage des données en fonction de vos paramètres.

Pour configurer la planification du nettoyage des données :

  1. Ouvrez le Gestionnaire de stockage.
  2. Accédez à Groupe de stockage > Nettoyage des données.
  3. Cliquez sur Définir la programmation.
  4. Cochez Activer la planification du nettoyage des données.
  5. Précisez la fréquence, la date de début et la session temporelle.
  6. Accédez à l'onglet Sélectionner une cible.
  7. Sélectionnez le groupe de stockage sur lequel vous souhaitez effectuer le nettoyage des données.
  8. Cliquez sur OK.

Remarque :

  • Pour être ajouté à une planification, un groupe de stockage doit être en état Normal et prendre en charge le nettoyage des données.
  • Si votre système est mis à jour à partir de la version 6.1, les groupes de stockage ayant la capacité d'exécuter le nettoyage de données sont ajoutés par défaut à la planification.
  • Si vous avez configuré votre planification dans la version 6.1, nous activerons automatiquement votre planification en fonction de vos paramètres précédents.

Pour ajouter un groupe de stockage à une planification ou supprimer un groupe de stockage d'une planification :

  1. Cliquez sur Définir la programmation.
  2. Accédez à l'onglet Sélectionner une cible.
  3. Cochez ou décochez la case d'un groupe de stockage pour décider si vous souhaitez l'ajouter ou le supprimer d'une planification.
  4. Cliquez sur OK.

Remarque :

  • Un groupe de stockage ne peut pas être ajouté à ou supprimé d'une planification lorsque le nettoyage des données est exécuté manuellement sur ce groupe de stockage.
  • Pour supprimer un groupe de stockage qui subit un nettoyage de données planifié, la progression de ce groupe de stockage sera annulée.

Pour modifier les paramètres de la planification :

  1. Cliquez sur Définir la programmation
  2. Modifiez la fréquence, la date de début et la session temporelle selon vos besoins.
  3. Cliquez sur OK.

Remarque :

  • La modification de la date de début réinitialisera la progression de la planification et le nettoyage des données redémarrera en fonction de la nouvelle date de début.

Pour mettre en pause une planification :

  1. Sélectionnez un groupe de stockage sur lequel une planification de nettoyage des données est en cours.
  2. Cliquez sur Mettre en pause la planification à partir de la liste déroulante Action.

Remarque :

  • Vous devez cliquer sur Reprendre la planification à partir du menu déroulant Action pour reprendre la planification du nettoyage des données. Sinon, la planification demeure en pause.

Exécution manuelle du nettoyage des données :

Lorsqu'une anomalie est détectée par le système au niveau de l'espace de stockage, vous pouvez essayer de réparer vos données en exécutant manuellement le nettoyage des données.

Pour exécuter manuellement le nettoyage des données :

  1. Ouvrez le Gestionnaire de stockage.
  2. Accédez à Groupe de stockage > Nettoyage des données.
  3. Sélectionnez le groupe de stockage sur lequel vous souhaitez effectuer le nettoyage des données.
  4. Cliquez sur Exécution manuelle dans le menu déroulant Action.

Remarque :

  • Si le système est arrêté avant la fin du nettoyage des données, une partie de la progression peut être perdue et vous devrez recommencer le nettoyage des données manuellement.
  • Si le moment de l'exécution manuelle du nettoyage des données recouvre un nettoyage des données planifié, cette cession de planification n'est pas effectuée tant le nettoyage manuel des données n'est pas terminé.
  • Comme l'exécution du nettoyage des données peut avoir une influence sur les performances de votre système, nous vous suggérons de ne pas l'exécuter sur plusieurs groupes de stockage, à moins que cela ne soit nécessaire.

Pour mettre en pause une planification sur un groupe de stockage soumis à un nettoyage des données manuel :

  1. Sélectionnez un groupe de stockage que vous souhaitez mettre en pause.
  2. Cliquez sur Pause dans le menu déroulant Action.
