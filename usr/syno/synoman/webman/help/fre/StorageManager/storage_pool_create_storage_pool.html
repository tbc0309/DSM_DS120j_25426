Créer un nouveau groupe de stockage

Important :

  • Avant de démarrer, assurez-vous qu'il n'y a pas de données importantes sur les disques sur lesquels le groupe de stockage va être créé. Toutes les données existantes seront effacées au cours du processus de création.
  • Accédez à la page HDD/SSD et assurez-vous que l'état de chaque disque est Initialisé ou Non initialisé.
  • Nous vous recommandons d'utiliser des disques figurant dans la Liste de compatibilité des produits Synology pour éviter tout problème. L'utilisation de disques incompatibles pourrait réduire les performances système, voire provoquer une perte de données sur votre Synology NAS.

Pour créer un groupe de stockage :

  1. Ouvrez le Gestionnaire de stockage.
  2. Accédez à Groupe de stockage et cliquez sur le bouton Créer.
  3. Choisissez le type de groupe de stockage :
    • Meilleures performances : ce type de groupe de stockage ne prend en charge qu'un volume unique, mais offre de meilleures performances.
    • Plus grande flexibilité : ce type de groupe de stockage prend en charge plusieurs volumes, Synology Hybrid RAID (SHR), et la fonctionnalité RAID Group.
  4. Configurez les propriétés du groupe de stockage :
    • Description du groupe de stockage : une description de ce groupe de stockage.
    • Type de RAID : différents types de RAID offrent différents niveaux de protection des données. Pour plus d'informations, reportez-vous à Généralités.
    • Nombre minimum de disques par RAID : le nombre minimum de disques requis pour le type de RAID sélectionné.
    • Nombre maximum de disques par RAID : un nombre plus important de disques par RAID augmente la probabilité d'une panne du RAID. Nous vous recommandons de régler la limite du nombre de disques sur un RAID unique et d'utiliser RAID Group pour trouver un juste milieu entre la sécurité des données et la capacité de stockage du groupe.
    • Fonctionnalité RAID Group : si la fonctionnalité RAID Group est prise en charge par le type de RAID sélectionné. Alors qu'un RAID unique est restreint au niveau du nombre de disques, la fonctionnalité RAID Group vous permet de composer votre bloc de stockage de plusieurs RAID, ce qui augmente sa capacité et la redondance des disques. RAID Group est pris en charge par RAID 5, RAID 6 et RAID F1.
  5. Suivez les étapes restantes de l'assistant pour terminer la création du groupe de stockage.

Remarque :

  • SHR (Synology Hybrid RAID) n'est disponible que sur certains modèles. Pour plus d'informations, accédez au site Web de Synology.
  • Le nombre maximum de disques par RAID et la fonctionnalité RAID Group ne sont disponibles que sur certains modèles. Pour plus d'informations, accédez au site Web de Synology.
  • Le nombre maximum de disques par RAID ne peut être modifié que si le type de RAID sélectionné prend en charge la fonctionnalité RAID Group.
  • Si le type de RAID du groupe de stockage est RAID 5, RAID 6, RAID F1, SHR (plus de trois disques) ou SHR-2, le système effectue une resynchronisation du RAID sur le groupe de stockage afin d'améliorer la vitesse d'accessibilité et la résilience des données.
