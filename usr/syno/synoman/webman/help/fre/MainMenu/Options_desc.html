Options

Vous pouvez trouver le menu Options en cliquant sur l'icône en forme de personne dans le coin supérieur droit du bureau DSM. Il vous permet de faire ce qui suit :

  • Modifier les options des comptes personnels
  • Déconnexion
  • Arrêter le système (sur EDS14) ou d'éteindre le système (sur tous les autres modèles)
  • Redémarrer le système

Après avoir affiché le menu, cliquez sur Personnel pour modifier les paramètres de compte personnel.
