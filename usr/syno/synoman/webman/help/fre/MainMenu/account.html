Compte

L'onglet Compte fournit les options permettant de modifier les paramètres de compte basiques tels que la description, le mot de passe, l'email et la vérification en 2 étapes. Pour accéder à cet onglet, cliquez sur l'icône en forme de personnage, affichée en haut à droite du bureau et sélectionnez Personnel.

Modifier les paramètres de compte

Les paramètres de compte suivants peuvent être modifiés :

  • Description : Entrez une description pour votre compte. Les descriptions de l'utilisateur peuvent inclure jusqu'à 64 caractères Unicode affichables.
  • Nouveau mot de passe : Si vous souhaitez changer le mot de passe de votre compte, entrez un nouveau mot de passe ici et confirmez-le dans le champ Confirmer le mot de passe ci-dessous. Les mots de passe doivent respecter la casse et peuvent être composés de caractères affichables (127 maximum), tels que lettres, nombres, symboles et espaces.
  • Email : Entrez une adresse email (uniquement pour les utilisateurs non admin).
  • Langue d'affichage : Spécifiez la langue d'affichage applicable à votre compte. Les paramètres de langue d'affichage que vous définissez ici auront la priorité sur ceux définis dans Panneau de configuration > Options régionales > Langue.
  • Format de date : Spécifiez le format de date, en fonction de vos préférences et de votre situation géographique.
  • Format de l'heure : Spécifiez si l'heure doit être affichée au format 12 heures ou 24 heures.

Après avoir modifié les paramètres de compte, cliquez sur OK pour enregistrer les modifications.

Vérification en 2 étapes

La vérification en 2 étapes offre une sécurité optimisée à votre compte DSM. Si la vérification en 2 étapes est activée, vous devrez entrer votre mot de passe en plus d'un code de vérification unique lorsque vous vous connectez à DSM. Les codes de vérification sont obtenus depuis une application d'authentification installée sur votre périphérique mobile. Par conséquent, si quelqu'un veut accéder à votre compte, il devra non seulement connaître votre nom d'utilisateur et mot de passe, mais aussi être en possession de votre périphérique mobile.

Configuration requise :

La vérification en 2 étapes nécessite un périphérique mobile et une application d'authentification qui prend en charge le protocole TOTP (Time-based One-Time Password). Les applications d'authentification comprennent Google Authenticator (Android/iPhone/BlackBerry) ou Authenticator (Windows Phone).

Pour activer la vérification en 2 étapes, procédez comme suit :

  1. Accédez à Personnel > onglet Compte et activez la case à cocher Activez la vérification en 2 étapes pour lancer l'assistant de configuration de la vérification en 2 étapes.
  2. Sur le premier écran, entrez une adresse e-mail. Des codes de vérification d'urgence peuvent être envoyés à cette adresse email en cas de perte de votre périphérique mobile.
  3. Téléchargez et installez une application d'authentification telle que Google Authenticator (Android/iPhone/BlackBerry) ou Authenticator (Windows Phone).
  4. Sur l'écran suivant, ouvrez votre application d'authentification et numérisez le code QR. Vous pouvez également cliquer sur le lien manuel pour entrer une Secret Key.
  5. Votre application d'authentification génèrera ensuite un code de vérification à 6 chiffres. Entrez ce code dans le champ de texte de l'assistant afin de confirmer que les configurations sont correctes. Si une erreur se produit, veuillez-vous assurer que l'heure système de votre périphérique mobile est synchronisée avec l'heure système de DSM. En outre, les codes de vérification étant mis à jour régulièrement, il convient de vérifier que le code saisi n'est pas expiré.
  6. Une fois l'assistant de configuration terminé, cliquez sur OK pour enregistrer les paramètres.

Pour vous connecter à DSM avec la vérification en 2 étapes, procédez comme suit :

Quand la vérification en 2 étapes est activée, vous êtes invité à entrer un code de vérification à 6 chiffres lorsque vous vous connectez à DSM.

  1. Sur l'écran de connexion de DSM, entrez votre nom d'utilisateur et le mot de passe, comme d'habitude.
  2. Quand vous êtes invité à entrer un code de vérification, ouvrez l'application d'authentification sur votre périphérique mobile.
  3. Trouvez et entrez le code de vérification à 6 chiffres pour votre compte. Si votre périphérique mobile est perdu, vous pouvez cliquer sur le lien Vous avez perdu votre téléphone ? et un code de vérification d'urgence sera alors envoyé à votre adresse email.

Remarque :

  • Paramètres SMTP : Pour recevoir des codes d'urgence par e-mail, les paramètres du serveur SMTP doivent être correctement configurés. Vous pouvez accéder à Panneau de configuration > Notification ou Personnel > Compte email pour vérifier si vos paramètres sont corrects.
  • Limite du code d'urgence : Chaque utilisateur a une limite de 5 codes d'urgence. Si vous dépassez votre limite, vous devrez désactiver et activer la vérification en 2 étapes avant de pouvoir recevoir d'autres codes d'urgence.
  • Il est possible de forcer la vérification en 2 étapes sous DSM et toutes les applications mobiles Synology connexes (par ex., DSM mobile et DS file).
  • La vérification en 2 étapes s'applique également à tous les services en rapport avec SSH, tels que SSH terminal, SFTP et rsync avec le chiffrement du transfert SSH. Toutefois, comme Hyper Backup et Shared Folder Sync via rsync avec le chiffrement du transfert SSH ne prennent pas en charge la vérification en 2 étapes, ces deux services ne peuvent pas fonctionner normalement après l'activation de cette vérification.

Pour réinitialiser les paramètres de la vérification en 2 étapes des utilisateurs :

Les utilisateurs appartenant au groupe administrators peuvent réinitialiser les paramètres de la vérification en 2 étapes des autres utilisateurs.

  1. Accédez à Panneau de configuration > Utilisateur.
  2. Double-cliquez sur l'utilisateur dont vous voulez réinitialiser la vérification en 2 étapes.
  3. Sur l'onglet Informations utilisateur, cliquez sur le bouton Réinitialiser la vérification en 2 étapes.

Remarque :

  • Si un utilisateur appartenant au groupe administrators n'est pas disponible, vous pouvez appuyer sur le bouton de réinitialisation physique sur votre Synology NAS pour réinitialiser les paramètres DSM et désactiver la vérification en 2 étapes.

Activité du compte

L'activité du compte indique l'activité de connexion de votre compte DSM, notamment les connexions actuelles, les périphériques mémorisés et l'historique de connexion.

Pour vérifier l'activité de votre compte :

Cliquez sur le bouton Activité du compte. Une fenêtre contextuelle s'affiche et présente les onglets suivants :

  • Utilisateurs connectés : Cet onglet affiche les informations suivantes :
    • Date et heure : Date et heure du dernier accès au compte.
    • Nom du client et IP : Nom et adresse IP du périphérique qui a accédé au compte. Le nom du client apparaît uniquement lorsque le compte est accédé avec le protocole CIFS.
    • Service : Protocole utilisé pour accéder au compte.
    • Ressources : Ressources système utilisées par le compte.
  • Périphériques mémorisés : Cet onglet affiche la page de gestion des périphériques mémorisés :
    • Le périphérique actuel n'est pas mémorisé / est mémorisé : Cliquez sur Mémoriser ce périphérique si vous choisissez de mémoriser le périphérique actuel. La vérification en 2 étapes ne sera plus nécessaire lorsque vous vous connecterez à DSM sur ce périphérique.
    • Cesser de mémoriser d'autres périphériques : Vous pouvez arrêter la mémorisation d'autres périphériques (par exemple d'autres ordinateurs ou périphériques mobiles) qui ont été précédemment mémorisés. Cela signifie que la vérification en 2 étapes sera nécessaire lorsque vous vous connecterez à DSM sur ces périphériques.
  • Historique de connexion : Cet onglet affiche les sessions de connexion récentes et les informations suivantes :
    • Date et heure : Date et heure du dernier accès au compte.
    • Événement : Indique le nom d'utilisateur et l'adresse IP.

Remarque :

Le périphérique que vous avez marqué comme mémorisé sera mémorisé uniquement lorsque vous vous connecterez avec le même compte et à partir du même navigateur. Vous serez invité à saisir un OTP (mot de passe à usage unique) pour la vérification en 2 étapes dans les scénarios suivants, même si vous avez précédemment marqué un périphérique comme mémorisé :

  • Vous vous connectez avec le même compte, mais sur un navigateur différent.
  • Vos cookies ont été supprimés du navigateur.
  • Vous marquez un périphérique comme mémorisé lors de la connexion en tant qu'utilisateur A, puis vous marquez le même périphérique comme mémorisé lorsque vous vous connectez en tant qu'utilisateur B. Lorsque vous vous connecterez à nouveau en tant qu'utilisateur A, vous devrez encore saisir un mot de passe pour la vérification en 2 étapes.
