Prestazioni

Nella pagina Prestazioni, è possibile monitorare l'uso delle risorse su entrambi i server, attivo e passivo, lato per lato. I due server possono essere distinti facilmente per il nome server e i rispettivi ruoli (attivo o passivo). Quando il server passivo è spento, l'uso delle risorse nel server passivo non sarà visualizzato.

CPU

È possibile controllare l'uso generale della CPU in base a sistema, utenti e coda I/O. Dopo avere aperto la pagina, il carico CPU potrebbe essere più elevato in quanto il sistema sta caricando la pagina e, contemporaneamente, raccoglie le informazioni sull'uso del sistema.

Memoria

Sarà visualizzato l'uso generale della memoria fisica. Se l'uso di memoria resta alto, è possibile che il sistema archivi i dati usati di frequente nella cache per accedervi rapidamente, senza alcun accesso al disco. Quando la memoria generale non è sufficiente, il sistema rilascia la memoria della cache.

Rete

Nel grafico del traffico di rete viene mostrata la velocità di trasferimento dati in KB al secondo, con colori diversi:

  • In blu è indicata la velocità caricamento.
  • In verde è indicata la velocità di download.
  • Quando le velocità di caricamento e di download si sovrappongono, l'area di sovrapposizione sarà indicata in verde scuro.

Disco

Nel grafico viene visualizzato l'uso generale del disco.
