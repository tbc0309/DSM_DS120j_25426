Storage Pool

Una o più unità possono essere aggregate in una singola unità di archiviazione denominata storage pool, che può essere protetta mediante RAID. Diversi tipi di RAID offrono vari livelli di protezione dei dati. Per un elenco generale di tipi RAID supportati da Synology NAS, fare riferimento a Informazioni generali. Dopo avere creato uno storage pool, sarà possibile creare su questo uno o più volumi. Un volume è un'unità di archiviazione dati e può essere ampliato se lo storage pool associato contiene spazio allocabile. Prima di iniziare ad archiviare dati in Synology NAS, è necessario creare almeno un volume.

Nota:

  • Si consiglia di utilizzare le unità nell'Elenco compatibilità dei prodotti Synology per evitare problemi di compatibilità. L'utilizzo di unità incompatibili potrebbe ridurre le prestazioni di sistema e persino causare la perdita di dati.
  • Uno storage pool deve essere composto di unità dello stesso tipo. Quindi, uno storage pool non può essere creato con una combinazione di unità SATA e SAS.
  • Per i modelli Synology NAS che supportano RAID Group, uno storage pool RAID 5, RAID 6 o RAID F1 con supporto multi-volume può contenere più Array RAID. Ogni array RAID che appartiene allo stesso storage pool avrà lo stesso tipo RAID (ad esempio RAID 5, RAID 6 o RAID F1). Il numero massimo di unità per ciascun array RAID è 6, 12 o 24. È possibile impostare il numero unità massimo durante la creazione dello storage pool. Il numero non potrà essere modificato in seguito. Dopo avere assegnato il numero massimo di unità all'array RAID, l'unità successiva aggiunta allo storage pool potrebbe essere assegnata a un nuovo array RAID. Il nuovo array RAID deve contenere un numero minimo di dischi richiesto per il tipo RAID specifico (ad esempio tre dischi per RAID 5 e RAID F1 e quattro per RAID 6).
  • Se si utilizza una determinata unità per la creazione di uno storage pool, ma l'unità non appare nella Procedura guidata di creazione dello storage pool, andare alla pagina the HDD/SSD per controllare lo stato dell'unità.
