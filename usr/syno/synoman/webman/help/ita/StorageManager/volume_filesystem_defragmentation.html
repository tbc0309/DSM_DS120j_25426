Deframmentazione del file system

Se i dati vengono modificati di continuo, il numero di dati frammentati aumenterà nel file system, causando un rallentamento delle prestazioni di accesso dei file nelle unità. La deframmentazione del file system può migliorare tali prestazioni.

Per eseguire la deframmentazione del file system:

  1. Aprire Storage Manager.
  2. Andare su Volume.
  3. Selezionare il volume in cui eseguire la deframmentazione del file system.
  4. Selezionare Deframmentazione file system dal menu a discesa Azione.

Nota:

  • La deframmentazione del file system è disponibile solo su alcuni modelli.
  • La deframmentazione del file system può essere eseguita solo su volumi nel file system Btrfs.
  • La deframmentazione del file system può essere eseguita solamente quando lo stato del volume è Integro.
  • Non spegnere il sistema prima del termine della deframmentazione del file system. Se il sistema si spegne prima del completamento, sarà necessario riavviare dall'inizio la deframmentazione del file system.
  • L'uso del volume potrebbe aumentare se il volume contiene istantanee di cartelle condivise.
