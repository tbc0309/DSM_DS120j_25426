Espandere le dimensioni del volume

È possibile modificare le dimensioni di un volume se questo è creato in uno storage pool che supporta più volumi. Tuttavia, se un volume viene creato in uno storage pool che supporta un volume singolo, è ancora possibile espandere le dimensioni volume aggiungendo unità allo storage pool per espandere prima la capacità di quest'ultimo.

Per uno storage pool multi-volume:

Per espandere le dimensioni di un volume, procedere come segue:

  1. Andare alla pagina Volume e selezionare il volume da espandere.
    Nota: è possibile aggiungere unità allo storage pool corrispondente per espandere prima la capacità dello storage pool se necessario. Per ulteriori dettagli, consultare Aggiungere unità per espandere la capacità dello storage pool.
  2. Fare clic su Configura dal menu a discesa Azione.
  3. Andare alla scheda Generale e immettere le nuove dimensioni volume nel campo Modifica dimensioni assegnate (GB) oppure fare clic sul pulsante Max per inserire automaticamente le dimensioni massime assegnabili.
  4. Fare clic su OK per terminare.

Per uno storage pool a volume singolo:

Per espandere le dimensioni di un volume, procedere come segue:

  1. Aggiungere unità a uno storage pool per espanderne la capacità.
    Nota: Per ulteriori dettagli, consultare Aggiungere unità per espandere la capacità dello storage pool.
  2. Le dimensioni del volume corrispondente saranno estese automaticamente dopo l'ampliamento della capacità dello storage pool.

Nota:

  • Per informazioni sulle dimensioni volume singole massime del modello di Synology NAS in uso, vedere qui.
