Volumi

I volumi forniscono lo spazio di archiviazione di base per Synology NAS. Tutti gli elementi dell'utente -- come cartelle condivise, documenti e dati pacchetto -- sono archiviate sui volumi. Quindi, prima di iniziare ad archiviare elementi su Synology NAS, è necessario creare almeno un volume.

Nota:

Se i volumi sono usati come target replicati di Snapshot Replication, le prestazioni di replica possono essere migliorate regolando le impostazioni di Frequenza di registrazione ora di accesso file nella finestra Modifica dopo la creazione dei volumi:

  • Giornalmente: l'ora di accesso al file viene aggiornata ogni giorno oppure quando il file viene modificato, proprio come nella maggior parte dei sistemi operativi. Le funzionalità di alcune applicazioni si basano sulla precisione dell'ora di accesso al file. Selezionando Giornaliero si influirà sulle prestazioni di replica; tuttavia, quest'opzione è consigliata se si utilizza una delle seguenti funzioni o applicazioni frequentemente:
    • Ora di accesso meno recente in Storage Analyzer
    • SSD Cache Advisor in Storage Manager
    • Qualsiasi applicazione di terzi basata su Registrazione ora di accesso file.
  • Mensile: l'ora di accesso al file viene aggiornata mensilmente oppure quando il file viene modificato. Quest'opzione comporta un piccolo miglioramento nelle prestazioni di replica, mentre le applicazioni che si basano su Registrazione ora di accesso file saranno in grado di funzionare, anche se con minore precisione.
  • Mai: consigliato per ottenere le migliori prestazioni di replica e non sono necessarie applicazioni che si basano su Registrazione ora di accesso file nell'uso quotidiano.

Formato volume

A seconda del file system in uso, sono presenti due formati di volume:

  • Formato nuovo volume: questo formato utilizza il file system Btrfs. Le dimensioni minime di un volume in questo formato devono essere di almeno 10 GB. Supporta le seguenti nuove funzioni con cui è possibile gestire i dati in modo più semplice:
    • Elenco cartella condivisa: l'istantanea registra lo stato di tutti i dati nel momento in cui viene acquisita. Se un utente modifica o elimina accidentalmente i dati in un volume con istantanee, sarà possibile ripristinare rapidamente i dati al periodo precedente, in cui è stata acquisita l'istantanea.
    • Quota cartella condivisa: è possibile impostare la capacità massima per ogni cartella.
    • Quota utente in base alla cartella condivisa: è possibile controllare la quantità di spazio utilizzabile in una cartella condivisa da ogni utente.
    • Protezione dell'integrità dei dati avanzata: la cartella condivisa è protetta con le strategie somma di controllo CRC32 e copy-on-write.
  • Formato volume tradizionale: questo formato utilizza il file system ext4. Le dimensioni minime di un volume in questo formato devono essere di almeno 10 GB. Il file system ext4 non è supportato dalla maggior parte dei prodotti Synology, quindi sarà possibile migrare facilmente il volume in uso in questo formato in un altro prodotto Synology.

Nota:

  • Btrfs è disponibile solo su alcuni modelli. Per verificare se il proprio modello supporta Btrfs, consultare questo articolo.
