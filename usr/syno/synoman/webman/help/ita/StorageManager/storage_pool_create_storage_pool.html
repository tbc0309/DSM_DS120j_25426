Crea nuovo storage pool

Importante:

  • prima di iniziare, verificare che non siano presenti dati importanti nei dischi rigidi su cui verrà creato lo storage pool. Tutti i dati esistenti saranno cancellati durante il processo di creazione.
  • Andare alla pagina HDD/SSD e verificare che lo stato di ogni unità sia Inizializzato o Non inizializzato.
  • Si consiglia di utilizzare le unità nell'Elenco compatibilità dei prodotti Synology per evitare eventuali problemi. L'utilizzo di unità incompatibili potrebbe ridurre le prestazioni di sistema o persino causare la perdita di dati in Synology NAS.

Per creare uno storage pool:

  1. Aprire Storage Manager.
  2. Andare su Storage Pool e fare clic sul pulsante Crea.
  3. Scegliere il tipo di storage pool:
    • Migliori prestazioni: questo tipo di storage pool supporta solo un volume singolo ma fornisce prestazioni migliori.
    • Maggiore flessibilità: questo tipo di storage pool supporta più volumi, Synology Hybrid RAID (SHR) e la funzione RAID Group.
  4. Configurare proprietà storage pool:
    • Descrizione storage pool: una descrizione di questo storage pool.
    • Tipo di RAID: Diversi tipi di RAID offrono vari livelli di protezione dei dati. Per maggiori informazioni, vedere Informazioni generali.
    • Numero minimo di unità per RAID: Il numero minimo di dischi rigidi richiesti per il Tipo di RAID selezionato.
    • Numero massimo di unità per RAID: Un numero maggiore di unità per RAID aumenta la probabilità di guasti RAID. Si consiglia di regolare il numero limite di unità in un singolo RAID e di usare RAID Group per creare un equilibrio tra la sicurezza dei dati e la capacità dello storage pool.
    • Funzione RAID Group: Se la funzione RAID Group è supportata dal tipo RAID selezionato. Mentre un RAID singolo presenta una restrizione sul numero di unità, la funzione RAID Group consente al blocco storage di essere composto da più RAID, incrementandone così la capacità e la ridondanza dell'unità. RAID Group è supportato da RAID 5, RAID 6 e RAID F1.
  5. Seguire i passaggi restanti della procedura guidata per terminare la creazione dello storage pool.

Nota:

  • SHR (Synology Hybrid RAID) è disponibile solo su alcuni modelli. Per maggiori informazioni, visitare il sito Web Synology.
  • Il numero massimo di unità per RAID e la funzione RAID Group sono disponibili solo in modelli specifici. Per maggiori informazioni, visitare il sito Web Synology.
  • Il numero massimo di unità per RAID può essere modificato se il tipo di RAID selezionato supporta la funzione RAID Group.
  • Se il tipo di RAID dello storage pool è RAID 5, RAID 6, RAID F1, SHR (superiore a tre unità) o SHR-2, il sistema eseguirà la risincronizzazione RAID sullo storage pool per migliorare la velocità di accesso e la resilienza dati.
