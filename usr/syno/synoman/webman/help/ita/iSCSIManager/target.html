Destinazione

A seconda del modello, Synology NAS supporta la creazione di più target su cui possono essere mappati i LUN.

Nota: il numero di target supportati varia a seconda dei modelli Synology NAS.

Per creare un target:

  1. Fare clic sul pulsante Crea.
  2. Inserire le informazioni target, incluso Nome, IQN e se abilitare l'Autenticazione CHAP.
  3. Fare clic su Avanti.
  4. Impostare la mappatura iSCSI LUN. Selezionare Crea nuovo iSCSI LUN o Mappare iSCSI LUN esistenti.
  5. Fare clic su Avanti.
  6. Fare clic su Indietro per modificare o su Applica per confermare le impostazioni.

Terminologie

  • IQN: IQN (Nome qualificato iSCSI) è un gruppo di nomi unici per un target. Esso non deve essere in conflitto con i Target IQN su altri server. Il nome qualificato iSCSI (IQN) non deve essere inferiore a 128 caratteri e può includere lettere (senza maiuscole/minuscole), numeri, due punti, trattini e punti. Il formato del nome suggerito è iqn.[data].[dominio]:[identificatore del dispositivo], ad esempio: iqn.2000-04.com.synology:storage.diskarray.1.
  • Autenticazione CHAP mutuo: in CHAP mutua, devono autenticarsi sia il cliente che il server. Qualsiasi password CHAP utilizzata per l'autenticazione del client non deve essere configurata per l'autenticazione di alcun server, e qualsiasi password CHAP utilizzata per l'autenticazione del server non deve essere configurata per l'autenticazione di alcun client.

    • Nota:il processo di Rilevamento su iniziatore iSCSI potrebbe non riuscire se l'autenticazione CHAP nel NAS è abilitata. Completare prima il Rilevamento, quindi abilitare l'autenticazione CHAP prima di connettersi a una destinazione. In alternativa è possibile connettersi a una destinazione con IP e IQN.

Per rimuovere un target:

È possibile rimuovere un target e ogni LUN mappato. Quando si rimuove un iSCSI LUN, tutti i dati su iSCSI LUN saranno cancellati.

  1. Selezionare il target da rimuovere.
  2. Fare clic su Rimuovi.
  3. Per rimuovere i LUN, selezionare Elimina anche i LUN mappati e selezionare quelli da rimuovere.
  4. Fare clic su OK.

Per modificare un target:

  1. Selezionare il target da modificare.
  2. Fare clic su Modifica dal menu a discesa Azione.
  3. Modificare le impostazioni secondo le preferenze. Fare clic su OK per salvare le modifiche.

Terminologie

  • Somma di controllo CRC: abilitare la Somma di controllo CRC per controllare gli errori verificatisi durante la trasmissione dei dati.
  • Sessioni multiple: consente sessioni multiple da uno o più iniziatori di connettersi allo stesso target allo stesso momento. Per proteggere i file da corruzione o perdita dei dati, si consiglia di utilizzare un file di sistema condiviso o un cluster come VMware Virtual Machine File System (VMFS) e Oracle Cluster File System (OCFS).
  • Mascheramento: il mascheramento consente l'accesso esclusivo a un target da iniziatori iSCSI specificati IQN.

Per abilitare o disabilitare un target:

  1. Selezionare il target da abilitare o disabilitare.
  2. Scegliere Abilita o Disabilita nel menu a discesa Azione.
