AFP

In Pannello di controllo > Servizi file > SMB/AFP/NFS, è possibile impostare il servizio AFP per rendere Synology NAS un server file Mac (server AFP).

Per abilitare il servizio file Mac:

  1. Spuntare Abilita servizio AFP per rendere Synology NAS un server Mac file (server AFP).
  2. Fare clic su Applica.

Abilitazione della registrazione di trasferimento

Se attivata, quest'opzione consente di registrare le operazioni di accesso ai file, fra cui creazione, eliminazione, lettura, scrittura, spostamento e rinomina di file o cartelle tramite AFP, denominato anche come protocollo Apple Filing Protocol.

Impostazioni avanzate

Applica autorizzazioni UNIX predefinite:

abilitando questa opzione si applicano le autorizzazioni UNIX predefinite quando si caricano o creano file e cartelle. Le autorizzazioni applicate sono le stesse applicate dal comando UNIX umask. Quando questa opzione è abilitata, le autorizzazioni UNIX sono 644 per i file e 755 per le cartelle. Quando questa opzione è disabilitata, le autorizzazioni UNIX sono 666 per i file e 777 per le cartelle. Il valore umask predefinito è 022.

Nota:

l'abilitazione di questa funzione può causare problemi di autorizzazioni non coerenti tra diversi protocolli. Per evitare incoerenze, consigliamo di lasciare questa opzione disabilitata.

Rilasciare subito le risorse dopo la disconnessione:

L'attivazione di quest'opzione comporta il rilascio immediato delle risorse in caso di disconnessione imprevista del server AFP. Se, durante l'utilizzo di Time Machine, viene visualizzato il messaggio di errore "Volume in uso", abilitare l'opzione.

Per ricercare i contenuti file tramite Finder:

Dopo avere completamente indicizzato e montato le cartelle nel computer Mac, è possibile usare Finder per ricercare i file contenuti e i relativi contenuti.

  1. Andare su Universal Search > Preferenze > Indicizzazione file > Elenco cartelle indicizzate > Crea per indicizzare specifiche cartelle e file contenuti per tipo di file.
  2. Fare clic su OK per iniziare l'indicizzazione.
  3. Montare le cartelle indicizzate sul computer Mac.
  4. Nel computer Mac, fare clic sulla cartella montata desiderata nel pannello di sinistra di Finder e usare Finder per ricercare i contenuti dei file contenuti.

Nota:

  • i modelli NVR (network video recorder) non supportano la ricerca di contenuti di file tramite Mac Finder.
  • usare Finder per la ricerca nelle cartelle montate. Spotlight su desktop Mac può essere usato solo per ricercare i file locali.
  • Su Mac OS X 10.9 e versioni successive, Finder supporta la ricerca la ricerca all'interno di cartelle condivise per nome tag e per categoria.
  • Quando si accede alle cartelle condivise tramite AFP, saranno visualizzate solo 255 cartelle condivise (in ordine alfabetico) se il numero delle cartelle condivise è maggiore di 255.
