Ora

È possibile controllare l'ora attuale, impostare la data e l'ora del server manualmente o consentire l'impostazione automatica utilizzando un server dell'ora di rete.

Per impostare il formato data:

scegliere il formato della data in base alle preferenze e alla posizione geografica.

Per impostare il formato ora:

scegliere l'ora di visualizzazione nel formato di 12 ore o 24 ore.

Per impostare il fuso orario:

Scegliere l'opzione Fuso orario dal menu.

Per impostare data e ora manualmente:

Scegliere Manualmente e selezionare la data e l'ora.

Per sincronizzare la data e l'ora con il server NTP:

  1. Scegliere Sincronizza con server NTP, quindi immettere l'indirizzo del server orario di rete o scegliere un server dal menu a discesa.
  2. Fare clic su Aggiorna adesso per verificare che Synology NAS sincronizza correttamente l'ora con il server.
  3. Fare clic su Applica.

Nota: per sincronizzare la data e l'ora con un server orario di rete, verificare di avere inserito un IP valido del server DNS su Menu principale > Pannello di controllo > Rete e che Synology NAS sia collegato a Internet.

Informazioni sul server Network Time Protocol (NTP)

Lo scopo principale di Network Time Protocol (NTP) è la sincronizzazione degli orologi del sistema del computer sulla rete di dati.

Per maggiori informazioni sul server NTP, visitare http://support.ntp.org/bin/view/Servers/WebHome
