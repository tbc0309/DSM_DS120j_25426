E-mail

Abilitando le notifiche e-mail consente a Synology NAS di inviare messaggi alla propria e-mail quando si verifica un evento specifico, come errori di sistema, perdita connessione di rete, comportamento ventola, interruzione dell'alimentazione o guasto attività.

Nota:

  • l'evento del funzionamento della ventola non è disponibile nei modelli senza ventola come EDS14. Per maggiori informazioni, visitare il sito Web Synology.

Per abilitare le notifiche e-mail:

  1. Andare su Pannello di controllo > Notifica > E-mail.
  2. Spuntare Abilita notifiche e-mail.
  3. Inserire uno o più indirizzi e-mail nel campo Indirizzo e-mail destinatario. Notifiche di sistema saranno inviate agli indirizzi inseriti qui. Sono consentiti due indirizzi. Separare più indirizzi con un punto e virgola (;).
  4. Se necessario, aggiungere un prefisso oggetto nel campo Prefisso oggetto. Questo prefisso sarà aggiunto all'oggetto di ogni messaggio inviato da Synology NAS, aiutando quindi a identificare e filtrare i messaggi.
  5. Se si desidera utilizzare un server e-mail di terzi (ad es. Gmail od Outlook) per trasmettere i messaggi, si veda la sezione "Servizio e-mail di terzi" seguente. Se si vuole utilizzare un server personalizzato, si veda la sezione "Server SMTP personalizzato".

Per utilizzare un servizio e-mail di terzi:

  1. Nel menu Provider di servizi, scegliere un servizio e-mail di terzi, come Gmail od Outlook.
  2. Inserire il nome utente e la password per il servizio e-mail selezionato.
  3. Fare clic su Applica per salvare le impostazioni.
  4. È possibile fare clic su Invia una e-mail di prova per vedere se le impostazioni sono corrette.

Per utilizzare un server SMTP personalizzato:

  1. Nel menu Provider di servizi, selezionare Server SMTP personalizzato.
  2. Inserire il nome server SMTP o l'indirizzo IP nel campo Server SMTP.
  3. Inserire il numero di porta SMTP nel campo Porta SMTP. (Se si è specificata una porta SMTP diversa mentre si configurano le regole di invio porta, inserire tale numero.)
  4. Se il server SMTP richiede l'autenticazione SMTP, selezionare Autenticazione richiesta e inserire Nome utente e Password per il server e-mail.
  5. Se il server SMTP richiede la connessione SSL/TLS, selezionare È richiesta una connessione sicura (SSL/TLS).
  6. Per personalizzare il nome del mittente indicato nei messaggi e-mail inviati da Synology NAS, inserire il nome nel campo Nome mittente. Se si desidera personalizzare l'indirizzo e-mail del mittente, immettere l'indirizzo e-mail nel campo E-mail mittente. Se il campo è vuoto, il primo indirizzo e-mail nel campo Indirizzo e-mail destinatario sarà utilizzato.
  7. Fare clic su Applica per salvare le impostazioni.
  8. È possibile fare clic su Invia una e-mail di prova per vedere se le impostazioni sono corrette.

Nota:

  • Se si desidera inviare un messaggio di benvenuto per ciascun account utente creato di recente, selezionare Invia messaggio di benvenuto ai nuovi utenti, e ogni utente riceverà un messaggio di benvenuto quando il suo account è creato.
  • La lingua per le notifiche e-mail può essere impostata su Lingua notifica. Per cambiare la lingua di notifica e-mail, andare sul Menu principale > Pannello di controllo > Opzioni regionali, fare clic sulla scheda Lingua, e scegliere la lingua preferita dal menu a discesa nella sezione Lingua notifica.
  • Se si sta utilizzando il server SMTP di Gmail e si è abilitata la verifica in due passaggi, ricordarsi di utilizzare la "password specifica per l'applicazione" fornita da Google per l'autenticazione SMTP.

Nome dominio server SMTP:

Il server SMTP può essere un indirizzo IP o un nome di dominio. Se è il nome di un dominio, verificare di aver inserito un IP valido del server DNS nella pagina Menu principale > Pannello di controllo > Rete e verificare che il server sia su Internet.

Nota:

a seguito di una modifica della politica API di Google Gmail API, gli utenti devono scegliere Personalizza impostazioni server SMTP dal menu a discesa Provider servizi per inviare e-mail per seguenti tre scopi. Per ulteriori informazioni sulla configurazione, consultare questo tutorial.

  • Usare il binario ssmtp per inviare e-mail, ad esempio l'esecuzione di scritpt definiti dall'utente nella Utilità di pianificazione.
  • Creare un sito Web con Web Station e inviare e-mail utilizzando PHP. È incluso anche l'invio di e-mail da pacchetti di siti Web di terzi, come Wordpress, MantisBT, ecc.
  • Inviare le notifiche mail quando qualcuno lascia un commento a una foto o un articolo in Photo Station.
