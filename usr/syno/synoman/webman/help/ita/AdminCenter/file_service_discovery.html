Avanzate

Nella scheda Avanzate, è possibile abilitare la funzione Clonazione veloce file per salvare spazio di archiviazione creando cloni file. Inoltre, è possibile configurare le impostazioni per consentire agli utenti di trovare Synology NAS nella rete tramite i servizi Bonjour e SSDP (Simple Service Discovery Protocol).

Per eseguire la clonazione file veloce in Synology NAS:

Con la funzione di clonazione veloce dei file, i blocchi di dati fisici verranno copiati solo alla modifica dei file clonati, consentendo così di ottimizzare lo spazio di archiviazione.

  1. Nella sezione Clonazione veloce file, spuntare la casella di controllo Abilita clonazione veloce file.
  2. Fare clic su Applica per salvare le impostazioni.

Per rendere Synology NAS rilevabile mediante il servizio Bonjour:

Il servizio Bonjour consente agli utenti di individuare Synology NAS ed eseguire attività di stampa e di backup dal computer Mac.

  1. Nella sezione Bonjour, spuntare la casella di controllo Abilita rilevamento servizio Bonjour.
  2. Selezionare le opzioni necessarie associate a Bonjour:
    • Abilita Bonjour Printer Broadcast: Selezionare per consentire ai computer Mac di individuare la stampante USB connessa a Synology NAS per la stampa di documenti.
    • Abilitare la trasmissione Bonjour Time Machine tramite SMB: Selezionare per consentire a Mac Time Machine di individuare Synology NAS ed eseguire il backup di file nella cartella condivisa DSM tramite SMB.
    • Abilitare la trasmissione Bonjour Time Machine tramite AFP: Selezionare per consentire a Mac Time Machine di individuare Synology NAS ed eseguire il backup di file nella cartella condivisa DSM tramite AFP.
  3. Se è stata selezionare un'opzione associata a Time Machine, fare clic su Imposta cartelle Time Machine per specificare le cartelle di destinazione per i backup di Time Machine.
  4. Fare clic su Applica per salvare le impostazioni.

Nota:

  • se si disabilita il servizio SMB/AFP (su Pannello di controllo > Servizi file > SMB/AFP/NFS), l'opzione Bonjour associata sarà disabilitata automaticamente.
  • Quando l'opzione Abilitare la trasmissione Bonjour Time Machine tramite SMB è selezionata, saranno abilitate le opzioni seguenti (su Pannello di controllo > Servizi file > SMB/AFP/NFS > Impostazioni avanzate):
    • Abilitare SMB 3 (per Protocollo max per il servizio file Windows)
    • Opportunistic Locking
    • Abilita versione SMB2
    • Abilita gestioni SMB durevoli (il blocco file protocollo trasversale sarà disabilitato)

Per rendere Synology NAS rilevabile mediante SSDP:

Il servizio SSDP consente agli utenti di individuare Synology NAS tramite il rilevamento di rete Windows.

  1. Nella sezione SSDP, spuntare la casella di controllo Abilitare il rilevamento di rete Windows per consentire l'accesso tramite browser Web.
  2. Fare clic su Applica per salvare le impostazioni.

Per rendere Synology NAS rilevabile mediante WS-Discovery:

Il servizio WS-Discovery consente agli utenti di individuare il servizio SMB in Synology NAS tramite il rilevamento di rete Windows.

  1. Nella sezione WS-Discovery, spuntare la casella di controllo Abilitare il rilevamento di rete Windows per consentire al file di accedere tramite SMB.
  2. Fare clic su Applica per salvare le impostazioni.
