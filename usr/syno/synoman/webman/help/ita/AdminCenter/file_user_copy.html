Copia utente

Copiare le impostazioni di un utente esistente su un nuovo utente. In questo modo sarà possibile creare rapidamente un nuovo utente senza dovere configurare di nuovo le stesse impostazioni.

  1. Seleziona l'utente da copiare.
  2. Fare clic su Crea > Copia utente. Verrà lanciata la finestra della Procedura guidata di creazione utente.
  3. Le impostazioni dell'utente selezionato saranno applicate al nuovo utente per impostazione predefinita (escluse informazioni personali quali nome utente ed e-mail).
