Utente

Con le opzioni gestione utente flessibili, è possibile creare account utente per i singoli membri della propria famiglia o attività. La creazione di utenti consente di definire e gestire permessi per ogni persona, come permessi accesso cartella condivisa, quote archiviazione/cartella condivisa o limitazioni di banda.

    Nota:
  1. la gestione della cartella condivisa è applicabile solo alle piatteforme Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake, Denverton. Per verificare se il proprio modello supporta questa funzione, consultare questo articolo.
  2. La gestione della quota di cartelle condivise/archiviazione non è applicabile a Docker DSM.
