Dominio/LDAP

Synology NAS può unirsi a un servizio di directory esistente, ad esempio un dominio Windows o un client LDAP. È possibile gestire i privilegi di accesso dell'utente del dominio o LDAP alle cartelle condivise e alle applicazioni DSM e abilitarne il servizio sede utente. Andare su Pannello di controllo > Dominio/LDAP per unire Synology NAS a un servizio directory.

Nota:

  • la funzione di dominio e LDAP non è disponibile sui prodotti EDS14 e Synology NVR.
