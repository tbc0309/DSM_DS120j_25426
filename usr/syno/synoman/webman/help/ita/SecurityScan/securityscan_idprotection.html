Analisi accesso

L'analisi accesso consente agli amministratori di sistema di controllare le informazioni di sicurezza sul sistema e gli account correnti.

È possibile disabilitare l'analisi di accesso o modificare le regole su Security Advisor > Avanzate > Personalizza checklist.

Nuove attività di accesso

Security Advisor analizza le informazioni di accesso di ciascun utente, compresi HTTP user-agent, indirizzo IP e posizione geografica. Quando viene rilevata una nuova attività di accesso, Security Advisor invierà una notifica all'amministratore di sistema e all'utente in questione.

Dopo avere confermato che uno specifico record di accesso è anomalo, si consiglia di utilizzare una password più forte, abilitare la verifica in due passaggi o anche limitare gli IP consentiti per migliorare la sicurezza dell'account.

Rilevamento della password (attacchi forzati)

Security Advisor analizza i tentativi di accesso non riusciti su diversi IP e protocolli. Quando qualcuno cerca di accedere provando a inserire delle password, ma fallisce più volte, Security Advisor invierà una notifica all'amministratore di sistema e all'utente in questione. L'amministratore di sistema può migliorare la sicurezza dell'account abilitando Blocco automatico o Protezione account (su Pannello di controllo > Sicurezza > Account).
