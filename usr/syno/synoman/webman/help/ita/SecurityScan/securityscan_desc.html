Security Advisor

Security Advisor è un'applicazione per la sicurezza DSM che scansiona le impostazioni di DSM e Synology NAS. Security Advisor consente di controllare le impostazioni e suggerisce le modifiche che contribuiscono alla sicurezza di Synology NAS. Di seguito è riportato un elenco delle funzioni offerte:

  • Security Advisor eseguirà diversi controllo per rilevare le impostazioni di DSM che potrebbero presentare rischi per la sicurezza e qualsiasi attività sospetta.
  • Security Advisor analizzerà le attività di accesso anomale e i tentativi di inserimento password da parte di utenti terzi (attacchi forzati).
  • Security Advisor fornisce risultati dettagliati, con report che spiegano le azioni consigliate da intraprendere e i vantaggi associati.
  • Il database delle definizioni di sicurezza interna viene aggiornato automaticamente, restando aggiornato sulle nuove minacce cibernetiche.
  • Security Advisor genera report mensili o giornalieri per fornire le azioni consigliate.

Per lanciare Segnalatore sicurezza procedere come segue:

  • Andare su Menu principale > Security Advisor.
