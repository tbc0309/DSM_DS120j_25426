Avanzate

È possibile visualizzare e modificare le impostazioni avanzate per il Segnalatore sicurezza.

Base della sicurezza

Scegliere le regole di sicurezza che saranno verificate da Security Advisor selezionando una Base della sicurezza. Il numero e i tipi di regole variano per ogni Base della sicurezza. Scegliere in base alle specifiche esigenze. Synology NAS è utilizzato per lavoro e attività business oppure in casa per uso personale? È possibile scegliere dalle seguenti opzioni:

  • Per uso domestico e personale: Security Advisor eseguirà dei controlli essenziali per proteggere Synology NAS da minacce cibernetiche comuni.
  • Per lavoro e azienda: Security Advisor eseguirà dei controlli completi per soddisfare i requisiti di conformità di sicurezza per le organizzazioni.
  • Personalizzato: selezionare le regole di sicurezza che dovranno essere controllate dal Segnalatore sicurezza.

Programmazione scansione

È possibile definire una programmazione regolare della scansione facendo clic su Abilita programmazione scansione regolare e scegliere l'ora in cui il Segnalatore sicurezza dovrà eseguire la scansione. Segnalatore sicurezza controllerà le regole in base alla Base della sicurezza scelta.

Impostazioni report

È possibile configurare le impostazioni di report per ricevere report da Security Advisor. Selezionare una cartella condivisa come posizione di archiviazione per i report. Quindi selezionare Abilita report mensile o Abilita report giornaliero. Per ricevere i report per e-mail, andare su Pannello di Controllo > Notifica > E-mail e abilitare prima le notifiche e-mail-
