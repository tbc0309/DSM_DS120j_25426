Opzioni

Per visualizzare il menu Opzioni fare clic sull'icona a forma di persona nell'angolo in alto a destra del desktop DSM. Questo menu consente di:

  • Modificare le opzioni dell'account personale
  • Disconnettersi
  • Arrestare il sistema (su EDS14) o spegnere il sistema (in tutti gli altri modelli)
  • Riavviare il sistema

Dopo aver aperto il menu, fare clic su Personale per modificare le impostazioni dell'account personale.
