Grupo de almacenamiento

Se pueden agregar una o más unidades a una única unidad de almacenamiento denominada grupo de almacenamiento, que puede estar protegido por RAID. Cada tipo de RAID ofrece un nivel de protección de datos diferente. Para obtener una lista general de los tipos de RAID compatibles con Synology NAS, consulte la sección General. Después de crear un grupo de almacenamiento, puede crear uno o varios volúmenes dentro del mismo. Un volumen es una unidad de almacenamiento de datos y se puede ampliar si el grupo de almacenamiento asociado contiene espacio asignable. Se debe crear al menos un volumen para poder empezar a almacenar datos en su Synology NAS.

Observación:

  • Recomendamos el uso de unidades incluidas en la Lista de compatibilidad con productos de Synology para evitar problemas. El uso de unidades incompatibles puede reducir el rendimiento del sistema e incluso provocar la pérdida de datos.
  • Un grupo de almacenamiento se debe componer de unidades del mismo tipo. Es decir, no se puede crear un grupo de almacenamiento con una combinación de unidades SATA y SAS.
  • Para los modelos de Synology NAS compatibles con RAID Group, un grupo de almacenamiento RAID 5, RAID 6 o RAID F1 compatible con múltiples volúmenes puede contener varias matrices RAID. Cada una de las matrices RAID que pertenezca al mismo grupo de almacenamiento será del mismo tipo (p. ej., RAID 5, RAID 6 o RAID F1). El número máximo de unidades para cada matriz RAID es 6, 12 o 24. Puede fijar el número máximo de unidades mientras crea el grupo de almacenamiento y el número no podrá modificarse posteriormente. Cuando se asigna un número máximo de unidades a una matriz RAID, la siguiente unidad que se añada al grupo de almacenamiento puede asignarse a una nueva matriz RAID. La nueva matriz RAID debe contener el número mínimo de unidades necesarias para ese tipo de RAID concreto (p. ej., tres unidades para RAID 5 y RAID F1, y cuatro unidades para RAID 6).
  • Si desea utilizar una unidad determinada para crear un grupo de almacenamiento, pero la unidad no aparece en el Asistente para la creación de un grupo de almacenamiento, vaya a la página HDD/SSD para comprobar el estado de la unidad.
