Depuración de datos

Realizar la depuración de datos

La depuración de datos es una función de mantenimiento de datos que inspecciona los grupos de almacenamiento. Los siguientes dos tipos de Depuración de datos se ejecutarán de forma secuencial:

  • Depuración del sistema de archivos: esta función comprobará los volúmenes del sistema de archivos Btrfs. Si se detectan datos incoherentes con la suma de comprobación, el sistema intentará utilizar una copia de seguridad para reparar los datos y la ruta del archivo se registrará en el Centro de registros.
  • Depuración de RAID: esta función puede usarse con grupos de almacenamiento que implementen SHR (compuestos por tres o más unidades), RAID 5 o RAID 6. Puede reparar cualquier inconsistencia detectada en los datos.

Le recomendamos que realice regularmente la depuración de datos para garantizar la consistencia de estos y evitar la pérdida de datos importantes en caso de que una unidad falle.

Observación:

  • La depuración de datos y las funciones relacionadas que se describen en este artículo solo están disponibles en modelos específicos.
  • La Depuración de datos solo se puede realizar cuando el estado de los grupos de almacenamiento es Averiado o Degradado.
  • Si la suma de comprobación para la integridad de datos avanzada no está habilitada para una carpeta compartida, la depuración del sistema de archivos no podrá comprobar y reparar los datos de la carpeta debido a la falta de sumas de comprobación.

Configuración del programa de depuración de datos

Puede establecer un programa de depuración de datos. El sistema ejecutará la depuración de datos en función de la configuración.

Para establecer un programa de depuración de datos:

  1. Inicie el Administrador de almacenamiento.
  2. Vaya a Grupo de almacenamiento > Depuración de datos.
  3. Haga clic en Establecer el programa.
  4. Marque Habilitar el programa de depuración de datos.
  5. Especifique la Frecuencia, la Fecha de inicio y la Sesión de tiempo.
  6. Vaya a la pestaña Seleccionar objetivo.
  7. Seleccione el grupo de almacenamiento en el que desea realizar la depuración de datos.
  8. Haga clic en OK.

Observación:

  • Para que se pueda agregar a un programa, el grupo de almacenamiento debe estar en estado Normal y admitir la depuración de datos.
  • Si su sistema se actualiza de la versión 6.1, los grupos de almacenamiento que permitan la depuración de datos se agregarán de forma predeterminada al programa.
  • Si ha establecido el programa en la versión 6.1, habilitaremos el programa automáticamente según la configuración anterior.

Para agregar un grupo de almacenamiento a un programa o eliminar un grupo de almacenamiento de un programa:

  1. Haga clic en Establecer el programa
  2. Vaya a la pestaña Seleccionar objetivo.
  3. Marque o desmarque la casilla de verificación de un grupo de almacenamiento para decidir si desea agregarlo o quitarlo de un programa.
  4. Haga clic en OK.

Observación:

  • Un grupo de almacenamiento no se puede agregar a ni quitar de un programa cuando la depuración de datos se está ejecutando manualmente en ese grupo de almacenamiento.
  • Al eliminar un grupo de almacenamiento en el que se está ejecutando una depuración de datos programada, el progreso de dicho grupo de almacenamiento se cancelará.

Para modificar la configuración del programa:

  1. Haga clic en Establecer el programa.
  2. Modifique la frecuencia, la fecha de inicio y la sesión de tiempo en función de sus necesidades.
  3. Haga clic en OK.

Observación:

  • Modificar la Fecha de inicio restablecerá el progreso del programa y la depuración de datos se reiniciará en función de la nueva fecha de inicio.

Para pausar un programa:

  1. Seleccione un grupo de almacenamiento en el que se esté realizando la depuración de datos.
  2. Haga clic en Pausar programa, en el menú desplegable Acción.

Observación:

  • Haga clic en Continuar programa en el menú desplegable Acción para reanudar el programa de depuración de datos. En caso contrario, el programa permanecerá en pausa.

Ejecución manual de la depuración de datos:

Si el sistema detecta anomalías en el sistema, puede intentar reparar los datos ejecutando la depuración de datos manualmente.

Para realizar la Depuración de datos manualmente:

  1. Abra Administrador de almacenamiento.
  2. Vaya a Grupo de almacenamiento > Depuración de datos.
  3. Seleccione el grupo de almacenamiento en el que desea realizar la depuración de datos.
  4. Haga clic en Ejecución manual en el menú desplegable Acción.

Observación:

  • Si el sistema se apaga antes de que se complete la depuración de datos, se pueden perder algunos progresos y deberá reiniciar manualmente la Depuración de datos.
  • Si la hora en que se realiza manualmente la depuración de datos se superpone con la depuración de datos programada, esta fase del programa no se completará hasta que finalice la depuración de datos manual.
  • La depuración de datos puede afectar al rendimiento del sistema. Le sugerimos que no ejecute la depuración de datos en varios grupos de almacenamiento a menos que sea necesario.

Para pausar un programa en un grupo de almacenamiento en el que se está ejecutando una depuración de datos manual:

  1. Seleccione un grupo de almacenamiento que desee poner en pausa.
  2. Haga clic en Pausar en el menú desplegable Acción.
