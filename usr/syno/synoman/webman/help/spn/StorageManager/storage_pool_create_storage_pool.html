Crear un nuevo grupo de almacenamiento

Importante:

  • Antes de comenzar, asegúrese de que no hay datos importantes en las unidades en las que se va a crear el grupo de almacenamiento. Todos los datos existentes se eliminarán durante el proceso de creación.
  • Vaya a la página HDD/SSD y asegúrese de que el estado de cada unidad es Iniciado o No iniciado.
  • Recomendamos el uso de unidades incluidas en la lista de compatibilidad de productos de Synology para evitar problemas. El uso de unidades incompatibles puede reducir el rendimiento del sistema o incluso provocar la pérdida de datos en su Synology NAS.

Para crear un grupo de almacenamiento:

  1. Abra Administrador de almacenamiento.
  2. Vaya a Grupo de almacenamiento y haga clic en el botón Crear.
  3. Seleccione el tipo de grupo de almacenamiento:
    • Mejor rendimiento: este tipo de grupo de almacenamiento solo admite un volumen, pero ofrece un mejor rendimiento.
    • Mayor flexibilidad: este tipo de grupo de almacenamiento admite varios volúmenes, Synology Hybrid RAID (SHR) y la función RAID Group.
  4. Configure las propiedades del grupo de almacenamiento:
    • Descripción del grupo de almacenamiento: descripción de este grupo de almacenamiento.
    • Tipo de RAID: cada tipo de RAID ofrece un nivel de protección de datos diferente. Para obtener más información, consulte General.
    • Número mínimo de unidades por RAID: el número mínimo de unidades necesario para el tipo de RAID seleccionado.
    • Número máximo de unidades por RAID: un mayor número de unidades por RAID aumenta la probabilidad de error del RAID. Le recomendamos que ajuste el límite de unidades en un único RAID y utilice RAID Group para establecer un equilibrio entre la seguridad de los datos y la capacidad del grupo de almacenamiento.
    • Función de RAID Group: si la función RAID Group es compatible con el tipo de RAID seleccionado. Mientras que un solo RAID tiene un límite de unidades, la función de RAID Group permite que su bloque de almacenamiento se componga de varios RAID, aumentando su capacidad y redundancia de la unidad. RAID Group es compatible con RAID 5, RAID 6 y RAID F1.
  5. Siga los pasos restantes del asistente para finalizar la creación del grupo de almacenamiento.

Observación:

  • SHR (Synology Hybrid RAID) está disponible solo en modelos específicos. Para obtener más información, visite el sitio web de Synology.
  • Las funciones Número máximo de unidades por RAID y RAID Group solo están disponibles en modelos específicos. Para obtener más información, visite el sitio web de Synology.
  • El número máximo de unidades por RAID que se pueden cambiar solo si el tipo de RAID seleccionado es compatible con la función RAID Group.
  • Si el tipo de RAID del grupo de almacenamiento es RAID 5, RAID 6, RAID F1, SHR (más de tres unidades) o SHR-2, el sistema ejecutará la resincronización RAID en el grupo de almacenamiento para mejorar la velocidad de acceso y la resistencia de los datos.
