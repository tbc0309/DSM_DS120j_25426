Usuarios conectados

En esta pestaña puede visualizar y administrar los clientes que están conectados al sistema y están accediendo a los recursos en estos momentos. Esta sección muestra los clientes conectados a través de HTTP, TELNET, SSH, SMB/CIFS, AFP, FTP y NFS.

Para administrar las conexiones actuales, deberá realizar una de las siguientes opciones:

  • Haga clic en Guardar para exportar la lista actual a un archivo HTML.
  • Haga clic en Actualizar para actualizar la lista.
  • Haga clic en Desconectar conexión para no permitir que el usuario acceda al recurso. El usuario desconectado no vuelve a ser autorizado para tener acceso al recurso.
  • Haga clic en Deshabilitar usuario para deshabilitar el usuario. Para volver a habilitarlo, vaya a Menú principal > Panel de control > Usuario.

Nota:

  • La opción Desconectar conexión no funciona con clientes NFS.
  • Si los clientes NFS no desmontan carpetas compartidas correctamente, es posible que todavía aparezcan en la lista de usuarios conectados.
  • Si Synology NAS se reinicia, los clientes NFS que estaban conectados previamente no aparecerán en la lista de usuarios conectados.
