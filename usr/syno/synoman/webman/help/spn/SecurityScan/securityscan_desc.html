Consejero de seguridad

El Consejero de seguridad es una aplicación que examina las configuraciones de DSM y Synology NAS. Comprobará sus configuraciones y recomendará cambios que le ayudarán a mantener la seguridad de Synology NAS. A continuación se presenta una lista de funciones que ofrece:

  • El Consejero de seguridad llevará a cabo diferentes comprobaciones para detectar cualquier configuración de DSM que pueda suponer un riesgo de seguridad, así como cualquier actividad sospechosa.
  • El Consejero de seguridad analizará las actividades de inicio de sesión anómalas e intentos de adivinación de contraseña por parte de un tercero malicioso (ataques de fuerza bruta).
  • El Consejero de seguridad ofrece resultados detallados y proporciona informes que explican las acciones recomendadas y las ventajas que estas suponen.
  • La base de datos de definiciones de seguridad internas se actualiza automáticamente y se mantiene al día de los riesgos cibernéticos cambiantes.
  • El Consejero de seguridad genera informes diarios o mensuales para recomendar acciones.

Para abrir el Consejero de seguridad haga lo siguiente:

  • Vaya a Menú principal > Consejero de seguridad.
