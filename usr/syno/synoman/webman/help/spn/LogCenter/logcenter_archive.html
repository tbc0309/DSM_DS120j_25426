Configuración de archivo

La página de Configuración de archivo ofrece opciones para configurar cómo y dónde se archivan los registros recibidos por el Centro de registros. Utilice las opciones siguientes para especificar la ubicación de almacenamiento y las reglas para crear archivos de registros automáticamente.

Destino de almacenamiento para archivos

El destino se refiere a la carpeta donde se almacenan y archivan los registros recibidos por el Centro de registros. Haga clic en Seleccionar ubicación para elegir la carpeta.

Si desea que los registros locales de Synology NAS se archiven en el mismo destino, marque la casilla designada Archivar registros locales en la ubicación de almacenamiento especificada anteriormente. En la mayoría de casos, esto proporciona más espacio de almacenamiento para los registros locales, lo que le permite mantener un sistema de registros más completo.

Observación:

Los registros locales (registros de sistema, de conexión y de transferencia de archivos) se guardan en una partición del sistema que tiene un almacenamiento limitado. Cuando se habilita Archivar los registros locales a la ubicación de almacenamiento indicada anteriormente, todos los registros se archivan en el destino indicado de acuerdo con las Reglas de archivo. Para examinar los registros locales, haga clic en Abrir archivo de registros y seleccione SYNOSYSLOGDB__LOCALARCH.DB.

Reglas de archivo

Los registros se archivan automáticamente según los criterios definidos en la sección de Reglas de archivo. Los registros se pueden archivar automáticamente cuando el tamaño de la base de datos de registros alcance un tamaño especificado, el número de registros individuales alcance un número específico o los registros alcancen una antigüedad específica.

Si no habilitó la función del archivo de registro, el registro anterior se eliminará cuando el registro acumulado exceda el número de registro máximo que se puede conservar de la base de datos del registro. Consulte la tabla siguiente para ver los números de registro máximos que se pueden conservar de cada protocolo.

Protocolo Registro máximo que se puede conservar
AFP 20.000
File Station 20.000
FTP 20.000
SMB 100.000
WebDAV 20.000

Formato de archivo

Esta sección incluye las siguientes opciones.

  • Archivar registros con formato de texto además del formato predeterminado: Guarda archivos como archivos de solo texto, además de los archivos SQLite predeterminados.
  • Comprimir registros de archivos: Guarda archivos de registros como archivos comprimidos en formato ZIP.
  • Archivar registros de manera separada según el dispositivo: Guarda archivos separados para dispositivos cliente individuales que envían registros a Synology NAS.
