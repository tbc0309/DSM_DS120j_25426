Instantánea

Para tomar una instantánea de LUN:

Una instantánea de un LUN es una copia puntual de solo lectura de un LUN.

  1. Seleccione el LUN del que desea sacar instantáneas.
  2. Haga clic en Instantánea y elija Tomar una instantánea.
  3. Introduzca una descripción para esta instantánea.
  4. Elija uno de los siguientes tipos de instantáneas de acuerdo con sus necesidades:
    • Coherente con la aplicación: este tipo de instantánea solo se puede tomar cuando se ha instalado Synology Storage Console o Synology Snapshot Manager en su servidor Windows o VMware vCenter Server, según su entorno operativo para comunicarse con DSM. Cuando se activan instantáneas coherentes con aplicaciones en DSM, las aplicaciones recibirán una notificación y pasarán todos los datos de la memoria al LUN en el que se toma la instantánea para garantizar la coherencia de los mismos.
    • Coherente frente a bloqueos: si no se ha instalado Storage Console o Snapshot Manager en ninguno de sus clientes, solo podrá capturar instantáneas consistentes con averías de los LUN. Cuando se activan, las aplicaciones no recibirán notificaciones para pasar los datos de la memoria al LUN y por tanto no se garantizará la consistencia de los datos.
  5. Haga clic en OK.

Observación:

  • Se recomienda instalar Synology Storage Console o Snapshot Manager para evitar posibles inconsistencias en los datos.

Terminología:

  • Synology Storage Console: Synology Storage Console es un complemento de gestión de almacenamiento en VMware vSphere Client o Windows Server que simplifica la gestión y la supervisión de varios sistemas de almacenamiento de Synology. También le permite tomar instantáneas de aplicaciones consistentes directamente desde DSM.
    • Para entornos VMware, el complemento solo es compatible con VMware vCenter Server Appliance 6.5 U2 o versiones posteriores.
    • Puede descargar el complemento en el Centro de descargas; consulte Synology Storage Console for VMware y Synology Storage Console for Windows para obtener más información.
  • Synology Snapshot Manager: Synology Snapshot Manager es un complemento de software que permite tomar instantáneas de aplicaciones consistentes directamente desde DSM.
    • Para entornos VMware, este complemento solo es compatible con VMware vCenter Server 6.0 o versiones anteriores.

Para eliminar una instantánea:

  1. Seleccione el LUN del que ha tomado instantáneas.
  2. Haga clic en Instantánea > Lista de instantáneas.
  3. Seleccione la instantánea que desea eliminar y haga clic en Eliminar.

Observación:

  • La recuperación de espacio tarda tiempo en procesarse después de eliminar las instantáneas.

Para restaurar un LUN:

Seleccione una instantánea puntual para restaurar los datos de un LUN.

  1. Seleccione el LUN que desea restaurar.
  2. Haga clic en Instantánea y elija Lista de instantáneas.
  3. Seleccione la instantánea que desea restaurar y haga clic en Restaurar.

Para clonar un LUN a partir de instantáneas:

realice una copia grabable del LUN o la instantánea del LUN.

  • Seleccione una instantánea de LUN de Lista de instantáneas y haga clic en Clonar.

Observación:

  • Las funciones de instantánea, restauración y clonación de LUN solo están disponibles en LUN con comandos de aceleración de almacenamiento.
  • Si ha tomado una instantánea o ha creado un clon, y el estado ha pasado a ser Anormal/Averiado, elimine dicha instantánea/clon y cree otra/o.
  • Todos los cambios realizados en un LUN hasta después de crear una instantánea se perderán si restaura el LUN en esta instantánea.
  • Las funciones para restaurar los LUN, clonar los LUN y tomar y clonar instantáneas de LUN no están disponibles cuando un volumen tiene menos de 1 GB de espacio libre.

Política de retención

Esta opción proporciona una solución ideal para ayudarle a especificar las cantidades máximas de versiones de instantáneas y ahorrar espacio en el volumen, pero es posible que necesite retener las instantáneas durante periodos de tiempo más largos. Debe tener en cuenta algunas cuestiones acerca de la política de retención:

  • Synology utiliza la política de retención GFS (abuelo-padre-hijo). Puede configurar las cantidades máximas de versiones de instantáneas que se pueden retener para los siguientes rangos de tiempo respectivamente: horario, diario, semanal, mensual y anual.
  • Una instantánea puede cumplir varias reglas de retención al mismo tiempo. Por ejemplo, una instantánea puede cumplir las reglas diarias, semanales y mensuales.
  • Las instantáneas más antiguas se conservarán si toma muchas instantáneas dentro de un periodo determinado. Por ejemplo, si establece una regla de retención de 10 instantáneas semanales y su programación de instantáneas está establecida para tomar 1 instantánea al día, Snapshot Replication conservará la instantánea más antigua de cada semana durante las últimas 10 semanas.
  • Si no ha tomado instantáneas durante un periodo determinado, el sistema conservará las instantáneas más recientes. Por ejemplo, si la regla de instantáneas diarias está establecida en 15 y no se han tomado instantáneas durante los últimos 30 días, el sistema conservará las instantáneas tomadas hace entre 30 y 45 días.
  • Todas las versiones de instantáneas tomadas en el periodo de una hora antes de la instantánea más reciente se conservarán de manera predeterminada, para que le resulte más fácil encontrar y restaurar desde versiones recientes de instantáneas. También puede bloquear una instantánea para evitar que la eliminen automáticamente las reglas de retención.

Ejemplo 1:
Si desea conservar instantáneas diarias de los últimos 3 meses, instantáneas semanales de los últimos 4-6 meses e instantáneas mensuales de los últimos 7-9 meses, debe introducir 0 para las instantáneas cada hora, 90 para las instantáneas diarias, 24 para las instantáneas semanales, 9 para las instantáneas mensuales y 0 para las instantáneas anuales.

Ejemplo 2:
Si desea conservar instantáneas cada hora de los últimos 2 días, instantáneas diarias de la última semana, instantáneas semanales del último mes e instantáneas mensuales del último año, debe introducir 48 para las instantáneas cada hora, 7 para las instantáneas diarias, 4 para las instantáneas semanales, 12 para las instantáneas mensuales y 0 para las instantáneas anuales.
