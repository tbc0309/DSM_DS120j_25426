Avanzado

Puede modificar los siguientes parámetros de seguridad en Panel de control > Seguridad > Avanzado para sesiones de exploración de DSM:

Compresión HTTP

Puede habilitar Compresión HTTP para Synology NAS. Al reducir el ancho de banda del tráfico de red, puede aumentar la velocidad de carga de las páginas web.

Para habilitar/deshabilitar la compresión HTTP:

  • Marque/desmarque Compresión HTTP en Panel de control > Seguridad > Avanzado.

Nivel del perfil TLS/SSL

En el perfil TLS/SSL, puede elegir el nivel de seguridad de las conexiones HTTPS cifradas. Ofrece los tres niveles siguientes:

  • Compatibilidad con versiones actuales: solo compatible con los dispositivos móviles y exploradores más recientes.
  • Compatibilidad intermedia: entre la compatibilidad con versiones actuales y la compatibilidad con versiones anteriores. Este es el valor predeterminado del sistema.
  • Compatibilidad con versiones anteriores: esta opción no es segura, pero es compatible con la mayoría de exploradores y dispositivos móviles.

Para configurar el nivel del perfil TLS/SSL para todos los servicios:

Puede seleccionar un nivel del perfil TLS/SSL en la página principal, el cual se aplicará a todos los servicios que usen el Perfil predeterminado.

  1. Seleccione un nivel adecuado para el perfil TLS/SSL.
  2. Haga clic en OK.

Para configurar el nivel del perfil TLS/SSL para un servicio específico:

Puede cambiar el nivel del perfil para un servicio en función de los requisitos de seguridad de este.

  1. Haga clic en Configuración personalizada para ver todos los servicios y su nivel del perfil correspondiente.
  2. Haga clic en el nivel del perfil actual del servicio deseado.
  3. Seleccione el nivel del perfil adecuado en el menú desplegable.
  4. Haga clic en OK.

Observación:

  • El nivel de perfil predeterminado del sistema se aplicará a la conexión que no se encuentra en la lista de servicios.
