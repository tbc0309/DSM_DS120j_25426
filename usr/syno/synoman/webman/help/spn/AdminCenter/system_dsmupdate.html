Actualización de DSM

De manera periódica, Synology publica actualizaciones gratuitas para DSM. Las actualizaciones pueden incluir características nuevas, mejoras a funciones y mejoras de rendimiento. Las actualizaciones de DSM pueden presentarse en forma de actualizaciones secundarias o principales. el sistema muestra un mensaje junto a Estado si la actualización es secundaria. Cuando existan versiones de DSM principales, habrá una sección en Estado en la que se proporcionará información sobre esta versión, además de un vínculo de descarga a la versión de actualización principal. Normalmente, las actualizaciones secundarias incluyen correcciones de errores críticos y parches de seguridad para la estabilidad del sistema. Por su parte, las actualizaciones principales incluyen sobre todo funciones nuevas y mejoras de rendimiento, junto con correcciones de errores críticos y parches de seguridad.

Observación:

  • La actualización de DSM no afecta a los datos de Synology NAS. Sin embargo, para mayor seguridad de los datos, se recomienda encarecidamente hacer una copia de seguridad de los datos antes de empezar.
  • Después de actualizar a nuevas versiones de DSM, no podrá volver a las versiones anteriores de DSM.
  • Synology NAS ejecuta automáticamente una lista de comprobación para determinar si las nuevas versiones son relevantes para su sistema en función de los servicios y las configuraciones del sistema. Por lo tanto, las actualizaciones disponibles y las versiones más recientes pueden variar en función de sus configuraciones actuales de DSM.

Realizar la actualización manual de DSM

Puede descargar el archivo de actualización .pat desde el Centro de descargas de Synology y realizar la actualización manual de DSM.

Para realizar una actualización manual de DSM:

  1. Visite el Centro de descargas para descargar la nueva actualización de DSM en su equipo local. No es necesario extraer el archivo .pat.
  2. Haga clic en Actualización DSM manual para acceder a la página Actualización DSM manual
  3. Haga clic en Examinar para seleccionar el archivo de actualización.
  4. Haga clic en Aplicar para comenzar a actualizar DSM. El sistema se reiniciará de forma automática cuando finalice la actualización.

Resolución de problemas:

Si no puede actualizar DSM, las siguientes sugerencias pueden ayudarle a solucionar el problema:

  • Compruebe si el proceso de actualización ha sido interrumpido por los usuarios, o si se han producido otros eventos inesperados, como un corte eléctrico.
  • Compruebe la versión de DSM. Sólo puede actualizar el sistema con una versión más reciente de DSM.
  • Descargue de nuevo la actualización de DSM, pues es posible que el archivo de actualización esté dañado.

Si continúa sin resolver el problema, póngase en contacto con el Soporte técnico de Synology.

Configuración de las preferencias de actualización e instalación automáticas de DSM

Para ajustar la configuración de instalación de actualizaciones y comprobar el programa, haga clic en Configuración de actualizaciones.

Para que el sistema le notifique si hay una nueva actualización de DSM disponible:

La opción Notificarme y dejarme decidir si se instala la nueva actualización permite al sistema notificarle únicamente cuando hay una nueva versión disponible. Puede optar por descargar la actualización después de recibir la notificación.

Se mostrará una notificación en el escritorio cuando la actualización de DSM se haya descargado y esté lista para la instalación. Para evitar que DSM se actualice automáticamente, puede cancelar esta opción en las notificaciones de escritorio. Además, las actualizaciones de DSM se instalarán antes de los apagados programados si ha habilitado Apagar en Programa de energía.

Para instalar automáticamente la nueva actualización de DSM:

La opción Instalar automáticamente la nueva actualización permite que el sistema instale automáticamente la nueva actualización de DSM si el resultado de la comprobación determina que hay una nueva actualización disponible. Esto se aplica únicamente a las actualizaciones secundarias que se muestran en Estado y no a las actualizaciones principales.

Para configurar un programa de comprobación:

La opción Comprobar programa le permite decidir cuándo debe comprobar el sistema las actualizaciones disponibles. Seleccione los días de la semana y la hora del día de los días especificados para buscar actualizaciones. Por ejemplo, de martes a viernes, a las 11:00 h.

Actualización principal

Cuando haya una versión de DSM principal disponible, esta información se mostrará en la parte inferior de la sección Actualización de DSM. Podrá descargar la versión de actualización principal haciendo clic en Descargar. Cuando finalice la descarga, haga clic en Actualizar ahora para completar el proceso.

Resolución de problemas:

Si ve mensajes de error en texto rojo durante la comprobación o descarga de la nueva versión de DSM, compruebe lo siguiente:

  • Compruebe que Synology NAS tiene conexión a Internet.
  • Es posible que el servidor de descargas de Synology esté ocupado en este momento. Vuelva a intentar la descarga más adelante.

Si continúa sin resolver el problema, póngase en contacto con el Soporte técnico de Synology.

Actualización de firmware de las unidades de expansión

Si tiene unidades de expansión SAS, puede que sea necesario actualizar el firmware de vez en cuando. El sistema comprobará automáticamente si hay actualizaciones de firmware y mostrará uno de estos estados: Es necesario actualizar el firmware de inmediato., El firmware de todas las unidades de expansión está actualizado o Ninguna unidad de expansión conectada.

  • Es necesario actualizar el firmware de inmediato: cuando se muestre este estado, recomendamos encarecidamente actualizar el firmware de la unidad de expansión lo antes posible. Sus unidades de expansión dejarán de estar disponibles hasta que se complete la actualización. Tampoco podrá crear nuevos volúmenes en las unidades de expansión durante este tiempo.
  • El firmware de todas las unidades de expansión está actualizado: este estado indica que su sistema ha completado la actualización y está funcionando en un entorno estable.
  • Ninguna unidad de expansión conectada: Este estado indica que su Synology NAS no tiene unidades de expansión conectadas. Por lo tanto, no es necesario realizar ninguna actualización de firmware.

Importante: Normalmente, la actualización del firmware de la unidad de expansión no afecta a los datos de Synology NAS. Sin embargo, para mayor seguridad de los datos, se recomienda encarecidamente hacer una copia de seguridad de los datos antes de empezar.

Observación:

  • Solo es necesario actualizar el firmware de las unidades de expansión en modelos de SAS específicos.
  • La actualización de firmware de las unidades de expansión solo está disponible en DSM 6.2.2 o superiores.

Para actualizar el firmware de las unidades de expansión:

La actualización de firmware de las unidades de expansión solo tarda unos minutos y no requiere que se reinicie el sistema. El resto de servicios en ejecución no se interrumpirán durante la actualización.

  1. Haga clic en Actualizar ahora para iniciar la actualización.
  2. Aparecerá un cuadro de diálogo de advertencia. Seleccione cualquiera de las siguientes opciones:
    • Para continuar con la actualización, haga clic en Aceptar. Aparecerá una barra de progreso.
    • Para realizar la actualización más adelante, haga clic en Cancelar.
  3. El sistema mostrará una de las siguientes notificaciones de escritorio para informarle de si la descarga se ha realizado correctamente: Actualización de firmware de las unidades de expansión completada o Error al actualizar el firmware de las unidades de expansión.
  4. Se mostrará uno de los mensajes de estado siguientes en Estado:
    • El firmware de todas las unidades de expansión está actualizado: este mensaje se muestra cuando la actualización se completa correctamente. El sistema funciona en un entorno estable y las unidades de expansión están disponibles para su uso.
    • Error de actualización: si la actualización de firmware de las unidades de expansión falla, haga clic en Contactar con el servicio de asistencia para resolver el problema.

Observación:

  • Para garantizar la estabilidad del sistema, el servicio de las unidades de expansión deja de estar disponible cuando el sistema detecta que es necesario actualizar el firmware. El servicio volverá a estar disponible cuando se complete la actualización de firmware.
  • Si la actualización de firmware de las unidades de expansión falla, no intente reiniciar ni retirar las unidades o las unidades de expansión. Para resolver este problema, consulte este artículo o póngase en contacto con el departamento de soporte técnico de Synology para obtener más ayuda.
