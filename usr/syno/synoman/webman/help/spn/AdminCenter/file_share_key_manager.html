Gestionar las claves de las carpetas compartidas cifradas

Puede utilizar el administrador de claves para gestionar las claves de las carpetas compartidas y eliminar el cifrado de varias carpetas compartidas cifradas al mismo tiempo.

Explicación de los términos:

  • Almacén de claves: un almacén de claves es cualquier dispositivo externo o partición del sistema compatible con Synology NAS, que se utiliza como clave física para eliminar el cifrado de carpetas compartidas.
  • Cifrado: un cifrado es un método para codificar las claves de las carpetas compartidas. El administrador de claves ofrece dos tipos de cifrado:
    • Frase de contraseña: cualquiera que conozca la frase de contraseña puede eliminar el cifrado de las claves mediante este método.
    • Clave de equipo: tan solo el Synology NAS vinculado puede eliminar el cifrado de las claves cifradas mediante una clave de equipo.

Para iniciar un almacén de claves:

  1. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  2. Seleccione un dispositivo externo o una partición del sistema como almacén de claves en Ubicación del almacén de claves.
  3. Introduzca una frase de contraseña para este almacén de claves en el campo Frase de contraseña.

Sugerencias de seguridad:

  • Con fines de seguridad y gestión, es más seguro almacenar el archivo cifrado y la clave correspondiente en distintos dispositivos.
  • Le recomendamos que seleccione un dispositivo externo como Ubicación del almacén de claves para mejorar la seguridad de los datos.

Para agregar una nueva clave a un almacén de claves:

  1. Asegúrese de que ha iniciado un almacén de claves.
  2. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  3. Haga clic en Agregar.
  4. Seleccione una carpeta compartida cifrada.
  5. Seleccione el tipo de cifrado para la clave de cifrado. Puede elegir entre Frase de contraseña o Clave de equipo.
  6. Escriba o importe la clave de cifrado.
  7. Haga clic en OK para guardar la configuración.

Para eliminar manualmente el cifrado de varias carpetas compartidas cifradas:

  1. Asegúrese de que ha añadido claves al almacén de claves.
  2. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  3. Seleccione las carpetas compartidas cifradas de las que desea eliminar el cifrado.
  4. Haga clic en Montar.

Para eliminar el cifrado de varias carpetas compartidas cifradas automáticamente al arrancar:

  1. Asegúrese de que ha añadido claves al almacén de claves.
  2. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  3. Marque Montar al arrancar para las carpetas compartidas cifradas de las que quiere eliminar el cifrado al arrancar.
  4. Haga clic en OK para guardar la configuración.

Para expulsar un almacén de claves después del arranque:

  1. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  2. Haga clic en Configurar.
  3. Marque Expulsar dispositivo después del arranque.
  4. Haga clic en OK para guardar la configuración.

Para migrar claves de cifrado existentes a un almacén de claves para todas las carpetas compartidas montadas automáticamente:

Migrar claves durante la inicialización del almacén de claves:

  1. Al inicializar un almacén de claves, marque Migrar las claves de cifrado a este almacén de claves para todas las carpetas compartidas existentes montadas automáticamente (recomendado).

Migrar claves manualmente:

  1. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  2. Haga clic en Configurar.
  3. Haga clic en Migrar ahora.
  4. En el caso de claves duplicadas, puede elegir Sobrescribir claves duplicadas.
  5. Haga clic en OK para guardar la configuración.

Para clonar un almacén de claves en el modo High Availability:

  1. Vaya a Panel de control > Carpeta compartida > Acción > Administrador de claves.
  2. Haga clic en Clonar.
  3. Seleccione un almacén de claves de origen en el servidor activo desde Almacén de claves de origen.
  4. Seleccione un almacén de claves de destino en el servidor pasivo desde Almacén de claves de destino.
  5. Haga clic en Aplicar para guardar la configuración.
