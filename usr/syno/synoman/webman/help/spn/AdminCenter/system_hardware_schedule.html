Programa de energía

En Panel de control > Hardware y alimentación > Programa avanzado, puede configurar el sistema para que se encienda o se apague automáticamente a una hora determinada.

Para agregar un nuevo programa de encendido y apagado:

  1. Haga clic en el botón Crear para abrir el cuadro de diálogo Agregar programa.
  2. Elija Arranque o Apagar.
  3. Ajuste la Fecha y Hora para arrancar o apagar.
  4. Haga clic en OK.
  5. Haga clic en Guardar.

El nuevo programa se agregará a la lista de programas.

Para ver el resumen del programa:

Haga clic en Resumen para examinar el programa semanal de encendido/apagado.

Observación:

  • Puede especificar hasta 100 programas para un Synology NAS.
  • En los siguientes casos, es posible que el sistema no consiga arrancar automáticamente:
    • El Administrador de almacenamiento está realizando tareas.
    • El sistema está cifrando las carpetas compartidas.
    • El sistema está realizando Secure Erase en los discos duros.
    • DSM se está actualizando.
    • El sistema está reparando volúmenes de RAID.
    • El Centro de paquetes está realizando tareas.
    • El sistema está actualizando bases de datos.
    • El sistema está realizando tareas de copia de seguridad de datos.

Observación:

  • Los programas de energía no se aplicarán si se produce una interrupción temporal de la alimentación de su Synology NAS. Para garantizar la alimentación ininterrumpida de Synology NAS, le recomendamos encarecidamente que utilice un SAI SNMP o USB.
  • Solo los modelos de Synology NAS con arquitectura de paquetes como Denverton, Grantley, Broadwell (incluidos Broadwellnk, Broadwellntb y Broadwellntbap), Bromolow, Apollolake y Purley seguirán aplicando el programa de arranque después de que se produzca una interrupción temporal de la alimentación. Consulte este artículo para obtener más información sobre la arquitectura de paquete de su Synology NAS.
  • EDS14 no es compatible con el arranque/apagado automático. Para obtener más información sobre cómo programar el encendido y apagado automático de la salida de CC en EDS14, consulte este artículo de ayuda.
