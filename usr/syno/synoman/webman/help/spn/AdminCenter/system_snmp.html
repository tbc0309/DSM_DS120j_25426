SNMP

El servicio SNMP (en Panel de control > Terminal y SNMP > SNMP) que permite a los usuarios controlar Synology NAS con el software de administración de red. En la actualidad, se admiten los protocolos SNMPv1, SNMPv2c y SNMPv3.

Habilitar servicio SNMP

Marque Habilitar servicio SNMP y siga estos pasos.

Para habilitar el servicio SNMPv1, SNMPv2c:

  1. Marque Servicio SNMPv1, SNMPv2c.
  2. Escriba un nombre de Comunidad.
  3. Haga clic en Aplicar.

Para habilitar el servicio SNMPv3:

  1. Marque Servicio SNMPv3.
  2. Introduzca un Nombre de usuario y Contraseña. Seleccione el Protocolo que desea utilizar.
  3. Haga clic en Aplicar.

Para habilitar la privacidad SNMP:

  1. Marque Habilitar la privacidad SNMP.
  2. Seleccione el Protocolo que desea utilizar. A continuación, introduzca una Contraseña.
  3. Haga clic en Aplicar.

Además, si introduce Nombre, Ubicación y Contacto, le resultará más fácil identificar su Synology NAS. Estos campos son opcionales.

Observación:

  • IPv4 y IPv6 son compatibles con el servicio SNMP.
  • El servicio SNMP admite SNMPv1, SNMPv2c y SNMPv3 al mismo tiempo.

Limitación del nombre de comunidad de SNMPv1, SNMPv2c:

El nombre de comunidad debe contener entre 1 y 64 caracteres visibles. Los siguientes caracteres no están permitidos:
" ' \ y espacio.

Limitación del nombre de usuario de SNMPv3:

El nombre de usuario debe contener entre 1 y 64 caracteres visibles. Los siguientes caracteres no están permitidos:
" ' \ y espacio.

Limitación de la contraseña de SNMPv3:

La contraseña distingue entre mayúsculas y minúsculas, y debe contener entre 8 y 127 caracteres visibles, incluyendo letras, números y signos. Los siguientes caracteres no están permitidos:
" ' \ y espacio.

Limitación de la contraseña de la privacidad SNMP:

La contraseña distingue entre mayúsculas y minúsculas, y debe contener entre 8 y 127 caracteres visibles, incluyendo letras, números y signos. Los siguientes caracteres no están permitidos:
" ' \ y espacio.

Descargar archivos MIB de Synology

Synology ofrece la posibilidad de controlar Synology NAS a través de sistemas de gestión de red, incluyendo sistema, disco y estado de volúmenes RAID.

Observación:

  • RAID no está disponible en un modelo sin disco, como EDS14. Para obtener más información, visite el sitio web de Synology.

Visite el sitio web de Synology para obtener más información y descargar archivos relacionados.

Observación:

  • Los sistemas de gestión de red se basan en archivos MIB para las definiciones de los dispositivos controlados. Para obtener más información, haga clic: aquí.
