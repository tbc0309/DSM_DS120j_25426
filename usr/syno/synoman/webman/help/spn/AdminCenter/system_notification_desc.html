Notificación

Puede configurar su Synology NAS para que le envíe notificaciones cuando se produzcan determinados eventos o errores, avisándole por correo electrónico, SMS, dispositivos móviles o exploradores web. Por ejemplo, el sistema puede enviar una notificación automáticamente cuando se pierde la conexión con la red, el comportamiento del ventilador de refrigeración cambia (según corresponda), se produce un corte de alimentación, el volumen de almacenamiento se queda sin espacio o cuando no se completa una tarea de copia de seguridad y restauración.
