Zona activa*

El dispositivo Synology NAS admite la creación de zonas activas para compartir su conexión de Internet con otros dispositivos inalámbricamente si ha instalado un adaptador USB Wi-Fi o si su Synology NAS tiene Wi-Fi incorporada. Puede configurar su Synology NAS para que actúe como un punto de acceso (AP) o un enrutador inalámbrico, dependiendo de cómo quiera utilizar Internet.

Modo de zona activa Descripción
AP inalámbrico
  • Crea un puente entre su LAN y Wi-Fi.
  • Sus clientes LAN e inalámbricos estarán en la misma red local.
  • Se puede aplicar cuando ya tenga un enrutador y desee utilizar su Synology NAS como un punto de acceso adicional para una red existente.
  • Permite el uso compartido de los recursos de red, como dispositivos DLNA, impresoras (no aplicable a las plataformas Grantley; para ver si su modelo es compatible con esta función, consulte este artículo), Synology Web Assistant u otros servicios basados en la difusión de red.
  • El servidor DHCP se inhabilitará en los dispositivos Synology NAS que actúen como AP. Para reactivar el servidor DHCP en su Synology NAS, quite los adaptadores Wi-Fi y reinicie Synology NAS.
  • En modelos con varios puertos LAN, solo se utilizará LAN 1 para crear el puente con el AP inalámbrico.
Enrutador inalámbrico
  • Configura una red inalámbrica en la que los dispositivos se encuentran en una subred privada.

Nota:

  1. Algunos modelos requieren la conexión de adaptadores inalámbricos para crear zonas activas. Para ver una lista de los adaptadores inalámbricos compatibles, visite www.synology.com.
  2. En modelos que no sean DS213air, la opción de zona activa inalámbrica en DSM 4.3 se ha denominado "Enrutador inalámbrico" en DSM 5.0 y posteriores.

Administrar la configuración de la zona activa

En esta sección se explica cómo habilitar y administrar la configuración de la zona activa de su Synology NAS.

Para convertir Synology NAS en una zona activa:

  1. Vaya al Panel de control > Inalámbrico > Mapa de red. Elija AP inalámbrico o Enrutador inalámbrico.
  2. Vaya a la pestaña Wi-Fi y haga clic en Editar para administrar las siguientes configuraciones:
    • Nombre (SSID): escriba un nombre para la red inalámbrica.
    • Mostrar/Ocultar: elija si desea que la red inalámbrica esté visible para otros dispositivos.
    • Modo inalámbrico: seleccione un estándar inalámbrico. Cada estándar admite varios anchos de banda y frecuencias.
    • Ancho de canal: seleccione un ancho de canal (solo disponible al utilizar el modo inalámbrico 802.11n).
    • Canal: seleccione una frecuencia para la red inalámbrica. Para un mejor rendimiento, seleccione una frecuencia diferente de las utilizadas por redes cercanas.
    • Potencia de transmisión: seleccione una potencia de señal de la red inalámbrica.
    • Conexiones máx.: seleccione el número de dispositivos cliente que podrán conectarse a la red. El número mayor de conexiones es de 256.
    • Habilitar WMM QoS: QoS (Calidad de servicio) de Wi-Fi Multimedia (WMM) por Wi-Fi. Esta opción permite a la red inalámbrica priorizar el tráfico y optimizar el modo en que se comparten los recursos entre las aplicaciones.

Para configurar un programa inalámbrico:*

Al habilitar un programa de red inalámbrica, podrá controlar cuándo estará activa y accesible la red inalámbrica de su Synology NAS.

  1. Vaya a la pestaña Wi-Fi y haga clic en Editar.
  2. Marque la casilla Habilitar programa.
  3. Haga clic en Establecer el programa.
  4. En la pantalla del programa, seleccione cuándo estará activa la red inalámbrica.

Parámetros de seguridad

Panel de control > Inalámbrico > Wi-Fi proporciona opciones para editar la configuración de seguridad de su red inalámbrica.

Para configurar la seguridad inalámbrica:

  1. Haga clic en Editar.
  2. Elija uno de los siguientes protocolos de seguridad del menú desplegable Configuración de seguridad:
    • WEP: un protocolo de seguridad básica que obliga a los usuarios a introducir una contraseña para acceder a la red inalámbrica, pero ofrece protección mínima.
    • WPA/WPA2 Personal: recomendado. Obliga a los usuarios a introducir una contraseña para acceder a la red inalámbrica y es más seguro que el protocolo WEP.
    • WPA/WPA2 Empresa:* un protocolo de seguridad diseñado para empresas que requiere la autenticación de los usuarios con un servidor de autenticación externo.

Para habilitar Wi-Fi Protected Setup:*

WPS (Wi-Fi Protected Setup) es un estándar promovido por la Wi-Fi Alliance para simplificar la configuración de seguridad inalámbrica para los usuarios. Puede utilizar un pulsador o un número PIN para que su red Wi-Fi permanezca segura. Haga clic en WPS para habilitar Wi-Fi Protected Setup.

Red de invitados*

Además de crear una red inalámbrica de host, su Synology NAS también permite crear una red inalámbrica de invitados. La configuración de la red de invitados se puede limitar a restringir el acceso de los usuarios, lo que resulta útil para empresas u otras instituciones que quieran ofrecer una red inalámbrica para visitantes de forma que no puedan acceder a la red de host.

Para habilitar la red de invitados:

  1. Vaya al Panel de control > Inalámbrico > Wi-Fi.
  2. Haga clic en Editar para administrar las siguientes configuraciones en la pestaña Red de invitados:
    • Nombre (SSID): escriba un nombre para la red inalámbrica.
    • Mostrar/Ocultar: elija si desea que la red inalámbrica esté visible para otros dispositivos.
    • Conexiones máx.: seleccione el número de dispositivos cliente que podrán conectarse a la red. El número máximo de conexiones es de 256.

Para configurar la seguridad de la red de invitados:

Elija uno de los siguientes protocolos de seguridad del menú desplegable Configuración de seguridad:

  • WEP: un protocolo de seguridad básica que obliga a los usuarios a introducir una contraseña para acceder a la red inalámbrica, pero ofrece protección mínima.
  • WPA/WPA2 Personal: recomendado. Obliga a los usuarios a introducir una contraseña para acceder a la red inalámbrica y es más seguro que el protocolo WEP.

Para configurar un programa de red de invitados:

Al habilitar un programa de red de invitados, podrá controlar cuándo estará activa y accesible la red inalámbrica de invitados de su Synology NAS.

  1. Marque la casilla Habilitar programa de red de invitados.
  2. Haga clic en Establecer programa de invitados.
  3. En la pantalla del programa, seleccione cuándo estará activa la red inalámbrica de invitados.

Nota:

  • Esta función solo está disponible en DS213air.
  • La red inalámbrica de invitados no puede estar activa durante aquellos periodos de tiempo en los que la red inalámbrica de host esté desactivada.

Configuración de filtro MAC*

El filtro MAC (en Panel de control > Inalámbrico) especifica qué direcciones MAC tienen permiso para acceder a la red inalámbrica, de forma que pueda bloquear o permitir el acceso a determinados dispositivos. Esta función solo controla el acceso a la red, ninguna otra configuración de cifrado.

Para crear reglas de filtro MAC:

  1. Introduzca la descripción y la dirección MAC de un dispositivo.
  2. Elija una política de acceso:
    • Denegar: limita el acceso a la red inalámbrica a la dirección MAC especificada.
    • Permitir: permite el acceso a la red inalámbrica a la dirección MAC especificada.

Control parental*

El control parental le permite supervisar el momento en que cada dispositivo cliente conectado a la red local de Synology NAS puede acceder a Internet. Por ejemplo, si los ordenadores o los dispositivos móviles de sus hijos se conectan a Internet a través de Synology NAS, puede configurar el control parental para restringir el uso de Internet conforme a dispositivos y horas específicos.

Para habilitar el control parental:

  1. Vaya al Panel de control > Inalámbrico > Configuración de control parental.
  2. Haga clic en el botón de Agregar para agregar un dispositivo cliente.
    • Nombre del cliente: elija un dispositivo que esté conectado a Synology NAS o introduzca un nombre manualmente (por ejemplo, Portátil de Luisa).
    • Dirección MAC: introduzca la dirección MAC del dispositivo cuyo uso de Internet desea controlar.
    • Establecer el programa: elija a qué hora el dispositivo especificado podrá acceder a Internet a través de Synology NAS.

Lista de conexiones*

La lista de conexiones (en Panel de control > Inalámbrico) muestra información referente a los dispositivos conectados a Synology NAS. Los dispositivos se pueden añadir directamente desde aquí a la lista de filtros MAC.

Para añadir dispositivos a la lista de filtros MAC:

  1. Active la casilla que hay junto al dispositivo que desea añadir al filtro MAC.
  2. Haga clic en Agregar a la Lista de filtros MAC.
