Dispositivos externos

Vaya a Panel de control > Dispositivos externos > Dispositivos externos para ver una lista de dispositivos externos de almacenamiento conectados a su Synology NAS.

Unidades externas

Al conectar una unidad externa al sistema, podrá compartir su capacidad de unidad a través de una carpeta compartida creada por el sistema llamada usbshare[número] (para unidades USB, si corresponde) o satashare[número] (para unidades eSATA, si corresponde). La carpeta compartida se quitará automáticamente cuando la unidad externa se expulse del sistema.

Synology NAS reconoce los siguientes formatos: Btrfs, ext3, ext4, FAT, FAT32, exFAT, HFS, HFS Plus y NTFS. Será necesario formatear cualquier unidad externa que no se reconozca para poder utilizarla en el sistema.

Observación:

  1. Algunos modelos son compatibles con HFS/HFS Plus con solo lectura.
  2. Journal no es compatible con HFS/HFS Plus.
  3. Deberá instalar exFAT Access desde el Centro de paquetes para habilitar la compatibilidad de Synology NAS con exFAT. Para ver la lista de modelos de productos Synology que son compatibles con el formato exFAT, visite esta página.
  4. Asegúrese de expulsar la unidad externa antes de desconectarla.
  5. Btrfs solo está disponible en modelos específicos. Para saber si su modelo es compatible con Btrfs, consulte este artículo.

Para acceder a las imágenes del disco:

Synology NAS admite el montaje de unidades virtuales para acceder al contenido de imágenes de disco en dispositivos externos con File Station. Si desea obtener más información sobre cómo montar unidades virtuales, consulte este artículo.

Observación:

  • Formato de imagen de archivo compatible: archivos ".iso" con ISO 9960 y sistema de archivos UDF.

Para formatear la unidad:

Haga clic en Formatear y siga las instrucciones del asistente. Puede formatear unidades externas con los formatos ext4 y FAT32 con Synology NAS.

Observación:

  1. Recuerde realizar una copia de seguridad de los datos antes de formatear la unidad externa. Se eliminarán todos los datos durante el procedimiento y no se podrán restaurar.
  2. Para dar a la unidad externa el formato ext3 o ext4, la capacidad de la unidad debe ser superior a 1 GB.
  3. Si la unidad tiene el formato ext3 o ext4, solo la reconocerá Synology NAS. Si desea leer datos directamente de su unidad externa con un PC y Mac, elija el formato FAT32.

Para ejecutar la prueba S.M.A.R.T., :

La prueba S.M.A.R.T. examina sus unidades de disco duro y notifica su estado para avisarle con antelación de errores inminentes en las unidades. Si se detecta algún error, le recomendamos que cambie la unidad.
Para ejecutar la prueba S.M.A.R.T., simplemente haga clic en S.M.A.R.T. S.M.A.R.T. y siga las instrucciones del asistente.
Para comprobar los resultados de la prueba, sencillamente haga clic en S.M.A.R.T.. Información.

Observación:

  • El tiempo de la prueba varía dependiendo del entorno del sistema.

Limitaciones

El formateado de unidades no está disponible en las siguientes situaciones:

  • La unidad externo se está iniciando, es decir, cuando se acaba de conectar el dispositivo externo.
  • La capacidad de la unidad externa es menor de 1 GB (formato ext3 o ext4).
  • La unidad externa se está formateando.
  • Si el sistema está haciendo copias de seguridad de archivos en la unidad externa.
  • La unidad externa está dañada.

Solución de problemas

  1. Error al formatear la unidad externa. ¿Por qué?
    • Compruebe si la unidad externa está insertada correctamente.
    • Compruebe si la unidad externa está protegida contra escritura. Si es así, retírelo antes de formatear.
  2. El sistema de archivos de la unidad externa se muestra como No admitido, o el sistema no crea la carpeta compartida usbshare[número] o satashare[número] de la unidad externa. ¿Qué puedo hacer?
    • El sistema actual reconoce los formatos Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus y NTFS. Las unidades externas no formateadas con los tipos anteriores no se reconocerán y el estado se mostrará como No admitido.
  3. No puedo acceder a la carpeta compartida usbshare[número] o satashare[número]. ¿Qué puedo hacer?
    • Esto podría ser el resultado de un problema en el sistema de archivos de su unidad externa. Para solucionarlo, conecte la unidad externa a un PC y realice una comprobación del sistema de archivos.
  4. ¿Puedo conectar mi unidad externa con formato ext3 o ext4 al PC y a Mac y acceder a ella directamente?
    • No, los formatos ext3 o ext4 solo los reconoce Synology NAS. Si desea leer datos directamente de su unidad externa con un PC y Mac, elija el formato FAT32.
  5. ¿Por qué veo una carpeta denominada "lost+found" en mi unidad externa?
    • Se trata de una carpeta predeterminada del sistema que se crea automáticamente cuando se utiliza el sistema de archivos ext3/ext4. Es normal ver la carpeta en la unidad externa. Aunque los administradores tienen los derechos para acceder y administrar su contenido, se recomienda encarecidamente no manipular la carpeta.

Observación:

  • Btrfs solo está disponible en modelos específicos. Para saber si su modelo es compatible con Btrfs, consulte este artículo.

Para configurar los permisos predeterminados para todos los dispositivos externos:

Puede configurar los permisos predeterminados haciendo clic en Configuración > Permisos predeterminados y especificando los usuarios o grupos que pueden acceder, visualizar o modificar dispositivos externos y sus contenidos. Cuando un dispositivo externo se conecta a Synology NAS, los permisos que haya especificado se asignaran a dicho dispositivo externo automáticamente.

En la ventana de permisos haga lo siguiente para configurar los permisos predeterminados:

  1. Elija una de las siguientes opciones del menú desplegable:
    • Usuario interno del sistema: asigne permisos para usuarios predeterminados del sistema, como el usuario Anonymous FTP/WebDAV.
    • Usuarios locales: asigne permisos para usuarios locales (incluido guest).
    • Grupos locales: asigne permisos para grupos locales.
  2. Marque o desmarque las casillas de verificación para personalizar los permisos de acceso de los dispositivos externos para cada usuario o grupo:
    • Lectura/Escritura: el usuario o grupo pueden acceder a los archivos y las carpetas en dispositivos externos y hacer cambios.
    • Solo lectura: el usuario o el grupo pueden acceder a los archivos y las carpetas en dispositivos externos.
    • Sin acceso: el usuario o grupo no puede acceder a los archivos y las carpetas en dispositivos externos.
