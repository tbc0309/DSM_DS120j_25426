Usuario

Con opciones de administración flexible de usuarios, puede crear cuentas de usuarios para miembros individuales de su familia o empresa. La creación de usuarios le permite definir y administrar permisos para cada persona, como permisos de acceso a carpetas compartidas, cuotas de almacenamiento/carpetas compartidas o limitaciones de ancho de banda.

    Nota:
  1. La gestión de carpetas compartidas solo es aplicable a las plataformas Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake y Denverton. Para ver si su modelo es compatible con esta función, consulte este artículo.
  2. La gestión de la cuota de almacenamiento/carpetas compartidas no es aplicable a Docker DSM.
