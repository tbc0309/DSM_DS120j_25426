Copiar usuario

Puede copiar la configuración de un usuario existente a un nuevo usuario. Esto le permite crear rápidamente un usuario nuevo sin tener que configurar los mismos ajustes otra vez.

  1. Seleccione el usuario que desea copiar.
  2. Haga clic en Crear > Copiar usuario. La ventana del Asistente de creación de usuario se abrirá.
  3. La configuración del usuario seleccionado se aplicará al nuevo por defecto (sin incluir información personal como el nombre de usuario y el correo electrónico).
