Opciones

El menú Opciones se puede encontrar haciendo clic en el icono con forma de persona situado en la parte superior derecha del escritorio de DSM. Le permite realizar las siguientes acciones:

  • Modificar las opciones de una cuenta personal
  • Cerrar sesión
  • Detener el sistema (en EDS14) o apagar el sistema (en todos los demás modelos)
  • Reiniciar el sistema

Tras revelar el menú, haga clic en Personal para editar la configuración de las cuentas personales.
