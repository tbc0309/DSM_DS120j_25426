Criar um Volume

Este artigo explica como criar um volume no seu Synology NAS.Ao criar um novo volume, pode escolher um dos seguintes métodos:

  • Rápido: Esta é a melhor opção para principiantes.Escolha este método se pretender optimizar automaticamente o desempenho e a capacidade de armazenamento, sem preocupações com os aspectos técnicos, como os níveis de RAID.
  • Personalizar: Para utilizadores avançados que pretendem controlar os aspectos mais detalhados da criação do volume, como os tipos de RAID e a alocação de capacidade.Esta opção suporta diferentes tipos de RAID, permite-lhe criar volumes únicos ou múltiplos e disponibiliza opções para alocar montantes específicos de capacidade para cada volume (apenas na criação de volumes múltiplos em RAID).

Volumes Únicos ou Múltiplos em RAID?

Ao criar um novo volume no modo personalizado, pode escolher uma das seguintes opções:

  • Volume Único em RAID
    • Cria um volume único, utilizando toda a capacidade disponível dos discos rígidos seleccionados.
    • Permite uma velocidade e desempenho melhores.
    • Suporta diferentes tipos de RAID.
  • Volumes Múltiplos em RAID
    • Permite-lhe criar múltiplos volumes num grupo de discos.
    • Suporta a alocação de montantes específicos de capacidade a cada volume.
    • Oferece maior flexibilidade de gestão do armazenamento.
    • Suporta diferentes tipos de RAID.

Antes de Começar:

  • Antes de começar, certifique-se de que não existem dados importantes nos discos rígidos.Todos os dados existentes serão eliminados durante o processo de criação.
  • Aceda à página HDD/SSD e certifique-se de que o estado de cada um dos discos rígidos é Normal ou Não inicializado.

Nota:

  • O tamanho mínimo para um volume Btrfs é 10GB.
  • Volumes Btrfs reservam 4% de espaço para metadados.

Criar um volume:

  1. Abra o Gestor de armazenamento.
  2. Aceda a Volume e clique em Criar.
  3. Será apresentado o assistente para criação do volume.Seleccione Rápido ou Personalizado, dependendo das suas necessidades.
  4. Modifique as definições pretendidas.
  5. Escolha se deseja registar a hora de acesso de ficheiros:
    • Ativar a opção Gravar Hora de Acesso de Ficheiros é o equivalente a usar a opção de montagem relatime nos sistemas Linux.A hora de acesso de ficheiros será apenas atualizada quando a hora do último acesso (atime) for anterior à hora de modificação atual (mtime) ou hora de alteração (ctime).A hora de acesso será também atualizada caso a hora do último acesso seja superior a um dia.
    • Desativar a opção Gravar Hora de Acesso de Ficheiros é o equivalente a usar a opção de montagem noatime nos sistemas Linux.A hora de acesso a ficheiros e pastas não será atualizada, resultando em reduzida utilização do sistema de ficheiros.Tenha em conta que algumas aplicações podem depender da hora de acesso de ficheiros para funcionar corretamente.
  6. Clique no botão Aplicar para confirmar as definições.
