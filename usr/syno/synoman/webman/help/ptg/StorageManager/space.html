Volume

Os volumes são espaços de armazenamento básico nos quais o utilizador pode criar pastas partilhadas, iSCSI LUN (ficheiros normais), guardar dados ou instalar pacotes.Antes de criar qualquer volume, certifique-se de que já criou um RAID Group.Para mais informações sobre RAID Groups, consulte RAID Group.

Criar um volume:

  1. Certifique-se que Synology NAS tem um RAID Group existente com espaço que possa ser atribuído.
  2. Clique no botão Criar.
  3. Selecione um RAID Group no qual criar o volume.
  4. Modifique as definições pretendidas.
  5. Defina a atribuição de tamanho para o volume.(A definição da atribuição do tamanho não está disponível para volumes criados no RAID Group para volumes únicos ou iSCSI LUN.)
  6. Escolha se deseja registar a hora de acesso de ficheiros:
    • Ativar a opção Gravar Hora de Acesso de Ficheiros é o equivalente a usar a opção de montagem relatime nos sistemas Linux.A hora de acesso de ficheiros será apenas atualizada quando a hora do último acesso (atime) for anterior à hora de modificação atual (mtime) ou hora de alteração (ctime).A hora de acesso será também atualizada caso a hora do último acesso seja superior a um dia.
    • Desativar a opção Gravar Hora de Acesso de Ficheiros é o equivalente a usar a opção de montagem noatime nos sistemas Linux.A hora de acesso a ficheiros e pastas não será atualizada, resultando em reduzida utilização do sistema de ficheiros.Tenha em conta que algumas aplicações podem depender da hora de acesso de ficheiros para funcionar corretamente.
  7. Clique no botão Aplicar para confirmar as definições.

Nota:

  • O tamanho máximo de volume que pode ser alocado é 200 TB.
  • O tamanho mínimo alocado a um volume Btrfs tem de ser pelo menos 10 GB.
  • Volumes Btrfs reservam 4% de espaço para metadados.

Remover um volume:

  1. Seleccione o volume que pretende remover.
  2. Clique no botão Remover e siga o assistente para concluir o processo.

Nota:

  • A remoção de um volume irá apagar todos os dados existentes num determinado volume.

Para forçar a montagem de um volume:

Se não existir a cache de um volume, o sistema não irá montar o volume durante o arranque.Se a cache não for recuperável e pretende aceder ao volume, pode clicar em Forçar montagem.

Nota:

  • Esta função apenas está disponível em determinados modelos.

Para reparar um volume:

Se o estado do volume indicar Degradado, aceda ao separador RAID Group e encontre o RAID Group apropriado.Depois, siga as instruções sugeridas para o reparar.

Para expandir o tamanho de um volume

  • Para volumes criados num RAID Group para múltiplos volumes ou iSCSI LUNs (Nível de Bloqueio).
    1. Selecione o volume que pretende expandir.
    2. Clique em Editar.
    3. Introduza o novo tamanho e clique em OK para expandir.
  • Para volumes criados num RAID Group para volume único ou iSCSI LUN (Nível de Bloqueio), consulte a secção Para expandir o tamanho de um RAID Group do RAID Group, o tamanho do volume será expandido depois de o RAID Group subjacente ser expandido.

SSD TRIM

Se um volume for totalmente constituído por SSDs (Unidades de Estado Sólido), recomendamos a ativação de SSD TRIM.Esta funcionalidade melhora o desempenho de leitura e escrita de volumes criados em SSDs, melhorando a eficiência, bem como prolongando a duração dos SSDs.

Para ativar o SSD TRIM:

  1. Abra Gestor de Armazenamento e aceda a Volume.
  2. Selecione um volume que seja totalmente constituído por SSD e depois clique no botão Configurar.
  3. Na janela apresentada, aceda ao separador SSD TRIM e marque a caixa Ativar TRIM.
  4. Clique no botão Configurar agendamento para escolher quando o sistema irá executar o SSD TRIM.
  5. Clique em OK.

Nota:

  • O SSD TRIM só está disponível em determinados modelos do Synology NAS e tipos de RAID.
  • Alguns modelos SSD não são capazes de executar SSD TRIM em volumes RAID 5, e RAID 6, e RAID F1.Consulte a lista de compatibilidades no site oficial da Synology.
  • O SSD TRIM não pode ser ativado nas seguintes situações:
    • Se o volume estiver montado com cache SSD.
    • Se o pacote Time Backup estiver instalado.
  • O SSD TRIM não pode ser ativado nos iSCSI LUNs de nível de bloqueio.
  • O desempenho do sistema será afetado durante a execução do SSD TRIM.O tempo necessário para concluir a execução do TRIM varia de acordo com o tamanho do volume e modelo SSD.
  • Quando o TRIM está ativado num volume, também será ativado noutros volumes criados no mesmo Grupo de Discos ou RAID Group.

Análise do sistema de ficheiros*

A análise do sistema de ficheiros é uma funcionaliadde de manutenção de dados que inspeciona volumes.Se for detetada alguma inconsistência de dados com o checksum, o caminho do ficheiros será gravado no Centro de Registos, e recomenda-se que use uma cópia de segurança para reparar os dados.

Recomendamos que realize regularmente análise do sistema de ficheiros para garantir a consistência dos dados.

Para realizar a análise do sistema de ficheiros:

  1. Abra Gestor de Armazenamento e aceda a Volume.
  2. Selecione o volume no qual realizar a análise do sistema de ficheiros.
  3. Clique em Gerir.
  4. Selecione Análise do sistema de ficheiros.

Nota:

  • A análise do sistema de ficheiros pode ser apenas realizada em volumes no sistema de ficheiros Btrfs.
  • A análise do sistema de ficheiros pode apenas ser realizada quando o estado do volume for Normal.
  • Não encerre o sistema antes que esteja concluída a análise do sistema de ficheiros.Se o sistema for encerrado antes da sua conclusão, terá de reiniciar manualmente a análise do sistema de ficheiros desde o início.

Desfragmentação do sistema de ficheiros*

Se os seus dados forem modificados de forma constante, aumentará a quantidade de ficheiros fragmentados no sistema de ficheiros, resultando em desempenho lento no acesso a ficheiros nos discos rígidos.Realizar a desfragmentação do sistema de ficheiros pode aumentar o desempenho.

Para realizar a desfragmentação do sistema de ficheiros:

  1. Abra Gestor de Armazenamento e aceda a Volume.
  2. Selecione o volume no qual realizar a desfragmentação do sistema de ficheiros.
  3. Clique em Gerir.
  4. Selecione Desfragmentação do sistema de ficheiros.

Nota:

  • A desfragmentação do sistema de ficheiros pode ser apenas realizada em volumes no sistema de ficheiros Btrfs.
  • A desfragmentação do sistema de ficheiros pode apenas ser realizada quando o estado do volume for Normal.
  • Não encerre o sistema antes que esteja concluída a desfragmentação do sistema de ficheiros.Se o sistema for encerrado antes da sua conclusão, terá de reiniciar manualmente a desfragmentação do sistema de ficheiros desde o início.
  • A utilização do volume poderá aumentar se este possuir imagens instantâneas de pastas partilhadas.
