Criar um Utilizador

Pode criar e gerir diversas contas de utilizador no seu Synology NAS.

Para criar um utilizador:

  1. Aceda a Painel de Controlo > Utilizador.
  2. Clique no botão Criar seguido de Criar utilizador. Esta ação inicia o Assistente de Criação de Utilizador.
  3. Na página Informações do utilizador, especifique as seguintes opções:
    • Nome: Introduza um nome para o utilizador. Esta secção é obrigatória.
    • Descrição: Introduza uma breve descrição para o utilizador.
    • E-mail: Introduza um endereço de e-mail para o utilizador. As notificações do sistema, tais como as mensagens de reposição da palavra-passe, são enviadas para o endereço especificado aqui.
    • Enviar um e-mail de notificação para o novo utilizador criado: Se esta opção estiver ativada, o sistema irá enviar uma mensagem para o utilizador para notificá-lo sobre a nova conta. Esta opção requer a ativação das notificações do sistema por e-mail em Menu Principal > Painel de Controlo > Notificação > E-mail. Por predefinição, a palavra-passe do utilizador não é apresentada na mensagem de notificação. No entanto, se pretender apresentar a palavra-passe na mensagem de notificação, marque a caixa Mostrar palavra-passe de utilizador no e-mail de notificação.
    • Não permitir ao utilizador alterar a palavra-passe da conta: Se esta opção estiver ativada, o utilizador não poderá alterar a respetiva palavra-passe utilizada para iniciar sessão no DSM.
  4. Na página Grupo > Criar > Assistente de Criação de Grupo, especifique os grupos aos quais o novo utilizador irá pertencer (por exemplo, users, administrators, etc.).
  5. Na página Atribuir permissões de pastas partilhadas, escolha quais as pastas partilhadas que podem ser acedidas pelo novo utilizador modificando os respetivos privilégios de acesso. A prioridade dos privilégios é a seguinte: Sem acesso > Leitura/Escrita > Só de leitura.
    • Pré-visualizar: Esta coluna apresenta os privilégios atuais do utilizador de acesso de pasta partilhada, de acordo com as definições e grupos atuais, aos quais o utilizador pertence.
    • Permissões de grupo: Esta coluna apresenta os privilégios de acesso de pasta partilhada atribuídos de acordo com os grupos de utilizador, aos quais o utilizador pertence.
  6. Na página Definição da quota do utilizador, pode controlar a quantidade de espaço de armazenamento que pode ser utilizado pelo novo utilizador. Para restringir a quantidade de espaço que pode ser utilizado pelo utilizador, marque a caixa em Ativar quota para cada volume/pasta partilhada. Em seguida, introduza a quantidade máxima de espaço que pode ser utilizado no campo Quota (por exemplo, 10 GB).
  7. Na página Atribuir permissões para aplicação, pode controlar quais os serviços que podem ser acedidos pelo utilizador.
  8. Na página Definição de Limite de Velocidade de Grupo, pode ativar um limite de velocidade para diferentes serviços (por exemplo, File Station, FTP, etc.) para restringir a quantidade de largura de banda consumida pelo utilizador durante a transferência de ficheiros. Para cada serviço, pode selecionar uma das seguintes opções:
    • Nenhum: As definições do limite de velocidade serão iguais às definições do grupo ao qual pertence o utilizador.
    • Constante: Será aplicado um limite de velocidade fixo e constante. Pode especificar os limites de velocidade de carregamento e transferência nos campos à esquerda.
    • Variável: É possível especificar e aplicar dois limites de velocidade diferentes ao utilizador de acordo com um agendamento. Clique no botão Personalizar para modificar as definições do limite de velocidade e o agendamento.
  9. Depois de confirmar as definições, clique em Aplicar para concluir.

Limitação do nome de utilizador:

  • O nome de utilizador não é sensível a maiúsculas e minúsculas e pode ter entre 1 e 64 caracteres Unicode, mas não pode conter os seguintes símbolos:
    ! " # $ % & ' ( ) * + ,/: ; < = > ? @ [ ] \ ^ ` { } | ~
  • O primeiro carácter não pode ser um sinal de menos ou um espaço e o último carácter não pode ser um espaço.
  • Os seguintes nomes de utilizador estão reservados para utilização do sistema: mailer-daemon, postmaster.
  • A descrição do utilizador pode incluir até 64 caracteres Unicode apresentáveis.

Limitação da palavra-passe:

  • A palavra-passe é sensível a maiúsculas e minúsculas e está limitada a 127 caracteres apresentáveis, incluindo letras, números, sinais e espaços.

Limitação de quota:

  • A quota máxima de utilização para armazenamento que pode ser introduzida para cada utilizador é de, aproximadamente, 4 TB (4095 GB).
