Gerir chaves de pastas partilhadas encriptadas

Poderá usar o Gestor de chaves para gerir chaves de pastas partilhadas e desencriptar múltiplas pastas encriptadas ao mesmo tempo.

Explicação dos termos:

  • Armazenamento de chaves:Um armazenamento de chaves é qualquer dispositivo externo suportado por Synology NAS e é usado como chave física para desencriptar pastas partilhadas.
  • Encriptação:Uma encriptação é um método de encriptar chaves de pastas partilhadas.O Gestor de chaves fornece dois tipos de encriptação:
    • Frase-passe:As chaves encriptadas por uma frase-passe podem ser desencriptadas por qualquer pessoa que saiba a frase-passe.
    • Chave de máquina:As chaves encriptadas por uma chave de máquina apenas podem ser desencriptadas pelo Synology NAS vinculado.

Para inicializar um Armazenamento de chaves:

  1. Aceda a Painel de controlo > Pasta partilhada > Ação > Gestor de chaves.
  2. Selecione um dispositivo externo como armazenamento de chaves a partir da Localização do Armazenamento de chaves.
  3. Introduza uma frase-passe no campo Frase-passe para este armazenamento de chaves.

Para adicionar uma nova chave a um Armazenamento de chaves:

  1. Certifique-se de que inicializou um Armazenamento de chaves.
  2. Aceda a Painel de controlo > Pasta partilhada > Ação > Gestor de chaves.
  3. Clique em Adicionar.
  4. Selecione uma pasta partilhada encriptada.
  5. Selecione a encriptação para a chave de encriptação.Poderá selecionar Frase-passe ou Chave de máquina.
  6. Introduza ou importe a chave de encriptação.
  7. Clique em OK para guardar as definições.

Para desencriptar manualmente múltiplas pastas partilhadas encriptadas:

  1. Certifique-se de que adicionou chaves ao Armazenamento de chaves.
  2. Aceda a Painel de controlo > Pasta partilhada > Ação > Gestor de chaves.
  3. Selecione as pastas partilhadas encriptadas que pretende desencriptar.
  4. Clique em Montar.

Para desencriptar automaticamente múltiplas pastas partilhadas encriptadas no arranque:

  1. Certifique-se de que adicionou chaves ao Armazenamento de chaves.
  2. Aceda a Painel de controlo > Pasta partilhada > Ação > Gestor de chaves.
  3. Assinale Montar no arranque para as pastas partilhadas encriptadas que pretende desencriptar no arranque.
  4. Clique em OK para guardar as definições.

Para ejetar um Armazenamento de chaves após o arranque:

  1. Aceda a Painel de controlo > Pasta partilhada > Ação > Gestor de chaves.
  2. Clique em Configurar.
  3. Assinale Ejetar dispositivo após arranque.
  4. Clique em OK para guardar as definições.

Para migrar automaticamente chaves de encriptação para um armazenamento de chaves para pastas partilhadas montadas automaticamente:

Migrar chaves durante inicialização de armazenamento de chaves:

  1. Quando inicializa um armazenamento de chaves, assinale Migrar automaticamente chaves de encriptação para este armazenamento de chaves para todas as pastas partilhadas montadas (recomendado).

Migrar chaves manualmente:

  1. Aceda a Painel de controlo > Pasta partilhada > Ação > Gestor de chaves.
  2. Clique em Configurar.
  3. Clique em Migrar Agora.
  4. No caso de chaves duplicadas, pode escolher Substituir chaves duplicadas.
  5. Clique em OK para guardar as definições.
