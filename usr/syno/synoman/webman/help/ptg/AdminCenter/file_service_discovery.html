Deteção de serviços

A deteção de serviços permite aos utilizadores encontrar este Synology NAS na rede através de SSDP (Simple Service Discovery Protocol) e serviços Bonjour.

Para tornar o seu Synology NAS detetável via SSDP:

O serviço SSDP permite aos utilizadores encontrar o Synology NAS através de deteção de rede Windows.

  1. Aceda a Painel de controlo > Serviços de ficheiros > Deteção de serviços > SSDP.
  2. Selecione Ativar deteção de rede do Windows.
  3. Clique em Aplicar para guardar as definições.

Para tornar o Synology NAS detetável através do serviço Bonjour:

O serviço Bonjour permite aos utilizadores encontrar o Synology NAS e realizar tarefas de impressão e cópia de segurança a partir do computador Mac.

  1. Aceda a Painel de controlo > Serviços de ficheiros > Deteção de serviços > Bonjour.
  2. Selecione Ativar deteção de serviços Bonjour.
  3. Poderá selecionar opções relacionadas com Bonjour para satisfazer as suas necessidades:
    • Ativar Emissão de impressora Bonjour: Selecione para permitir a computadores Mac encontrar a impressora USB ligada ao seu Synology NAS para impressão de documentos.
    • Ativar a emissão Bonjour Time Machine via SMB: Selecione para permitir à Mac Time Machine encontrar o seu Synology NAS e efetuar uma cópia de segurança de ficheiros para pastas DSM partilhads via SMB.
    • Ativar a emissão Bonjour Time Machine via AFP: Selecione para permitir à Mac Time Machine encontrar o seu Synology NAS e efetuar uma cópia de segurança de ficheiros para pastas DSM partilhads via AFP.
  4. Se selecionou qualquer opção relacionada com Time Machine, clique em Definir pastas Time Machine para especificar pastas de destino para as cópias de segurança de Time Machine.
  5. Clique em Aplicar para guardar as definições.

Nota:

  • Se desativar o serviço SMB/AFP (em Painel de controlo > Serviços de ficheiros > SMB/AFP/NFS), a opção Bonjour associada será automaticamente desativada.
  • Quando Ativar a emissão Bonjour Time Machine via SMB estiver selecionado, as seguintes opções serão ativadas (em Painel de controlo > Serviços de ficheiros > SMB/AFP/NFS > Definições avançadas):
    • Ative SMB 3 (para Protocolo máximo para serviço de ficheiros Windows)
    • Opportunistic Locking
    • Ativar autorização SMB2
    • Ative alcunhas duráveis SMB (Será desativado o bloqueio de ficheiros entre múltiplas plataformas)
