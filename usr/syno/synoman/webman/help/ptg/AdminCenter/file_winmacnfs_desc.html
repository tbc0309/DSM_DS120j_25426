SMB/AFP/NFS

Os utilizadores de Windows, Mac e Linux podem aceder ao seu Synology NAS a partir da rede, se tiver ativado as opções em Painel de controlo > Serviços de ficheiros > SMB/AFP/NFS.

São suportados os seguintes protocolos de partilha de ficheiros:

  • Para Windows: SMB/CIFS (Os Meus Locais na Rede), FTP, WebDAV
  • Para Mac: SMB, FTP, AFP, WebDAV
  • Para Linux: SMB, FTP, NFS, WebDAV
