Copiar um utilizador

Poderá copiar as definições de um utilizador existente para um novo utilizador. Isto permitirá criar rapidamente um novo utilizador sem ter de configurar novamente as mesmas definições.

  1. Selecione o utilizador que pretende copiar.
  2. Clique em Criar > Copiar utilizador. A janela do Assistente de Criação de Utilizador será executada.
  3. As definições do utilizador selecionado serão aplicadas a um novo utilizador por predefinição (excluindo informação pessoal, tal como nome de utilizador e email).
