Bloqueio Automático

A funcionalidade de bloqueio automático ajuda a melhorar a segurança do seu Synology NAS, bloqueando os endereços IP de clientes com demasiadas tentativas falhadas de início de sessão. Isto ajuda a reduzir o risco de contas serem assaltadas através de ataques violentos.

Também pode criar e gerir uma lista autorizada para adicionar endereços IP nos quais confia ou uma lista de bloqueio para impedir sempre que determinados endereços IP iniciem sessão.

Nota:

  • Vários serviços e pacotes suportam o bloqueio automático, como os seguintes: DSM, SSH, Telnet, rsync, cópia de segurança de rede, sincronização de pasta partilhada, FTP, WebDAV, File Station, Photo Station, Audio Station, Video Station, Download Station, Mail Server, Mail Station, Time Backup, VPN Server, Cloud Station e aplicações móveis da Synology.

Para ativar o bloqueio automático:

  1. Abra o Painel de controlo e aceda a Segurança > Bloqueio automático.
  2. Assinale Ativar bloqueio automático.
  3. Introduza um número de tentativas falhadas de início de sessão no campo Tentativas de início de sessão e um número de minutos no campo No espaço de (minutos). Um endereço IP será bloqueado quando exceder o número de tentativas falhadas de início de sessão no número de minutos especificado.
  4. Assinale a caixa de verificação Ativar validade do bloqueio e introduza um número para remover um endereço IP bloqueado após o número especificado de dias.
  5. Clique em Aplicar para guardar as definições.

Gerir Lista de Bloqueio

Clique em Lista de permissões/bloqueios e aceda ao separador Lista de Bloqueio.

Para adicionar endereços IP:

Escolha qualquer uma das seguintes a partir do menu pendente Criar:

  • Adicionar endereço IP: Introduza um endereço IP e defina a hora de validade.
  • Importar lista de endereços IP: Importe um ficheiro de texto com os endereços IP listados para adicionar vários endereços IP e definir todas as horas de validade ao mesmo tempo. Assinale a caixa de verificação se pretender substituir endereços IP existentes. Se esta opção não estiver selecionada, serão ignorados endereços IP duplicados
    Se o ficheiro selecionado tiver um formato correto, verá todos os endereços IP na área de pré-visualização. Clique em OK para importar.

Nota:

Para importar um ficheiro, este deverá cumprir os seguintes critérios:

  • O ficheiro deverá ser um ficheiro de texto simples.
  • Cada linha do ficheiro pode apenas conter um endereço IP.
  • As linhas de comentários começam com um # (cardinal).

Para remover endereços IP bloqueados:

Selecione os endereços IP que pretende remover da lista e clique em Remover.

Gerir Lista de permissões

Clique em Lista de permissões/bloqueios e aceda ao separador Lista de permissões.

Para adicionar endereços IP:

Escolha qualquer uma das seguintes a partir do menu pendente Criar:

  • Adicionar endereço IP: Introduza um endereço IP, intervalo IP, sub-rede ou nome de domínio.
  • Importar lista de endereços IP: Importe um ficheiro de texto com os endereços IP listados para adicionar todos os endereços IP ao mesmo tempo. Assinale a caixa de verificação se pretender substituir endereços IP existentes. Se esta opção não estiver selecionada, serão ignorados endereços IP duplicados
    Se o ficheiro selecionado tiver um formato correto, verá todos os endereços IP na área de pré-visualização. Clique em OK para importar.

Nota:

Para importar um ficheiro, este deverá cumprir os seguintes critérios:

  • O ficheiro deverá ser um ficheiro de texto simples.
  • Cada linha do ficheiro pode conter apenas um endereço IP, um intervalo IP (192.168.1.1 - 192.168.1.10), uma sub-rede (192.168.1.1/255.255.0.0) ou um nome de domínio (www.synology.com).
  • As linhas de comentários começam com um # (cardinal).

Para remover endereços IP:

Selecione os endereços IP que pretende remover da lista e clique em Remover.

Nota:

  • Se o seu dispositivo Synology NAS estiver atrás de um servidor proxy inverso, adicione o endereço IP desse servidor proxy inverso à lista autorizada.
