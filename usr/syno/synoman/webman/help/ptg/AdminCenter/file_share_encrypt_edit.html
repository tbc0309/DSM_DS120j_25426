Editar Pasta Encriptada

Após a encriptação da pasta, pode utilizar o menu pendente Encriptação para editar a pasta.

Os itens mostrados no menu dependem do estado da pasta partilhada seleccionada:

  • Se seleccionar uma pasta partilhada não encriptada, o menu pendente Encriptação será desactivado.
  • Se seleccionar uma pasta partilhada encriptada que esteja montada, pode seleccionar Desmontar no menu pendente Encriptação para desmontar a pasta ou Exportar chave para guardar a chave de encriptação.
  • Se seleccionar uma pasta partilhada encriptada que esteja desmontada, pode escolher Montar no menu pendente Encriptação e, de seguida, introduzir ou importar a chave de encriptação para montar a pasta para acesso.
  • Se selecionar múltiplas pastas partilhadas encriptadas que não estejam montadas e as chaves de encriptação sejam geridas por Gestor de chaves, pode escolher Montar do menu de contexto Encriptação, e depois introduza a frase-passe de Gestor de chaves para montar estas pastas para acesso.
    Nota:
  1. É muito importante lembrar-se da chave de encriptação. É fortemente recomendado que exporte e guarde a chave de encriptação.Se não se lembrar ou perder a chave, não existe forma de recuperar os dados encriptados.
  2. Se uma pasta partilhada encriptada estiver montada, não pode mudar o nome nem alterar a sua localização no volume.Só poderá realizar alterações após ser desmontada.
