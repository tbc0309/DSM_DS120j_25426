Pakete installieren oder kaufen

Das Paketzentrum bietet eine reiche Vielfalt von Webanwendungen, die die Funktionalität Ihrer Synology NAS erweitern. Im Paketzentrum können Sie Pakete mit nur wenigen einfachen Klicks herunterladen, installieren und starten.

Im folgenden Artikel wird erläutert, wie Sie Pakete über das Paketzentrum auf Ihrer Synology NAS installieren. Informationen zu einzelnen Paketen finden Sie hier.

Pakete installieren oder kaufen:

  1. Öffnen Sie das Paket-Zentrum.
  2. Gehen Sie zur Seite Alle Pakete, um alle verfügbaren Pakete zu sehen.

    3. Anmerkung:

    • Wenn Sie das gewünschte Paket nicht finden können, benötigt es möglicherweise eine spätere Version von DSM oder ist nur auf Modellen mit bestimmten Hardware-Anforderungen verfügbar. Weitere Informationen erhalten Sie vom Entwickler des jeweiligen Pakets.
  3. Gehen Sie zu dem Paket, das Sie installieren möchten, und klicken Sie auf Installieren. (Klicken Sie für Bezahlpakete auf Kaufen oder Ausprobieren, um sich mit dem Synology-Konto zu verbinden, und bezahlen Sie mit Kreditkarte, oder verwenden Sie die Demoversion zum Testen.)
  4. Nach der erfolgreichen Installation des Pakets müsste es im Hauptmenü angezeigt werden.

Kaufen von Paketen:

Beachten Sie die nachstehenden Hinweise, bevor Sie Pakete kaufen.

  • Registrieren Sie ein Synology-Konto, um Zugriff auf weitere Funktionen zu haben, wie etwa Vorabinformationen zur Abrechnung und Kaufverlauf. Die Angaben, die Sie hier machen, werden nur zur künftigen Verbesserung der Dienste verwendet.
  • Paketpreise werden standardmäßig in US-Dollar angegeben. Wenn Sie sich bei Ihrem Synology-Konto anmelden, werden die Preise in der Ihren Einstellungen entsprechenden Währung angezeigt.

Haftungsausschluss bei Währungsumrechnung:

Für Transaktionen, die eine Währungsumrechnung beinhalten, kann der Aussteller Ihrer Kredit-/Debitkarte zusätzliche Gebühren verlangen, die nicht im angezeigten Preis enthalten sind. Für Käufe könnten Gebühren für Fremdwährung oder grenzüberschreitende Zahlungen vom Kartenaussteller verlangt werden. Sämtliche anfallenden Gebühren werden von Ihrem Kartenaussteller festgelegt. Synology kann dafür keine Rückzahlung leisten. Weitere Informationen erhalten Sie vom Aussteller Ihrer Kredit-/Debitkarte.

Pakete manuell installieren:

  1. Wenn Sie ein Paket durch Hochladen einer .spk-Datei (im Download-Zentrum von Synology oder auf externen Webseiten verfügbar) installieren möchten, klicken Sie auf Manuelle Installation und folgen Sie dem Assistenten, um das neue Paket zu installieren.

Nach einem Paket suchen:

  1. Geben Sie in die Suchleiste oben links ein Schlüsselwort ein und drücken Sie die Eingabetaste, um mit der Suche zu beginnen.

Sortierung von Paketen auswählen:

Sie können die Reihenfolge der anzuzeigenden Pakete im Paketzentrum ändern. Klicken Sie auf das Dropdown-Menü unter dem Banner und wählen Sie die Sortierung von Paketen mit einer der folgenden Optionen aus:

  • Sortieren nach Name: Klicken Sie auf diese Option, um Pakete in alphabetischer Reihenfolge zu sortieren.
  • Sortieren nach aktueller Beliebtheit: Klicken Sie auf diese Option, um Pakete nach der aktuellen Anzahl der Installationen zu sortieren, wobei die beliebtesten zuerst angezeigt werden.

Installierte Pakete ausblenden:

Wählen Sie eine Kategorie aus, klicken Sie auf das Dropdown-Menü unter dem Banner und wählen Sie Installierte Pakete ausblenden. Dadurch werden bereits installierte Pakete ausgeblendet und nur jene angezeigt, die noch auf Installation, Aktualisierung oder Reparatur warten.
