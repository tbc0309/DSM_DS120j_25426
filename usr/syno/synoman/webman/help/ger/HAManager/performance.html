Leistung

Auf der Seite Leistung können Sie die Ressourcennutzung von aktiven und passiven Servern nebeneinander überwachen. Die beiden Server lassen sich einfach anhand des Servernamens und ihrer jeweiligen Rollen (aktiv oder passiv) unterscheiden. Wenn der passive Server ausgeschaltet ist, wird die Ressourcennutzung auf dem passiven Server nicht angezeigt.

CPU

Sie können die gesamte CPU-Auslastung anzeigen, die das System, Benutzer und I/O-Warteschlangen beanspruchen. Nachdem Sie die Seite geöffnet haben, kann die CPU-Auslastung höher sein, da das System die Seite ladet und gleichzeitig die Daten der Systemauslastung sammelt.

Arbeitsspeicher

Die Gesamtnutzung des physischen Speichers wird angezeigt. Eine konstant hohe Speicherauslastung kann bedeuten, dass das System häufig genutzte Daten im Cache speichert, um Daten rasch abzurufen und nicht auf die Festplatte zugreifen zu müssen. Bei unzureichendem Gesamtspeicher gibt das System Zwischenspeicher frei.

Netzwerk

Das Netzwerkverkehrsdiagramm zeigt die Datenübertragungsrate in KB pro Sekunde in verschiedenen Farben an:

  • Blau steht für die Upload-Rate.
  • Grün steht für die Download-Rate.
  • Wenn sich Upload- und Download-Raten überschneiden, wird der Überschneidungsbereich dunkelgrün angezeigt.

Festplatte

Das Diagramm zeigt die gesamte Festplattennutzung an.
