Daten mit Synology Drive Client auf meinem Computer sichern

Überblick

Synology Drive Server ist nicht nur eine hochentwickelte plattformübergreifende Synchronisierungssoftware, sondern auch ein intelligentes Tool zur Sicherung der Daten von PC-Benutzern. Ihr Desktop-Dienstprogramm Synology Drive Client ist eine Lösung, die eine sofortige Sicherung in Echtzeit ermöglicht, Bandbreite spart und mit bis zu 32 früheren Versionen einer Datei hevorragenden Schutz bietet.

Synology Drive Server auf Synology NAS einrichten und Synology Drive Client herunterladen

  1. Gehen Sie zum Paketzentrum, suchen Sie Synology Drive Server, und klicken Sie anschließend auf Installieren. Es werden insgesamt drei Pakete heruntergeladen, Synology Drive Admin-Konsole, Synology Drive ShareSync und Synology Drive.
  2. Starten Sie Synology Drive. Unten rechts wird eine Eingabeaufforderung zum Herunterladen von Synology Drive Client angezeigt.

    Oder klicken Sie unten links auf das Benutzersymbol und wechseln Sie zu Client > Client herunterladen.

    Wahlweise können Sie Synology Drive Client auch   im Download-Zentrum von Synology herunterladen.
  3. Starten Sie das Installationsprogramm auf Ihrem Computer und folgen Sie den Anweisungen zur Installation von Synology Drive Client. Führen Sie nach der Installation Synology Drive Client aus, indem Sie auf Jetzt starten klicken.
  4. Wählen Sie Sicherungsaufgabe.
  5. Geben Sie die Adresse oder QuickConnect ID für den Synology NAS, auf dem Synology Drive Server läuft, sowie den Benutzernamen und das Kennwort ein. Sie können auch rechts auf das Suchsymbol klicken, sodass Synology Drive Client automatisch nach einem Synology NAS in Ihrem lokalen Netzwerk sucht. Klicken Sie auf Weiter.
  6. Wählen Sie Ihre Sicherungsquelle aus und deaktivieren Sie alle Unterordner, die Sie nicht synchronisieren möchten. Synology Drive Client erstellt auf dem ausgewählten Ziel einen Sicherungsordner mit dem Namen Ihres Computers. Standardmäßig wird der Ordner home als Zielordner ausgewählt. Wenden Sie sich an Ihren Systemadministrator, wenn Sie einen Ordner verwenden möchten, der nicht in der Liste angeführt ist. Systemadministratoren finden weitere Informationen dazu auf der Hilfe-Seite der Synology Drive Admin-Konsole. Sie können auch auf Datensicherungsregeln klicken, wenn Sie Synchronisierungsfilter oder -regeln festlegen möchten. Klicken Sie auf Weiter.
  7. Wählen Sie einen Sicherungsmodus aus und klicken Sie auf Weiter.
  8. Sie können eine Übersicht Ihrer Datensicherungseinstellungen anzeigen. Klicken Sie auf Zurück, um Änderungen vorzunehmen, oder auf Fertig, um die Einrichtung abzuschließen.
  9. Der Fortschritt der Datensicherung wird im Hauptfenster angezeigt.

Eine frühere Version einer synchronisierten Datei auf Synology Drive Client herunterladen oder wiederherstellen

  1. Gehen Sie im Synology Drive Client zu Sicherungsaufgaben > Wiederherstellen und suchen Sie die Datei bzw. den Ordner, die oder den Sie wiederherstellen möchten.
  2. Wählen Sie die Datei aus, die Sie abrufen möchten, und klicken Sie auf Frühere Versionen durchsuchen. Wenn eine Datei auf Ihrem Computer versehentlich gelöscht oder verschoben wurde, können Sie auf Wiederherstellen klicken, um sie wiederherzustellen.
  3. Wählen Sie die Version der Datei, die Sie herunterladen möchten, und klicken Sie auf Herunterladen.

Eine frühere Version einer synchronisierten Datei auf der Synology Drive Admin-Konsole herunterladen oder wiederherstellen

  1. Gehen Sie in DSM zu Synology Drive Admin-Konsole > Team-Ordner, klicken Sie auf den Ordner, in dem sich die wiederherzustellende Datei befindet, und klicken Sie auf Versionsexplorer.
  2. Klicken Sie im Fenster Versionsexplorer auf die Datei, die Sie abrufen möchten, und anschließend auf Frühere Versionen durchsuchen.
  3. Wählen Sie die Version der Datei, die Sie herunterladen möchten, und klicken Sie auf Herunterladen. Falls Sie sich allerdings sicher sind, dass die ausgewählte Version wiederhergestellt werden soll, klicken Sie auf Wiederherstellen, dann wird Ihre aktuelle Version überschrieben.
