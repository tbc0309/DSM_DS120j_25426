Leistung

Klicken Sie auf die Registerkarte Leistung, um die Ressourcennutzungsdaten anzuzeigen. Details zu Datum, Uhrzeit und Informationen zum Diagramm werden angezeigt, wenn Sie den Mauszeiger darauf richten.

Auf der Registerkarte Überblick werden Diagramme für CPU, GPU, Systemspeicher, GPU-Speicher, Netzwerk, Festplatte, Volume und iSCSI angezeigt.

CPU

Sie können die gesamte CPU-Auslastung unter Benutzer, System und I/O Warten anzeigen. Die CPU-Auslastung sollte hoch sein, wenn der Ressourcenmonitor das erste Mal gestartet wird, denn das System muss die Ressourcendaten erst sammeln und gleichzeitig die UI-Seite laden.

  • Benutzer zeigt den Prozentsatz der CPU an, der für die Ausführung von Benutzeranwendungen verwendet wird.
  • System zeigt den Prozentsatz der CPU an, der für den Systemkern verwendet wird, sowie Hardware/Software-IRQs.
  • I/O Warten zeigt den Prozentsatz der CPU an, der für „I/O Warten“ genutzt wird.
  • Auslastung ist die Summe der CPU-Auslastung durch Benutzer und System.
  • Die Durchschnittsleistung innerhalb von 1, 5 und 15 Minuten wird mit der Zahl der ausführbaren Prozesse (Run-Queue Length) angezeigt (d. h. die Anzahl der Prozesse, die gegenwärtig ausgeführt werden, plus die Anzahl der wartenden Prozesse).

Erklärung der Begriffe

  • I/O Warten: I/O Warten zeigt an, dass sich die CPU im Leerlauf befindet und nicht von einem anderen Programm verwendet wird.

    • I/O Warten ist ein normales Verhalten. Da die Geschwindigkeit der CPU um ein Vielfaches höher als die von I/O ist, ist es normal, dass die CPU häufig auf die Übertragungen von Festplatten oder aus dem Netzwerk warten muss. Wenn der Prozentsatz von "I/O Warten" allerdings extrem hoch ist, bedeutet dies, dass die CPU die meiste Zeit mit "I/O Warten" verbringt und womöglich nicht gut ausgelastet sein ist. Sie können eine effizientere Festplattenkonfiguration (z. B. SHR oder RAID 5) oder Netzwerkkonfigurationen (z. B. Bonding) nutzen, um eine bessere Systemleistung zu erreichen.

GPU

Hier wird die gesamte GPU-Auslastung angezeigt. Im Dropdown-Menü können Sie zwischen GPU und GPU-Speicher wechseln.

  • Auslastung ist die Summe der GPU-Auslastung durch Benutzer und System.
  • Unter GPU-Speicher können Sie die Speicherauslastung nach benutztem und freiem Speicher anzeigen.

Anmerkung:

  • Nur auf Modellen verfügbar, die GPU unterstützen.

Systemspeicher

Die gesamte Nutzung des physikalischen Speichers und des Auslagerungsspeichers wird angezeigt. Die Speicherauslastung bleibt hoch, weil das System häufig aufgerufene Daten im Zwischenspeicher behält, damit die Daten schnell aufgerufen werden können, ohne dass auf die Festplatte zugegriffen werden muss. Der Zwischenspeicher wird geleert, wenn der Gesamtspeicher nicht mehr ausreicht.

Eine hohe Auslastung des Auslagerungsspeichers weist auf zu wenig Systemspeicher hin und wirkt sich auf die Leistung des Systems aus. Sie können die Werte für „Swap in“ und „Swap out“ durch Auswahl von Swap im oberen Dropdown-Menü anzeigen.

Netzwerk

Das Netzwerk-Durchflussdiagramm zeigt die Daten, die in KB pro Sekunde gesendet und empfangen werden. Wenn PPPoE aktiviert ist, wird die Übertragungsrate auch auf dem Diagramm angezeigt.

Festplatte

Sie können die gesamte Festplattennutzung, die Übertragungsrate oder IOPS anzeigen, indem Sie oben im Diagramm auf das Dropdown-Menü klicken. Sie können unten auf Alle anzeigen klicken, um weitere Datenträger durch Markieren der Kästchen auszuwählen und deren Daten zur Darstellung hinzuzufügen.

Volume

Diese Seite zeigt den Gesamtübertragungsstatus von Volumes an. Klicken Sie auf das Dropdown-Menü über dem Diagramm, um die Übertragungsauslastung, Übertragungsrate oder IOPS anzuzeigen. Klicken Sie auf Alle anzeigen, um einzelne Volumes auszuwählen und zum Diagramm hinzuzufügen.

iSCSI

Diese Seite zeigt den Gesamtstatus der aktuell ausgeführten iSCSI-LUNs an. Klicken Sie auf das Dropdown-Menü über dem Diagramm, um eine LUN auszuwählen und ihren Status anzuzeigen. Drei Diagramme werden standardmäßig angezeigt: Durchsatz, IOPS und Latenz. Für detailliertere Informationen klicken Sie auf Diagramme auswählen, um weitere Diagramme anzuzeigen.
