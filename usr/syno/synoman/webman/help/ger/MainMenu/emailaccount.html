E-Mail-Konto

Sie können Ihre E-Mail-Konten unter Persönlich > E-Mail-Konto hinzufügen und können in File Station gespeicherte Dateien von hier hinzugefügten E-Mail-Adressen als Anhänge versenden. Darüber hinaus können Sie Ihr E-Mail-Konto auch unter Einstellungen > E-Mail-Konto in Drive und Calendar hinzufügen. Nachdem das Konto hinzugefügt wurde, können Sie E-Mail-Einladungen zu Ereignissen über Calendar und E-Mail-Benachrichtigungen bei der Freigabe von Dateien auf Drive versenden.

Zu den unterstützten E-Mail-Anbietern gehören Gmail, Outlook und benutzerdefinierte SMTP-Server.

E-Mail-Konto hinzufügen:

  1. Klicken Sie auf Hinzufügen und wählen Sie den Anbieter für das E-Mail-Konto, das Sie einrichten möchten.
  2. Geben Sie Ihre E-Mail-Adresse, Passwort und den Absendernamen ein.
  3. Markieren Sie Dieses E-Mail-Konto als Standard verwenden, wenn Sie möchten, dass File Station, Calendar und Drive standardmäßig dieses E-Mail-Konto nutzen, wenn Sie mehrere Konten eingerichtet haben.
  4. Klicken Sie auf Kontakte synchronisieren, um Ihre E-Mail-Kontakte mit Synology NAS zu synchronisieren, wenn Ihr E-Mail-Anbieter (Gmail oder Outlook) diese Funktion unterstützt. Beachten Sie, dass die Kontakte, die synchronisiert werden, eher aus dem E-Mail-Konto stammen, das Sie im Authentifizierungsfenster festgelegt haben, als aus demjenigen, dass Sie im vorherigen Schritt eingegeben haben. Wenn daher ein anderes oder unerwünschtes Konto im Authentifizierungsfenster angezeigt wird, wechseln Sie bitte manuell auf das gewünschte E-Mail-Konto oder auf das, das Sie zuvor eingegeben haben.
  5. Klicken Sie auf Anwenden, um die Einstellungen zu speichern.

E-Mail-Konto bearbeiten:

Wählen Sie das gewünschte E-Mail-Konto aus der Liste aus und klicken Sie auf Bearbeiten.

E-Mail-Konto löschen:

Wählen Sie das gewünschte E-Mail-Konto aus der Liste aus und klicken Sie auf Löschen.

Anmerkung:

  • Das Standard-Konto kann nur gelöscht werden, wenn keine anderen Konten vorhanden sind. Möchten Sie ein E-Mail-Konto löschen, das derzeit als Standard festgelegt ist, dann legen Sie bitte ein anderes Konto als Ihr Standardkonto fest, bevor Sie fortfahren.

Standardkonto ändern:

Wählen Sie aus der Liste das E-Mail-Konto aus, dass Sie als Standard festlegen möchten, und klicken Sie auf Als Standard festlegen.

Dateien als Anhänge senden:

  1. Rechtsklicken Sie in File Station auf Dateien und wählen Sie Als E-Mail-Anhänge senden.
  2. Wenn Sie mehr als ein E-Mail-Konto hinzugefügt haben, können Sie Ihre gewünschte Absender-E-Mail-Adresse in der Dropdown-Liste auswählen, um die Dateien an einen angegebenen Empfänger zu senden.
