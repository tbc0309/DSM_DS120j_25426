Optionen

Sie können das Menü Optionen anzeigen, indem Sie das Symbol einer Person oben rechts auf dem DSM-Desktop anklicken. Hier haben Sie folgende Möglichkeiten:

  • Persönliche Kontooptionen ändern
  • Abmelden
  • Halten Sie das System an (auf EDS14) oder fahren Sie es herunter (auf allen anderen Modellen)
  • Das System neu starten

Nach dem Anzeigen des Menüs klicken Sie auf Persönlich, um die persönlichen Kontoeinstellungen zu bearbeiten.
