Target

Je nach Modell unterstützt Synology NAS die Erstellung mehrerer Targets, denen LUNs zugeordnet werden können.

Anmerkung: Die Anzahl unterstützter Targets unterscheidet sich je nach Synology NAS-Modell.

Ein Target erstellen:

  1. Klicken Sie auf die Schaltfläche Erstellen.
  2. Geben Sie die Informationen zum Target ein, wie Name, IQN und ob die CHAP-Authentifizierung aktiviert werden soll.
  3. Klicken Sie auf Weiter.
  4. Richten Sie die iSCSI LUN-Zuordnung ein. Wählen Sie Neue iSCSI LUN erstellen oder Vorhandene iSCSI LUNs zuordnen aus.
  5. Klicken Sie auf Weiter.
  6. Klicken Sie auf Zurück, um Änderungen vorzunehmen, oder auf Übernehmen, um die Einstellungen zu bestätigen.

Glossar

  • IQN: IQN (iSCSI Qualifizierter Name) ist ein spezifischer Name für ein Target. Er darf nicht mit Target-IQNs auf anderen Servern in Konflikt stehen. IQN muss weniger als 128 Zeichen umfassen und darf Buchstaben (Groß- oder Kleinschreibung), Ziffern, Doppelpunkte, Bindestriche und Punkte enthalten. Das vorgeschlagene Benennungsformat ist iqn.[date].[domain]:[device identifier], also z. B.: iqn.2000-04.com.synology:storage.diskarray.1.
  • Gegenseitige CHAP-Authentifizierung: Unter Gegenseitiger CHAP authentifizieren Client und Server sich gegenseitig. Ein zur Client-Authentifizierung verwendetes CHAP-Kennwort darf nicht für die Authentifizierung eines Servers konfiguriert werden und ein zur Server-Authentifizierung verwendetes CHAP-Kennwort darf nicht für die Authentifizierung eines Clients konfiguriert werden.

    • Anmerkung：Die Erkennung auf dem iSCSI-Initiator kann fehlschlagen, wenn die CHAP-Authentifizierung auf Ihrem NAS aktiviert ist. Bitte führen Sie erst die Erkennung durch und aktivieren Sie danach die CHAP-Authentifizierung, bevor Sie sich mit einem Target verbinden. Alternativ können Sie sich auch mit IP und IQN mit einem Target verbinden.

Ein Target entfernen:

Sie können ein Target und beliebige zugeordneten LUNs entfernen. Wenn Sie eine iSCSI LUN entfernen, werden allen Daten in der iSCSI LUN gelöscht.

  1. Wählen Sie das Target aus, das Sie entfernen möchten.
  2. Klicken Sie auf Entfernen.
  3. Wenn Sie zugeordnete LUNs entfernen möchten, wählen Sie Zusätzlich iSCSI LUNs Zuordnungen löschen und wählen Sie diejenigen aus, die Sie entfernen möchten.
  4. Klicken Sie auf OK.

Ein Target bearbeiten:

  1. Wählen Sie das Target aus, das Sie bearbeiten möchten.
  2. Wählen Sie Bearbeiten im Dropdown-Menü Aktion aus.
  3. Bearbeiten Sie die Einstellungen nach Ihren Wünschen. Klicken Sie auf OK, um die Änderungen zu speichern.

Glossar

  • CRC Prüfsumme: Aktivieren Sie die CRC Prüfsumme, um zu prüfen, ob bei der Datenübertragung ein Fehler aufgetreten ist.
  • Mehrere Sitzungen: Ermöglicht mehreren Sitzungen von mehreren Initiatoren die gleichzeitige Verbindung mit demselben Target. Um Ihre Dateien vor Beschädigung oder Verlust zu schützen, sollten Sie ein freigegebenes Datensystem oder Cluster-System wie z. B. VMware Virtual Machine File System (VMFS) und Oracle Cluster File System (OCFS) verwenden.
  • Maskierung: Maskierung ermöglicht es, auf ein Target ausschließlich mit spezifischen iSCSI Initiator IQNs zuzugreifen.

Ein Target aktivieren oder deaktivieren:

  1. Wählen Sie das Target aus, das Sie aktivieren oder deaktivieren möchten.
  2. Wählen Sie Aktivieren oder Deaktivieren im Dropdown-Menü Aktion aus.
