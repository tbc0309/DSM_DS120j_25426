Schnappschuss

LUN-Schnappschuss erstellen:

Ein Schnappschuss einer LUN ist eine zu einem bestimmten Zeitpunkt erstellte schreibgeschützte Kopie einer LUN.

  1. Wählen Sie die LUN aus, für die Sie Schnappschüsse erstellen möchten.
  2. Klicken Sie auf Schnappschuss und wählen Sie Einen Schnappschuss machen.
  3. Geben Sie eine Beschreibung für diesen Schnappschuss ein.
  4. Wählen Sie je nach Bedarf einen der folgenden Schnappschuss-Typen aus:
    • Anwendungskonsistent: Dieser Schnappschuss-Typ kann nur erstellt werden, wenn Synology Storage Console oder Synology Snapshot Manager für die Kommunikation mit DSM je nach Betriebsumgebung auf Ihrem Windows-Server bzw. VMware vCenter Server installiert ist. Wenn ein anwendungskonsistenter Schnappschuss in DSM ausgelöst wird, werden Anwendungen benachrichtigt, die daraufhin alle Daten aus dem Speicher zur LUN übertragen, auf der der Schnappschuss erstellt wird, um die Konsistenz der Daten zu gewährleisten.
    • Absturzkonsistent: Wenn auf keinem Ihrer Clients Storage Console oder Snapshot Manager installiert ist, können Sie nur absturzkonsistente Schnappschüsse von LUNs erstellen. Wenn diese ausgelöst werden, werden Anwendungen nicht angewiesen, Daten vom Speicher zur LUN zu übertragen, sodass die Datenkonsistenz nicht gewährleistet ist.
  5. Klicken Sie auf OK.

Anmerkung:

  • Es wird empfohlen, Synology Storage Console oder Snapshot Manager zu installieren, um mögliche Dateninkonsistenzen zu vermeiden.

Glossar:

  • Synology Storage Console: Synology Storage Console ist ein Plugin zur Speicherverwaltung im VMware vSphere Client oder Windows-Server, das die Verwaltung und Überwachung mehrerer Synology-Speichersysteme vereinfacht. Sie können damit auch anwendungskonsistente Schnappschüsse direkt von DSM aus erstellen.
    • Für VMware-Umgebungen ist das Plugin nur mit VMware vCenter Server Appliance 6.5 U2 oder höheren Versionen kompatibel.
    • Sie können das Plugin im Download-Zentrum herunterladen und weitere Informationen in der Synology Storage Console for VMware und Synology Storage Console for Windows finden.
  • Synology Snapshot Manager: Synology Snapshot Manager ist ein Software-Plugin, mit dem Sie anwendungskonsistente Schnappschüsse direkt von DSM aus erstellen können.
    • Für VMware-Umgebungen ist dieses Plugin nur mit VMware vCenter Server 6.0 oder älteren Versionen kompatibel.

So löschen Sie einen Schnappschuss:

  1. Wählen Sie die LUN aus, für die Sie Schnappschüsse erstellt haben.
  2. Klicken Sie auf Schnappschuss > Schnappschussliste.
  3. Wählen Sie den Schnappschuss aus, den Sie löschen möchten, und klicken Sie auf Entfernen.

Anmerkung:

  • Die Verarbeitung der Speicherplatzrückgewinnung nimmt etwas Zeit in Anspruch, nachdem Schnappschüsse gelöscht wurden.

Eine LUN wiederherstellen:

Wählen Sie einen Schnappschuss zu einem bestimmten Zeitpunkt aus, um die Daten auf einer LUN wiederherzustellen.

  1. Wählen Sie die LUN aus, die Sie wiederherstellen möchten.
  2. Klicken Sie auf Schnappschuss und wählen Sie Schnappschussliste.
  3. Wählen Sie einen Schnappschuss, den Sie wiederherstellen möchten und klicken Sie auf Wiederherstellen.

Eine LUN aus Schnappschüssen klonen:

Erstellen Sie eine nicht schreibgeschützte Kopie von LUN oder LUN-Schnappschuss.

  • Wählen Sie einen LUN Schnappschuss aus der Schnappschussliste aus und klicken Sie auf Klonen.

Anmerkung:

  • Die Funktionen „LUN-Schnappschuss“, „Wiederherstellen“ und „Klonen“ sind nur auf LUNs mit Speicherbeschleunigungsbefehlen verfügbar.
  • Wenn Sie einen Schnappschuss oder einen Klon erstellt haben und der Status Abnormal/Abgestürzt lautet, löschen Sie bitte diesen Schnappschuss/Klon und erstellen Sie einen anderen.
  • Alle nach dem Erstellen eines Schnappschusses an einer LUN vorgenommenen Änderungen gehen verloren, wenn Sie die LUN mit diesem Schnappschuss wiederherstellen.
  • Die Funktionen, einschließlich dem Wiederherstellen und Klonen von LUNs sowie dem Erstellen und Klonen von LUN-Schnappschüssen, sind nicht verfügbar, wenn ein Volume weniger als 1 GB freien Speicherplatz hat.

Aufbewahrungsrichtlinie

Diese Option bietet eine ideale Lösung zum Festlegen der maximalen Anzahl von Schnappschussversionen, um Volume-Speicherplatz zu sparen. Möglicherweise müssen Sie Ihre Schnappschüsse aber für längere Zeit aufbewahren. Einige Dinge, die Sie zur Aufbewahrungsrichtlinie wissen sollten:

  • Synology nutzt die GFS-Aufbewahrungsrichtlinie („Grandfather-Father-Son“). Sie können die maximale Anzahl von aufzubewahrenden Schnappschussversionen für die folgenden Zeiträume konfigurieren: stündlich, täglich, wöchentlich, monatlich und jährlich.
  • Ein Schnappschuss kann gleichzeitig mehrere Aufbewahrungsrichtlinien erfüllen. Beispielsweise kann ein Schnappschuss die Regeln für die tägliche, wöchentliche und monatliche Schnappschüsse erfüllen.
  • Wenn Sie zahlreiche Schnappschüsse innerhalb eines bestimmten Zeitraums erstellen, werden die frühesten Schnappschüsse aufbewahrt. Beispiel: Wenn Sie eine Aufbewahrungsrichtlinie für 10 wöchentliche Schnappschüsse einrichten und gemäß Ihrem Zeitplan 1 Schnappschuss pro Tag aufgenommen wird, bewahrt Snapshot Replication den frühesten Schnappschuss aus jeder der letzten 10 Wochen auf.
  • Wenn Sie innerhalb eines bestimmten Zeitraums keine Schnappschüsse erstellt haben, bewahrt das System die jüngsten Schnappschüsse auf. Beispiel: Wenn als Regel für tägliche Schnappschüsse 15 eingestellt ist und in den letzten 30 Tagen keine Schnappschüsse erstellt wurden, bewahrt das System die vor 30 bis 45 Tagen erstellten Schnappschüsse auf.
  • Damit Sie die aktuellsten Schnappschussversionen schnell finden und wiederherstellen können, werden alle Schnappschussversionen innerhalb einer Stunde nach Aufnahme des neuesten Schnappschusses standardmäßig aufbewahrt. Um zu verhindern, dass ein Schnappschuss aufgrund der Aufbewahrungsregeln automatisch entfernt wird, können Sie ihn sperren.

Beispiel 1:
Wenn Sie für die letzten 3 Monate tägliche Schnappschüsse, für die letzten 4 bis 6 Monate wöchentliche Schnappschüsse und für die letzten 7 bis 9 Monate monatliche Schnappschüsse aufbewahren möchten, müssen Sie „0“ für stündliche Schnappschüsse, „90“ für tägliche Schnappschüsse, „24“ für wöchentliche Schnappschüsse, „9“ für monatliche Schnappschüsse und „0“ für jährliche Schnappschüsse eingeben.

Beispiel 2:
Wenn Sie für die letzten 2 Tage stündliche Schnappschüsse, für die letzte Woche tägliche Schnappschüsse, für den letzten Monat wöchentliche Schnappschüsse und für das letzte Jahr monatliche Schnappschüsse aufbewahren möchten, müssen Sie „48“ für stündliche Schnappschüsse, „7“ für tägliche Schnappschüsse, „4“ für wöchentliche Schnappschüsse, „12“ für monatliche Schnappschüsse und „0“ für jährliche Schnappschüsse eingeben.
