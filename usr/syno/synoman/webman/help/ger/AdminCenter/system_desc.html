System

In diesem Abschnitt können Sie die Systemeinstellungen für Synology NAS verwalten. Sie können den Anmeldebildschirm anpassen, Zeitzone und Spracheinstellungen ändern, einen Zeitplan für das automatische Starten/Herunterfahren (DC-Ausgang ein/aus für EDS14) festlegen, externe Geräte verwalten, die DSM-Version aktualisieren oder das Gerät auf die Werkseinstellungen zurücksetzen.
