Erweitert

In der Registerkarte Erweitert können Sie die Funktion Datei-Schnellklonen aktivieren, um beim Klonen von Dateien Speicherplatz zu sparen. Außerdem können Sie die Einstellungen so konfigurieren, dass Benutzer diesen Synology NAS im Netzwerk über die Bonjour- und SSDP (Simple Service Discovery Protocol)-Dienste finden können.

Datei-Schnellklonen auf Ihrem Synology NAS ausführen:

Mit der Funktion „Datei-Schnellklonen“ werden physische Datenblöcke nur kopiert, wenn die geklonten Dateien geändert wurden. Dies hilft, Speicherplatz zu sparen.

  1. Aktivieren Sie im Bereich Datei-Schnellklonen das Kontrollkästchen Datei-Schnellklonen aktivieren.
  2. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

So machen Sie Ihren Synology NAS über den Bonjour-Dienst erkennbar:

Mit dem Bonjour-Dienst können Nutzer den Synology NAS finden und Druckaufgaben sowie Sicherungsaufgaben über den Mac-Computer ausführen.

  1. Aktivieren Sie im Bereich Bonjour das Kontrollkästchen Bonjour-Diensterkennung aktivieren.
  2. Sie können Bonjour-bezogene Optionen ganz nach Wunsch auswählen:
    • Bonjour Printer Broadcast aktivieren: Lassen Sie zu, dass Mac-Computer den mit Ihrem Synology NAS verbundenen USB-Drucker zum Ausdrucken von Dokumenten finden.
    • Bonjour Time Machine Broadcast über SMB aktivieren: Lassen Sie zu, dass Mac Time Machine Ihren Synology NAS findet und Dateien in freigegebenen DSM-Ordnern über SMB sichert.
    • Bonjour Time Machine Broadcast über AFP aktivieren: Lassen Sie zu, dass Mac Time Machine Ihren Synology NAS findet und Dateien in freigegebenen DSM-Ordnern über AFP sichert.
  3. Wenn Sie eine Option für Time Machine gewählt haben, klicken Sie auf Time Machine-Ordner einrichten, um Zielordner für Time Machine-Sicherungen festzulegen.
  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Anmerkung:

  • Wenn Sie den SMB/AFP-Dienst deaktivieren (unter Systemsteuerung > Dateidienste > SMB/AFP/NFS), wird die zugehörige Bonjour-Option automatisch deaktiviert.
  • Wenn Bonjour Time Machine Broadcast über SMB aktivieren ausgewählt ist, werden die folgenden Optionen aktiviert (unter Systemsteuerung > Dateidienste > SMB/AFP/NFS > Erweiterte Einstellungen):
    • SMB 3 aktivieren (für Max. Protokoll für Windows-Dateidienst)
    • Opportunistic Locking
    • SMB2 Lease aktivieren
    • SMB permanente Handles aktivieren (die protokollübergreifende Dateisperre wird deaktiviert)

So machen Sie Ihren Synology NAS über SSDP erkennbar:

Mit dem SSDP-Dienst können Benutzer den Synology NAS über Windows-Netzwerkerkennung finden.

  1. Aktivieren Sie im Bereich SSDP das Kontrollkästchen Windows-Netzwerkerkennung aktivieren, um Zugriff über Web-Browser zu erlauben.
  2. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

So machen Sie Ihren Synology NAS über WS-Discovery erkennbar:

Mit dem Dienst WS-Discovery können Benutzer den SMB-Dienst auf dem Synology NAS über die Windows-Netzwerkerkennung finden.

  1. Aktivieren Sie im Bereich WS-Discovery das Kontrollkästchen Windows-Netzwerkerkennung aktivieren, um den Dateizugriff über SMB zu erlauben.
  2. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.
