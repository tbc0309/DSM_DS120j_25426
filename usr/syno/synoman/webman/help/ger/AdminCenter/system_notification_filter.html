Erweitert

Sie können die Arten von Ereignissen auswählen, welche das System veranlassen, Benachrichtigungen zu senden, sowie das Medium (E-Mail, SMS und Mobilgeräte), über das Sie Nachrichten empfangen.

Für jedes der Ereignisse gibt es eine eigene Standardbenachrichtigung, aber Sie haben auch die Option, die Inhalte der Benachrichtigung zu ändern. Die Standardnachricht kann später wiederhergestellt werden, wenn Sie ihre Änderungen rückgängig machen möchten.

Nach Benachrichtigungsereignissen suchen:

  1. Geben Sie einen Suchbegriff in das Suchfeld oben rechts ein.
  2. Es werden alle Ereignisse in Zusammenhang mit dem eingegebenen Suchbegriff angezeigt.

Benachrichtigungsmedien für Benachrichtigungsereignisse auswählen:

  1. Markieren Sie die Kontrollkästchen der Ereignisse, um die bevorzugten Benachrichtigungsmedien auszuwählen.
  2. Klicken Sie auf Speichern, um die Einstellungen zu übernehmen.

Benachrichtigungen bearbeiten:

  1. Wählen Sie ein Ereignis aus der Ereignisliste aus und klicken Sie auf Nachricht bearbeiten.
  2. Im eingeblendeten Fenster können Sie den Betreff und den Haupttext bearbeiten, um die Benachrichtigung individuell anzupassen.
  3. Sie können dabei auch die folgenden Variablen ändern. Aktualisierte Variablen werden für alle Benachrichtigungen übernommen:
    • %COMPANY_NAME%: Die Signatur, die am Ende einer Benachrichtigung angezeigt wird. Der Standardwert ist Synology DiskStation, Synology RackStation, Synology FlashStation oder Synology Unified Controller, je nach Modell.
    • %HTTP_URL%: Ein Hyperlink zu Ihrem Synology NAS. Wird kein Wert angegeben, führt das System alle möglichen Links in der Benachrichtigung auf.
  4. Klicken Sie auf Speichern, um die Änderungen zu übernehmen.
  5. Standardeinstellungen der Benachrichtigung wiederherstellen:
    1. Wählen Sie ein Ereignis aus der Ereignisliste aus und klicken Sie auf Nachricht bearbeiten.
    2. Klicken Sie im angezeigten Fenster auf Standard wiederherstellen und anschließend im Bestätigungsdialogfeld auf Ja.

Anmerkung:

  • Es wird empfohlen, dass Sie die Textzeilensegmente mit '%' sowohl im Betreff als auch im Haupttext unverändert belassen. Der Textteil wird durch tatsächliche Systeminformationen ersetzt, wenn eine Benachrichtigung gesendet wird.
