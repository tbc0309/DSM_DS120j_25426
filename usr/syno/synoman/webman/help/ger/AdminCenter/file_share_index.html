Datei-Indizierung

Durch Indizieren eines Ordners oder seiner Unterordner können Sie die darin enthaltenen Dateien und Inhalte bei der Dateisuche schneller finden.

Hinweis:

Remote-Ordner, Cloud-Ordner oder an Ihr Synology NAS angehängte .iso-Ordner können nicht indiziert werden.

Inhalt von Ordnern und Dateien indizieren:

Wenn Sie einen Ordner indizieren, werden die darin enthaltenen Unterordner und Dateien von File Station nach Name indiziert, sodass Sie mit dem Dateinamen nach gewünschten Elementen suchen können. Wenn Sie eine weitere Indizierung nach Dateityp vornehmen, indiziert File Station die entsprechenden Dateien nicht nur nach Name, sondern auch nach Inhalt; so können Sie sowohl nach Dateiname wie auch nach Dateiinhalt suchen.

  1. Gehen Sie zu Systemsteuerung > Indizierungsdienst > Dateiindizierung und klicken Sie auf Liste indizierter Ordner.
  2. Klicken Sie im eingeblendeten Fenster auf Erstellen > Auswählen, um die Ordner für die Indizierung anzuzeigen.
  3. Wählen Sie den gewünschten Ordner aus und klicken Sie auf Auswählen. Um einen neuen Ordner für die Indizierung zu erstellen, klicken Sie auf Ordner erstellen und fahren Sie fort.
  4. Klicken Sie auf den Ordner, den Sie indizieren möchten, und dann auf Auswählen.
  5. Wenn Sie bestimmte Dateitypen indizieren möchten, gehen Sie zu Dateityp und wählen Sie Bild, Audio, Video oder Dokument aus. Wenn ein Dateityp (z. B. Audio) nicht ausgewählt wurde, können Sie Dateien dieses Typs (z. B. Audiodateien) nur nach Dateiname suchen.
  6. Klicken Sie zum Beenden der Indizierung auf OK.

Hinweis:

  • Sie können nur bis zu 1.000 Ordner für die Indizierung auswählen, wobei Unterordner in den ausgewählten Ordnern nicht mitgezählt werden.
  • Wenn Sie in File Station Durchsuchen des Dateiinhalts aktivieren auswählen, können Sie den Dateiinhalt in den indizierten Ordnern durchsuchen.
  • Das Volume, das den indizierten Ordner hostet, muss mehr als 10 MB freien Speicherplatz haben, ansonsten wird die Indizierung unterbrochen. Erweitern Sie das Volume und klicken Sie auf Neu indizieren, um die Indizierung fortzusetzen.
  • Wenn Sie einen Ordner und dessen Unterordner gleichzeitig indiziert haben, indiziert das System nur den (die) Unterordner, um Systemressourcen einzusparen.
  • Nachdem Ordner vollständig indiziert und über SMB- oder AFP-Protokoll an einen Mac-Computer angehängt wurden, können Sie mit Finder die Dateiinhalte durchsuchen.

Indizierte Ordner löschen:

  1. Gehen Sie zu Systemsteuerung > Indizierungsdienst > Dateiindizierung > Liste indizierter Ordner und wählen Sie den (die) gewünschten indizierten Ordner aus.
  2. Klicken Sie auf Löschen, um die Indizierung des Ordners bzw. der Ordner zu stoppen.

Dateien neu indizieren

Eine erneute Indizierung von Dateien ist unter normalen Umständen nicht erforderlich. Stimmen Dateien in einem Ordner mit bestimmten Suchkriterien überein, wurden aber während der Suche nicht angezeigt, müssen Sie die Dateien möglicherweise neu indizieren, um die Indexstruktur der Dateien neu zu aufzubauen.

Dateien in einem freigegebenen Ordner neu indizieren:

  1. Gehen Sie zu Systemsteuerung > Indizierungsdienst > Dateiindizierung > Liste indizierter Ordner.
  2. Wählen Sie den (die) aufgeführten Ordner aus, den (die) Sie neu indizieren möchten.
  3. Klicken Sie auf Neu indizieren.

Hinweis:

File Station unterstützt keine Dateiindizierung von Remote-Ordnern oder auf externen Speichergeräten (z. B. USB-Geräte/SD-Karten).

Dateiindizierung über den Desktop verwalten

Wenn die Dateiindizierung beginnt, wird der Indizierungsfortschritt in der Taskleiste angezeigt (Vergrößerungsglas). Da die Dateiindizierung mehr CPU-Ressourcen benötigt, können Sie damit die Nutzung der CPU-Ressourcen optimieren.

  • Klicken Sie auf die Schaltfläche Wiedergabe, um die Dateiindizierung anzuhalten bzw. fortzusetzen.
  • Wenn Sie die Dateiindizierung bis zu einem passenden Zeitpunkt verzögern möchten, wählen Sie im Dropdown-Menü die gewünschte Option aus.
  • Um anzuzeigen, welche Ordner indiziert werden, klicken Sie auf Liste der indizierten Ordner anzeigen.

Hinweis:

  • Wenn Sie die Indizierung anhalten, können Sie sie nur durch Anklicken der Schaltfläche Wiedergabe in der Taskleiste am Desktop fortsetzen.
