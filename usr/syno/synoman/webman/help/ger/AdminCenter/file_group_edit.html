Eine Gruppe bearbeiten

Sie können Gruppeninformationen sowie die Rechte für den Zugriff auf freigegebene Ordner bearbeiten.

Gruppeninformationen bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten, und bearbeiten Sie die Gruppeninformationen.
  3. Klicken Sie zum Beenden auf OK.

Gruppenzugriffsrechte für freigegebene Order bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und Berechtigungen.
  3. Aktivieren oder deaktivieren Sie die folgenden Rechte, um der Gruppe Zugriffsberechtigungen zuzuweisen:
    1. Lesen/Schreiben: Die Gruppe hat Zugriff und kann Änderungen am freigegebenen Ordner vornehmen.
    2. Nur Lesen: Die Gruppe hat Zugriff, kann jedoch keine Änderungen am freigegebenen Ordner vornehmen.
    3. Kein Zugriff: Die Gruppe hat keinen Zugriff auf den freigegebenen Ordner.
  4. Klicken Sie zum Beenden auf OK.

Anmerkung:

Beachten Sie bei Berechtigungskonflikten folgende Priorität: Kein Zugriff > Lesen/Schreiben > Nur Lesen.

Kontingent einer Gruppe bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und gehen Sie zur Registerkarte Quote.
  3. Aktivieren Sie für jedes Volume bzw. jeden freigegebenen Ordner das Kontrollkästchen unter Kontingent aktivieren. Geben Sie dann in das Feld Quote ein Kontingent ein (z. B. 10 GB).

Anmerkung:

  • Jedes Gruppenmitglied übernimmt das angegebene Gruppenkontingent. Das Gruppenkontingent wird nicht auf die einzelnen Gruppenmitglieder aufgeteilt (nicht anwendbar für EDS14 oder DDSM).
  • Das maximale für jede Gruppe einstellbare Speicherkontingent beträgt ca. 4 TB (4095 GB).
  • Das Bearbeiten des Kontingents einer Gruppe durch freigegebene Ordner wird nur in VDSM und von Modellen mit folgender Paketarchitektur unterstützt: Apollo Lake, Avoton, Braswell, Broadwell, Bromolow, Cedarview und Grantley. Welche Paketarchitektur Ihr Modell unterstützt erfahren Sie in diesem Artikel.
  • DDSM unterstützt diese Funktion nicht.

Gruppenzugriffsrechte auf Anwendungen bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten, und gehen Sie zur Registerkarte Anwendungen.
  3. Für jeden Dienst gibt es die folgenden Optionen:
    • Zulassen: Die Gruppe kann auf die Anwendung zugreifen.
    • Verweigern: Die Gruppe kann nicht auf die Anwendung zugreifen.
    • Benutzerdefiniert: Zugriffsrechte über die IP-Adresse verwalten.
  4. Klicken Sie zum Beenden auf OK.

Anmerkung:

  • Beachten Sie bei Berechtigungskonflikten folgende Priorität: Zulassen > Verweigern.
  • Nicht alle Pakete und Dienste unterstützen die Zugriffsrechte-Einstellungen Nach IP.

Geschwindigkeitseinschränkung für eine Gruppe bearbeiten:

Sie können für verschiedene Dienste (z. B. File Station, FTP usw.) eine Geschwindigkeitseinschränkung für eine Gruppe festlegen, um die von den Gruppenmitgliedern für die Datenübertragung über die Dienste verbrauchte Bandbreite zu beschränken. Geschwindigkeitseinschränkungen für Gruppen gelten für alle Gruppenmitglieder.

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und gehen Sie zur Registerkarte Geschwindigkeitseinschränkung.
  3. Für jeden Dienst gibt es die folgenden Optionen:
    • Keine: Für die Nutzung des Dienstes gibt es für die Gruppenmitglieder keine Geschwindigkeitseinschränkung.
    • Geschwindigkeitsbegrenzung einrichten: Für alle Gruppenmitglieder gilt eine feste, konstante Geschwindigkeitseinschränkung. Geben Sie die Upload- und Download-Geschwindigkeitseinschränkungen in den Feldern links an.
    • Grenzwertbereich anpassen: Für die Gruppenmitglieder gelten zwei unterschiedliche Geschwindigkeitseinschränkungen gemäß einem Zeitplan. Klicken Sie auf die Schaltfläche Anpassen, um die Einstellungen für die Geschwindigkeitseinschränkung und den Zeitplan zu ändern.
  4. Klicken Sie zum Beenden auf OK.

Geschwindigkeitseinschränkung für Benutzer, die zu mehreren Gruppen gehören

Wenn ein Benutzer zu mehreren Gruppen gehört, hängt die Geschwindigkeitseinschränkung von der Gruppe mit der höchsten verfügbaren Bandbreite zum Zeitpunkt der Übertragung von Dateien ab. Da die Bandbreite gleichmäßig auf Benutzer, die Dateien übertragen, verteilt wird, hängt die verfügbare Bandbreite davon ab, wie viele Benutzer gleichzeitig die Bandbreite der Gruppe verwenden.

Der Benutzer "John" gehört z. B. zur "Gruppe 1" (Geschwindigkeitseinschränkung 500 KB/s; insgesamt fünf Benutzer) und "Gruppe 2" (Geschwindigkeitseinschränkung 400 KB/s; insgesamt zwei Benutzer).

  • Wenn alle Benutzer in Gruppe 1 und Gruppe 2 gleichzeitig Dateien übertragen, beträgt die Geschwindigkeitseinschränkung für John 200 KB/s und sein Verbrauch wird auf die gemeinsame Bandbreite von Gruppe 2 angerechnet.
  • Wenn die anderen Benutzer in Gruppe 1 mit der Übertragung von Dateien fertig sind, wird die Geschwindigkeitseinschränkung von John sofort auf 500 KB/s umgestellt und sein Verbrauch auf die gemeinsame Bandbreite von Gruppe 1 angerechnet.
