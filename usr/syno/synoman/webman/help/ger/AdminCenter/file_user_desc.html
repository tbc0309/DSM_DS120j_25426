Benutzer

Mit den flexiblen Benutzerverwaltungsoptionen können Sie Benutzerkonten für einzelne Familienmitglieder oder Mitarbeiter eines Unternehmens erstellen. Durch das Erstellen von Benutzern können Berechtigungen für jeden festgelegt und verwaltet werden, beispielsweise Zugriffsberechtigungen für freigegebene Ordner, Quoten für Speicher und freigegebene Ordner oder Bandbreitenbeschränkungen.

    Anmerkung:
  1. Verwaltung freigegebener Ordner ist nur auf folgenden Plattformen verfügbar: Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake, Denverton. In diesem Artikel erfahren Sie, ob Ihr Modell diese Funktion unterstützt.
  2. Quotenverwaltung für Speicher und freigegebene Ordner ist für Docker DSM nicht anwendbar.
