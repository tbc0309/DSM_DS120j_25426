SMS

Sie können SMS-Benachrichtigungen aktivieren, damit Synology NAS Ihnen Benachrichtigungen per SMS zusendet, wenn es zu Statuswechseln oder Fehlern im System kommen sollte, z. B. unterbrochene Verbindung, verändertes Verhalten des Lüfters, Stromausfall, zur Neige gehender Speicherplatz auf dem Volume oder Fehler bei Sicherung und Wiederherstellung usw.

Anmerkung:

  • Das Ereignis „Verändertes Lüfterverhalten“ ist bei lüfterlosen Modellen wie EDS14 nicht verfügbar. Weitere Informationen erhalten Sie auf der Synology-Website.

SMS-Benachrichtigungen aktivieren:

Markieren Sie SMS-Benachrichtigungen aktivieren und legen Sie folgende Einstellungen fest:

  1. Wählen Sie einen SMS-Anbieter aus dem Dropdown-Menü der SMS-Serviceanbieter. Die Standardanbieter sind „clickatell“, „clickatell-2017“ und „SendinBlue-v3“.
  2. Klicken Sie auf SMS-Serviceanbieter hinzufügen, um eine neue SMS-Anbieterkonfiguration zu erstellen, oder klicken Sie auf Löschen, um die ausgewählte SMS-Anbieterkonfiguration zu löschen.
  3. Wenn Ihr SMS-Serviceanbieter Benutzername und Passwort für die Authentifizierung nutzt, wählen Sie bitte die entsprechenden Kategorien für die Parameter aus. Geben Sie in das Feld Benutzername den Benutzernamen Ihres SMS-Dienstkontos ein. Geben Sie in die Felder Passwort und Passwort bestätigen das Passwort Ihres SMS-Kontos ein und bestätigen Sie es.
  4. Wenn Ihr SMS-Serviceanbieter einen API Key für die Authentifizierung nutzt, wählen Sie bitte die Kategorie API Key für den entsprechenden Parameter aus. Geben Sie im Feld API Key den von ihrem SMS-Serviceanbieter bereitgestellten API Key ein.
  5. Wenn Ihnen Ihr SMS-Serviceanbieter einen Parameter gibt, um den Absendernamen zu verändern, wählen Sie bitte die Kategorie Absender für den entsprechenden Parameter aus. Geben Sie im Feld Absender den Namen des Absenders gemäß den Regeln des SMS-Serviceanbieters ein.
  6. Geben Sie Ihre Vorrangige Telefonnummer oder Zweite Telefonnummer ein. Wurden beide Nummern eingegeben, wird die SMS-Benachrichtigung an beide Nummern gesendet.
  7. Wenn Sie nicht zu viele Nachrichten erhalten möchten, markieren Sie SMS-Intervallgrenze aktivieren. Die Anzahl der Benachrichtigungen wird dann auf eine Nachricht pro festgelegtem Zeitintervall begrenzt.
  8. Klicken Sie auf Test-SMS senden, um eine Test-SMS an die festgelegte Telefonnummer zu senden. Wenn Sie keine Nachricht empfangen, ist die Konfiguration nicht korrekt.
  9. Klicken Sie auf Übernehmen, um die Konfiguration zu speichern, oder auf Abbrechen, um die Einrichtung abzubrechen.

Anmerkung:

  • Die Sprache der SMS-Benachrichtigung wird unter Benachrichtigungssprache eingestellt. Um die Sprache für SMS-Benachrichtigungen zu ändern, gehen Sie zu Hauptmenü > Systemsteuerung > Regionale Optionen, klicken Sie auf die Registerkarte Sprache, und wählen Sie die gewünschte Sprache aus dem Dropdown-Menü im Bereich Benachrichtigungssprache aus.
  • Alle SMS-Serviceanbieter senden Benachrichtigungen in englischer Sprache, ausgenommen die beiden Standard-SMS-Anbieter „clickatell“ und „clickatell-2017“.

Einschränkungen:

  • Benutzername: Groß- und Kleinschreibung wird berücksichtigt, der Name darf zwischen 1 und 32 Unicode-Zeichen enthalten.
  • Passwort: Groß- und Kleinschreibung wird berücksichtigt, muss 0 bis 128 anzeigbare Zeichen enthalten, einschließlich Buchstaben, Zahlen, sonstige Zeichen und Leerzeichen.
  • API Key: Groß- und Kleinschreibung wird berücksichtigt, muss zwischen 1 und 256 Unicode-Zeichen enthalten.
  • Absender: Groß- und Kleinschreibung wird berücksichtigt, muss zwischen 1 und 64 Unicode-Zeichen enthalten.
  • Die SMS-Intervallgrenze muss zwischen 1 und 10000 Minuten liegen.

So erstellen Sie eine SMS-Anbieterkonfiguration:

  1. Klicken Sie auf SMS-Serviceanbieter hinzufügen.
  2. Geben Sie in das Feld Name des Anbieters den Namen Ihres SMS-Serviceanbieters ein.
  3. Geben Sie im Feld SMS-URL die SMS-URL ein. Sie erhalten die SMS-URL von Ihrem SMS-Dienstanbieter.
  4. Wählen Sie GET oder POST unter HTTP-Methode. Diese Informationen finden Sie in den API-Dokumenten Ihres SMS-Serviceanbieters.
  5. Klicken Sie auf Weiter.
  6. Unter Header der HTTP-Anforderung bearbeiten müssen Sie die in den API-Dokumenten Ihres SMS-Serviceanbietern angeführten Parameter hinzufügen.
  7. Klicken Sie auf Weiter.
  8. Wenn Sie POST unter HTTP-Methode ausgewählt haben, müssen Sie unter Text der HTTP-Anforderung bearbeiten die in den API-Dokumenten Ihres SMS-Serviceanbietern angeführten Parameter hinzufügen.
  9. Klicken Sie auf Weiter.
  10. Das System unterteilt SMS-URL sowie Header und Text der HTTP-Anforderung in mehrere Parameter. Legen Sie jeden Parameter fest, indem Sie die entsprechende Kategorie auswählen. Jede Parameterkategorie kann nur einmal verwendet werden, mit Ausnahme von Sonstige.

Einschränkungen:

  1. Name des Anbieters: Groß- und Kleinschreibung wird berücksichtigt, der Name darf zwischen 1 und 32 Unicode-Zeichen enthalten.
  2. SMS-URL: Groß- und Kleinschreibung wird berücksichtigt, muss zwischen 1 und 500 Unicode-Zeichen enthalten.
    Diese SMS-URL dient nur für Einrichtungszwecke. Nach Abschluss der Einrichtung wird keine SMS-Nachricht gesendet. Das Format dieser URL wird vom SMS-Serviceanbieter vorgegeben.
    Bei Clickatell wäre eine mögliche URL beispielsweise: https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world
  3. Parameter: Groß- und Kleinschreibung wird berücksichtigt, muss zwischen 1 und 32 Unicode-Zeichen enthalten.
  4. Wert: Groß- und Kleinschreibung wird berücksichtigt, muss zwischen 1 und 256 Unicode-Zeichen enthalten.
  5. Sie können maximal 10 Parameter hinzufügen.
  6. Beachten Sie, dass im Parameter „Inhalt der Nachricht“ das Leerzeichen zwischen „Hello“ und „world“ durch ein vom Serviceanbieter definiertes Zeichen ersetzt wird.
    Bei Clickatell ist der Wert des Parameters „Inhalt der Nachricht“ beispielsweise „Hello+world“, da das „+“-Zeichen bei Clickatell als Leerzeichen dient.
