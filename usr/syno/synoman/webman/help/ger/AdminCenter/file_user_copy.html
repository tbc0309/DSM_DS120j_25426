Benutzer kopieren

Sie können die Einstellungen eines bestehenden Nutzers zu einem neuen Nutzer kopieren. Auf diese Weise können Sie einen neuen Nutzer schnell erstellen, ohne dieselben Einstellungen erneut konfigurieren zu müssen.

  1. Wählen Sie den Nutzer aus, den Sie kopieren möchten.
  2. Klicken Sie zuerst auf Erstellen> Benutzer kopieren. Das Fenster des Assistenten Benutzererstellung wird gestartet.
  3. Die Einstellungen des ausgewählten Nutzers werden standardmäßig für den neuen Nutzer verwendet (mit Ausnahme persönlicher Informationen wie Benutzername und E-Mail).
