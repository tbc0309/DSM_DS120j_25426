Hotspot*

Ihr Synology NAS-Gerät unterstützt die Einrichtung eines WLAN-Hotspots, um die Internetverbindung mit anderen Geräten kabellos zu teilen, wenn Sie einen WLAN-USB-Adapter installiert haben oder der Synology NAS ein integriertes WLAN besitzt. Sie können Ihren Synology NAS so einstellen, dass er als WLAN-Zugangspunkt (AP) oder WLAN-Router funktioniert, je nachdem, wie Sie das Internet nutzen möchten.

Hotspot-Modus Beschreibung
WLAN-AP
  • Schließt Ihr LAN mit dem WLAN-Netzwerk zusammen.
  • Ihre LAN- und WLAN-Clients befinden sich im selben lokalen Netzwerk.
  • Trifft zu, wenn Sie bereits über einen Router verfügen und den Synology NAS als zusätzlichen Zugangspunkt in einem bestehenden Netzwerk verwenden möchten.
  • Ermöglicht die gemeinsame Nutzung von Netzwerkressourcen wie DLNA-Geräten, Druckern (nicht anwendbar für Grantley-Plattformen; ob Ihr Modell diese Funktion unterstützt, verrät Ihnen dieser Artikel), Synology Web Assistant oder andere Dienste, welche die Übertragung über das Netzwerk nutzen.
  • Der DHCP Server wird auf Synology NAS-Geräten deaktiviert, die als WLAN-APs fungieren. Zur erneuten Aktivierung des DHCP-Servers des Synology NAS entfernen Sie die WLAN-Dongles und starten Sie Synology NAS neu.
  • Bei Modellen mit mehreren LAN-Ports wird nur LAN 1 zur Verbindung mit dem WLAN-AP verwendet.
WLAN-Router
  • Richtet ein WLAN-Netzwerk ein, in dem sich die Geräte in einem privaten Subnetz befinden.

Anmerkung:

  1. Einige Modelle erfordern den Anschluss eines WLAN-Dongles, um Hotspots zu erstellen. Eine komplette Liste der unterstützten WLAN-Dongles finden Sie unter www.synology.com.
  2. Für Nicht-DS213air-Modelle wird die Option WLAN-Hotspot von DSM 4.3 in DSM 5.0 und höher zu "Wireless Router" umbenannt.

Hotspot-Einstellungen verwalten

In diesem Abschnitt wird beschrieben, wie Sie die Hotspot-Einstellungen Ihres Synology NAS aktivieren und verwalten.

Synology NAS in einen Hotspot verwandeln:

  1. Gehen Sie zu Systemsteuerung > WLAN-Netzwerk > Netzwerkabbildung. Wählen Sie WLAN-AP oder WLAN-Router.
  2. Gehen Sie auf die Registerkarte WLAN, und klicken Sie auf Bearbeiten, um die folgenden Einstellungen zu verwalten:
    • Name (SSID): Geben Sie einen Namen für das WLAN-Netzwerk ein.
    • Anzeigen/Ausblenden: Wählen Sie, ob das WLAN-Netzwerk für andere Geräte sichtbar sein soll.
    • WLAN-Modus: Wählen Sie einen WLAN-Standard. Jeder Standard unterstützt unterschiedliche Bandbreiten und Frequenzen.
    • Kanalbreite: Wählen Sie eine Kanalbreite aus (nur verfügbar, wenn der 802.11n WLAN-Modus verwendet wird).
    • Kanal: Geben Sie eine Frequenz für das WLAN-Netzwerk ein. Wählen Sie für eine bessere Leistung eine Frequenz aus, die sich von Netzwerken in der Nähe unterscheidet.
    • Übertragungsleistung: Wählen Sie eine Signalstärke des WLAN-Netzwerks aus.
    • Max. Verbindungen: Wählen Sie, wie viele Client-Geräte sich mit dem Netzwerk verbinden können. Die maximale Anzahl an Verbindungen ist 256.
    • WMM QoS aktivieren: WLAN Multimedia (WMM) QoS (Quality of Service) über WLAN. Mit dieser Option wird dem WLAN-Netzwerk ermöglicht, Verkehr zu priorisieren und zu optimieren, wie Ressourcen zwischen Anwendungen geteilt werden.

So richten Sie einen WLAN-Zeitplan ein:*

Die Aktivierung eines WLAN-Zeitplans ermöglicht Ihnen, zu bestimmen, wann das WLAN-Netzwerk von Synology NAS aktiv und erreichbar ist.

  1. Gehen Sie zur Registerkarte WLAN und klicken Sie auf Bearbeiten.
  2. Markieren Sie das Kontrollkästchen Zeitplan aktivieren.
  3. Klicken Sie auf Zeitplan einstellen.
  4. Wählen Sie im Zeitplanfenster, wann das WLAN-Netzwerk aktiv sein soll.

Sicherheitseinstellungen

Systemsteuerung > WLAN-Netzwerk > WLAN bietet Optionen, um die Sicherheitseinstellungen Ihres WLAN-Netzwerks zu bearbeiten.

WLAN-Sicherheit einrichten:

  1. Klicken Sie auf Bearbeiten.
  2. Wählen Sie eine der folgenden Sicherheitsprotokolle aus dem Dropdown-Menü Sicherheitseinstellungen:
    • WEP: Ein grundlegendes Sicherheitsprotokoll, bei dem der Benutzer ein Passwort für den Zugriff auf das WLAN-Netzwerk eingeben muss, aber nur geringen Schutz bietet.
    • WPA/WPA2 Personal: Empfehlenswert. Dieses Protokoll erfordert die Eingabe eines Kennworts für den Zugriff auf das WLAN-Netzwerk und es ist sicherer als WEP.
    • WPA/WPA2 Enterprise:* Ein Sicherheitsprotokoll für Unternehmen, das eine Autorisierung von Benutzern über einen externen Autorisierungsserver verlangt.

So wird Wi-Fi Protected Setup aktiviert:*

WPS (Wi-Fi Protected Setup) ist eine Norm der Wi-Fi Alliance, um die WLAN-Sicherheitseinstellungen für Benutzer zu vereinfachen. Sie können eine Drucktaste oder eine PIN-Nummer verwenden, um Ihr WLAN-Netzwerk sicher zu halten. Klicken Sie auf WPS, um Wi-Fi Protected Setup zu aktivieren.

Gastnetzwerk*

Zusätzlich zum Hinzufügen eines Host-WLAN-Netzwerks unterstützt Synology NAS auch die Erstellung eines Gast-WLAN-Netzwerks. Die Einstellungen des Gast-Netzwerks können den Benutzerzugang einschränken, was für Unternehmen oder anderen Einrichtungen hilfreich ist, die ein WLAN-Netzwerk für Besucher anbieten, aber keine Zugriffe auf das Host-Netzwerk erlauben möchten.

Gastnetzwerk aktivieren:

  1. Gehen Sie zu Systemsteuerung > WLAN-Netzwerk > WLAN.
  2. Klicken Sie auf Bearbeiten, um die folgenden Einstellungen auf der Registerkarte Gastnetzwerk zu verwalten:
    • Name (SSID): Geben Sie einen Namen für das WLAN-Netzwerk ein.
    • Anzeigen/Ausblenden: Wählen Sie, ob das WLAN-Netzwerk für andere Geräte sichtbar sein soll.
    • Max. Verbindungen: Wählen Sie, wie viele Client-Geräte sich mit dem Netzwerk verbinden können. Die maximale Anzahl an Verbindungen ist 256.

Gastnetzwerk-Sicherheit einrichten:

Wählen Sie eine der folgenden Sicherheitsprotokolle aus dem Dropdown-Menü Sicherheitseinstellungen:

  • WEP: Ein grundlegendes Sicherheitsprotokoll, bei dem der Benutzer ein Passwort für den Zugriff auf das WLAN-Netzwerk eingeben muss, aber nur geringen Schutz bietet.
  • WPA/WPA2 Personal: Empfehlenswert. Dieses Protokoll erfordert die Eingabe eines Kennworts für den Zugriff auf das WLAN-Netzwerk und es ist sicherer als WEP.

So richten Sie einen Gastnetzwerk-Zeitplan ein:

Die Aktivierung eines Gastnetzwerk-Zeitplans ermöglicht Ihnen, zu bestimmen, wann das Gast-WLAN-Netzwerk von Synology NAS aktiv und erreichbar ist.

  1. Markieren Sie das Kontrollkästchen Gastnetzwerkplan aktivieren.
  2. Klicken Sie auf Gastplan einstellen.
  3. Wählen Sie im Zeitplanfenster, wann das Gast-WLAN-Netzwerk aktiv sein soll.

Anmerkung:

  • Die Funktion ist nur mit DS213air verfügbar.
  • Das Gast-WLAN-Netzwerk kann nicht aktiv sein, wenn das Host-WLAN-Netzwerk deaktiviert ist.

MAC-Filter-Einstellungen*

MAC-Filtereinstellungen (unter Systemsteuerung > WLAN-Netzwerk) gibt an, welchen MAC-Adressen der Zugriff auf das WLAN-Netzwerk erlaubt wird. Damit können Sie bestimmten Geräten den Zugriff verweigern oder gewähren. Diese Funktion steuert nur den Netzwerkzugriff und keine anderen Verschlüsselungseinstellungen.

MAC-Filterregeln erstellen:

  1. Geben Sie die Beschreibung und MAC-Adresse eines Geräts ein.
  2. Wählen Sie eine Zugriffsrichtlinie:
    • Verweigern: Verweigert der angegebenen MAC-Adresse den Zugriff auf das WLAN-Netzwerk.
    • Zulassen: Erlaubt der angegebenen MAC-Adresse den Zugriff auf das WLAN-Netzwerk.

Kindersicherung*

Mit der Kindersicherungsfunktion können Sie bestimmen, wann ein mit dem lokalen Netzwerk der Synology NAS verbundenes Client-Gerät auf das Internet zugreifen kann. Wenn zum Beispiel die Computer oder Mobilgeräte Ihrer Kinder die Verbindung zum Internet über die Synology NAS herstellen, können Sie die Einstellungen der Kindersicherung konfigurieren, um die Internetnutzung der Kinder nach Geräten und Zeit zu beschränken.

So aktivieren Sie die Kindersicherung:

  1. Gehen Sie zu Systemsteuerung > WLAN-Netzwerk > Einstellungen für Kindersicherung.
  2. Klicken Sie auf die Schaltfläche Hinzufügen, um ein Client-Gerät hinzuzufügen.
    • Client-Name: Wählen Sie ein Gerät, das mit Synology NAS verbunden ist, oder geben Sie den Namen manuell ein (z. B. Louisas Laptop).
    • MAC-Adresse: Geben Sie die MAC-Adresse des Geräts ein, dessen Internet-Nutzung Sie steuern möchten.
    • Zeitplan einstellen: Wählen Sie, zu welcher Zeit das angegebene Gerät über die Synology NAS auf das Internet zugreifen darf.

Verbindungsliste*

Die Verbindungsliste (unter Systemsteuerung > WLAN-Netzwerk) zeigt Informationen über die Geräte an, die mit Synology NAS verbunden sind. Von hier können Geräte direkt zur MAC-Filterliste hinzugefügt werden.

Geräte zur MAC-Filterliste hinzufügen:

  1. Aktivieren Sie das Kästchen neben dem Gerät, das Sie zum MAC-Filter hinzufügen möchten.
  2. Klicken Sie auf Zu MAC-Filterliste hinzufügen.
