Erweitert

Die folgenden erweiterten Einstellungen können unter Systemsteuerung > Sicherheit > Erweitert für DSM-Browser-Sitzungen geändert werden:

HTTP-Komprimierung

Sie können HTTP-Komprimierung für Synology NAS aktivieren. Durch Verringerung der Bandbreite des Netzwerkverkehrs können Webseiten schneller geladen werden.

HTTP-Komprimierung aktivieren/deaktivieren:

  • Aktivieren bzw. deaktivieren Sie HTTP komprimieren unter Systemsteuerung > Sicherheit > Erweitert.

TLS- / SSL-Profilebene

Im TLS- / SSL-Profil können Sie die Sicherheitsstufe von HTTPS-verschlüsselten Verbindungen wählen. Es werden die nachfolgenden drei Ebenen bereitgestellt:

  • Moderne Kompatibilität - Ist nur mit den neuesten Browsern und mobilen Geräten kompatibel.
  • Zwischenkompatibilität - Zwischen moderner Kompatibilität und alter Abwärtskompatibilität. Das ist der Standardwert des Sytems.
  • Alte Abwärtskompatibilität - Diese Option ist unsicher, wird jedoch von den meisten Browsern und mobilen Geräten unterstützt.

TLS- / SSL-Profilebene für alle Dienste konfigurieren:

Sie können eine TLS- /SSL-Profilebene auf der Hauptseite auswählen, die auf alle Dienste mit dem Standardprofil angewendet wird.

  1. Wählen Sie eine passende TLS - / SSL-Profilebene aus.
  2. Klicken Sie auf OK.

TLS- / SSL-Profilebene für bestimmte Dienste konfigurieren:

Sie können die Profilebene für einen Dienst gemäß der Sicherheitsanforderungen des Dienstes ändern.

  1. Klicken Sie auf Benutzerdefinierte Einstellungen, um alle Dienste und ihre entsprechende Profilebene anzuzeigen.
  2. Klicken Sie auf die aktuelle Profilebene des betreffenden Dienstes.
  3. Wählen Sie die richtige Profilebene aus dem Dropdown-Menü aus.
  4. Klicken Sie auf OK.

Anmerkung:

  • Die System-Standard-Profilebene wird für Verbindungen angewendet, die nicht in der Liste der Dienste angeführt sind.
