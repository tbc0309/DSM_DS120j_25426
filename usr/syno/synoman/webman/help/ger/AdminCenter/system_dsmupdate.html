DSM-Aktualisierung

Synology veröffentlicht regelmäßig kostenlose DSM-Aktualisierungen. Die Aktualisierungen können neue Funktionen, Funktionsverbesserungen und Leistungserweiterungen beinhalten. DSM-Aktualisierungen können in Form größerer oder kleinerer Updates erfolgen. Wenn es sich um ein kleineres Update handelt, zeigt das System eine Nachricht neben Status an. Wenn es sich um eine große neue DSM-Version handelt, werden in einem Bereich unter Status Informationen zu dieser Version angezeigt. Hier finden Sie auch einen Download-Link, um die neue Version herunterzuladen. Generell enthalten kleinere Updates wichtige Fehlerkorrekturen und Sicherheitsaktualisierungen, um die Systemstabilität zu verbessern; größere Updates enthalten neben wichtigen Fehlerkorrekturen und Sicherheitsaktualisierungen auch brandneue Funktionen und Leistungsverbesserungen.

Anmerkung:

  • Die Daten auf der Synology NAS sind von einer DSM-Aktualisierung nicht betroffen. Trotzdem empfehlen wir Ihnen, Ihre Daten zu sichern, bevor Sie mit der Aktualisierung beginnen.
  • Nach der Aktualisierung auf eine neue DSM-Update-Version können Sie nicht mehr zur vorherigen DSM-Update-Version zurückwechseln.
  • Synology NAS geht automatisch eine Checkliste durch, um basierend auf Diensten und Systemkonfiguration zu entscheiden, ob die neue Version für Ihr System relevant ist. Welche Updates und neuen Versionen verfügbar sind, kann sich daher je nach Ihrer DSM-Konfiguration unterscheiden.

Manuelle DSM-Aktualisierung durchführen

Sie können die .pat-Aktualisierungsdatei im Synology Download-Zentrum herunterladen und eine manuelle DSM-Aktualisierung durchführen.

So aktualisieren Sie DSM manuell:

  1. Besuchen Sie das Download-Zentrum, um das DSM-Update auf Ihren lokalen Computer herunterzuladen. Die .pat-Datei muss nicht extrahiert werden.
  2. Klicken Sie auf Manuelle DSM-Aktualisierung, um zur Seite „Manuelle DSM-Aktualisierung“ zu wechseln
  3. Klicken Sie auf Durchsuchen, um die Aktualisierungsdatei auszuwählen.
  4. Klicken Sie auf Anwenden, um die Aktualisierung Ihres DSM zu starten. Das System startet automatisch neu, sobald die Aktualisierung abgeschlossen ist.

Fehlerbehebung:

Wenn Sie DSM nicht aktualisieren können, helfen Ihnen die folgenden Vorschläge möglicherweise, das Problem zu beheben:

  • Prüfen Sie, ob der Aktualisierungsprozess von einem Benutzer oder anderen unerwarteten Ereignissen, z. B. Stromausfällen, unterbrochen wurde.
  • Überprüfen Sie die DSM-Version. Sie können das System nur mit einer neueren DSM-Version aktualisieren.
  • Laden Sie die DSM-Aktualisierung erneut herunter, denn die Aktualisierungsdatei könnte beschädigt sein.

Besteht das Problem weiterhin, kontaktieren Sie bitte den Technischen Support von Synology.

Automatische DSM-Aktualisierung und Installationseinstellungen einrichten

Klicken Sie auf Update-Einstellungen, um die Einstellungen für die Installation von Updates und den Zeitplan für die Suche nach Updates zu konfigurieren.

Einrichten, dass Sie vom System benachrichtigt werden, wenn ein neues DSM-Update verfügbar ist:

Wenn Sie die Option Mich benachrichtigen und mir die Entscheidung überlassen, ob das neue Update installiert werden soll auswählen, verständigt Sie das System nur, wenn eine neue Version verfügbar ist. Nach Erhalt der Benachrichtigung können Sie entscheiden, ob Sie das Update herunterladen möchten.

Ein Desktop-Benachrichtigung wird angezeigt, wenn ein DSM-Update heruntergeladen wurde und für die Installation bereit steht. Um die automatische DSM-Aktualisierung zu stoppen, können Sie sie in der Desktop-Benachrichtigung beenden. Darüber hinaus werden DSM-Updates vor dem geplanten Herunterfahren installiert, wenn Sie Herunterfahren unter Energie-Zeitplan aktiviert haben.

Das neue DSM-Update automatisch installieren:

Mit der Option Neues Update automatisch installieren kann das System ein verfügbares neues DSM-Update automatisch installieren. Dies gilt nur für in Status angezeigte kleinere Updates, nicht für größere Updates.

Zeitplan für die Suche nach Updates einrichten:

Mit Zeitplan kontrollieren können Sie bestimmen, wann das System nach verfügbaren Updates suchen soll. Wählen Sie aus, an welchen Wochentagen und zu welcher Uhrzeit nach Updates gesucht werden soll. Beispielsweise kann die Suche Dienstags und Freitags um 11 Uhr durchgeführt werden.

Wichtiges Update

Wenn eine große neue DSM-Version verfügbar ist, wird diese Information unten im Bereich DSM-Update angezeigt. Sie können auf Herunterladen klicken, um das große Update herunterzuladen. Klicken Sie nach dem Download auf Jetzt aktualisieren, um den Vorgang abzuschließen.

Fehlerbehebung:

Falls während der Suche nach der neuen DSM-Version oder während des Herunterladens Fehlermeldungen mit rotem Text angezeigt werden, kontrollieren Sie bitte Folgendes:

  • Stellen Sie sicher, dass Ihre Synology NAS mit dem Internet verbunden ist.
  • Möglicherweise ist der Download-Server von Synology derzeit ausgelastet. Versuchen Sie das Herunterladen zu einem späteren Zeitpunkt erneut.

Besteht das Problem weiterhin, kontaktieren Sie bitte den Technischen Support von Synology.

Firmware-Aktualisierung für Expansionseinheit

Wenn Sie über SAS-Expansionseinheiten verfügen, kann gelegentlich eine Firmware-Aktualisierung erforderlich sein. Das System führt automatisch eine Prüfung auf verfügbare Firmware-Updates durch und zeigt eines der folgenden Ergebnisse an: Firmware muss unverzüglich aktualisiert werden, Die Firmware aller Expansionseinheiten ist aktuell oder Keine verbundene Expansionseinheit.

  • Firmware muss unverzüglich aktualisiert werden: Wenn dies angezeigt wird, empfehlen wir dringend, die Firmware Ihrer Expansionseinheiten so schnell wie möglich zu aktualisieren. Ihre Expansionseinheiten sind nicht verfügbar, bis die Aktualisierung abgeschlossen wurde. Während dieser Zeit können Sie auch keine neuen Volumes auf den Expansionseinheiten erstellen.
  • Die Firmware aller Expansionseinheiten ist aktuell: Dies zeigt an, dass Ihr System aktuell ist und in einer stabilen Umgebung betrieben wird.
  • Keine verbundene Expansionseinheit: Dies zeigt an, dass keine Expansionseinheit mit Ihrer Synology NAS verbunden ist. Daher sind auch keine Firmware-Aktualisierungen erforderlich.

Wichtig: Normalerweise sind die Daten auf der Synology NAS von einer Firmware-Aktualisierung der Expansionseinheit nicht betroffen. Trotzdem empfehlen wir Ihnen, Ihre Daten zu sichern, bevor Sie mit der Aktualisierung beginnen.

Anmerkung:

  • Eine Firmware-Aktualisierung für Expansionseinheiten ist nur bei bestimmten SAS-Modellen erforderlich.
  • Eine Firmware-Aktualisierung für Expansionseinheiten ist nur in DSM 6.2.2 oder höher verfügbar.

Firmware-Aktualisierung für Expansionseinheiten durchführen:

Die Firmware-Aktualisierung für Expansionseinheiten dauert nur wenige Minuten. Das System muss nicht neu gestartet werden. Andere ausgeführte Dienste werden während der Aktualisierung nicht unterbrochen.

  1. Klicken Sie auf Jetzt aktualisieren, um die Aktualisierung zu starten.
  2. Eine Warnung wird angezeigt. Wählen Sie eine der folgenden Optionen aus:
    • Klicken Sie auf OK, um mit der Aktualisierung fortzufahren. Ein Fortschrittsbalken wird angezeigt.
    • Klicken Sie auf Abbrechen, um die Aktualisierung zu einem späteren Zeitpunkt durchzuführen.
  3. Das System zeigt eine der folgenden Desktop-Benachrichtigungen an, um Sie über das Ergebnis der Aktualisierung zu informieren: Firmware-Aktualisierung für Expansionseinheit(en) abgeschlossen oder Firmware-Aktualisierung für Expansionseinheiten fehlgeschlagen.
  4. Unter Status wird eine der folgenden Statusmeldungen angezeigt:
    • Die Firmware aller Expansionseinheiten ist aktuell: Diese Meldung wird angezeigt, wenn die Aktualisierung erfolgreich abgeschlossen wurde. Das System wird in einer stabilen Umgebung betrieben und Ihre Expansionseinheiten sind verfügbar.
    • Aktualisierung fehlgeschlagen: Sollte die Firmware-Aktualisierung für Expansionseinheiten fehlgeschlagen sein, klicken Sie auf Support kontaktieren, um das Problem zu beheben.

Anmerkung:

  • Um die Stabilität des Systems zu gewährleisten, sind Expansionseinheiten nicht mehr verfügbar, wenn das System erkennt, dass eine Firmware-Aktualisierung erforderlich ist. Die Expansionseinheiten sind wieder verfügbar, sobald die Firmware-Aktualisierung abgeschlossen wurde.
  • Sollte die Firmware-Aktualisierung für Expansionseinheiten fehlgeschlagen sein, versuchen Sie bitte nicht, einen Neustart durchzuführen oder Laufwerke oder Expansionseinheiten zu entfernen. Um das Problem zu beheben, siehe stattdessen bitte diesen Artikel und wenden Sie sich für weitere Unterstützung an den technischen Support von Synology.
