Benutzer erstellen

Sie können mehrere Benutzerkonten auf Ihrem Synology NAS erstellen und verwalten.

Benutzer erstellen:

  1. Gehen Sie zu Systemsteuerung > Benutzer.
  2. Klicken Sie auf die Schaltfläche Erstellen, und erstellen Sie anschließend den Benutzer. Dies startet den Assistent Benutzererstellung.
  3. Auf der Seite Benutzerinformationen geben Sie die folgenden Optionen an:
    • Name: Geben Sie einen Namen für den Benutzer ein. Dieser Bereich muss ausgefüllt werden.
    • Beschreibung: Geben Sie eine kurze Beschreibung des Benutzers ein.
    • E-Mail: Geben Sie eine E-Mail-Adresse des Benutzers ein. Systembenachrichtigungen, wie Benachrichtigungen über das Zurücksetzen des Kennworts, werden an die hier angegebene Adresse gesendet.
    • Eine Benachrichtigung an den neu erstellten Benutzer senden: Wenn diese Option aktiviert ist, sendet das System eine Benachrichtigung an den Benutzer, um ihn über das neue Konto zu informieren. Diese Option erfordert die Aktivierung der System-E-Mail-Benachrichtigungen unter Hauptmenü > Systemsteuerung > Benachrichtigung > E-Mail. Standardmäßig wird das Kennwort des Benutzers in der Benachrichtigung nicht angezeigt. Falls das Passwort in der Benachrichtigung angezeigt werden soll, markieren Sie bitte das Kästchen Benutzerpasswort in Benachrichtigungs-E-Mail anzeigen.
    • Lassen Sie nicht zu, dass der Benutzer das Konto-Passwort ändern kann: Wenn diese Option aktiviert ist, kann der Benutzer sein Passwort für die Anmeldung in DSM nicht ändern.
  4. Auf der Seite Gruppe > Erstellen > Gruppenerstellungsassistent geben Sie die Gruppen an, zu denen der neue Benutzer gehören soll (z. B. Benutzer, Administratoren usw.).
  5. Auf der Seite Berechtigungen für gemeinsame Ordner zuweisen wählen Sie durch die Einstellung seiner Zugriffsberechtigungen aus, auf welche gemeinsamen Ordner der neue Benutzer zugreifen kann. Die Berechtigungspriorität ist wie folgt: Kein Zugriff > Lesen/Schreiben > Nur Lesen.
    • Vorschau: Diese Spalte zeigt die aktuellen Zugriffsberechtigungen für gemeinsame Ordner gemäß den aktuellen Einstellungen und Gruppen an, zu welchen der Benutzer gehört.
    • Gruppenberechtigungen: Diese Spalte zeigt die Zugriffsberechtigungen auf den gemeinsamen Ordner gemäß den Benutzergruppen an, zu welchen der Benutzer gehört.
  6. Auf der Seite Benutzerkontingenteinstellung können Sie regeln, wie viel Speicherplatz der neue Benutzer verwenden darf. Um den Speicherplatz des neuen Benutzers zu beschränken, aktivieren Sie für jedes Volume bzw. jeden freigegebenen Ordner das Kontrollkästchen unter Kontingent aktivieren. Dann geben Sie im Feld Quota den maximalen Speicherplatz an (z. B. 10 GB), der verwendet werden darf.

    Anmerkung:

    • Die Option Kontingent aktivieren für jeden freigegebenen Ordner wird ausschließlich von VDSM und Modellen mit folgender Paketarchitektur unterstützt: Apollo Lake, Avoton, Braswell, Broadwell, Bromolow, Cedarview und Grantley. Welche Paketarchitektur Ihr Modell unterstützt erfahren Sie in diesem Artikel.
    • Bestimmte Optionen beim Erstellen eines Benutzers sind für EDS14 möglicherweise nicht anwendbar.
    • DDSM unterstützt Benutzerkontingenteinstellung nicht.
  7. Auf der Seite Berechtigungen für Anwendungen zuweisen können Sie regeln, auf welche Dienste der Benutzer zugreifen kann.
  8. Auf der Seite Gruppen-Geschwindigkeitseinschränkungs-Einstellungen können Sie eine Geschwindigkeitseinschränkung für verschiedene Dienste (z. B. File Station, FTP usw.) aktivieren, um die vom Benutzer für die Übertragung von Dateien genutzte Bandbreite zu beschränken. Für jeden Dienst steht Folgendes zur Auswahl:
    • Keine: Die Einstellungen für die Geschwindigkeitseinschränkung sind identisch mit der Gruppe, zu welcher der Benutzer gehört.
    • Konstant: Eine feste, konstante Geschwindigkeitseinschränkung wird angewendet. Sie können die Upload- und Download-Geschwindigkeitseinschränkungen in den Feldern links angeben.
    • Variabel: Für den Benutzer gelten zwei unterschiedliche Geschwindigkeitseinschränkungen gemäß einem Zeitplan. Klicken Sie auf die Schaltfläche Anpassen, um die Einstellungen für Geschwindigkeitseinschränkung und den Zeitplan zu ändern.
  9. Nach der Bestätigung der Einstellungen klicken Sie zum Beenden auf Übernehmen.

Beschränkungen für Benutzername:

  • Bei dem Benutzernamen wird die Groß-/Kleinschreibung nicht berücksichtigt, und er kann aus 1 bis 64 Unicode-Zeichen mit Ausnahme der folgenden Symbole bestehen:
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • Das erste Zeichen darf kein Minus- oder Leerzeichen sein und das letzte Zeichen darf kein Leerzeichen sein.
  • Die folgenden Benutzernamen werden vom System verwendet: mailer-daemon, postmaster.
  • Die Benutzerbeschreibung darf bis zu 64 anzeigbare Unicode-Zeichen enthalten.

Passworteinschränkungen:

  • Beim Kennwort wird die Groß-/Kleinschreibung berücksichtigt und es ist beschränkt auf 127 anzeigbare Zeichen einschließlich Buchstaben, Zahlen, Symbolen und Leerzeichen.

Quoten-Einschränkung:

  • Das maximale, für jeden Benutzer eingebbare Speicherkontingent beträgt ca. 4 TB (4095 GB).

    • Anmerkung:

    • Diese Funktion wird von EDS14 und DDSM nicht unterstützt.
