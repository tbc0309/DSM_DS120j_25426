Kontoschutz

Die Funktion „Kontoschutz“ verbessert die Sicherheit des Synology NAS durch Schützen von Konten vor vertrauensunwürdigen Clients mit zu vielen fehlgeschlagenen Anmeldeversuchen. Dies verringert das Risiko, dass bei Konten mit Brute-Force-Angriffen eingebrochen wird.

Anmerkung:

  • Unterstützte Dienste und Pakete: DSM, File Station, Audio Station, Video Station, Download Station, Mail Station, Cloud Station und Synology mobile Apps.

Kontoschutz aktivieren:

  1. Gehen Sie zu Systemsteuerung > Sicherheit > Kontoschutz.
  2. Aktivieren Sie Kontoschutz aktivieren.
  3. Geben Sie eine Zahl in das Feld Anmeldeversuche und innerhalb von (Minuten). Ein vertrauensunwürdiger Client wird blockiert, wenn die Anzahl der fehlgeschlagenen Anmeldeversuche innerhalb der unten eingegebenen Zeit erreicht werden.
  4. Geben Sie eine Zahl in das Feld Blockierung aufheben (nach Minuten) ein. Der Kontoschutz wird nach Ablauf der voreingestellten Zeit aufgehoben.
  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Zum Aufheben des Schutzes von bestimmten Konten

  1. Klicken Sie auf Geschützte Konten verwalten.
  2. Wählen Sie die aufzuhebenden Konten aus und klicken Sie auf Schutz aufheben.

Blockierung von vertrauenswürdigen Clients aufheben

  1. Klicken Sie auf Vertrauenswürdige Geräte verwalten.
  2. Wählen Sie die Clients aus, bei denen die Sperre aufgehoben werden soll und klicken Sie auf Blockierung aufheben.
