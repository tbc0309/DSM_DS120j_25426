Externe Geräte

Gehen Sie zu Systemsteuerung > Externe Geräte > Externe Geräte, um eine Liste von externen Speichergeräten anzuzeigen, die mit Synology NAS verbunden sind.

Externe Laufwerke

Durch Anschließen eines externen Laufwerks an das System können Sie dessen Laufwerkskapazität über einen vom System erstellten freigegebenen Ordner namens usbshare[Nummer] (für USB-Laufwerke) oder satashare[Nummer] (für eSATA-Laufwerke) nutzen. Der freigegebene Ordner wird automatisch entfernt, wenn das externe Laufwerk vom System getrennt wird.

Synology NAS erkennt die folgenden Formate: Btrfs, ext3, ext4, FAT, FAT32, exFAT, HFS, HFS Plus und NTFS. Ein nicht erkanntes externes Laufwerk muss zuerst formatiert werden, bevor es im System verwendet werden kann.

Anmerkung:

  1. Einige Modelle unterstützen HFS/HFS Plus nur schreibgeschützt.
  2. Journal wird auf HFS/HFS Plus nicht unterstützt.
  3. Sie müssen exFAT Access im Paketzentrum installieren, damit das Synology NAS exFAT unterstützen kann. Für eine Liste der Synology-Modelle, die das exFAT-Format unterstützen, siehe diese Seite.
  4. Stellen Sie sicher, dass Sie das externe Laufwerk auswerfen, bevor Sie es abstecken.
  5. Btrfs ist nur bei bestimmten Modellen verfügbar. In diesem Artikel erfahren Sie, ob Ihr Modell Btrfs unterstützt.

Auf Datenträger-Images zugreifen:

Das Synology NAS unterstützt die Bereitstellung virtueller Laufwerke, um auf externen Geräten mit File Station auf Inhalte von Datenträger-Images zuzugreifen. Wie Sie virtuelle Laufwerke bereitstellen können, erfahren Sie in diesem Artikel.

Anmerkung:

  • Unterstütztes Datenträger-Imageformat: „.iso"-Dateien unter Verwendung des Dateisystems ISO 9960 und UDF.

Laufwerk formatieren:

Klicken Sie auf Formatieren und folgen Sie dem Assistenten. Sie können externe Laufwerke mit dem Synology NAS in den Formaten ext4 und FAT32 formatieren.

Anmerkung:

  1. Achten Sie darauf, dass Sie Ihre Daten sichern, bevor Sie das externe Laufwerk formatieren. Während des Vorgangs werden alle Daten gelöscht und können nicht wiederhergestellt werden.
  2. Um das externe Laufwerk im Format ext3 oder ext4 zu formatieren, muss es größer als 1 GB sein.
  3. Wenn das Laufwerk als ext3 oder ext4 formatiert ist, wird es nur vom Synology NAS erkannt. Wenn Sie Daten direkt von Ihrem externen Laufwerk mit einem PC oder Mac lesen möchten, wählen Sie das Format FAT32.

S.M.A.R.T.-Test durchführen:

Der S.M.A.R.T.-Test prüft und meldet den Status Ihrer Festplatten und kann so frühzeitig vor drohendem Festplattenausfall warnen. Wir empfehlen, das Laufwerk auszutauschen, wenn Fehler erkannt werden.
Um den S.M.A.R.T.-Test durchzuführen, klicken Sie einfach auf die Schaltfläche S.M.A.R.T.-Test und folgen dem Assistenten.
Um die Testergebnisse zu sehen, klicken Sie einfach auf S.M.A.R.T.-Info.

Anmerkung:

  • Die Dauer des Tests hängt von der jeweiligen Systemumgebung ab.

Einschränkungen

Das Formatieren des Laufwerks ist in den folgenden Fällen nicht möglich:

  • Das externe Laufwerk wird gerade initialisiert, d. h. das externe Gerät wurde gerade erst angeschlossen.
  • Die Kapazität des externen Laufwerks beträgt weniger als 1 GB (Format ext3 oder ext4).
  • Das externe Laufwerk wird gerade formatiert.
  • Das System sichert gerade Dateien auf dem externen Laufwerk.
  • Das externe Laufwerk ist beschädigt.

Fehlerbehebung

  1. Das externe Laufwerk kann nicht formatiert werden. Warum?
    • Prüfen Sie, ob das externe Laufwerk ordnungsgemäß angeschlossen ist.
    • Prüfen Sie, ob das externe Laufwerk schreibgeschützt ist. Wenn ja, entfernen Sie den Schreibschutz vor der Formatierung.
  2. Das Dateisystem des externen Laufwerks wird als Nicht unterstützt angezeigt oder das System erstellt die freigegebenen Ordner usbshare[Nummer] oder satashare[Nummer] für das externe Laufwerk nicht. Was soll ich tun?
    • Das aktuelle System erkennt die Formate Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus und NTFS. Alle externen Laufwerke, die nicht nach den o. g. Typen formatiert wurden, werden nicht erkannt und als Status wird Nicht unterstützt angezeigt.
  3. Ich kann nicht auf den freigegebenen Ordner usbshare[Nummer] oder satashare[Nummer] zugreifen. Was soll ich tun?
    • Dies könnte durch ein Problem im Dateisystem auf dem externen Laufwerk verursacht werden. Um dies zu beheben, schließen Sie das externe Laufwerk an einen PC an und führen Sie eine Dateisystemprüfung aus.
  4. Kann ich mein externes Laufwerk im ext3- oder ext4-Format direkt mit meinem PC und Mac verbinden und direkt darauf zugreifen?
    • Nein, ext3 oder ext4 werden nur vom Synology NAS erkannt. Wenn Sie Daten direkt von Ihrem externen Laufwerk mit einem PC oder Mac lesen möchten, wählen Sie das Format FAT32.
  5. Warum sehe ich auf meinem externen Laufwerk einen Ordner mit dem Namen „lost+found“?
    • Dies ist ein Standard-Systemordner, der automatisch bei der Verwendung des ext3/ext4-Dateisystems erstellt wird. Es ist ganz normal, dass Sie den Ordner auf dem externen Laufwerk sehen. Auch wenn Administratoren das Recht haben, auf den Inhalt zuzugreifen und ihn zu verwalten, so wird doch dringend empfohlen, diesen Ordner nicht zu verändern.

Anmerkung:

  • Btrfs ist nur bei bestimmten Modellen verfügbar. In diesem Artikel erfahren Sie, ob Ihr Modell Btrfs unterstützt.

Standardberechtigungen für alle externen Geräte einstellen:

Sie können Standardberechtigungen festlegen, indem Sie auf Einstellungen > Standardberechtigungen klicken und angeben, welche Benutzer oder Gruppen externe Geräte und deren Inhalte anzeigen, ändern oder darauf zugreifen können. Wenn ein externes Gerät mit Ihrem Synology NAS verbunden wird, werden die von Ihnen festgelegten Berechtigungen automatisch auch dem externen Gerät zugewiesen.

Gehen Sie im Berechtigungsfenster wie folgt vor, um Standardberechtigungen festzulegen:

  1. Wählen Sie eine der folgenden Optionen aus dem Dropdown-Menü aus:
    • Interner Systembenutzer: Weist Berechtigungen für Standard-Systembenutzer zu, z. B. Anonymous FTP/WebDAV-Benutzer.
    • Lokale Benutzer: Weist Berechtigungen für lokale Benutzer (einschließlich guest) zu.
    • Lokale Gruppen: Weist Berechtigungen für lokale Gruppen zu.
  2. Setzen oder entfernen Sie Häkchen, um die Berechtigungen für den Zugriff auf externe Geräte für die einzelnen Benutzer oder Gruppen anzupassen:
    • Lesen/Schreiben: Der Benutzer oder die Gruppe hat Zugriff auf Dateien und Unterordner auf den externen Geräten und kann Änderungen an Dateien und Ordnern vornehmen.
    • Nur Lesen: Der Benutzer oder die Gruppe kann auf die Dateien und Ordner auf den externen Geräten zugreifen.
    • Kein Zugriff: Der Benutzer oder die Gruppe kann nicht auf die Dateien und Ordner auf den externen Geräten zugreifen.
