Speicherpool

Ein oder mehrere Laufwerke können zu einer Speichereinheit zusammengefasst werden, die Speicherpool genannt wird und durch RAID geschützt werden kann. Verschiedene RAID-Typen bieten unterschiedliche Ebenen des Datenschutzes. Eine allgemeine Liste über die von Ihrem Synology NAS unterstützten RAID-Typen finden Sie unter Allgemein. Nachdem ein Speicherpool erstellt wurde, können Sie ein oder mehrere Volumes darauf erstellen. Ein Volume ist eine Datenspeichereinheit und kann erweitert werden, wenn im zugehörigen Speicherpool zuweisbarer Speicherplatz verfügbar ist. Sie müssen mindestens ein Volume erstellen, bevor Sie Daten auf Ihrem Synology NAS speichern können.

Anmerkung:

  • Wir empfehlen, Laufwerke aus der Synology-Kompatibilitätsliste zu verwenden, um Probleme mit der Kompatibilität zu vermeiden. Die Verwendung nicht kompatibler Laufwerke kann die Systemleistung beeinträchtigen oder sogar Datenverluste verursachen.
  • Ein Speicherpool muss aus Laufwerken desselben Typs bestehen. Das bedeutet, ein Speicherpool kann nicht mit einer Kombination aus SATA- und SAS-Laufwerken erstellt werden.
  • Bei Synology NAS-Modellen, die RAID Group unterstützen, kann ein RAID 5-, RAID 6- oder RAID F1-Speicherpool mit Unterstützung mehrerer Volumes mehrere RAID Arrays enthalten. Jedes RAID Array, das zum selben Speicherpool gehört, ist vom selben RAID-Typ (z. B. RAID 5, RAID 6 oder RAID F1). Die maximale Anzahl an Laufwerken pro RAID Array kann 6, 12 oder 24 sein. Sie können die maximale Anzahl an Laufwerken beim Erstellen des Speicherpools festlegen. Diese Anzahl kann später nicht mehr geändert werden. Wurde einem RAID Array die maximale Anzahl an Laufwerken zugewiesen, kann das nächste zum Speicherpool hinzugefügte Laufwerk einem neuen RAID Array zugewiesen werden. Das neue RAID Array muss die Mindestanzahl an Laufwerken enthalten, die für diesen RAID-Typ benötigt werden (d. h. drei Laufwerke für RAID 5 und RAID F1 und vier Laufwerke für RAID 6).
  • Wenn Sie ein bestimmtes Laufwerk zur Erstellung eines Speicherpools verwenden möchten, dieses jedoch nicht im Assistent zur Speicherpool-Erstellung angezeigt wird, gehen Sie zur Seite HDD/SSD, um den Laufwerkstatus zu überprüfen.
