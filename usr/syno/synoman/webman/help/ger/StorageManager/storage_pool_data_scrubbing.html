Datenbereinigung

Datenbereinigung durchführen

Die Datenbereinigung ist eine Wartungsfunktion für Daten, die Speicherpools überprüft. Die nachfolgenden zwei Datenbereinigungsarten werden nacheinander ausgeführt:

  • Dateisystembereinigung: Diese Funktion überprüft die Volumes im Btrfs-Dateisystem. Wenn bei der Prüfsumme eine Dateninkonsistenz erkannt wird, versucht das System die Daten mit einem Backup zu reparieren, wobei der Dateipfad im Protokoll-Center erfasst wird.
  • RAID-Bereinigung: Diese Funktion kann bei Speicherpools verwendet werden, die SHR (bestehend aus drei oder mehr Laufwerken), RAID 5 oder RAID 6 nutzen. Sie kann alle gefundenen Datenunstimmigkeiten reparieren.

Wir empfehlen eine regelmäßige Datenbereinigung, um die Datenkonsistenz sicherzustellen und den Datenverlust im Fall eines Festplattenfehlers zu vermeiden.

Anmerkung:

  • Die Datenbereinigung und die entsprechenden in diesem Artikel erläuterten Funktionen sind nur bei bestimmten Modellen verfügbar.
  • Die Datenbereinigung kann nicht ausgeführt werden, wenn der Status der Speicherpools Abgestürzt oder Fehlerhaft lautet.
  • Wenn Daten-Prüfsumme für erweiterte Dateiintegrität für einen freigegebenen Ordner nicht aktiviert ist, kann die Dateisystembereinigung die Daten in dem Ordner aufgrund fehlender Prüfsummen nicht überprüfen oder reparieren.

Zeitplan für Datenbereinigung festlegen

Sie können einen Zeitplan für die Datenbereinigung festlegen. Das System wird die Datenbereinigung gemäß Ihren Einstellungen ausführen.

Einen Zeitplan für die Datenbereinigung festlegen:

  1. Starten Sie den Speicher-Manager.
  2. Gehen Sie zu Speicherpool > Datenbereinigung.
  3. Klicken Sie auf Zeitplan einstellen.
  4. Aktivieren Sie das Kontrollkästchen Zeitplan für Datenbereinigung aktivieren.
  5. Legen Sie Frequenz, Startdatum und Zeitsitzung fest.
  6. Gehen Sie zur Registerkarte Ziel auswählen.
  7. Wählen Sie den Speicherpool aus, in dem die Datenbereinigung ausgeführt werden soll.
  8. Klicken Sie auf OK.

Anmerkung:

  • Um zu einem Zeitplan hinzugefügt werden zu können, muss der Speicherpool die Datenbereinigung unterstützen und sein Status Normal lauten.
  • Wenn Ihr System von Version 6.1 aktualisiert wird, werden Speicherpools, die Datenbereinigung ausführen können, standardmäßig zum Zeitplan hinzugefügt.
  • Wenn Sie Ihren Zeitplan in Version 6.1 eingerichtet haben, werden wir Ihren Zeitplan automatisch gemäß Ihren früheren Einstellungen aktivieren.

Speicherpool einem Zeitplan hinzufügen oder aus einem Zeitplan entfernen:

  1. Klicken Sie auf Zeitplan einstellen
  2. Gehen Sie zur Registerkarte Ziel auswählen.
  3. Aktivieren oder deaktivieren Sie das Kontrollkästchen eines Speicherpools, um ihn zu einem Zeitplan hinzuzufügen oder zu entfernen.
  4. Klicken Sie auf OK.

Anmerkung:

  • Ein Speicherpool kann nicht zu einem Zeitplan hinzugefügt oder daraus entfernt werden, wenn auf ihm gerade manuell eine Datenbereinigung ausgeführt wird.
  • Wenn ein Speicherpool, auf dem eine geplante Datenbereinigung stattfindet, entfernt wird, geht der aktuelle Fortschritt auf diesem Speicherpool verloren.

Einstellungen für Zeitplan ändern:

  1. Klicken Sie auf Zeitplan einstellen.
  2. Ändern Sie je nach Bedarf Frequenz, Startdatum und Zeitsitzung.
  3. Klicken Sie auf OK.

Anmerkung:

  • Wird das Startdatum geändert, so wird der aktuelle Fortschritt des Zeitplans zurückgesetzt und die Datenbereinigung gemäß dem neuen Startdatum neu begonnen.

Einen Zeitplan anhalten:

  1. Wählen Sie einen Speicherpool aus, bei dem gerade eine geplante Datenbereinigung durchgeführt wird.
  2. Klicken Sie in der Dropdown-Liste Aktion auf Zeitplan anhalten.

Anmerkung:

  • Um den Datenbereinigungszeitplan fortzusetzen, müssen Sie im Dropdown-Menü Aktion auf Zeitplan fortsetzen klicken. Ansonsten bleibt der Zeitplan unterbrochen.

Datenbereinigung manuell ausführen:

Wenn das System eine Störung im Speicherplatz feststellt, können Sie versuchen, Ihre Daten durch eine manuell ausgeführte Datenbereinigung zu reparieren.

Datenbereinigung manuell ausführen:

  1. Öffnen Sie den Speicher-Manager.
  2. Gehen Sie zu Speicherpool > Datenbereinigung.
  3. Wählen Sie den Speicherpool aus, auf dem die Datenbereinigung ausgeführt werden soll.
  4. Klicken Sie in der Dropdown-Liste Aktion auf Manuelle Ausführung.

Anmerkung:

  • Wenn das System heruntergefahren wird, bevor die Datenbereinigung abgeschlossen wurde, kann Fortschritt verloren gehen und Sie müssen die Datenbereinigung manuell erneut starten.
  • Wenn der Zeitpunkt der manuell ausgeführten Datenbereinigung sich mit der geplanten Datenbereinigung überschneidet, wird diese Runde der geplanten Datenbereinigung erst abgeschlossen, nachdem die manuell ausgeführte Datenbereinigung beendet wurde.
  • Die Ausführung der Datenbereinigung kann Ihre Systemleistung beeinträchtigen. Wir empfehlen, die Datenbereinigung nur dann auf mehreren Speicherpools auszuführen, wenn es wirklich notwendig ist.

Einen Zeitplan auf einem Speicherpool anhalten, auf dem eine manuell ausgeführte Datenbereinigung stattfindet:

  1. Wählen Sie den Speicherpool aus, den Sie anhalten möchten.
  2. Klicken Sie in der Dropdown-Liste Aktion auf Pause.
