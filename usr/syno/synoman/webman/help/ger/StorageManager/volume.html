Volumes

Volumes stellen den grundlegenden Speicherplatz auf Ihrem Synology NAS bereit. Alle Ihre Daten, wie freigegebene Ordner, Dokumente und Paketdaten, werden auf Volumes gespeichert. Deshalb müssen Sie mindestens ein Volume erstellen, bevor Sie Daten auf Ihrem Synology NAS speichern können.

Anmerkung:

Wenn Volumes als replizierte Ziele von Snapshot Replication genutzt werden, können Sie die Replikationsleistung verbessern, indem Sie die Einstellungen für Dateizugriffsfrequenz aufzeichnen im Fenster Bearbeiten anpassen, nachdem Volumes erstellt wurden:

  • Täglich: Die Dateizugriffszeit wird wie in den meisten Betriebssystemen täglich oder bei Änderung der Datei aktualisiert. Die Funktionalität einiger Anwendungen hängt von der Präzision der Dateizugriffszeit ab. Die Einstellung Täglich beeinträchtigt die Replikationsleistung, wird jedoch empfohlen, wenn Sie die folgenden Funktionen oder Anwendungen regelmäßig nutzen:
    • Selten verwendet in Speicher-Analysator
    • SSD-Cache-Ratgeber in Speicher-Manager
    • Anwendungen von Drittanbietern, die Dateizugriffszeit aufzeichnen benötigen.
  • Monatlich: Die Dateizugriffszeit wird monatlich oder bei Änderung der Datei aktualisiert. Diese Option bewirkt eine geringe Verbesserung der Replikationsleistung und Anwendungen, die Dateizugriffszeit aufzeichnen benötigen, funktionieren, wenn auch mit geringerer Präzision.
  • Nie: Empfohlen, wenn Sie beste Replikationsleistung wünschen und im täglichen Gebrauch keine Anwendungen verwenden, die Dateizugriffszeit aufzeichnen benötigen.

Volume-Format

Je nach dem verwendeten Dateisystem sind zwei Volume-Formate verfügbar:

  • Neues Volume-Format: Dieses Format nutzt das Dateisystem Btrfs. Die Größe eines Volumes dieses Formats muss mindestens 10 GB betragen. Es unterstützt die folgenden neuen Funktionen, die die Dateiverwaltung erleichtern:
    • Schnappschuss von freigegebenem Ordner: Beim Erstellen des Schnappschusses wird der gesamte Datenstatus erfasst. Werden auf einem Volume mit Schnappschüssen versehentlich Daten geändert oder gelöscht, können sie in der Version, die vom Schnappschuss aufgezeichnet wurde, schnell wiederhergestellt werden.
    • Kontingent von freigegebenem Ordner: Für jeden Ordner kann eine maximale Kapazität festgelegt werden.
    • Benutzerquote basierend auf freigegebenem Ordner: Sie können festlegen, wie viel Speicherplatz in einem freigegebenen Ordner von jedem Benutzer verwendet werden darf.
    • Erweiterter Datenintegritätsschutz: Der freigegebene Ordner wird über die Strategien "CRC32-Prüfsumme" und "Kopieren-nach-Schreiben" geschützt.
  • Herkömmliches Volume-Format: Dieses Format nutzt das Dateisystem ext4. Die Größe eines Volumes dieses Formats muss mindestens 10 GB betragen. Da das Dateisystem ext4 von den meisten Synology-Produkten unterstützt wird, können Sie Ihr Volume in diesem Format mühelos zu einem anderen Synology-Produkt migrieren.

Anmerkung:

  • Btrfs ist nur bei bestimmten Modellen verfügbar. In diesem Artikel erfahren Sie, ob Ihr Modell Btrfs unterstützt.
