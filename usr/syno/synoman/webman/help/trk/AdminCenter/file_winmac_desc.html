SMB/AFP

Windows, Mac ve Linux kullanıcıları, Denetim Masası > Dosya Hizmetleri > SMB/AFP bölümündeki seçenekleri etkinleştirdikten sonra Synology NAS cihazınıza ağ üzerinden erişebilirler.

Aşağıdaki dosya paylaşım protokolleri desteklenmektedir:

  • Windows için: SMB/CIFS (Ağ Bağlantılarım), FTP, WebDAV
  • Mac için: SMB, FTP, AFP, WebDAV
  • Linux için: SMB, FTP, WebDAV
