Element docelowy

W zależności od modelu, Synology NAS obsługuje tworzenie wielu elementów docelowych, do których można odwzorować jednostki LUN.

Uwaga: Liczba obsługiwanych elementów docelowych zależy od modeli Synology NAS.

Aby utworzyć element docelowy:

  1. Kliknij przycisk Utwórz.
  2. Wprowadź informacje o elemencie docelowym, takie jak Nazwa, IQN, a także określ, czy włączyć funkcję Uwierzytelnianie CHAP.
  3. Kliknij Dalej.
  4. Skonfiguruj mapowanie iSCSI LUN. Wybierz opcję Utwórz nowy iSCSI LUN lub Mapuj istniejące LUN.
  5. Kliknij Dalej.
  6. Kliknij Wstecz, aby zmodyfikować, ewentualnie Zastosuj, aby potwierdzić ustawienia.

Terminologia

  • IQN: IQN (iSCSI Qualified Name — nazwa kwalifikowana iSCSI) to unikatowa nazwa elementu docelowego. Nie powinna ona kolidować z innymi nazwami IQN elementów docelowych na innych serwerach. IQN musi składać się z mniej niż 128 znaków i może zawierać litery (wielkość liter jest nieistotna), cyfry, dwukropki, łączniki i kropki. Zalecana konwencja nazewnictwa to iqn.[data].[domena]:[id urządzenia], na przykład: iqn.2000-04.com.synology:storage.diskarray.1.
  • Wzajemne uwierzytelnianie CHAP: W trybie Wzajemny CHAP zarówno klient, jak i serwer uwierzytelniają się nawzajem. Żadne z haseł CHAP używanych do uwierzytelniania klienta nie może być skonfigurowane do uwierzytelniania jakiegokolwiek serwera, a żadne z haseł CHAP używanych do uwierzytelniania serwera nie może być skonfigurowane do uwierzytelniania żadnego z klientów.

    • Uwaga：proces wykrywania po stronie inicjatora iSCSI może zakończyć się niepowodzeniem, jeśli na serwerze NAS jest włączone uwierzytelnianie CHAP. Przed nawiązaniem połączenia z elementem docelowym należy w pierwszej kolejności dokończyć proces wykrywania, a następnie włączyć uwierzytelnianie CHAP. Można także nawiązać połączenie z elementem docelowym przy użyciu IP i IQN.

Aby usunąć element docelowy:

Element docelowy i wszelkie zmapowane jednostki LUN można usunąć. Usunięcie jednostki iSCSI LUN spowoduje usunięcie wszystkich danych przechowywanych na tej jednostce iSCSI LUN.

  1. Wybierz element docelowy do usunięcia.
  2. Kliknij opcję Usuń.
  3. Jeśli chcesz usunąć zmapowane jednostki LUN, wybierz Usuń także zmapowane LUN, a następnie wybierz jednostki do usunięcia.
  4. Kliknij przycisk OK.

Aby edytować element docelowy:

  1. Wybierz element docelowy do edycji.
  2. Wybierz polecenie Edycja w menu rozwijanym Akcja.
  3. Dokonaj edycji ustawień zgodnie z preferencjami. Kliknij przycisk OK, aby zapisać zmiany.

Terminologia

  • Suma kontrolna CRC: Włącz sumę kontrolną CRC, aby sprawdzić, czy w czasie transmisji danych nie wystąpiły błędy.
  • Wiele sesji: Zezwól na jednoczesne łączenie wielu sesji z więcej niż jednego inicjatora z tym samym elementem docelowym. W celu ochrony plików przed uszkodzeniem lub nieoczekiwaną utratą danych zdecydowanie zaleca się używanie wspólnego systemu plików lub systemu klastrowego, takiego jak VMware Virtual Machine File System (VMFS) i Oracle Cluster File System (OCFS).
  • Maskowanie: Umożliwia dostęp do elementu docelowego wyłącznie określonym IQN inicjatorów iSCSI.

Aby włączyć lub wyłączyć element docelowy:

  1. Wybierz element docelowy, który zamierzasz włączyć lub wyłączyć.
  2. Z listy rozwijanej Akcja wybierz Włącz lub Wyłącz.
