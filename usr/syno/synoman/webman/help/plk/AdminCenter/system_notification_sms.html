SMS

Aby serwer Synology NAS wysyłał powiadomienia o zmianie statusu lub wystąpieniu błędów w systemie, na przykład o utracie połączenia, zmianie działania wentylatora, awarii zasilania, braku miejsca na wolumenie lub niepowodzeniu tworzenia kopii zapasowej i przywracania danych itp., można włączyć powiadomienia SMS.

Uwaga:

  • Zdarzenie zmiany zachowania wentylatorów nie jest dostępne w modelach bez wentylatora, takich jak EDS14. Więcej informacji można znaleźć na stronie internetowej Synology.

Aby włączyć powiadomienia SMS:

Zaznacz pole wyboru Włącz powiadomienia SMS i wypełnij następujące ustawienia:

  1. Wybierz dostawcę SMS z listy rozwijanej Dostawca usługi SMS. Domyślnymi dostawcami usługi są „clickatell”, „clickatell-2017” i „SendinBlue-v3”.
  2. Kliknij Dodaj dostawcę usługi SMS, aby utworzyć nową konfigurację dostawcy usługi SMS, ewentualnie kliknij Usuń, aby usunąć wybraną konfigurację dostawcy SMS.
  3. Jeśli dostawca usługi SMS korzysta z Nazwy użytkownika i Hasła w celu uwierzytelniania, wybierz odpowiednie kategorie parametrów. W polu Nazwa użytkownika wprowadź nazwę użytkownika swojego konta usługi SMS. W polach Hasło i Potwierdź hasło wprowadź nazwę użytkownika oraz hasło swojego konta usługi SMS.
  4. Jeśli dostawca usługi SMS korzysta z Klucza API w celu uwierzytelniania, wybierz kategorię Klucz API dla odpowiedniego parametru. W polu Klucz API wprowadź klucz API podany przez dostawcę usługi SMS.
  5. Jeśli dostawca usługi SMS podał parametr do modyfikacji nazwy nadawcy, wybierz kategorię Nadawca dla odpowiedniego parametru. W polu Nadawca wprowadź nazwę nadawcy, spełniającą regułę dostawcy usługi SMS.
  6. Wprowadź swój Podstawowy numer telefonu lub Drugi numer telefonu. Jeżeli zostaną wprowadzone oba numery, wiadomości SMS z powiadomieniem będą wysyłane na oba numery.
  7. Jeżeli nie chcesz otrzymywać zbyt wielu wiadomości, zaznacz Włącz limit interwału SMS i ogranicz ilość wiadomości powiadomień do jednej na określony czas.
  8. Kliknij Wyślij testową wiadomość SMS, aby wysłać testową wiadomość SMS na podany przez siebie numer telefonu. Jeśli nie otrzymasz wiadomości, oznacza to, że konfiguracja jest nieprawidłowa.
  9. Kliknij Zastosuj, aby zapisać konfigurację, ewentualnie Resetuj, aby anulować konfigurację.

Uwaga:

  • Do ustawiania języka powiadomień SMS służy pozycja Język powiadamiania. Aby zmienić język powiadomień SMS, przejdź do pozycji Menu główne > Panel sterowania > Opcje regionalne, kliknij kartę Język i wybierz preferowany język z
    menu rozwijanego w sekcji Język powiadamiania.
  • Wszyscy dostawcy usługi SMS będą wysyłać powiadomienia w języku Angielskim, za wyjątkiem domyślnych dostawców usługi SMS: „clickatell” i „clickatell-2017”.

Ograniczenia:

  • Nazwa użytkownika: rozróżniana wielkość liter, od 1 do 32 znaków Unicode.
  • Hasło: rozróżniana jest wielkość liter, od 0 do 128 widocznych znaków, z uwzględnieniem liter, cyfr, symboli i spacji.
  • Klucz API: rozróżniana jest wielkość liter, od 1 do 256 znaków Unicode.
  • Nadawca: rozróżniana wielkość liter, od 1 do 64 znaków Unicode.
  • Limit interwału SMS musi obejmować od 1 do 10000 minut.

Aby utworzyć konfigurację dostawcy SMS:

  1. Kliknij Dodaj dostawcę usługi SMS.
  2. W polu Nazwa dostawcy wprowadź nazwę dostawcy usługi SMS.
  3. W polu SMS URL wprowadź adres URL SMS. Adres URL SMS można uzyskać od swojego dostawcy usługi SMS.
  4. Wybierz opcję GET lub POST w polu Metoda HTTP. Informację tę można znaleźć w dokumencie interfejsu API dostawcy usługi SMS.
  5. Kliknij Następne.
  6. W polu Edytuj nagłówek żądania HTTP dodaj parametry wymienione w dokumencie interfejsu API dostawcy usługi SMS.
  7. Kliknij Następne.
  8. W polu Edytuj treść żądania HTTP, jeśli dla Metoda HTTP wybrano opcję POST, dodaj parametry wymienione w dokumencie interfejsu API dostawcy usługi SMS.
  9. Kliknij Następne.
  10. System podzieli adres URL SMS, nagłówek żądania HTTP i treść żądania HTTP na kilka parametrów. Zdefiniuj każdy parametr, wybierając odpowiednią kategorię. Każda kategoria parametru, oprócz kategorii Inne, może zostać użyta tylko raz.

Ograniczenia:

  1. Nazwa dostawcy: rozróżniana jest wielkość liter, od 1 do 32 znaków Unicode.
  2. SMS URL: rozróżniana wielkość liter, od 1 do 500 znaków Unicode.Ten adres URL SMS służy tylko do konfiguracji. Żadna wiadomość SMS nie zostanie wysłana po zakończeniu konfiguracji. Format tego adresu URL jest podany przez dostawcę usługi SMS.Przykładowo dla Clickatell możliwy adres URL to: https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world
  3. Parametr: rozróżniana wielkość liter, od 1 do 32 znaków Unicode.
  4. Wartość: rozróżniana wielkość liter, od 1 do 256 znaków Unicode.
  5. Możesz dodać do 10 parametrów.
  6. Zauważ, że w parametrze „treść wiadomości” spacja pomiędzy „Hello” a „world” zostanie zamieniona na określony znak zdefiniowany przez dostawcę.
    Przykładowo dla Clickatell wartość parametru „treść wiadomości” to „Hello+world”, ponieważ w przypadku Clickatell spacja jest zastępowana znakiem „+”.
