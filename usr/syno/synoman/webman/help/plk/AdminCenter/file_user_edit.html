Edycja konta użytkownika

Znajdź konto użytkownika (wprowadzając w polu Szukaj nazwę użytkownika), kliknij je dwukrotnie lub kliknij przycisk Edytuj, aby edytować dane konta.

Uwaga:
  1. Jeżeli konto zostanie wyłączone, przestanie ono działać.
  2. Wartość w polu Limit powinna być liczbą całkowitą większą lub równą 0 oraz mniejszą niż pojemność dysku/wolumenu. Jeżeli limit zostanie ustawiony na wartość 0, wartość ustawienia Obowiązujący limit zostanie wyznaczona na podstawie ustawienia Limit grupy.
  3. Jeżeli w systemie jest więcej niż jeden wolumen/folder współdzielony, można ustawić limit dla użytkownika na potrzeby każdego wolumenu/folderu współdzielonego.
  4. Informacje e-mail na stronie „Powiadomienia” muszą być ustawione poprawnie, aby możliwe było powiadamianie użytkownika pocztą elektroniczną o utworzeniu lub zmianie informacji konta.
  5. Podczas ustawiania limitu szybkości dla użytkownika, jeżeli do tego użytkownika ma już zastosowanie limit szybkości grupy, w polu Rezultat są wyświetlane odpowiednie wartości.
  6. Limit szybkości przypisany użytkownikowi nie będzie uwzględniany w limicie szybkości przypisanym grupie, do której ten użytkownik należy.

Uwaga:

  • Funkcja edytowania limitu nie jest obsługiwana przez modele EDS14 i DDSM.
  • Edycja limitu użycia grupy przez folder współdzielony jest obsługiwana wyłącznie przez VDSM i modele o następujących architekturach pakietów: Apollo Lake, Avoton, Braswell, Broadwell, Bromolow, Cedarview oraz Grantley. Aby sprawdzić, jaką architekturę pakietów obsługuje dany model, zapoznaj się z tym artykułem.
