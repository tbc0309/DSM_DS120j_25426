Użytkownik

Elastyczne opcje zarządzania użytkownikami umożliwiają tworzenie kont użytkowników dla poszczególnych członków rodziny lub pracowników firmy. Tworzenie użytkowników pozwala na definiowanie uprawnień i zarządzanie nimi w odniesieniu do każdej osoby, w tym uprawnień dostępu do folderu współdzielonego, limitów pamięci masowej/folderu współdzielonego oraz ograniczeń przepustowości łącza.

    Uwaga:
  1. Zarządzanie folderami współdzielonymi dotyczy wyłącznie platform Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake oraz Denverton. Informacje dotyczące modeli obsługujących tę funkcję można znaleźć w tym artykule.
  2. Zarządzanie limitami pamięci/folderów współdzielonych nie dotyczy Docker DSM.
