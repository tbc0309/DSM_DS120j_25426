Indeksowanie plików

Indeksowanie folderu współdzielonego lub jego podfolderów może pomóc w szybkim znajdowaniu plików umieszczonych w tych folderach oraz zawartości tych plików.

Uwaga:

Foldery zdalne, foldery w chmurze lub foldery .iso zamontowane na Synology NAS nie mogą być indeksowane.

Aby indeksować foldery i zawartość plików:

Aplikacja File Station indeksuje wybrany folder oraz jego podfoldery/pliki według nazwy oraz umożliwia wyszukiwanie żądanych elementów wg nazw plików. W przypadku dalszego indeksowania według typu pliku aplikacja File Station zindeksuje odpowiednie pliki nie tylko według nazwy, ale także według zawartości. W ten sposób będzie można wyszukiwać żądane elementy według nazwy pliku oraz jego zawartości.

  1. Wybierz Panel sterowania > Usługa indeksowania > Indeksowanie plików i kliknij opcję Lista indeksowanych folderów.
  2. W wyświetlonym oknie kliknij opcję Utwórz > Wybierz, aby wyświetlić foldery gotowe do indeksowania.
  3. Wybierz żądany folder i kliknij opcję Wybierz. Aby utworzyć nowy folder do indeksowania, kliknij opcję Utwórz folder i kontynuuj procedurę.
  4. Kliknij żądany folder, który chcesz indeksować, a następnie kliknij opcję Wybierz.
  5. Aby indeksować określone typy plików, wybierz Typ pliku i wybierz opcję Obraz, Dźwięk, Wideo lub Dokument. Jeżeli nie wybrano któregoś z typów plików (np. Dźwięk), pliki tego typu można wyszukiwać tylko według nazwy.
  6. Kliknij przycisk OK, aby zakończyć indeksowanie.

Uwaga:

  • Do indeksowania można wybrać maksymalnie 1000 folderów. Podfoldery w wybranych folderach nie są wliczane do limitu.
  • Jeśli wybrano opcję Włącz przeszukiwanie zawartości plików w File Station, możesz przeszukiwać zawartość plików w indeksowanych folderach.
  • Wolumen, który jest hostem dla indeksowanego folderu musi mieć ponad 10 MB wolnego miejsca. W innym przypadku indeksowanie zostanie wstrzymane. Aby wznowić indeksowanie, powiększ wolumen i kliknij opcję Ponownie indeksuj.
  • Jeśli jednocześnie indeksujesz folder i jego podfoldery, system będzie indeksował wyłącznie podfoldery, aby zoptymalizować wykorzystanie zasobów systemowych.
  • Po pełnym zindeksowaniu folderów i zamontowaniu ich na komputerze Mac za pośrednictwem protokołów SMB lub AFP można użyć aplikacji Finder, aby przeszukiwać zawartość plików.

Aby usunąć indeksowane foldery:

  1. Wybierz Panel sterowania > Usługa indeksowania > Indeksowanie plików > Lista indeksowanych folderów, a następnie wybierz żądane indeksowane foldery.
  2. Kliknij opcję Usuń, aby zatrzymać indeksowanie folderów.

Ponowne indeksowanie plików

W normalnych okolicznościach ponowne indeksowanie plików nie jest wymagane. Jeżeli jednak pliki w folderze pasują do określonego kryterium wyszukiwania, ale brakuje ich wśród wyników wyszukiwania, ponowne indeksowanie może być potrzebne w celu odbudowania struktury indeksu plików.

Aby ponownie indeksować pliki w folderze współdzielonym:

  1. Wybierz Panel sterowania > Usługa indeksowania > Indeksowanie plików > Lista indeksowanych folderów.
  2. Wybierz z listy foldery, które chcesz ponownie indeksować.
  3. Kliknij Ponownie indeksuj.

Uwaga:

File Station nie obsługuje indeksowania plików w folderach zdalnych lub zewnętrznych urządzeniach pamięci masowej (np. dysków USB/kart SD).

Zarządzaj indeksowaniem plików za pośrednictwem pulpitu

Gdy rozpocznie się indeksowanie plików, na pasku zadań będzie widoczny Postęp indeksowania (ikona unoszącego się szkła powiększającego). Ponieważ indeksowanie plików wymaga więcej zasobów procesora, ta funkcja pozwala oszczędniej dysponować dostępną mocą obliczeniową.

  • Aby przerwać/wznowić indeksowanie plików, kliknij przycisk z ikoną odtwarzania.
  • Aby opóźnić indeksowanie plików do odpowiedniego momentu, wybierz żądaną opcję z rozwijanego menu.
  • Aby wyświetlić indeksowane foldery, kliknij opcję Wyświetl listę indeksowanych folderów.

Uwaga:

  • W przypadku przerwania indeksowania można je wznowić tylko kliknięciem przycisku z ikoną odtwarzania na pasku zadań pulpitu.
