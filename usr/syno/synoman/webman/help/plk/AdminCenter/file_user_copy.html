Kopiuj użytkownika

Ustawienia istniejącego użytkownika można skopiować no nowego użytkownika. Dzięki temu możliwe jest szybkie utworzenie nowego użytkownika bez konieczności ponownego konfigurowania tych samych ustawień.

  1. Wybierz użytkownika do skopiowania.
  2. Kliknij Utwórz > Kopiuj użytkownika. Zostanie wyświetlone okno Kreator tworzenia użytkownika.
  3. Ustawienia wybranego użytkownika zostaną zastosowane do nowego użytkownika domyślnie (poza informacjami osobistymi, takimi jak nazwa użytkownika czy adres e-mail).
