Analiza logowania

Analiza logowania umożliwia administratorom systemu sprawdzanie informacji dotyczących zabezpieczeń w bieżącym systemie i kontach.

Możesz wyłączyć analizę logowania lub edytować reguły w Doradca ds. zabezpieczeń > Zaawansowane > Dostosuj listę kontrolną.

Nowe aktywności logowania

Program Security Advisor analizuje informacje logowania każdego użytkownika, w tym HTTP użytkownika-agenta, adres IP i położenie geograficzne. W przypadku wykrycia nowej aktywności logowania program Security Advisor wyśle powiadomienie do administratora systemu i przedmiotowego użytkownika.

Po potwierdzeniu, że dany rekord logowania jest nieprawidłowy, zaleca się użycie silniejszego hasła, włączenie weryfikacji dwuetapowej, a nawet ograniczenie liczby dozwolonych adresów IP w celu poprawy bezpieczeństwa konta.

Odgadywanie haseł (ataki brute force)

Program Security Advisor analizuje nieudane próby logowania w odniesieniu do różnych adresów IP i protokołów. Kiedy ktoś próbuje zalogować się, starając się odgadnąć hasło, ale nie powiedzie mu się kilka razy, program Security Advisor wyśle powiadomienie do administratora systemu i przedmiotowego użytkownika. Administrator systemu może poprawić bezpieczeństwo konta, włączając opcję Automatyczne blokowanie lub Ochrona konta (w obszarze Panel sterowania > Bezpieczeństwo > Konto).
