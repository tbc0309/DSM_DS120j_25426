Security Advisor

Program Security Advisor jest aplikacją bezpieczeństwa DSM, która skanuje ustawienia DSM oraz urządzenie Synology NAS. Program Security Advisor sprawdza ustawienia i zaleca zmiany pomagające zapewnić bezpieczeństwo urządzenia Synology NAS. Poniżej podano listę dostępnych funkcji.

  • Program Security Advisor przeprowadza różnego rodzaju czynności kontrolne w celu wykrycia ustawień DSM, które mogą powodować zagrożenie bezpieczeństwa, a także podejrzane działania.
  • Program Security Advisor będzie analizował nieprawidłowe logowania i odgadywanie haseł przez złośliwe oprogramowanie zewnętrzne (ataki brute force).
  • Program Security Advisor podaje szczegółowe wyniki oraz raporty z zalecanymi działaniami i wynikającymi z nich korzyściami.
  • Wewnętrzna baza danych definicji zabezpieczeń jest automatycznie aktualizowana, dzięki czemu zachowuje aktualność w stosunku do zmieniających się zagrożeń w cyberprzestrzeni.
  • Program Security Advisor generuje raporty miesięczne lub codzienne, aby podpowiadać zalecane działania.

Aby uruchomić program Security Advisor, wykonaj następujące czynności:

  • Wybierz Menu główne > Program Security Advisor.
