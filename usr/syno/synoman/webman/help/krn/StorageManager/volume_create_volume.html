볼륨 만들기

볼륨은 스토리지 풀에서 구축되므로 스토리지 풀을 만든 후 볼륨을 만들어야 합니다. 하지만 사용 중인 모델에서 빠른 생성 모드가 지원되는 경우 볼륨과 스토리지 풀을 동시에 만들 수 있으며, 볼륨 생성 마법사가 자동으로 저장소 용량과 성능을 최적화하므로 기술적인 측면에 대해 걱정을 할 필요가 없습니다. SHR(Synology Hybrid RAID)을 지원하는 모델에서만 빠른 생성 모드를 사용할 수 있습니다. 자세한 내용은 Synology 웹사이트를 참조하십시오.

빠른 생성 모드를 지원하는 모델에서 볼륨 만들기:

볼륨과 스토리지 풀을 동시에 만들 수 있습니다. 새 볼륨 생성 시 다음 방법 중 하나를 선택할 수 있습니다.

  • 빠름: 초보자에게 가장 적합한 옵션입니다. RAID 유형과 같은 기술적인 측면에 대해 걱정하지 않고 저장소 용량과 성능을 자동으로 최적화하려는 경우에 이 모드를 선택합니다.
  • 사용자 지정: RAID 유형과 용량 할당과 같은 볼륨 생성 고급 옵션을 구성하려는 사용자를 위한 옵션입니다. (스토리지 풀 구성 방법에 대해서는 여기를 참조하십시오.)

아래 단계를 따라 고속 생성 모드에서 볼륨을 만듭니다.

  1. 저장소 관리자를 엽니다.
  2. 볼륨으로 이동하고 생성을 클릭합니다.
  3. 볼륨 생성 마법사가 나타납니다. 필요에 따라 빠름 또는 사용자 지정을 선택합니다. 그런 다음 마법사 단계를 따라 볼륨 만들기를 완료합니다.

기타 모델에서 볼륨 만들기:

  1. Synology NAS에 할당 가능한 공간이 있는 기존 스토리지 풀이 있는지 확인합니다.
  2. 생성 버튼을 클릭합니다.
  3. 새로운 볼륨을 만들 스토리지 풀을 선택합니다.
  4. 볼륨의 할당 크기를 설정합니다.
  5. 적용을 클릭하여 설정을 확정합니다.

참고:

  • 스토리지 풀 생성 시 스토리지 풀로 보다 우수한 성능을 선택한 경우에는 이 스토리지 풀에서 생성된 볼륨의 할당 크기를 설정할 수 없습니다.
  • 최대 볼륨 크기는 모델에 따라 다릅니다. 자세한 내용은 모델 사양을 참조하십시오
  • 볼륨에 할당되는 최소 크기는 10GB 이상이어야 합니다.
  • Btrfs 볼륨은 메타데이터 공간의 5%를 차지합니다.
