볼륨

볼륨은 사용자가 공유 폴더, iSCSI LUN(일반 파일)을 만들고, 데이터를 저장하고, 패키지를 설치할 수 있는 기본 저장소 공간입니다. 볼륨을 만들기 전에 RAID Group을 미리 만드십시오. RAID Group에 대한 자세한 정보는 RAID Group을 참조하십시오.

볼륨 만들기:

  1. Synology NAS에 할당 가능한 공간이 있는 기존 RAID Group이 있는지 확인하십시오.
  2. 생성 버튼을 클릭합니다.
  3. 볼륨을 만들 RAID Group을 선택합니다.
  4. 원하는 설정을 수정합니다.
  5. 볼륨 크기 할당을 설정합니다. (단일 볼륨 또는 iSCSI LUN용 RAID Group에서 만든 볼륨에서는 크기 할당 설정을 사용할 수 없습니다.)
  6. 적용 버튼을 클릭하여 설정을 확인합니다.

참고:

  • 할당할 수 있는 최대 볼륨 크기는 200 TB입니다.
  • Btrfs 볼륨에 할당되는 최소 크기는 10 GB 이상이어야 합니다.
  • Btrfs 볼륨은 메타데이터 공간 중 4%를 차지합니다.

볼륨 제거하기:

  1. 제거할 볼륨을 선택합니다.
  2. 제거 버튼을 클릭하고 마법사를 따라 프로세스를 완료합니다.

참고:

  • 볼륨을 제거하면 해당 볼륨의 모든 기존 데이터가 지워집니다.

볼륨을 강제로 탑재하기:

볼륨에서 캐시가 손실된 경우, 시스템은 부팅 시 볼륨을 탑재하지 않습니다. 캐시를 복원할 수 없지만 볼륨에 액세스하려는 경우에는 강제 마운트를 클릭하면 됩니다.

참고:

  • 특정 모델에서만 이 기능을 사용할 수 있습니다.

볼륨 복구하기:

볼륨 상태가 저하됨으로 표시되면 RAID Group 탭으로 이동하고 적합한 RAID Group을 찾으십시오. 그런 다음 권장 지침을 따라 볼륨을 복구합니다.

볼륨 크기 확장하기

  • 여러 볼륨 또는 iSCSI LUN(블록 수준)용 RAID Group에서 만든 볼륨의 경우
    1. 확장할 볼륨을 선택합니다.
    2. 편집을 클릭합니다.
    3. 새 크기를 입력하고 확인을 클릭하여 확장합니다.
  • 단일 볼륨 또는 iSCSI LUN(블록 수준)용 RAID Group에서 만든 볼륨의 경우, RAID Group의 RAID Group 크기 확장하기 섹션을 참조하십시오. 볼륨 크기는 기본 RAID Group이 확장된 후에 확장됩니다.

참고:

볼륨이 Snapshot Replication의 복제 대상으로 사용되는 경우, 볼륨이 생성된 후 편집 창에서 파일 액세스 시간 기록 빈도 설정을 조정하여 복제 성능을 향상시킬 수 있습니다.

  • 매일: 대부분의 운영 체제와 마찬가지로 파일 액세스 시간이 매일 또는 파일 수정 시 업데이트됩니다. 일부 응용 프로그램 기능은 파일 액세스 시간의 정밀도에 따라 달라질 수 있습니다. 매일을 선택하면 복제 성능이 영향을 받습니다. 다음 기능 또는 응용 프로그램을 자주 사용하는 경우에만 권장됩니다.
    • 저장소 분석기의 오래 전에 액세스한 시간
    • 저장소 관리자의 SSD 캐시 어드바이저
    • 파일 액세스 시간 기록을 사용하는 타사 응용 프로그램
  • 월간: 파일 액세스 시간이 월간 또는 파일 수정 시 업데이트됩니다. 이 옵션은 복제 성능을 약간 향상시키므로, 파일 액세스 시간 기록을 사용하는 응용 프로그램이 낮은 정밀도로 작동할 수 있습니다.
  • 없음: 최상의 복제 성능을 원하며 일상에서 사용 시 파일 액세스 시간 기록을 사용하는 응용 프로그램이 필요하지 않은 경우에 권장됩니다.

SSD TRIM

볼륨 전체가 SSD(Solid State Drive)로 구성된 경우, SSD TRIM을 활성화하는 것이 좋습니다. 이 기능은 SSD에서 생성된 볼륨의 읽기 및 쓰기 성능을 향상시키므로, SSD의 효율성과 수명이 증가합니다.

SSD TRIM 활성화하기:

  1. 저장소 관리자를 열고 볼륨으로 이동합니다.
  2. 전체가 SSD로 구성된 볼륨을 선택한 후 구성 버튼을 클릭합니다.
  3. 표시되는 창에서 SSD TRIM 탭으로 이동하고 TRIM 활성화 확인란을 확인 표시합니다.
  4. 스케줄 설정 버튼을 클릭하여 시스템이 SSD TRIM을 수행할 시기를 선택합니다.
  5. 확인을 클릭합니다.

참고:

  • 특정 Synology NAS 모델 및 RAID 유형에서만 SSD TRIM을 사용할 수 있습니다.
  • 일부 SSD 모델은 RAID 5, RAID 6 및 RAID F1 볼륨에서 SSD TRIM을 수행할 수 없습니다. Synology 공식 웹사이트의 호환 목록을 참조하십시오.
  • 다음 상황에서는 SSD TRIM을 활성화할 수 없습니다.
    • 볼륨이 SSD 캐시를 통해 탑재된 경우.
    • Time Backup 패키지가 설치된 경우.
  • 블록 수준 iSCSI LUN에서는 SSD TRIM을 활성화할 수 없습니다.
  • SSD TRIM을 수행하면 시스템 성능이 영향을 받습니다. TRIM 수행을 완료하는 데 소요되는 시간은 볼륨 크기와 SSD 모델에 따라 다릅니다.
  • 볼륨에서 TRIM이 활성화된 경우, 같은 드라이브 그룹 또는 RAID Group에서 만든 다른 볼륨에서도 TRIM이 활성화됩니다.

파일 시스템 삭제*

파일 시스템 삭제는 볼륨을 검사하는 데이터 유지 보수 기능입니다. 체크섬과 일치하지 않는 데이터가 감지되면 파일 경로가 로그 센터에 기록됩니다. 백업을 사용하여 데이터를 복구하는 것이 좋습니다.

데이터 일관성을 보장하기 위해 파일 시스템 삭제를 정기적으로 수행하는 것이 좋습니다.

파일 시스템 삭제 수행하기:

  1. 저장소 관리자를 열고 볼륨으로 이동합니다.
  2. 파일 시스템 삭제를 수행할 볼륨을 선택합니다.
  3. 관리를 클릭합니다.
  4. 파일 시스템 삭제를 선택합니다.

참고:

  • Btrfs 파일 시스템의 볼륨에서만 파일 시스템 삭제를 수행할 수 있습니다.
  • 볼륨 상태가 정상인 경우에만 파일 시스템 삭제를 수행할 수 있습니다.
  • 파일 시스템 삭제가 완료되기 전에 시스템을 종료하지 마십시오. 완료 전에 시스템을 종료하면 수동으로 파일 시스템 삭제를 처음부터 다시 시작해야 합니다.

파일 시스템 조각 모음*

데이터가 계속 수정되는 경우, 파일 시스템에서 조각화된 파일 양이 증가하여 드라이브의 파일 액세스 성능이 저하됩니다. 파일 시스템 조각 모음을 수행하여 성능을 향상시킬 수 있습니다.

파일 시스템 조각 모음 수행하기:

  1. 저장소 관리자를 열고 볼륨으로 이동합니다.
  2. 파일 시스템 조각 모음을 수행할 볼륨을 선택합니다.
  3. 관리를 클릭합니다.
  4. 파일 시스템 조각 모음을 선택합니다.

참고:

  • Btrfs 파일 시스템의 볼륨에서만 파일 시스템 조각 모음을 수행할 수 있습니다.
  • 볼륨 상태가 정상인 경우에만 파일 시스템 조각 모음을 수행할 수 있습니다.
  • 파일 시스템 조각 모음이 완료되기 전에 시스템을 종료하지 마십시오. 완료 전에 시스템을 종료하면 수동으로 파일 시스템 조각 모음을 처음부터 다시 시작해야 합니다.
  • 볼륨에 공유 폴더 스냅샷이 있는 경우, 볼륨 사용량이 증가할 수 있습니다.

* 특정 모델에서만 사용할 수 있습니다.
