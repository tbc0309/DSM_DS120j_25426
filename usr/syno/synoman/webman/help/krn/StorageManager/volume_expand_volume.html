볼륨 크기 확장

여러 볼륨을 지원하는 스토리지 풀에서 볼륨을 만든 경우 볼륨 크기를 변경할 수 있습니다. 하지만 단일 볼륨만 지원하는 스토리지 풀에서 볼륨을 만든 경우 먼저 스토리지 풀 용량이 확장되도록 스토리지 풀에 드라이브를 추가하여 볼륨 크기를 확장할 수 있습니다.

여러 볼륨 스토리지 풀의 경우:

아래 단계를 수행하여 볼륨의 크기를 확장하십시오.

  1. 볼륨 페이지로 이동하고 확장할 볼륨을 선택합니다.
    참고: 필요한 경우 먼저 해당 스토리지 풀에 드라이브를 추가하여 스토리지 풀 용량을 확장할 수 있습니다. 자세한 내용은 드라이브를 추가하여 스토리지 풀 용량 확장을 참조하십시오.
  2. 동작 드롭다운 메뉴에서 구성을 클릭합니다.
  3. 일반 탭으로 이동하고 할당된 크기 수정(GB) 필드에 새 볼륨 크기를 입력하거나 최대 버튼을 클릭하여 할당 가능한 최대 크기를 자동으로 입력합니다.
  4. 확인을 클릭하여 완료합니다.

단일 볼륨 스토리지 풀의 경우:

아래 단계를 수행하여 볼륨의 크기를 확장하십시오.

  1. 드라이브를 스토리지 풀에 추가하여 스토리지 풀 용량을 확장합니다.
    참고: 자세한 내용은 드라이브를 추가하여 스토리지 풀 용량 확장을 참조하십시오.
  2. 스토리지 풀 용량이 확장되면 해당 볼륨 크기가 자동으로 확장됩니다.

참고:

  • 여기를 참조하여 Synology NAS 모델의 최대 단일 볼륨 크기를 확인하십시오.
