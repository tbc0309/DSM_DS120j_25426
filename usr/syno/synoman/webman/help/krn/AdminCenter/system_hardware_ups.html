UPS

UPS(무정전 전원 공급 장치)는 정전이 발생했을 때 Synology NAS이(가) 단기간 동안 계속 작동할 수 있도록 해주는 백업 전원 장치입니다. 제어판 > 하드웨어 및 전원 > UPS에 있는 이 기능은 전원이 차단되고 종료(EDS14의 경우 중지됨)되기 전에 Synology NAS이(가) 데이터를 저장하고 볼륨을 분리할 수 있는 충분한 시간을 제공하여 데이터 손실을 방지하는 데 유용합니다.

안전 모드

Synology NAS은(는) 안전 모드에 들어가면 UPS 장치에 전원이 부족할 때 데이터 손실을 방지하고 안전하게 종료(EDS14의 경우 중지됨)될 수 있도록 모든 서비스를 중지하고 볼륨을 분리합니다. 기본적으로 UPS 장치가 전원이 부족한 상태에서 실행되기 시작하면 시스템은 안전 모드로 전환됩니다. 또한 정전이 발생했을 때 Synology NAS이(가) 안전 모드로 전환되기까지의 시간을 지정할 수도 있습니다. 그러나 지정된 시간이 경과하기 전에 UPS 장치의 배터리가 부족해지면 시스템은 즉시 안전 모드로 전환됩니다.

안전 모드 중에 Synology NAS이(가) 종료되는 경우 제어판 > 하드웨어 및 전원 > 일반에서 전원 차단 후 자동으로 다시 시작 옵션을 활성화하면 전원이 복구될 때 자동으로 전원이 켜집니다.

로컬 UPS 지원

하드웨어 설치에 대한 자세한 내용은 UPS 장치의 사용 설명서를 참조하십시오. 설치가 완료되고 Synology NAS이(가) UPS 장치에 연결되면 아래 단계를 수행하여 UPS 지원을 활성화할 수 있습니다.

로컬 UPS 지원 활성화하기:

  1. UPS 장치를 Synology NAS의 USB 포트에 연결합니다.
  2. UPS 탭에서 UPS 지원 활성화 확인란을 선택합니다.
  3. 정전 발생 후 Synology NAS이(가) 안전 모드로 전환될 시기를 선택합니다.
  4. 시스템이 안전 모드로 전환될 때 UPS 장치로 종료 신호를 전송할지 여부를 선택합니다. 대부분의 UPS 장치는 전원이 복구되면 자동으로 켜집니다.
    • 이 옵션이 활성화된 경우 전원이 부족하면 Synology NAS이(가) 즉시 종료됩니다.
    • 이 옵션을 활성화한 경우 제어판 > 하드웨어 및 전원 > 일반에서 전원 차단 후 자동으로 다시 시작을 선택하여 전원 복구 시 Synology NAS이(가) 자동으로 다시 시작하는지 확인합니다.
  5. 적용을 클릭합니다.

SNMP UPS 지원

SNMP UPS는 SNMP를 통해 Synology NAS에 연결한 백업 전원 장치입니다. 하드웨어 설치에 대한 자세한 내용은 SNMP UPS 장치의 사용 설명서를 참조하십시오. Synology NAS을(를) 장치에 꽂고 동일한 네트워크에 연결한 후 아래 단계를 따라 UPS 지원을 활성화할 수 있습니다.

SNMP UPS 지원 활성화하기:

  1. UPS 탭에서 UPS 지원 활성화 확인란을 선택합니다.
  2. UPS 유형 드롭다운 메뉴에서 SNMP UPS를 선택합니다.
  3. Synology NAS이(가) 안전 모드로 전환될 때까지의 시간을 선택하여 Synology NAS이(가) 정전 후 안전 모드로 전환되는 시기를 사용자 지정합니다. 숫자 값을 입력하고 드롭다운 메뉴에서 초, 분 또는 시를 선택합니다. 선택하지 않으면 배터리가 부족할 때까지가 적용됩니다. 즉, UPS 배터리가 부족하면 Synology NAS은(는) 안전 모드로 자동 전환됩니다.

    4. 참고: 선택한 옵션에 관계없이 전원이 부족하면 Synology NAS은(는) 안전 모드로 자동 전환됩니다.

  4. SNMP UPS IP 주소 필드에 SNMP UPS 장치의 IP 주소를 입력합니다.
  5. 드롭다운 메뉴에서 SNMP MIB를 선택합니다.
  6. 드롭다운 메뉴에서 SNMP 버전을 선택합니다.
  7. SNMP UPS 장치의 커뮤니티 문자열을 SNMP 커뮤니티 필드에 입력합니다.
  8. Synology NAS이(가) SNMP를 통해 연결된 경우 네트워크 UPS 서버 활성화를 선택하면 네트워크 UPS 서버로 작동합니다. 허용된 Synology NAS 장치를 클릭하여 이 UPS에 연결할 수 있는 모든 Synology NAS 장치의 IP 주소를 입력합니다.

    9. 참고: SA3200D에서는 허용된 Synology NAS 장치 기능을 제공하지 않습니다.

  9. 적용을 클릭합니다.

참고:

  • Synology High Availability(SHA) 배포에 SNMP UPS를 사용하는 것이 좋습니다.
  • Synology NAS은(는) SNMPv1 및 SNMPv2c UPS 장치를 지원합니다.

Synology UPS 서버

UPS 포트 또는 네트워크를 통해 UPS 장치에 연결된 Synology 제품 하나와 UPS 장치를 사용하는 여러 Synology 제품이 있는 경우 USB 또는 네트워크를 통해 연결된 Synology NAS만 UPS 장치 상태 관련 정보를 가져올 수 있습니다. 이 경우 이 Synology NAS은(는) "네트워크 UPS 서버"로 작동합니다. 그러면 네트워크 UPS 서버는 UPS 정보를 수신하고 클라이언트 Synology NAS에 릴레이합니다.

Synology UPS 서버 활성화하기:

  1. (USB 또는 네트워크를 통해) UPS 장치에 연결된 Synology NAS에 로그인합니다.
  2. UPS 탭에서 네트워크 UPS 서버 활성화 확인란을 선택하여 Synology NAS이(가) UPS 정보를 클라이언트 Synology NAS에 릴레이할 수 있도록 합니다.
  3. 허용된 Synology NAS 장치 버튼을 클릭합니다.
  4. 표시된 창에 클라이언트 Synology NAS의 IP 주소를 입력합니다. 확인을 클릭합니다.
  5. 적용을 클릭하여 설정을 저장합니다.

이제 네트워크 UPS 서버는 UPS 정보를 가져와 UPS 장치를 사용하는 클라이언트 Synology NAS에 릴레이할 수 있습니다. 아래 지침을 참조하여 클라이언트 Synology NAS에서 UPS 지원을 활성화하십시오.

클라이언트 Synology NAS에서 UPS 지원 활성화하기:

  1. USB 또는 네트워크를 통해 연결되어 있지 않고 UPS 장치를 사용하는 클라이언트 Synology NAS에 로그인합니다.
  2. UPS 탭에서 UPS 지원 활성화 확인란을 선택합니다.
  3. UPS 유형 드롭다운 메뉴에서 Synology UPS 서버를 선택합니다.
  4. Synology NAS이(가) 안전 모드로 전환될 때까지의 시간을 선택하여 Synology NAS이(가) 정전 후 안전 모드로 전환되는 시기를 사용자 지정합니다. 숫자 값을 입력하고 드롭다운 메뉴에서 초, 분 또는 시를 선택합니다. 선택하지 않으면 서버와 동일이 적용됩니다. 즉, 서버가 종료되면 Synology NAS은(는) 안전 모드로 전환됩니다.

    5. 참고: 선택한 옵션에 관계없이 전원이 부족하면 Synology NAS은(는) 안전 모드로 자동 전환됩니다.

  5. 네트워크 UPS 서버 IP 필드에 네트워크 UPS 서버의 IP 주소를 입력합니다.
  6. 적용을 클릭하여 설정을 저장합니다.

네트워크 UPS 요구 사항:

  • UPS 장치 정보를 릴레이하려면 모든 Synology NAS이(가) 동일한 네트워크에 연결되어 있어야 합니다.
  • 각 Synology NAS이(가) 연결되어 있는 네트워크 장치(예: 스위치 또는 허브)는 동일한 UPS 장치에 연결되어야 합니다. 그렇지 않으면 정전이 발생했을 때 Synology NAS이(가) UPS 장치 정보를 릴레이할 수 없습니다.
