암호화된 공유 폴더의 키 관리

키 관리자를 사용하여 공유 폴더의 키를 관리하고 여러 공유 폴더 암호화를 동시에 해제할 수 있습니다.

용어 설명:

  • 키 저장소: 키 저장소는 Synology NAS에서 지원되는 외부 장치 또는 시스템 파티션이며 공유 폴더 암호화를 해제하는 실제 키로 사용됩니다.
  • 암호: 암호는 공유 폴더의 키를 암호화하는 방법입니다. 키 관리자는 다음 두 가지 유형의 암호를 제공합니다.
    • 패스프레이즈: 패스프레이즈를 아는 사람은 누구나 패스프레이즈로 암호화된 키 암호화를 해제할 수 있습니다.
    • 컴퓨터 키: 컴퓨터 키로 암호화된 키는 바인딩된 Synology NAS을(를) 통해서만 암호화 해제될 수 있습니다.

키 저장소 초기화하기:

  1. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  2. 키 저장소 위치에서 키 저장소로 외부 장치 또는 시스템 파티션을 선택합니다.
  3. 패스프레이즈 필드에 이 키 저장소의 패스프레이즈를 입력합니다.

보안 제안 사항:

  • 안전 및 관리를 위해 암호화된 파일과 해당 키를 여러 장치에 보관하는 것이 더욱 안전합니다.
  • 데이터 보안을 강화하기 위해 키 저장소 위치로 외부 장치를 선택하는 것이 좋습니다.

새 키를 키 저장소에 추가하기:

  1. 키 저장소를 초기화했는지 확인합니다.
  2. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  3. 추가를 클릭합니다.
  4. 암호화된 공유 폴더를 선택합니다.
  5. 암호화 키의 암호를 선택합니다. 패스프레이즈 또는 컴퓨터 키를 선택할 수 있습니다.
  6. 암호화 키를 입력하거나 가져옵니다.
  7. 확인을 클릭하여 설정을 저장합니다.

수동으로 여러 공유 폴더의 암호화 해제하기:

  1. 키를 키 저장소에 추가했는지 확인합니다.
  2. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  3. 암호화를 해제하려는 암호화된 공유 폴더를 선택합니다.
  4. 마운트를 클릭합니다.

부팅 시 자동으로 여러 공유 폴더의 암호화 해제하기:

  1. 키를 키 저장소에 추가했는지 확인합니다.
  2. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  3. 부팅 시 암호화를 해제하려는 암호화된 공유 폴더에 대해 부팅 시 마운트를 선택합니다.
  4. 확인을 클릭하여 설정을 저장합니다.

부팅 후 키 저장소 꺼내기:

  1. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  2. 구성을 클릭합니다.
  3. 부팅 후 장치 꺼내기를 선택합니다.
  4. 확인을 클릭하여 설정을 저장합니다.

기존 암호화 키를 마운트된 공유 폴더의 키 저장소에 자동으로 마이그레이션하기:

키 저장소 초기화 중에 키 마이그레이션:

  1. 키 저장소를 초기화하는 경우, 자동으로 마운트된 모든 기존 공유 폴더에 대해 암호화 키를 이 키 저장소로 마이그레이션(권장됨)을 선택합니다.

수동으로 키 마이그레이션:

  1. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  2. 구성을 클릭합니다.
  3. 지금 마이그레이션을 클릭합니다.
  4. 중복 키의 경우 중복 키 덮어쓰기를 선택할 수 있습니다.
  5. 확인을 클릭하여 설정을 저장합니다.

High Availability 모드에서 키 저장소 복제하기:

  1. 제어판 > 공유 폴더 > 동작 > 키 관리자로 이동합니다.
  2. 복제를 클릭합니다.
  3. 소스 키 저장소에서 활성 서버의 소스 키 저장소를 선택합니다.
  4. 대상 키 저장소에서 수동 서버의 대상 키 저장소를 선택합니다.
  5. 적용을 클릭하여 설정을 저장합니다.
