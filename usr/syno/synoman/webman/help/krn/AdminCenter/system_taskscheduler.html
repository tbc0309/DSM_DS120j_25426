작업 스케줄러

작업 스케줄러를 사용하여 미리 지정한 시간에 서비스 또는 스크립트를 예약하고 실행할 수 있습니다.

지원되는 작업

다음을 수행하기 위한 작업을 만들 수 있습니다.

  • 사용자 정의 스크립트: 사용자 정의 스크립트 또는 명령을 실행합니다.
  • 휴지통: 특정 공유 폴더의 휴지통을 비웁니다. 또한, 보존 정책 설정을 사용하여 휴지통을 비울 때 영구 삭제할 파일을 제어할 수 있습니다. 예를 들어, 크기, 삭제 시간 또는 파일 형식에 따라 파일을 삭제하거나 보존할 수 있습니다.
  • 신호음 제어: Synology NAS 신호음을 만듭니다.
  • 서비스: Windows 파일 서비스, FTP 및 자동 차단 등과 같은 네트워크 서비스를 시작하거나 중지합니다.

작업 관리

작업 스케줄러의 메인 페이지에 기존 작업이 모두 표시되고 작업을 생성, 편집 또는 실행할 수 있는 옵션을 제공합니다.

예약 작업 생성하기:

예약 작업은 지정한 날짜, 시간 또는 간격에 실행되는 작업입니다.

  1. 제어판 > 작업 스케줄러로 이동하고 생성을 클릭한 후 예약 작업을 선택합니다.
  2. 예약하려는 작업 유형을 사용자 정의 스크립트, 휴지통, 신호음 제어 또는 서비스.
  3. 작업 생성 창이 나타나면 일반을 클릭하고 다음 설정을 수정합니다.
    • 일반 설정: 작업 이름을 입력하고, 작업을 실행할 사용자를 선택하고, 작업을 활성화하거나 비활성화합니다.
  4. 스케줄을 클릭하고 다음 설정을 수정합니다.
    • 날짜: 작업 실행 시기를 지정합니다. 작업이 특정 요일(예: 매일, 주말 또는 매주 일요일) 또는 특정 날짜(예: 2015/10/21)에 실행되도록 예약할 수 있습니다.
    • 시간: 매일 작업이 실행되는 횟수와 처음 및 마지막 실행 시간을 지정합니다.
  5. 작업 설정을 클릭하여 작업별 설정을 수정합니다(특정 작업에만 해당).
    • 작업 실행 세부 사항을 수신하려면 실행 세부 사항을 이메일로 보내기를 선택합니다.
    • 비정상적인 상황 발생 시에만 알림을 받으려면 스크립트가 비정상적으로 종료된 경우에만 실행 세부 사항 보내기를 선택합니다.
    • 작업 내용을 사용자 지정하려면 사용자 정의된 스크립트에 스크립트를 입력합니다.
    • 2단계에서 휴지통 또는 서비스를 선택한 경우, 휴지통을 비우는 시기와 방법 또는 시작/중지할 서비스를 결정할 수 있습니다.
  6. 확인을 클릭하여 설정을 저장합니다.

트리거된 작업 생성하기:

트리거된 작업은 특정 이벤트 완료 전/후에 실행하는 작업입니다.

  1. 제어판 > 작업 스케줄러로 이동하고 생성을 클릭한 후 트리거된 작업을 선택합니다.
  2. 사용자 정의 스크립트를 선택합니다.
  3. 작업 생성 창이 나타나면 일반을 클릭하고 다음 설정을 수정합니다.
    • 일반 설정: 작업 이름을 입력하고, 작업을 실행할 사용자를 선택하고, 작업을 트리거하는 이벤트/사전 작업을 선택하고, 작업을 활성화하거나 비활성화합니다.
  4. 작업 설정을 클릭하여 작업별 설정을 수정합니다(특정 작업에만 해당).
    • 작업 실행 세부 사항을 수신하려면 실행 세부 사항을 이메일로 보내기를 선택합니다.
    • 비정상적인 상황 발생 시에만 알림을 받으려면 스크립트가 비정상적으로 종료된 경우에만 실행 세부 사항 보내기를 선택합니다.
    • 작업 내용을 사용자 지정하려면 사용자 정의된 스크립트에 스크립트를 입력합니다.
  5. 확인을 클릭하여 설정을 저장합니다.

참고:

  • 트리거된 작업을 생성하는 경우, 해당 사전 작업을 지정해야 할 수 있습니다. 생성된 작업은 사전 작업이 수행된 후에만 실행됩니다.
  • 스크립트가 비정상적으로 종료된 경우에만 실행 세부 사항 보내기를 활성화한 경우, 스크립트가 0으로 끝나면 시스템은 세부 사항을 전달하지 않습니다.

작업 설정 편집하기:

  1. 기존 작업을 선택합니다.
  2. 편집을 클릭합니다.
  3. 작업 설정을 수정할 수 있는 창이 나타납니다.

작업 삭제하기:

  1. 기존 작업을 선택합니다.
  2. 동작 > 삭제를 클릭합니다.

작업 활성화 또는 비활성화하기:

  1. 작업 스케줄러 메인 페이지에서 활성화 아래의 해당 확인란을 확인 표시하거나 확인 표시를 지워 작업을 활성화하거나 비활성화합니다.
  2. 저장을 클릭합니다.

작업을 수동으로 실행하기:

  1. 기존 작업을 1개 이상 선택합니다.
  2. 실행을 클릭합니다.

작업 스케줄러 제한 사항:

  • 작업 이름에는 표시 가능한 문자가 512자까지 포함될 수 있습니다.
  • 최대 작업 수: 1024.
  • 스크립트 경로 또는 명령줄 최대 길이: 4095.
  • 신호음 기간을 1~600 초로 설정할 수 있습니다.
  • 이메일 알림에는 최대 1 MB의 작업 결과가 포함됩니다.
