메모리 압축

메모리 압축을 활성화하여 메모리에서 최근에 가장 적게 사용된 데이터를 압축하여 로드 중인 일반 시스템 응답성을 향상시킬 수 있습니다.

실행 중인 응용 프로그램이 여러 개이고 메모리가 꽉 차기 시작하면 동적 메모리 압축 기능이 메모리에서 잘 사용되지 않는 항목의 크기를 줄여 메모리 사용을 최소화합니다. 이러한 항목들이 다시 필요하게 되면 즉시 압축을 풉니다. 이렇게 하면 디스크에서 메모리 교체 파일을 읽고 쓸 필요가 줄어들어 효율성과 응답성이 향상됩니다.

메모리 압축을 활성화하려면 해당 확인란을 선택하면 됩니다.

참고:

  • 특정 모델에서만 메모리 압축을 사용할 수 있습니다. 자세한 내용은 Synology 웹사이트를 참조하십시오.
