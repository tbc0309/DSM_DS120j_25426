AFP

제어판 > 파일 서비스 > SMB/AFP/NFS에서 AFP 서비스를 활성화하여 Synology NAS을(를) Mac 파일 서버(AFP 서버)로 지정할 수 있습니다.

Mac 파일 서비스 활성화하기:

  1. Synology NAS을(를) Mac 파일 서버(AFP 서버)로 지정하려면 AFP 서비스 활성화를 선택합니다.
  2. 적용을 클릭합니다.

전송 로깅 활성화

이 옵션을 선택하면 AFP(Apple Filing Protocol)를 통한 파일 또는 폴더의 생성, 삭제, 읽기, 쓰기, 이동 및 이름 변경을 비롯하여 파일 액세스 작업에 대한 로깅이 활성화됩니다.

고급 설정

기본 UNIX 권한 적용:

이 옵션을 활성화하면 파일 및 폴더를 업로드 또는 생성 시 기본 UNIX 권한이 적용됩니다. 적용된 권한은 UNIX 명령 umask에 의해 적용된 권한과 동일합니다. 이 옵션을 활성화하면 UNIX 권한은 파일의 경우 644, 폴더의 경우 755입니다. 이 옵션을 비활성화하면 UNIX 권한은 파일의 경우 666, 폴더의 경우 777입니다. 기본 umask 값은 022입니다.

참고:

이 옵션을 활성화하면 여러 프로토콜 사이에서 권한이 일관되지 않는 문제가 발생할 수 있습니다. 일관성을 유지하려면 이 옵션을 비활성화 상태로 유지하는 것이 좋습니다.

연결이 끊긴 후 즉시 리소스 해제:

이 옵션을 활성화하면 AFP 서버 연결이 예기치 않게 해제되면 리소스가 즉시 해제됩니다. Time Machine을 사용하는 동안 "볼륨 사용 중" 오류 메시지가 수신되면 이 옵션을 활성화하십시오.

Finder를 통해 파일 콘텐츠 검색하기:

폴더 전체를 인덱싱하고 Mac 컴퓨터에 마운트한 경우, Finder를 사용하여 포함된 파일의 콘텐츠를 검색할 수 있습니다.

  1. Universal Search > 기본 설정 > 파일 인덱싱 > 인덱싱된 폴더 목록 > 생성으로 이동하여 특정 폴더와 포함된 파일을 파일 유형별로 인덱싱합니다.
  2. 확인을 클릭하여 인덱싱을 수행합니다.
  3. 인덱싱된 폴더를 Mac 컴퓨터에 마운트합니다.
  4. Mac 컴퓨터의 Finder 왼쪽 패널에서 원하는 마운트된 폴더를 클릭하고, Finder를 사용하여 포함된 파일의 콘텐츠를 검색합니다.

참고:

  • Network Video Recorder(NVR) 모델은 Mac Finder를 사용한 파일 콘텐츠 검색을 지원하지 않습니다.
  • Finder를 사용하여 마운트된 폴더 안에서 검색합니다. Mac 데스크톱의 Spotlight는 로컬 파일을 검색할 때에만 사용될 수 있습니다.
  • Mac OS X 10.9 이후 버전에서는 Finder가 태그 이름 및 범주별로 마운트된 폴더 내 검색을 지원합니다.
  • AFP를 통해 공유 폴더에 액세스한 경우, 공유 폴더가 255개 이상이면 공유 폴더는 255개만 표시됩니다(알파벳 순).
