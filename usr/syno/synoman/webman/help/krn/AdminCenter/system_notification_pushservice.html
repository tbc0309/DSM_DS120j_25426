푸시 서비스

특정 이벤트나 오류 발생 시 Synology NAS이(가) 푸시 서비스를 사용하여 알림을 모바일 장치 또는 웹 브라우저로 발송할 수 있습니다.

포트:

푸시 서비스를 활성화하기 전에 Synology NAS이(가) 포트 8089를 사용하여 인터넷에 액세스할 수 있는지 확인하십시오.

Synology가 제공하는 이메일 알림 활성화하기:

  1. Synology의 이메일 서버를 통해 시스템 상태에 관한 알림을 보냅니다를 선택합니다.
  2. 받는 사람 필드에 이메일 주소를 1개 이상 입력합니다. 시스템 알림은 여기에 입력한 주소로 발송됩니다. 주소는 2개까지 허용됩니다. 세미콜론(;)을 사용하여 여러 주소를 구분합니다.
  3. 적용을 클릭합니다. 그러면 받는 사람 필드에서 지정된 주소로 확인 메일이 발송됩니다. 확인 메일에 있는 단계를 따르십시오.
  4. 검증이 완료되면 테스트 메시지 전송을 클릭한 후 확인을 클릭하여 테스트 알림을 발송할 수 있습니다.

모바일 기기 알림 활성화하기:

  1. 제어판 > 알림 > 푸시 서비스로 이동합니다.
  2. 모바일 기기 알림 활성화 상자를 선택한 후 적용을 클릭합니다.
  3. 모바일 기기에 DS finder를 설치 및 실행한 후 Synology NAS을(를) 즐겨찾기 목록에 추가하여 모바일 장치와 Synology NAS을(를) 연결합니다.
  4. 테스트 메시지 전송을 클릭한 후 확인을 클릭하여 테스트 알림을 발송합니다. 그런 다음 모바일 기기에서 DS finder를 실행하여 테스트 메시지가 수신됐는지 여부를 확인합니다.

연결된 모바일 장치 관리하기:

  1. 페어링된 장치 관리를 클릭하여 연결된 모든 모바일 장치 목록을 확인합니다.
  2. 장치를 연결 해제하려는 경우, 해당 장치를 선택하고 페어링 해제를 클릭합니다. (Shift 또는 Ctrl 키를 길게 눌러 여러 항목을 선택할 수 있습니다.) 장치를 다시 연결할 때까지 알림은 해당 장치에 발송되지 않습니다.

브라우저 알림 활성화하기:

  1. 모바일 기기 알림 활성화를 선택한 후 적용을 클릭하여 이 기능을 활성화합니다.
  2. 브라우저 알림 활성화를 클릭하여 Synology NAS을(를) 브라우저와 연결합니다.
  3. 표시되는 창에서 다음을 수행합니다.
    • Chrome의 경우: 설치하지 않은 경우, 지금 설치를 클릭하여 Chrome 확장명을 설치한 후 Google 계정에 로그인합니다.
    • Safari의 경우: 허용을 클릭하여 알림 서비스를 인증합니다.
  4. 다시 브라우저 알림 페이지로 리디렉션되면 페어링된 장치 탭을 클릭하여 연결된 장치의 정보를 확인하거나 브라우저에서 Synology NAS 연결을 해제합니다. 또는 이벤트 탭을 클릭하여 지난 7일 간의 브라우저 알림 기록을 확인합니다.

참고:

  • 브라우저 알림 서비스는 Chrome(35 이후) 및 Safari(OS X v10.9 이후)에서 지원됩니다.
  • iOS 및 Android 플랫폼에서만 DS finder의 푸시 알림 서비스를 사용할 수 있습니다.
