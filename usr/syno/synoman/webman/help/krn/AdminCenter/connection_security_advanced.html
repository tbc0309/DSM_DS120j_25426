고급

제어판 > 보안 > 고급에서 다음과 같은 DSM 검색 세션의 고급 설정을 수정할 수 있습니다.

HTTP 압축

Synology NAS에 대한 HTTP 압축을 활성화할 수 있습니다. 네트워크 트래픽의 대역폭을 낮춤으로써 웹 페이지의 로딩 속도를 향상시킬 수 있습니다.

HTTP 압축 활성화/비활성화하기:

  • 제어판 > 보안 > 고급에서 HTTP 압축을 선택/선택 취소합니다.

TLS / SSL 프로필 수준

TLS/SSL 프로필에서 HTTPS 암호화된 연결의 보안 수준을 선택할 수 있습니다. 다음과 같은 3가지 수준이 있습니다.

  • 최신 호환성 - 최신 브라우저 및 모바일 장치만 호환됩니다.
  • 중간 호환성 - 최신 호환성과 이전 역방향 호환성의 중간으로, 시스템 기본 값입니다.
  • 이전 역방향 호환성 - 이 옵션은 안전하지 않지만 대부분의 브라우저 및 모바일 장치를 지원합니다.

모든 서비스를 위한 TLS/SSL 프로필 구성하기:

메인 페이지에서 기본 프로필을 사용하는 모든 서비스에 적용할 TLS/SSL 프로필 수준을 선택할 수 있습니다.

  1. 적절한 TLS/SSL 프로필 수준을 선택합니다.
  2. 확인을 클릭합니다.

특정 서비스를 위한 TLS/SSL 프로필 수준 구성하기:

서비스의 보안 요구 사항을 토대로 서비스에 대한 프로필 수준을 변경할 수 있습니다.

  1. 사용자 지정 설정을 클릭하여 모든 서비스 및 해당 프로필 수준을 표시합니다.
  2. 대상 서비스의 현재 프로필 수준을 클릭합니다.
  3. 드롭다운 메뉴에서 적절한 프로필 수준을 선택합니다.
  4. 확인을 클릭합니다.

참고:

  • 시스템 기본 프로필 수준은 서비스 목록에 없는 연결에 적용됩니다.
