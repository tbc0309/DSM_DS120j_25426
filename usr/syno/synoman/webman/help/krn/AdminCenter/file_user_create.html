사용자 생성

Synology NAS에서 복수의 사용자 계정을 생성하고 관리할 수 있습니다.

사용자 생성하기:

  1. 제어판 > 사용자로 이동합니다.
  2. 생성 버튼을 클릭하고 사용자를 생성합니다. 그러면 사용자 생성 마법사가 시작됩니다.
  3. 사용자 정보 페이지에서 다음 옵션을 지정합니다.
    • 이름: 사용자 이름을 입력합니다. 이 섹션은 필수 섹션입니다.
    • 설명: 사용자에 대한 간단한 설명을 입력합니다.
    • 이메일: 사용자의 이메일 주소를 입력합니다. 패스워드 재설정 메시지 같은 시스템 알림이 여기에서 지정한 주소로 전달됩니다.
    • 새로 생성한 사용자에게 알림 메일 보내기: 이 옵션을 활성화하면 새 계정에 관한 알림 메시지가 사용자에게 전송됩니다. 이 옵션을 사용할 때는 메인 메뉴 > 제어판 > 알림 > 이메일에서 시스템 이메일 알림을 활성화해야 합니다. 기본적으로 사용자의 패스워드는 알림 메시지에 표시되지 않습니다. 알림 메시지에 패스워드를 표시하고 싶은 경우에는 알림 메일에 사용자 패스워드 표시 확인란을 선택합니다.
    • 사용자의 계정 패스워드 변경 허용 안 함: 이 옵션을 활성화하면 사용자가 DSM에 로그인할 때 사용하는 패스워드를 변경할 수 없습니다.
  4. 그룹 > 생성 > 그룹 생성 마법사 페이지에서 새 사용자가 들어갈 그룹을 지정합니다(예: 사용자, 관리자 등).
  5. 공유 폴더 권한 할당 페이지에서 액세스 권한을 수정하여 새 사용자가 액세스할 수 있는 공유 페이지를 선택합니다. 권한 우선 순위는 다음과 같습니다. 액세스 금지 > 읽기/쓰기 > 읽기 전용.
    • 미리보기: 이 열은 사용자가 속한 그룹과 현재 설정에 따라 해당 사용자의 실제 공유 폴더 액세스 권한을 표시합니다.
    • 그룹 권한: 이 열은 사용자가 속한 사용자 그룹에 따라 할당된 공유 폴더 액세스 권한을 표시합니다.
  6. 사용자 할당량 설정 페이지에서 새 사용자가 사용할 수 있는 저장소 공간 용량을 제어할 수 있습니다. 사용자가 사용할 수 있는 공간의 용량을 제한하려면, 각 볼륨/공유 폴더의 할당량 활성화 밑에 있는 확인란을 선택합니다. 그런 다음 할당량 필드에 사용할 수 있는 공간의 최대치를 입력합니다(예: 10GB).

    참고:

    • VDSM과 다음의 패키지 아키텍처가 있는 모델만 각 공유 폴더의 할당량을 지원합니다. Apollo Lake, Avoton, Braswell, Broadwell, Bromolow, Cedarview, Grantley 사용하고 있는 모델이 지원하는 패키지 아키텍처를 확인하려면 이 문서를 참조하십시오.
    • 사용자를 생성하는 특정 옵션은 EDS14에 적용되지 않을 수 있습니다.
    • DDSM은 사용자 할당량 설정을 지원하지 않습니다.
  7. 응용 프로그램 권한 할당 페이지에서 사용자가 액세스할 수 있는 서비스를 제어할 수 있습니다.
  8. 그룹 속도 제한 설정 페이지에서 여러 서비스(예: File Station, FTP 등)의 속도 제한을 활성화하여 파일을 전송할 때 사용자가 소비하는 대역폭 용량을 제한할 수 있습니다. 각 서비스에서 다음 중 하나를 선택할 수 있습니다.
    • 없음: 속도 제한 설정은 사용자가 속하는 그룹의 설정과 동일하게 됩니다.
    • 상시: 고정된 일정 속도 제한이 적용됩니다. 왼쪽 필드에서 업로드 및 다운로드 속도 제한을 지정할 수 있습니다.
    • 가변: 서로 다른 두 가지 속도 제한을 지정하여 사용자에게 일정에 따라 적용할 수 있습니다. 사용자 지정 버튼을 클릭하여 속도 제한 설정 및 일정을 수정합니다.
  9. 설정을 확인하고 적용을 클릭하여 종료합니다.

사용자 이름 제한:

  • 사용자 이름은 대소문자를 구분하지 않으며, 1 ~ 64자의 Unicode 문자를 포함하지만 다음 기호는 포함되지 않습니다.
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • 첫 번째 문자는 마이너스(-) 기호 또는 공백 문자를 사용할 수 없으며, 마지막 문자는 공백 문자를 사용할 수 없습니다.
  • 사용자 이름(mailer-daemon, postmaster)은 시스템 용도로 예약되어 있습니다.
  • 사용자 설명에는 최대 64개의 표시 가능한 Unicode 문자가 포함될 수 있습니다.

패스워드 제한:

  • 패스워드는 대소문자를 구분하며 문자, 숫자, 기호, 공백 문자를 포함, 표시 가능한 127자까지 입력 가능합니다.

할당량 제한:

  • 각 사용자에 대해 입력할 수 있는 최대 저장소 사용 할당량은 대략 4TB(4095GB)입니다.

    • 참고:

    • 이 기능은 EDS14 또는 DDSM에서 지원되지 않습니다.
