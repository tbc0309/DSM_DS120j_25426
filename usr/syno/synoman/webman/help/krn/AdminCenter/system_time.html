시간

현재 시간을 확인하고 서버의 날짜와 시간을 직접 설정하거나 네트워크 시간 서버를 사용하여 자동으로 설정할 수 있습니다.

날짜 형식 설정하기:

개인 선호도와 지리적 위치에 따라 날짜 형식을 선택합니다.

시간 형식 설정하기:

표시 시간을 12시간 또는 24시간 형식 중에서 선택합니다.

표준 시간대 설정하기:

표준 시간대 드롭다운 메뉴에서 옵션을 선택합니다.

날짜와 시간을 수동으로 설정하기:

수동을 선택하고 날짜와 시간을 선택합니다.

날짜와 시간을 NTP 서버와 동기화하기:

  1. NTP 서버와 동기화를 선택하고 네트워크 시간 서버의 주소를 입력하거나 드롭다운 메뉴에서 서버를 선택합니다.
  2. 지금 업데이트를 클릭하여 Synology NAS에서 서버와 시간을 동기화할 수 있는지 확인합니다.
  3. 적용을 클릭합니다.

참고: 네트워크 시간 서버와 날짜 및 시간을 동기화하려면 메인 메뉴 > 제어판 > 네트워크에서 올바른 DNS 서버 IP를 입력했는지와 Synology NAS을(를) 인터넷에 연결했는지 확인하십시오.

네트워크 시간 프로토콜(NTP) 서버 관련 정보

네트워크 시간 프로토콜(NTP)의 주요 용도는 데이터 네트워크에서 컴퓨터 시스템의 시계를 동기화하는 것입니다.

NTP 서버에 대한 자세한 내용은 http://support.ntp.org/bin/view/Servers/WebHome을 참조하십시오.
