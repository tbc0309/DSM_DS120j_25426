사용자

유연한 사용자 관리 옵션을 사용하여 가족 또는 회사의 각 구성원에 대해 사용자 계정을 생성할 수 있습니다. 사용자를 생성하면 각 개인에 대해 공유 폴더 액세스 권한, 저장소/공유 폴더 할당량, 대역폭 제한 같은 권한을 정의하고 관리할 수 있습니다.

    참고:
  1. 공유 폴더 관리는 Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake, Denverton 플랫폼에만 적용될 수 있습니다. 사용 모델이 Btrfs를 지원하는지 확인하려면 이 항목을 참조하십시오.
  2. 저장소/공유 폴더 할당량 관리는 Docker DSM에 적용될 수 없습니다.
