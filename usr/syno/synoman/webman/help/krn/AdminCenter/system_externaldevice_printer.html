프린터

Synology NAS은(는) 인쇄 서버로 변환되므로 클라이언트 컴퓨터나 모바일 장치가 Synology NAS에 연결된 프린터에 액세스할 수 있습니다. 또한 Apple 무선 인쇄가 지원되므로 iOS 장치를 사용하여 원격으로 인쇄할 수 있습니다.

호환되는 프린터는 www.synology.com을 참조하십시오.

프린터 설정

아래 지침을 참조하여 USB 또는 네트워크 프린터를 설정하고 Synology NAS에 연결합니다.

USB 프린터 설정하기:

  1. 전원을 켜고 프린터를 Synology NAS의 USB 포트에 연결합니다.
  2. 제어판 > 외부 장치 > 프린터로 이동합니다.
  3. 설정할 프린터를 선택합니다. Synology NAS은(는) USB 허브를 통해 연결된 프린터를 인식할 수 없다는 점에 유의하십시오.
  4. 프린터 관리자를 클릭하고 프린터 설정을 선택합니다.
  5. 대화 상자에서 다음 모드 중 하나를 선택합니다.
    • 네트워크 MFP: 로컬 네트워크 내의 클라이언트가 복합기의 인쇄, 스캔 및 팩스 기능을 사용할 수 있습니다. Windows 컴퓨터에서만 스캔과 팩스 기능을 사용할 수 있습니다.
    • 네트워크 프린터: 인쇄만 지원하는 프린터를 사용하는 경우나 클라이언트가 인쇄 기능만 사용할 수 있도록 하려는 경우에 이 모드를 선택합니다.
  6. Apple 무선 인쇄 설정을 구성합니다(선택 사항).
  7. 저장을 클릭합니다.

참고:

  • 클라이언트 하나에서만 동시에 네트워크 MFP에 액세스할 수 있습니다.

네트워크 프린터 설정하기:

  1. 전원을 켜고 네트워크 프린터를 Synology NAS과(와) 동일한 로컬 네트워크에 연결합니다.
  2. 제어판 > 외부 장치 > 프린터로 이동합니다.
  3. 네트워크 프린터 추가를 클릭하고 설정 마법사의 지침을 따릅니다.

참고:

  • 프린터 이름: 설정하는 동안 프린터 이름을 입력해야 하며, 이름은 문자, 숫자, 밑줄을 포함하여 최대 32자입니다.
  • 프로토콜: 일반적으로 HP 네트워크 프린터는 소켓을, Canon 네트워크 프린터는 BJNP를, 기타 대부분은 LPR을 사용합니다.
  • 대기열 이름: LPR 및 IPP 네트워크 프린터에는 대기열 이름을 입력해야 하지만, 소켓이나 BJNP 프린터에는 대기열 이름을 입력할 필요가 없습니다.
  • 네트워크 MFP 없음: Synology NAS에 연결된 네트워크 프린터는 현재 네트워크 MFP 모드를 사용할 수 없습니다.
  • USB 프린터 없음: Synology 제품에서 공유되는 USB 프린터를 네트워크 프린터로 설정할 수 없습니다.

Mac을 사용하여 PostScript USB 프린터 설정하기:

Mac OS 10.12.X의 경우:

  1. USB 프린터 설정하기에 있는 단계를 따릅니다.
  2. Mac에서 시스템 환경설정 > 프린터 및 스캐너로 이동한 후 +를 클릭하여 새 프린터를 추가합니다.
  3. 프린터 유형으로 IP를 선택합니다.
    • 프린터의 호스트 이름 또는 IP 주소를 입력합니다.
    • 프로토콜에 라인 프린터 데몬 - LPD를 선택합니다.
    • 대기열 필드에 "usbprinter"를 입력합니다. 대기열 이름을 찾을 수 없으면 DSM의 제어판 > 외부 장치 > 프린터로 이동합니다.
  4. 사용에 일반 PostScript 프린터를 선택합니다.
  5. 추가를 클릭하여 종료합니다.

Mac OS 10.6.X의 경우:

  1. USB 프린터 설정하기에 있는 단계를 따릅니다.
  2. Mac에서 시스템 환경설정 > 프린터 및 팩스로 이동한 후 +를 클릭하여 새 프린터를 추가합니다.
  3. IP를 선택합니다.
    • 프로토콜에 라인 프린터 데몬 - LPD를 선택합니다.
    • 프린터의 IP 주소를 입력합니다.
    • 대기열 필드에 "usbprinter"를 입력합니다.
    • 인쇄 사용에 일반 PostScript 프린터를 선택합니다.
    • 추가를 클릭하여 종료합니다.

참고:

  • AppleTalk는 더 이상 Mac OS 10.6.X에서 지원되지 않습니다. AppleTalk 프린터를 사용해야 하는 경우 관련 문서를 참조하여 Mac 컴퓨터 또는 로컬 네트워크에 프린터를 연결하는 방법을 확인하십시오. 프린터를 연결할 방법이 없으면 Mac OS 10.5.X 이하 버전의 Mac 컴퓨터에 연결하고 공유되도록 설정하십시오.

Mac OS 10.5.X의 경우:

  1. USB 프린터 설정하기에 있는 단계를 따릅니다.
  2. Mac에서 시스템 환경설정 > 프린터 및 팩스로 이동한 후 +를 클릭하여 새 프린터를 추가합니다.
  3. 방법을 선택합니다.
    • 방법 1: AppleTalk
      • 인쇄 사용에 일반 PostScript 프린터를 선택합니다.
      • 추가를 클릭하여 종료합니다.
    • 방법 2: IP
      • 프로토콜에 라인 프린터 데몬 - LPD를 선택합니다.
      • 프린터의 IP 주소를 입력합니다.
      • 대기열 필드에 "usbprinter"를 입력합니다.
      • 인쇄 사용에 일반 PostScript 프린터를 선택합니다.
      • 추가를 클릭하여 종료합니다.

프린터 관리

프린터 탭에는 다음과 같은 Synology NAS에 연결된 USB 또는 네트워크 프린터를 관리하는 옵션이 있습니다.

  • 정보 가져오기: 장치 이름, 제조업체, 상태 등과 같은 선택한 프린터와 관련된 정보를 표시합니다.
  • 제거: Synology NAS에서 프린터를 제거하고 연결을 끊습니다.
  • 프린터 관리자: 다음 옵션이 포함되어 있습니다.
    • 프린터 설정: 프린터 설정을 편집합니다. Apple 무선 인쇄를 설정하려면 다음 지침을 참조하십시오.
    • 스풀 공간 정리: 모든 프린터의 모든 인쇄 작업을 취소합니다.
    • 테스트 페이지 인쇄: 테스트 페이지를 인쇄하여 구성이 올바른지 확인합니다.
    • 프린터 해제: 프린터와 클라이언트 연결을 끊습니다(네트워크 MFP 모드만 해당). 클라이언트가 복합기에 연결되어 팩스를 수신하거나 파일을 스캔하면 작업이 오래 걸릴 수 있습니다. 필요한 경우, 대기 중인 다음 클라이언트가 프린터에 액세스할 수 있도록 연결을 중지할 수 있습니다.

Apple 무선 인쇄 활성화하기:

  1. 제어판 > 외부 장치 > 프린터로 이동합니다.
  2. 장치 목록에서 프린터를 선택합니다.
  3. 프린터 관리자 > 프린터 설정을 클릭합니다.
  4. Apple 무선 인쇄 활성화 확인란을 확인 표시합니다.
  5. 프린터의 브랜드와 드라이버를 선택합니다.
  6. 저장을 클릭합니다.

설정이 완료되면 프린터 관리자 > 테스트 페이지 인쇄를 클릭하여 테스트 페이지를 인쇄할 수 있습니다.

참고:

  • Apple 무선 인쇄를 활성화하면 DSM에 대한 Bonjour 프린터 브로드캐스트도 활성화됩니다.
  • iOS 4.2 이상에서 Apple 무선 인쇄를 사용할 수 있습니다.

프린터에 액세스

PC 또는 Mac을 사용하여 프린터에 액세스하려면 아래 섹션을 참조하십시오.

Windows PC를 사용하여 프린터에 액세스하기:

Synology NAS 장치의 인쇄 서비스에 액세스하는 가장 좋은 방법은 Synology Assistant를 사용하여 프린터에 대한 연결을 설정하는 것입니다. 자세한 지침은 해당 도움말을 참조하십시오.

컴퓨터에 Synology Assistant가 설치되지 않은 경우 Synology NAS 장치의 인쇄 서비스에 액세스하는 다른 방법은 아래를 참조하십시오.

유형 1. 공유 네트워크 프린터

  1. PC의 파일 탐색기에서 네트워크를 열고 Synology NAS을(를) 더블 클릭합니다.
  2. usbprinter 또는 네트워크 프린터 이름을 더블 클릭합니다. 시스템에서 올바른 드라이버를 설치하라는 메시지가 표시되면 예를 클릭합니다.
  3. 제조업체의 CD나 웹사이트에서 프린터 드라이버를 가져오십시오. 프린터의 제조업체와 모델을 선택하거나 디스크 있음을 클릭하여 드라이버 위치를 선택합니다. 그런 다음 확인을 클릭합니다.
  4. 시스템이 네트워크 프린터 설정을 시작합니다. 설정이 완료되면 프린터를 사용할 수 있습니다.

유형 2. LPR 포트(권장)

  1. PC에서 시작을 클릭하고 장치 및 프린터로 이동합니다.
  2. 프린터 추가를 클릭합니다.
  3. 네트워크, 무선 또는 Bluetooth 프린터 추가를 선택합니다.
  4. 원하는 프린터가 목록에 없습니다를 선택합니다.
  5. TCP/IP 주소 또는 호스트 이름으로 프린터 추가를 선택합니다. 다음을 클릭하여 계속 진행합니다.
  6. 장치 유형을 TCP/IP 포트 장치로 선택합니다.
  7. Synology NAS의 IP 주소를 입력합니다. 다음을 클릭하여 계속 진행합니다.
  8. 장치 유형: 사용자 지정 > 설정 > …을 선택합니다. 다음을 클릭하여 계속 진행합니다.
  9. 프로토콜로 LPR을 선택하고 대기열 이름으로 usbprinter 또는 네트워크 프린터 이름을 입력한 후 LPR 바이트 계산 사용 확인란을 확인 표시합니다. 확인을 클릭하여 계속 진행합니다.
  10. 제조업체의 CD나 웹사이트에서 프린터 드라이버를 가져오십시오. 프린터의 제조업체와 모델을 선택하거나 디스크 있음을 클릭하여 드라이버 위치를 선택합니다. 그런 다음 확인을 클릭합니다.
  11. 시스템이 네트워크 프린터 설정을 시작합니다. 설정이 완료되면 프린터를 사용할 수 있습니다.

Mac을 사용하여 프린터에 액세스하기:

Mac 사용자는 시스템 환경설정에서 DSM의 Bonjour 프린터 브로드캐스트를 활성화하고 프린터를 추가하여 Synology NAS 장치의 인쇄 서비스에 액세스할 수 있습니다.

계속하기 전에 Synology NAS이(가) Postscript 프린터를 사용 중인지 확인하십시오.

1단계. DSM의 Bonjour 프린터 브로드캐스트 활성화

  1. DSM에 admin 또는 administrators 그룹에 속하는 사용자로 로그인합니다.
  2. 제어판 > 파일 서비스 > Win/Mac/NFS로 이동합니다.
  3. Bonjour 프린터 브로드캐스트 활성화를 선택합니다.

2단계 Mac을 사용하여 프린터에 액세스

  1. Mac에서 시스템 환경설정 > 인쇄 및 스캔으로 이동한 후 +를 클릭합니다.
  2. 드롭다운 메뉴에서 usbprinter[number] 또는 네트워크 프린터 이름을 선택합니다. Mac에서 프린터가 자동으로 설정됩니다.

참고:

  • RC18015xs+는 USB 프린터를 지원하지 않습니다.
