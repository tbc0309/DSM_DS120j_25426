고급

고급 탭에서 파일 빠른 복제 기능을 활성화하여 파일을 복제 시 저장소 공간을 절약할 수 있습니다. 또한, 사용자가 네트워크에서 Bonjour 및 SSDP(Simple Service Discovery Protocol) 서비스를 통해 이 Synology NAS을(를) 찾을 수 있도록 설정을 구성할 수 있습니다.

Synology NAS에서 파일 빠른 복제 수행하기:

파일 빠른 복제 기능을 이용하면, 복제된 파일이 변경된 때만 물리적 데이터 블록이 복사되고, 이는 저장 공간을 절약하는 데 도움이 됩니다.

  1. 파일 빠른 복제 섹션 아래에서 파일 빠른 복제 활성화 확인란을 선택합니다.
  2. 적용을 클릭하여 설정을 저장합니다.

Synology NAS이(가) Bonjour 서비스를 통해 검색되도록 허용:

Bonjour 서비스를 이용하면 사용자가 Synology NAS을(를) 찾아 Mac 컴퓨터에서 인쇄 및 백업 작업을 수행할 수 있습니다.

  1. Bonjour 섹션에서 Bonjour 서비스 검색 활성화를 선택합니다.
  2. 필요에 따라 Bonjour 관련 옵션을 선택할 수 있습니다.
    • Bonjour 프린터 브로드캐스트 활성화: 이 옵션을 선택하면 Mac 컴퓨터에서 Synology NAS에 연결된 USB 프린터를 찾아 문서를 인쇄할 수 있습니다.
    • SMB를 통한 Bonjour Time Machine 브로드캐스트 활성화: 이 옵션을 선택하면 Mac Time Machine에서 Synology NAS을(를) 찾아 SMB를 통해 파일을 DSM 공유 폴더에 백업할 수 있습니다.
    • AFP를 통한 Bonjour Time Machine 브로드캐스트 활성화: 이 옵션을 선택하면 Mac Time Machine에서 Synology NAS을(를) 찾아 AFP를 통해 파일을 DSM 공유 폴더에 백업할 수 있습니다.
  3. Time Machine과 관련된 옵션을 선택한 경우, Time Machine 폴더 설정을 클릭하여 Time Machine 백업을 위한 대상 폴더를 지정합니다.
  4. 적용을 클릭하여 설정을 저장합니다.

참고:

  • SMB/AFP 서비스를 비활성화한 경우(제어판 > 파일 서비스 > SMB/AFP/NFS), 연관된 Bonjour 옵션이 자동으로 비활성화됩니다.
  • SMB를 통한 Bonjour Time Machine 브로드캐스트 활성화를 선택하면 다음 옵션이 활성화됩니다(제어판 > 파일 서비스 > SMB/AFP/NFS > 고급 설정).
    • SMB 3 활성화(Windows 파일 서비스용 최대 프로토콜)
    • Opportunistic Locking
    • SMB2 임대 활성화
    • SMB 지속 가능 핸들 활성화(프로토콜 간 파일 잠금이 비활성화됨)

Synology NAS이(가) SSDP를 통해 발견되도록 허용:

SSDP 서비스를 이용하면 사용자가 Windows 네트워크 찾기를 통해 Synology NAS을(를) 검색할 수 있습니다.

  1. SSDP 섹션에서 웹 브라우저를 통해 액세스할 수 있도록 Windows 네트워크 검색 활성화 확인란을 선택합니다.
  2. 적용을 클릭하여 설정을 저장합니다.

Synology NAS이(가) WS-Discovery를 통해 발견되도록 허용:

WS-Discovery 서비스를 이용하면 사용자가 Windows 네트워크 찾기를 통해 Synology NAS에서 SMB 서비스를 검색할 수 있습니다.

  1. WS-Discovery 섹션에서 SMB를 통해 파일에 액세스할 수 있도록 Windows 네트워크 검색 활성화를 선택합니다.
  2. 적용을 클릭하여 설정을 저장합니다.
