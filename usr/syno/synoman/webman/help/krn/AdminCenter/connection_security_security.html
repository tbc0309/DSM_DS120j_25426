보안

제어판 > 보안 > 보안에서 DSM 검색 세션에 대해 다음 보안 설정을 수정할 수 있습니다.

  • 로그아웃 타이머(분): 여기에서 지정한 기간에 사용자가 비활성되면 자동으로 DSM에서 로그아웃됩니다. 1~65535 사이의 값을 입력합니다.
  • IP 검사를 건너뛰어 브라우저 호환성 개선: HTTP 프록시를 통해 Synology NAS에 액세스할 때 임의 로그아웃 문제가 발생한 경우 이 옵션을 활성화하여 IP 검사를 건너뛸 수 있습니다.
  • 교차 사이트 요청 위조 공격에 대한 보호 기능 향상: 이 옵션은 교차 사이트 스크립팅 공격으로부터 시스템의 보호 성능을 강화합니다. 이 옵션은 다음에 DSM에 로그인할 때 적용됩니다.
  • HTTP 콘텐츠 보안 정책(CSP) 헤더로 보안 향상: 이 옵션은 신뢰할 수 있는 소스의 데이터만 허용하고 인라인 스크립트 실행을 제한하여 교차 사이트 스크립팅(XSS) 공격에 대한 시스템 보안을 강화합니다.
  • iFrame과 함께 DSM이 포함되도록 허용하지 않음: 이 옵션을 활성화하면 다른 웹사이트가 iFrame을 사용하여 다른 웹 페이지에 DSM을 포함하지 못하도록 제한하여 악의적인 웹사이트로부터의 특정 유형 공격을 방지할 수 있습니다. 허용된 웹사이트 버튼을 클릭하여 특정 웹사이트가 iFrame을 통해 DSM을 포함하도록 허용할 수 있습니다.
  • 시스템 재시작 시 저장된 모든 사용자 로그인 세션 지우기: 이 옵션은 사용자가 로그인된 상태이고 시스템이 준비된 상태에서 시스템에서 작업을 수행하는 경우에 발생하는 예상치 못한 시스템 오류를 방지합니다. 이 옵션을 활성화하면 모든 사용자는 시스템 다시 시작 후 다시 로그인해야 합니다.
  • 현재 IP 변경 시 DSM 데스크톱에 알림 표시: 현재 연결된 사용자의 IP가 변경되면 해당 사용자에게 데스크톱 알림이 발송됩니다.
  • 신뢰할 수 있는 프록시: DSM은 신뢰할 수 있는 프록시 서버를 통과한 원격 IP 정보를 사용합니다. 신뢰할 수 있는 프록시 서버를 변경할 수 있습니다.
