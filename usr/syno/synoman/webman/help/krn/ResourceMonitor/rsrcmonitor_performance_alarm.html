성능 알람*

성능 알람 보기:

  • 성능 알람으로 이동하여 트리거된 모든 성능 알람을 확인합니다.
  • 새로 고침을 클릭하여 최신 업데이트된 알람을 가져옵니다.

성능 알람 내보내기:

  1. 성능 알람으로 이동합니다.
  2. 내보내기 버튼 옆에 있는 화살표를 클릭합니다.
  3. HTML 또는 CSV으로 알람을 내보낼 수 있습니다.

참고:

  • 성능 알람에 대한 지원은 Synology NAS 모델에 따라 다릅니다. 자세한 내용은 www.synology.com을 참조하십시오.

*특정 모델에서만 사용할 수 있습니다.
