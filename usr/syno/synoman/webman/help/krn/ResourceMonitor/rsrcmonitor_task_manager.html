작업 관리자*

작업 관리자 탭에서 각 서비스와 프로세스의 CPU 및 메모리 사용을 볼 수 있습니다.

서비스

서비스 탭을 클릭하여 CPU 사용, CPU 시간, 메모리 사용 및 디스크 읽기/쓰기 상태를 포함한 각 서비스의 다양한 값을 모니터링합니다.

프로세스

프로세스 탭을 클릭하여 CPU 부하를 줄이기 위해 각 프로세스의 CPU와 메모리 사용을 확인할 수 있습니다.

프로세스 상태는 실행 중, 수면 중 및 정지됨으로 표시됩니다. Linux의 프로세스 상태는 실행 중, 정지됨/추적 중, 수면 중이 사용됩니다.

수면 중 뒤에는 자세한 정보를 나타내는 문자가 올 수 있습니다(예: 'D'는 디스크 수면, 'Z'는 좀비, 'X'는 오류).

공유 메모리는 프로세스에서 사용되며 다른 프로세스와 공유할 수 있는 물리적 메모리의 크기를 표시합니다. 비공유 메모리는 프로세스에서 사용되며 다른 프로세스와 공유할 수 없는 물리적 메모리의 크기를 표시합니다.

*일부 모델에서만 사용할 수 있습니다.
