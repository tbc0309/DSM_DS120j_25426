보안 어드바이저

보안 어드바이저는 DSM 설정 및 Synology NAS을(를) 스캔하는 DSM 보안 응용 프로그램입니다. 보안 어드바이저는 설정을 검사하고 Synology NAS 안전 유지를 지원하는 변경 사항을 추천합니다. 아래는 제공되는 기능의 목록입니다.

  • 보안 어드바이저는 보안 위험이나 의심스러운 활동이 나타날 수 있는 DSM 설정을 감지하기 위해 여러 가지 검사를 수행합니다.
  • 보안 어드바이저는 비정상적인 로그인 활동과 악의적인 제3자(무차별 패스워드 대입 공격)가 추측한 패스워드를 분석합니다.
  • 보안 어드바이저는 상세 결과를 제공하고, 취해야 할 권장 조치 및 이로 인한 이점을 설명하는 보고서를 제공합니다.
  • 내부 보안 정의 데이터베이스는 자동으로 업데이트되어 변화하는 사이버 위협에 최신 상태로 대처합니다.
  • 보안 어드바이저는 권장 조치 사항이 기재된 월간 또는 일일 보고서를 생성합니다.

보안 어드바이저를 실행하려면 다음을 수행합니다.

  • 메인 메뉴 > 보안 어드바이저로 이동합니다.
