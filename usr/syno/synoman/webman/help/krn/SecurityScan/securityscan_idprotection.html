로그인 분석

시스템 관리자는 로그인 분석을 사용하여 현재 시스템과 계정의 보안 정보를 확인할 수 있습니다.

보안 어드바이저 > 고급 > 검사 목록 사용자 지정에서 로그인 분석을 비활성화하거나 규칙을 편집할 수 있습니다.

새로운 로그인 활동

보안 어드바이저는 HTTP 사용자 에이전트, IP 주소 및 지리적 위치를 포함한 각 사용자의 로그인 정보를 분석합니다. 새 로그인 활동이 감지되면 보안 어드바이저는 시스템 관리자와 문제가 되는 사용자에게 알림을 발송합니다.

특정 로그인 기록이 비정상으로 확인되면 계정 보안을 강화하기 위해 더욱 복잡한 패스워드 사용, 2단계 검증 활성화 및 허용된 IP 제한을 수행할 것을 강력히 권장합니다.

패스워드 추측(무차별 패스워드 대입 공격)

보안 어드바이저는 서로 다른 IP와 프로토콜에서 실패한 로그인 시도를 분석합니다. 누군가 패스워드를 입력하여 로그인을 시도했지만 수 차례 실패한 경우, 보안 어드바이저는 시스템 관리자와 문제가 되는 사용자에게 알림을 발송합니다. 시스템 관리자는 자동 차단 또는 계정 보호(제어판 > 보안 > 계정에서)를 활성화하여 계정 보안을 강화할 수 있습니다.
