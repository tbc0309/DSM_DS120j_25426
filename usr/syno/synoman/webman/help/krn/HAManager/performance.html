성능

성능 페이지에서 활성 서버와 수동 서버의 리소스 사용량을 각각 모니터링할 수 있습니다. 두 서버를 서버 이름과 각각의 역할(활성 또는 수동)로 간편하게 구분할 수 있습니다. 수동 서버 전원이 꺼지면 수동 서버의 리소스 사용량은 표시되지 않습니다.

CPU

시스템, 사용자 및 I/O 큐의 전체 CPU 사용량을 확인할 수 있습니다. 페이지를 열면 시스템이 페이지를 불러 오면서 동시에 시스템 사용량 정보를 수집하므로 CPU 부하가 높아질 수 있습니다.

메모리

물리적 메모리의 전체 사용량이 표시됩니다. 메모리 사용량이 계속 높으면 시스템은 디스크에 액세스할 필요 없이 데이터에 신속하게 액세스할 수 있도록 자주 사용하는 데이터를 캐시에 저장할 수 있습니다. 전체 메모리가 충분하지 않으면 시스템은 캐시 메모리를 해제합니다.

네트워크

네트워크 트래픽 그래프는 여러 가지 색상으로 데이터 전송 속도(KB/s)를 표시합니다.

  • 파란색은 업로드 속도를 나타냅니다.
  • 녹색은 다운로드 속도를 나타냅니다.
  • 업로드 속도와 다운로드 속도가 겹치면 겹친 영역은 진한 녹색으로 표시됩니다.

디스크

그래프는 전체 디스크 사용량을 표시합니다.
