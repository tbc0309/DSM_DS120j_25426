Backup de dados em meu PC/Mac usando o Cloud Station

Visão geral

O Cloud Station Server não é apenas um sofisticado software de sincronização entre plataformas. Os usuários podem usá-lo como um meio mais inteligente de fazer backup de seus computadores pessoais. Uma solução de backup em tempo real, o Cloud Station Backup é instantâneo e economiza largura de banda, oferecendo uma proteção melhor com até 32 versões históricas de um único arquivo.

Configuração do Cloud Station Server no Synology NAS

  1. Vá para Centro de pacotes, encontre o Cloud Station Server e em seguida clique em Instalar.
  2. Abra o Cloud Station Server.
  3. Vá para Visão geral e clique em Baixar agora em Cloud Station Backup. (Você também pode baixar o Cloud Station Backup  do Centro de Download Synology.)
  4. Inicie o instalador em seu computador e siga as instruções para instalar o Cloud Station Backup.
  5. Após a instalação, execute o Cloud Station Backup em seu computador. Clique em Iniciar Agora.
  6. Digite o endereço do Synology NAS executando o Cloud Station Server (ou QuickConnect ID), nome de usuário e senha. (Você também pode clicar no ícone à direita e o Cloud Station Backup pesquisará automaticamente outro Synology NAS dentro de sua LAN). Clique em Avançar.
  7. Selecione sua origem de backup e desmarque qualquer subpasta que não deseje sincronizar.
  8. Selecione uma pasta em seu Synology NAS clicando em Selecionar. O Cloud Station Backup criará uma pasta de backup em seu destino selecionado com o nome do seu computador. A pasta home será selecionada como pasta de destino por padrão. Se você desejar usar uma pasta que não está na lista, entre em contato com seu administrador de sistemas. Os administradores de sistemas podem consultar a seção Gerenciar o recurso de compartilhamento do Cloud Station Server na ajuda do Cloud Station Server.
  9. Clique em Regras de backup se desejar definir filtros ou regras de sincronização.
  10. Clique em Avançar para analisar suas configurações de backup, em seguida clique em  Concluído.
  11. Você pode visualizar o andamento do seu backup no aplicativo principal.

Baixar ou restaurar uma versão anterior de um arquivo sincronizado no Cloud Station Server

  1. Vá para Cloud Station Server > Histórico de versões e localize o arquivo ou a pasta que deseja restaurar.
  2. Selecione o arquivo que deseja recuperar e clique em Ação. Clique em Procurar versões anteriores. Se um arquivo for excluído ou removido acidentalmente de seu computador, você pode clicar em Restaurar para restaurá-lo.
  3. Selecione a versão do arquivo que deseja baixar e clique em Download. Ou, se você tem certeza de que quer a versão selecionada restaurada, clique em Restaurar e ela substituirá a versão atual.
