Criação do volume

Este artigo explica como criar volumes em seu Synology NAS. Ao criar um novo volume, é possível selecionar um dos seguintes métodos:

  • Rápido: Essa é a melhor opção para iniciantes. Selecione esse método se você desejar otimizar automaticamente a capacidade de armazenamento e desempenho sem se preocupar com os aspectos técnicos, como níveis de RAID.
  • Personalizado: Para usuários avançados que queiram controlar os aspectos mais detalhados da criação do volume, como tipos de RAID e atribuição de capacidade. Essa opção oferece suporte a diferentes tipos de RAID, permite que você crie volumes únicos ou múltiplos e fornece opções para alocar quantidades específicas de capacidade para cada volume (somente ao criar vários volumes em RAID).

Volumes Únicos ou Múltiplos em RAID?

Ao criar um volume com o modo personalizado, é possível selecionar uma das seguintes opções:

  • Volume único no RAID
    • Cria um único volume usando toda a capacidade disponível dos discos rígidos selecionados.
    • Proporciona melhor velocidade e desempenho.
    • Suporta diferentes tipos de RAID.
  • Volumes múltiplos no RAID
    • Permite que você crie vários volumes em um grupo de discos.
    • Suporta a alocação de quantidades específicas da capacidade para cada volume.
    • Fornece excelente flexibilidade de gerenciamento de armazenamento.
    • Suporta diferentes tipos de RAID.

Antes de Começar:

  • Antes de começar, verifique se não há dados importantes nos discos rígidos. Todos os dados existentes serão apagados durante o processo de criação.
  • Vá para a página HDD/SSD e verifique se o status de cada disco rígido é Normal ou Não inicializado.

Nota:

  • O tamanho mínimo de um volume Btrfs é 10 GB.
  • Os volumes Btrfs reservam 4% do espaço para metadados.

Para criar um volume:

  1. Abra o Gerenciador de armazenamento.
  2. Vá para Volume e clique em Criar.
  3. Você verá o assistente de criação de volumes. Selecione Rápida ou Personalizada de acordo com suas necessidades.
  4. Modifique as configurações desejadas.
  5. Escolha se deseja registrar a hora de acesso ao arquivo:
    • Ativar a opção Registrar hora de acesso ao arquivo é equivalente a usar a opção de montagem relatime em sistemas Linux. A hora de acesso ao arquivo será atualizada somente quando a hora do último acesso (atime) for anterior à hora de modificação atual (mtime) ou hora de mudança (ctime). A hora de acesso também será atualizada se a hora do último acesso tiver ocorrido há mais de um dia.
    • Desativar a opção Registrar hora de acesso ao arquivo é equivalente a usar a opção de montagem noatime em sistemas Linux. A hora de acesso de arquivos e pastas não será atualizada, resultando em menor uso do sistema de arquivos. Observe que alguns aplicativos podem depender da hora de acesso ao arquivo para funcionar corretamente.
  6. Clique no botão Aplicar para confirmar as configurações.
