Indexação de arquivos

A indexação de uma pasta compartilhada ou de suas subpastas pode ajudá-lo a encontrar rapidamente os arquivos contidos e os conteúdos durante a pesquisa de arquivos.

Nota:

Pastas remotas, pastas na nuvem ou pastas .iso montadas em seu Synology NAS não podem ser indexadas.

Para indexar pastas e o conteúdo do arquivo:

Quando você indexar uma pasta, o File Station indexará a pasta e seus arquivos/subpastas por nome, permitindo que você pesquise os itens desejados pelo nome do arquivo. Se você também indexar por tipo de arquivo, o File Station indexará os arquivos correspondentes não só por nome, mas também por conteúdo. Assim, você poderá pesquisar os itens pelo nome e pelo conteúdo do arquivo.

  1. Vá para Painel de controle > Serviço de indexação > Indexação de arquivo e clique em Lista de pastas indexadas.
  2. Na janela pop-up, clique em Criar > Selecionar para exibir as pastas prontas para indexação.
  3. Selecione a pasta desejada e clique em Selecionar. Para criar uma nova pasta para indexação, clique em Criar pasta e prossiga.
  4. Clique na pasta que deseja indexar e clique em Selecionar.
  5. Para indexar tipos específicos de arquivo, vá para Tipo de arquivo e selecione Imagem, Áudio, Vídeo ou Documento para a indexação. Se algum tipo de arquivo (por exemplo, Áudio) não for selecionado, você só poderá pesquisar arquivos desse tipo (ou seja, de áudio) pelo nome do arquivo.
  6. Clique em OK para concluir a indexação.

Nota:

  • Você pode selecionar no máximo 1.000 pastas para indexação. As subpastas dentro das pastas selecionadas não estão incluídas na contagem.
  • Se você selecionar Habilitar pesquisa de conteúdo do arquivo no File Station, pode pesquisar o conteúdo dos arquivos dentro das pastas indexadas.
  • O volume que hospeda a pasta indexada deve ter um espaço disponível maior do que 10 MB. Caso contrário, a indexação será pausada. Para retomar a indexação, expanda o volume e clique em Reindexar.
  • Se você indexou uma pasta e suas subpastas ao mesmo tempo, o sistema indexará apenas as subpastas para economizar recursos do sistema.
  • Após as pastas serem totalmente indexadas e depois montadas em um computador Mac através do protocolo SMB ou AFP, você pode usar o Finder para pesquisar o conteúdo dos arquivos.

Para excluir pastas indexadas:

  1. Vá para Painel de controle > Serviço de indexação > Indexação de arquivo e clique em Lista de pastas indexadas e selecione as pastas indexadas desejadas.
  2. Clique em Excluir para interromper a indexação das pastas.

Reindexar Arquivos

A reindexação de arquivos não é necessária sob circunstâncias normais. Entretanto, se os arquivos em uma pasta correspondem aos critérios específicos mas de alguma forma parecem estar ausentes durante a pesquisa de arquivos, pode ser necessário reindexar os arquivos para se reconstruir a estrutura do índice.

Para reindexar arquivos em uma pasta compartilhada:

  1. Vá para Painel de controle > Serviço de indexação > Indexação de arquivo > Lista de pastas indexadas.
  2. Selecione as pastas listadas para reindexar.
  3. Clique em Reindexar.

Nota:

O File Station não suporta a indexação de arquivos em pastas remotas ou dispositivos de armazenamento externos (ex. unidades USB e cartões SD).

Gerenciar a indexação de arquivos através da área de trabalho

Quando a indexação de arquivos for iniciada, você poderá encontrar o Andamento da indexação (um ícone de lupa flutuante) na barra de tarefas. Como a indexação de arquivos exige mais recursos da CPU, isso permite economizar recursos da CPU.

  • Para pausar/retomar a indexação de arquivos, clique no botão de reprodução.
  • Para adiar a indexação de arquivos até o momento adequado, selecione a opção desejado do menu suspenso.
  • Para visualizar quais pastas estão sendo indexadas, clique em Exibir lista de pastas indexadas.

Nota:

  • Se você pausar a ação de indexação, só será possível retomá-la clicando no botão de reprodução na barra de tarefas da área de trabalho.
