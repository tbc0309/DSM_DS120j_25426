SMB/AFP/NFS

Os usuários do Windows, Mac e Linux podem acessar o Synology NAS a partir da rede, caso as opções em Painel de Controle > Serviços de Arquivo > SMB/AFP/NFS estejam habilitadas.

Os seguintes protocolos de compartilhamento de arquivo são suportados:

  • Para Windows: SMB/CIFS (Meus Locais de Rede), FTP, WebDAV
  • Para Mac: SMB, FTP, AFP, WebDAV
  • Para Linux: SMB, FTP, NFS, WebDAV
