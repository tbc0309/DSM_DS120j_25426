Gerenciar chaves de pastas compartilhadas criptografadas

Você pode usar o Gerenciador de chaves para gerenciar chaves de pastas compartilhadas e descriptografar pastas compartilhadas criptografadas ao mesmo tempo.

Explicação dos termos:

  • Armazenamento de chaves: Um armazenamento de chaves é qualquer dispositivo externo suportado pelo Synology NAS e usado para descriptografar pastas compartilhadas.
  • Criptografia: Uma criptografia é um método de criptografar chaves de pastas compartilhadas. O Gerenciador de chaves oferece dois tipos de criptografias:
    • Senha: As chaves criptografadas por uma senha podem ser descriptografadas por quem souber a senha.
    • Chave de máquina: As chaves criptografadas por uma chave de máquina só podem ser descriptografadas pelo Synology NAS vinculado.

Para inicializar um armazenamento de chaves:

  1. Vá até Painel de Controle > Pasta Compartilhada > Ação > Gerenciador de chaves.
  2. Selecione um dispositivo externo como armazenamento de chave em Local do armazenamento de chaves.
  3. Digite uma senha no campo Senha para esse armazenamento de chaves.

Para adicionar uma nova chave a um armazenamento de chaves:

  1. Verifique se você inicializou um armazenamento de chaves.
  2. Vá até Painel de Controle > Pasta Compartilhada > Ação > Gerenciador de chaves.
  3. Clique em Adicionar.
  4. Selecione uma pasta compartilhada criptografada.
  5. Selecione a criptografia para a chave de criptografia. Você pode selecionar Senha ou Chave de máquina.
  6. Digite ou importe a chave de criptografia.
  7. Clique em OK para salvar as configurações.

Para descriptografar várias pastas compartilhadas criptografadas manualmente:

  1. Verifique se você adicionou chaves ao armazenamento de chaves.
  2. Vá até Painel de Controle > Pasta Compartilhada > Ação > Gerenciador de chaves.
  3. Selecione a pasta compartilhada criptografada que você deseja descriptografar.
  4. Clique em Montar.

Para descriptografar pastas compartilhadas criptografadas na inicialização automaticamente:

  1. Verifique se você adicionou chaves ao armazenamento de chaves.
  2. Vá até Painel de Controle > Pasta Compartilhada > Ação > Gerenciador de chaves.
  3. Marque Montar na inicialização para as pastas compartilhadas criptografadas que você deseja descriptografar na inicialização.
  4. Clique em OK para salvar as configurações.

Para ejetar um armazenamento de chaves após a inicialização:

  1. Vá até Painel de Controle > Pasta Compartilhada > Ação > Gerenciador de chaves.
  2. Clique em Configurar.
  3. Marque Ejetar dispositivo após inicialização.
  4. Clique em OK para salvar as configurações.

Para migrar chaves de criptografia existentes para um armazenamento de chaves para as pastas compartilhadas montadas automaticamente:

Migrar chaves durante a inicialização do armazenamento de chaves:

  1. Ao inicializar um armazenamento de chaves, marque Migrar chaves de criptografia para este armazenamento de chaves para todas as pastas compartilhadas existentes montadas automaticamente (recomendado).

Migrar chaves manualmente:

  1. Vá até Painel de Controle > Pasta Compartilhada > Ação > Gerenciador de chaves.
  2. Clique em Configurar.
  3. Clique em Migrar agora.
  4. Em caso de chaves duplicadas, você pode escolher Sobrescrever chaves duplicadas.
  5. Clique em OK para salvar as configurações.
