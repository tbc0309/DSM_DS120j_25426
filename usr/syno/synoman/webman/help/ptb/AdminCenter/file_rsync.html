rsync

Você pode permitir outros usuários façam backup de dados através do rsync de um Synology NAS ou clientes compatíveis com rsync com seu Synology NAS.

Habilitação do serviço Time Backup rsync

Para habilitar o serviço rsync:

Vá para Painel de controle > Serviços de arquivos > rsynce selecione Habilitar serviços rsync.

Para atribuir privilégios de backup rsync:

Atribua os privilégios de backup rsync adequados aos usuários para que esses usuários possam fazer backup dos dados de um Synology NAS ou um rsync compatível com o servidor com o Synology NAS de destino.

  1. Vá até Painel de controle > ﻿Privilégios.
  2. Selecione rsync (Sincronização da pasta compartilhada, Backup LUN em modelos suportados) e clique em Editar para atribuir privilégios.
  3. Atribuir privilégios Permitir para os usuários desejados (usuários do DSM para rsync criptografado ou contas rsync para rsync não criptografado) ou especifique ainda os endereços IP permitidos para usuários específicos.

Nota:

Para realizar o backup do rsync de um Synology NAS que executa uma versão do DSM anterior à 3.0 ou um cliente que não é um Synology NAS e manter as informações de proprietário e grupo dos dados de origem, você precisará adicionar as contas do rsync ao grupo administrators e fazer backup dos dados na pasta compartilhada NetBackup no modo de daemon.

Habilitação das contas rsync

Se você realizar o backup rsync de um cliente não Synology NAS, é necessário criar contas rsync no Synology NAS de destino para que os usuários rsync possam fazer o backup dos dados em seu Synology NAS.

Para habilitar contas rsync:

  1. Vá para Painel de controle > Serviços de arquivos > rsync e selecione Habilitar conta rsync.
  2. Clique em Editar conta rsync para definir os usuários e as senhas correspondentes para autenticação.
  3. Clique em Aplicar.

Nota:

  • Se o serviço rsync for criptografado (isto é, se tiver uma porta de criptografia SSH especificada), o sistema aplicará diretamente os nomes de usuário e senhas do usuário do DSM como suas contas rsync.
  • Se o serviço rsync for criptografado, você precisará criar contas rsync correspondentes para eles.

Uso da configuração rsync personalizada

É possível personalizar a configuração do rsync e configurar privilégios do usuário e o destino do backup em seu Synology NAS. Outras pessoas podem então realizar o backup rsync a partir de seus dispositivos Synology NAS ou clientes compatíveis com rsync para pastas compartilhadas de destino (ou “módulos de backup”) em seu Synology NAS.

Para usar configuração rsync personalizada:

  1. Faça login no seu dispositivo Synology NAS através de Telnet ou SSH.
  2. Especifique o módulo de backup e os usuários autorizados no arquivo de configuração do rsync “/etc/rsyncd.conf”. Para obter informações detalhadas, consulte a página do manual de rsyncd.conf no seu cliente rsync ou aqui.

Nota:

Se o módulo de backup e a pasta compartilhada compartilharem o mesmo nome, a configuração do módulo de backup será usada.

Ativar o limite de velocidade

Você pode controlar a largura de banda máxima utilizável ou o backup rsync para usuários e grupos.

Para ativar o limite de velocidade:

  1. Vá até Painel de controle > Serviços de arquivo > rsync. Selecione uma das opções a seguir:
    • Não há limite de velocidade
    • Habilitar limite de velocidade
    • Habilitar o limite de velocidade de acordo com uma programação (Clique em Definir programação para realizar outras modificações.)

    Nota:

    Esta é uma configuração para todo o servidor e anulará as configurações de usuários e grupos individuais. Por exemplo, escolher Nenhum limite de velocidade desabilita as configurações de limite de velocidade de todos os usuários e grupos.

  2. Se você habilitou o limite de velocidade, clique em Configurações do limite de velocidade e clique no usuário/grupo desejado para modificar as configurações:
    • Nenhum: Nenhum limite de velocidade será aplicado quando os membros do grupo transferirem arquivos com o serviço.
    • Aplicar configurações de grupo: O usuários compartilhará as mesmas configurações de limite de velocidade com o grupo afiliado.
    • Constante: Um limite de velocidade fixo e constante será aplicado ao usuário ou membros do grupo. Os limites de velocidade de upload e download podem ser especificados nos campos à esquerda.
    • Personalizado: Dois limites de velocidade diferentes podem ser especificados e aplicados ao usuário ou aos membros de grupo, de acordo com uma programação. Clique em Personalizar para modificar as configurações de limite de velocidade e definir uma programação.
  3. Clique em Aplicar.

Nota:

A seção Resultado (UL/DL) exibe os limites de carregamento e download atuais do usuário de acordo com suas configurações individuais e de grupo. Por exemplo, se o usuário "Achilles" tiver um limite de download individual de 100 KB/s, mas pertence a um grupo com um limite de 500 KB/s, a seção Resultado (UL/DL) mostrará "0/500".
