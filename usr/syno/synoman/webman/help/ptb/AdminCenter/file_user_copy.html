Copiar um usuário

Você pode copiar as configurações de um usuário existente para um novo usuário. Isso permite criar rapidamente um novo usuário sem precisar ajustar as mesmas configurações novamente.

  1. Selecione o usuário que você deseja copiar.
  2. Clique em Criar > Criar usuário. A janela do Assistente de criação de usuário será iniciado.
  3. As configurações do usuário selecionado serão aplicadas ao novo usuário por padrão (exceto informações pessoais, como nome de usuário e e-mail).
