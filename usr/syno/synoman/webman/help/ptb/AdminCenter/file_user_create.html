Criar usuário

Você pode criar e gerenciar várias contas de usuário em seu Synology NAS.

Para criar um usuário:

  1. Vá até Painel de controle > ﻿Usuário.
  2. Clique no botão Criar e depois em Criar usuário. Essa ação abre o Assistente de criação de usuário.
  3. Na página Informações do usuário, especifique as seguintes opções:
    • Nome: Digite um nome para o usuário. Essa seção é obrigatória.
    • Descrição: Digite uma descrição breve para o usuário.
    • E-mail: Digite um endereço de e-mail para o usuário. As notificações do sistema, como mensagens de redefinição da senha, são enviadas ao endereço especificado aqui.
    • Enviar um e-mail de notificação para o usuário recém criado: Se essa opção estiver ativada, o sistema irá enviar uma mensagem para o usuário avisando sobre a nova conta. Essa opção exige a ativação das notificações por e-mail do sistema em Menu principal > Painel de controle > Notificação > E-mail. Por padrão a senha do usuário não é exibida na mensagem de notificação. Entretanto, se você desejar exibir a senha na mensagem de notificação, marque a caixa Exibir senha de usuário no e-mail de notificação.
    • Não permitir que o usuário altere a senha da conta: Se essa opção estiver ativada, o usuário não poderá alterar sua senha usada para o login no DSM.
  4. Na página Grupo > Criar > Assistente de criação de grupo, especifique o grupo ao qual o novo usuário deve pertencer (por exemplo, usuários, administrators, etc.).
  5. Na página Atribuir permissões de pastas compartilhadas, escolha as pastas compartilhadas que o novo usuário pode acessar, modificando seus privilégios de acesso. A prioridade dos privilégios é a seguinte: Nenhum acesso > Leitura/Gravação > Somente leitura.
    • Visualizar: Esta coluna exibe os privilégios reais de acesso da pasta compartilhada do usuário de acordo com os grupos e configurações atuais aos quais o usuário pertence.
    • Permissões de grupo: Esta coluna exibe os privilégios de acesso da pasta compartilhada atribuídos de acordo com os grupos de usuários aos quais o usuário pertence.
  6. Na página Configuração da cota do usuário, você pode controlar o espaço de armazenamento que pode ser usado pelo novo usuário. Para restringir a quantidade de espaço que o usuário pode usar, marque a caixa em Ativar cota para cada volume/pasta compartilhada. Em seguida, insira a quantidade máxima de espaço que pode ser usado, no campo Cota (por exemplo, 10 GB).
  7. Na página Atribuir permissões de aplicativos, você pode controlar quantos serviços o usuário pode acessar.
  8. Na página Configuração do Limite de Velocidade de Grupo, você pode habilitar um limite de velocidade para diferentes serviços (por exemplo: File Station, FTP, etc.), para restringir a quantidade de largura de banda consumida pelo usuário ao transferir arquivos. Para cada serviço, você pode selecionar uma das seguintes opções:
    • Nenhum: As configurações do limite de velocidade serão iguais às do grupo ao qual o usuário pertence.
    • Constante: Um limite de velocidade fixo e constante será aplicado. Você pode especificar os limites de velocidade de upload e download nos campos à esquerda.
    • Variável: Dois limites de velocidade diferentes podem ser especificados e aplicados ao usuário, de acordo com uma programação. Clique no botão Personalizar para modificar as configurações de limite de velocidade e a programação.
  9. Depois de confirmar as configurações, clique em Aplicar para encerrar.

Limitação do nome de usuário:

  • O nome de usuário diferencia maiúsculas e minúsculas e deve ter de 1 a 64 caracteres Unicode, mas não pode conter os símbolos a seguir:
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • O primeiro caractere não pode ser um sinal de menos nem um espaço e o último caractere não pode ser um espaço.
  • Os seguintes nomes de usuário são reservados para uso do sistema: mailer-daemon, postmaster.
  • A descrição do usuário pode incluir até 64 caracteres Unicode visíveis.

Limitação da senha:

  • A senha diferencia maiúsculas de minúsculas e é limitada até 127 caracteres visíveis, inclusive letras, números, sinais e espaços.

Limitação de cota:

  • A cota de uso de armazenamento máxima que pode ser inserida para cada usuário é de aproximadamente 4 TB (4095 GB).
