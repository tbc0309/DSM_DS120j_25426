Descoberta de serviço

A descoberta de serviço permite que os usuários encontrem este Synology NAS na rede através do protocolo SSDP e dos serviços Bonjour.

Para possibilitar a descoberta do seu Synology NAS através do protocolo SSDP:

O serviço SSDP permite aos usuários encontrar o Synology NAS através da descoberta de rede do Windows.

  1. Vá para Painel de controle > Serviços de arquivos > Descoberta de serviço > SSDP.
  2. Selecione Ativar a descoberta de redes do Windows.
  3. Clique em Aplicar para salvar as configurações.

Para possibilitar a descoberta do seu Synology NAS pela através do serviço Bonjour:

O serviço Bonjour permite aos usuários encontrar o Synology NAS e executar tarefas de impressão e backup no computador Mac.

  1. Vá para Painel de controle > Serviços de arquivos > Descoberta de serviço > Bonjour.
  2. Selecione Habilitar descoberta do serviço Bonjour.
  3. Você pode selecionar opções relacionadas ao Bonjour para atender às suas necessidades:
    • Ativar o Bonjour Printer Broadcast: Selecione para permitir que computadores Mac encontrem a impressora USB conectada ao seu Synology NAS para impressão de documentos.
    • Ativar transmissão do Bonjour Time Machine via SMB: Selecione para permitir que o Time Machine do Mac encontre seu Synology NAS e faça backup dos arquivos nas pastas compartilhadas do DSM via SMB.
    • Ativar transmissão do Bonjour Time Machine via AFP: Selecione para permitir que o Time Machine do Mac encontre seu Synology NAS e faça backup dos arquivos nas pastas compartilhadas do DSM via AFP.
  4. Se você tiver selecionado qualquer opção relacionada ao Time Machine, clique em Definir pastas do Time Machine para especificar pastas de destino para backups do Time Machine.
  5. Clique em Aplicar para salvar as configurações.

Nota:

  • Se você desativar o serviço SMB/AFP (em Painel de controle > Serviços de arquivos > SMB/AFP/NFS), a opção do Bonjour associada será desabilitada automaticamente.
  • Quando a opção Ativar transmissão do Bonjour Time Machine via SMB for selecionada, as opções a seguir serão habilitadas (em Painel de controle > Serviços de arquivos > SMB/AFP/NFS > Configurações avançadas):
    • Habilitar SMB 3 (para Protocolo máximo para serviço de arquivos do Windows)
    • Opportunistic Locking
    • Habilitar concessão SMB2
    • Habilitar identificadores duráveis SMB (o bloqueio de arquivos entre protocolos será desabilitado)
