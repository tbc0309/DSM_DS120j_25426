Edição de pasta criptografada

Após a pasta ter sido criptografada, você pode usar o menu suspenso Criptografia para editar a pasta.

Os itens mostrados no menu dependem do status da pasta compartilhada selecionada:

  • Se você selecionar uma pasta compartilhada não criptografada, o menu suspenso Criptografia será desativado.
  • Se você selecionar uma pasta compartilhada criptografada que estiver montada, pode escolher Desmontar no menu suspenso Criptografia para desmontar a pasta, ou Exportar chave para salvar a chave de criptografia.
  • Se você selecionar uma pasta compartilhada criptografada que estiver desmontada, pode escolher Montar no menu suspenso Criptografia e em seguida digitar ou importar a chave de criptografia para montar a pasta para acesso.
  • Se você selecionar várias pastas compartilhadas criptografadas desmontadas e suas chaves de criptografia forem gerenciadas pelo Gerenciador de chaves, pode escolher Montar no menu suspenso Criptografia e, em seguida, digitar a senha do Gerenciador de chaves para montar essas pastas para acesso.
    Nota:
  1. É muito importante lembrar a chave de criptografia. Recomendamos enfaticamente que você exporte e salve a chave de criptografia. Se você esquecer ou perder a chave, não há modo de recuperar os dados criptografados.
  2. Quando uma pasta compartilhada criptografada é montada, não é possível renomeá-la ou mudar o local do seu volume. Você pode fazer alterações apenas depois de desmontá-la.
