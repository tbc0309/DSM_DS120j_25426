進階

您可以在進階頁籤中啟動檔案快速複製功能，以便在複製檔案時節省儲存空間。此外，您也可以調整設定來讓使用者利用 Bonjour 及 SSDP (Simple Service Discovery Protocol) 服務找到網路上的 Synology NAS。

若要在 Synology NAS 上進行檔案快速複製：

檔案快速複製功能讓已進行複製的檔案僅在發生任何變更時才會複製實體的資料區塊，如此即可幫助節省儲存空間。

  1. 在檔案快速複製區塊下，勾選啟動檔案快速複製核取方塊。
  2. 按一下套用來儲存設定。

若要讓 Synology NAS 可透過 Bonjour 服務找到：

Bonjour 服務可讓使用者找到 Synology NAS，並且可從 Mac 電腦執行列印及備份任務。

  1. 在 Bonjour 區塊中，勾選啟動 Bonjour 服務探索核取方塊。
  2. 即可選取 Bonjour 相關選項來符合所需：
    • 啟動 Bonjour 印表機共享：選取此選項後，即可讓 Mac 電腦找到連接至 Synology NAS 的 USB 印表機來列印文件。
    • 啟動透過 SMB 進行 Bonjour Time Machine 廣播：選取此選項後，即可讓 Mac Time Machine 找到 Synology NAS，並透過 SMB 來備份檔案至 DSM 共用資料夾。
    • 啟動透過 AFP 進行 Bonjour Time Machine 廣播：選取此選項後，即可讓 Mac Time Machine 找到 Synology NAS，並透過 AFP 來備份檔案至 DSM 共用資料夾。
  3. 如果已選取與 Time Machine 相關的任何選項，按一下設定 Time Machine 資料夾，即可指定 Time Machine 備份的目的地資料夾。
  4. 按一下 套用 來儲存設定。

注意：

  • 如果停用 SMB/AFP 服務 (位於控制台 > 檔案服務 > SMB/AFP/NFS)，相關的 Bonjour 選項便會立即停用。
  • 選取啟動透過 SMB 進行 Bonjour Time Machine 廣播後，下列選項將一併啟動 (位於控制台 > 檔案服務 > SMB/AFP/NFS > 進階設定)：
    • 啟動 SMB 3 (Windows 檔案服務優先使用的通訊協定的選項)
    • Opportunistic Locking
    • 啟動 SMB2 租用
    • 啟動 SMB 耐久控制代碼 (同時停用跨協定的檔案鎖定)

若要讓 Synology NAS 可透過 SSDP 找到：

SSDP 服務可讓使用者利用 Windows 網路探索來找到 Synology NAS。

  1. 在 SSDP 區塊中，勾選啟動 Windows 網路探索，允許透過網頁瀏覽器進行存取核取方塊。
  2. 按一下套用來儲存設定。

若要讓 Synology NAS 可透過 WS-Discovery 找到：

WS-Discovery 服務可讓使用者利用 Windows 網路探索來找到 Synology NAS 上的 SMB 服務。

  1. 在 WS-Discovery 區塊中，勾選啟動 Windows 網路探索，允許透過 SMB 進行檔案存取核取方塊。
  2. 按一下套用來儲存設定。
