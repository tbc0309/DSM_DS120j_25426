安全性

您可於控制台 > 安全性 > 安全性為 DSM 瀏覽工作階段修改下列安全性設定：

  • 登入逾時 (分鐘)：如果閒置時間到達指定的時間，使用者便會自動從 DSM 登出。 輸入 1 到 65535 之間的數值。
  • 忽略 IP 檢查來加強瀏覽器的相容性：如果您透過 HTTP 代理伺服器存取 Synology NAS 時遭遇隨機登出的問題，便可啟動此選項來忽略 IP 檢查。
  • 強化抵禦跨網站要求偽造攻擊的能力：此選項可加強系統抵抗網頁惡意程式碼攻擊的能力。此選項會於您下次登入 DSM 時生效。
  • 啟動 HTTP 內容安全政策 (CSP) 標頭來提昇安全性：僅接受來自可信賴來源的資料並限制內嵌指令碼執行，藉此提昇系統安全性，加強抵禦網頁惡意程式碼攻擊的能力。
  • 禁止 DSM 被 iFrame 嵌入：您可以啟動此選項來限制其他網站使用 iFrame 將 DSM 嵌入其網頁，以避免 DSM 遭受外來網站的惡意攻擊。按一下允許網站清單可以指定允許使用 iFrame 將 DSM 嵌入的網站。
  • 系統重新啟動時，清除所有使用者登入連線：若使用者於系統準備就緒時保持登入狀態並在系統上執行操作，此選項可防止意外的系統錯誤。啟動此選項後，系統將重新啟動，所有的使用者皆必須重新登入。
  • 當目前的 IP 變更時，顯示 DSM 桌面通知：當登入連線中的使用者發生 IP 位置異動時，發送桌面通知給該使用者。
  • 信任的代理伺服器：DSM 會使用從信任的代理伺服器傳來的遠端 IP 資訊。您可以更改受信任的代理伺服器。
