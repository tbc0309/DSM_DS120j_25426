電子郵件

啟動電子郵件通知後，當指定事件發生時 (例如系統錯誤、網路連線中斷、風扇行為改變、電源中斷或備份任務失敗)，Synology NAS 便會發送訊息至您的電子郵件信箱。

注意：

  • 風扇相關事件不適用於無風扇機種，例如 EDS14。請前往 Synology 網站以了解更多資訊。

若要啟動電子郵件通知：

  1. 請前往控制台 > 通知設定 > 電子郵件。
  2. 勾選啟動電子郵件通知。
  3. 在收件人電子郵件地址欄位中輸入一個或多個電子郵件位址。系統將會寄送通知訊息至您所輸入的位址。您最多只能輸入兩個電子郵件信箱，您可以使用分號 (;) 來分隔。
  4. 如有需要，請在標題前置文字欄位中加入文字。此前置文字將會加在 Synology NAS 所發送的郵件標題之前，以便您辨識及過濾訊息。
  5. 如果您想要使用第三方電子郵件伺服器 (例如 Gmail、Outlook) 來轉送訊息，請參閱下方的「第三方電子郵件服務」說明。如果您想要使用自訂的伺服器，請參閱「自訂的 SMTP 伺服器」說明。

若要使用第三方電子郵件服務：

  1. 在服務提供者選單中，請選擇一個第三方電子郵件服務，例如 Gmail 或是 Outlook。
  2. 輸入所選電子郵件服務的帳號及密碼。
  3. 按一下套用來儲存設定。
  4. 您可以按一下寄送測試電子郵件來檢查您的設定是否正確。

若要使用自訂的 SMTP 伺服器：

  1. 在服務提供者選單中，請選擇自訂 SMTP 伺服器。
  2. 在 SMTP 伺服器欄位中輸入 SMTP 伺服器名稱或 IP 位址。
  3. 在 SMTP 連結埠欄位中輸入 SMTP 連接埠號碼。(如果設定連接埠轉送規則時已指定另一個 SMTP 埠，請輸入該連接埠號碼。)
  4. 如果您的 SMTP 伺服器需要驗證，請勾選需要驗證，並輸入您電子郵件伺服器的使用者帳號及密碼。
  5. 如果您的 SMTP 伺服器需要 SSL/TLS 安全連線，請勾選需要安全連線 (SSL/TLS)。
  6. 如果您要自訂 Synology NAS 發送的電子郵件上所顯示的寄件人名稱，請在寄件人名稱欄位中輸入。如果您要自訂寄件人的電子郵件位址，請在寄件人信箱欄位中輸入。若此欄位留空，系統將會使用收件人電子郵件地址欄位中的第一個電子郵件位址。
  7. 按一下套用來儲存設定。
  8. 您可以按一下寄送測試電子郵件來檢查您的設定是否正確。

注意：

  • 若您想要寄送歡迎信給每位新增的使用者，請勾選寄送歡迎訊息給新使用者，使用者建立帳號後便會收到歡迎信。
  • 您可以在系統通知語言中設定電子郵件通知的語言。若要變更電子郵件通知的語言，請前往主選單 > 控制台 > 區域選項，按一下語言頁籤，從系統通知語言區塊中的下拉式選單來選擇您偏好的語言。
  • 如果您使用 Gmail 的 SMTP 伺服器且已啟動兩步驟驗證，請記得使用 Google 提供給 SMTP 驗證的「應用程式專用密碼」。

SMTP 伺服器網域名稱：

SMTP 伺服器可以是 IP 位址或網域名稱。如果是網域功能變數名稱，請確認您已在主選單 > 控制台 > 網路中輸入有效的 DNS 伺服器 IP。

注意：

由於 Google Gmail API 政策的變更，使用者必須從服務提供者的下拉式選單中，選擇自訂 SMTP 伺服器才可在下列三項服務中寄送電子郵件。若要了解更多相關設定的資訊，請參閱此篇應用教學。

  • 使用 ssmtp 寄送電子郵件，例如：在任務排程表中執行使用者定義指令碼。
  • 使用 Web Station 架設網站並利用 PHP 寄送電子郵件，亦包含從第三方網站套件 (例如：WordPress 及 MantisBT) 寄送電子郵件。
  • 使用電子郵件寄送 Photo Station 中的相片或文章留言通知。
