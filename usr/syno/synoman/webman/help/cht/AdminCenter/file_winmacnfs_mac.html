AFP

在控制台 > 檔案服務 > SMB/AFP/NFS 啟動 AFP 服務，便可讓 Synology NAS 成為 Mac 檔案伺服器 (AFP 伺服器)。

若要啟動 Mac 檔案服務：

  1. 勾選啟動 AFP 服務便可讓 Synology NAS 成為 Mac 檔案伺服器 (AFP 伺服器)。
  2. 按一下套用。

啟動傳輸日誌記錄

勾選此選項後便可紀錄透過 AFP (Apple Filing Protocol) 進行的檔案存取作業，包含建立、刪除、讀取、寫入、移動及重新命名檔案或資料夾。

進階設定

套用 UNIX 預設權限：

啟動此選項後，上傳或建立檔案及資料夾時將套用 UNIX 預設權限。所套用的權限和透過 UNIX 指令 umask 設定的權限完全相同。此選項啟動時，檔案的 UNIX 權限為 644，資料夾為 755。此選項停用時，檔案的 UNIX 權限為 666，資料夾為 777。預設的 umask 值為 022。

注意：

啟動此選項可能會導致不同通訊協定間出現權限不一致的情況。若要避免不一致的情況，建議您停用此選項。

斷線後立即釋放資源：

啟動此選項後，在 AFP 伺服器連線意外中斷時，將立即釋放資源。若您在使用 Time Machine 時收到「儲存使用中」的錯誤訊息，請啟動此選項。

若要透過 Finder 來搜尋檔案內容：

當資料夾建立完整索引並掛載至 Mac 電腦上後，即可使用 Finder 來搜尋內部檔案的內容。

  1. 請前往 Universal Search > 使用偏好 > 檔案索引 > 索引資料夾清單 > 新增來索引特定資料夾，並可依檔案類型索引內部檔案。
  2. 按一下確定來進行索引。
  3. 掛載已建立索引的資料夾至 Mac 電腦上。
  4. 在 Mac 電腦上，按一下 Finder 左側面板上的掛載資料夾，然後使用 Finder 來搜尋內部檔案。

注意：

  • Network video recorder (NVR) 機種不支援透過 Mac Finder 搜尋檔案內容。
  • 使用 Finder 來在掛載資料夾內進行搜尋。Mac 桌面上的 Spotlight 僅用來搜尋本機檔案。
  • 在 Mac OS X 10.9 及更新版本中，Finder 支援依標籤名稱及種類來搜尋掛載資料夾內部檔案。
  • 透過 AFP 存取共用資料夾時，若共用資料夾數量超過 255，僅會顯示 255 個共用資料夾 (依字母排序)。
