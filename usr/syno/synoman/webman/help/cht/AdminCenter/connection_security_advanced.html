進階

您可於控制台 > 安全性 > 進階為 DSM 瀏覽工作階段修改下列進階設定。

HTTP 壓縮管理

您可以替 Synology NAS 開啟 HTTP 壓縮，降低網路頻寬的使用量，進而加速網頁讀取速度。

若要啟動 / 停用 HTTP 壓縮管理：

  • 前往控制台 > 安全性 > 進階，勾選 / 反勾選 HTTP 壓縮管理。

TLS / SSL 安全性等級

您可依照需求自行選擇 HTTPS 加密連線的安全性等級。共有下列三個等級：

  • 現代相容性 - 僅相容於較新版的瀏覽器與行動裝置。
  • 中級相容性 - 介於現代相容性與回溯相容性之間，為系統預設值。
  • 回溯相容性 - 此選項不安全，但支援絕大多數的瀏覽器與行動裝置。

修改 TLS / SSL 安全性等級至所有服務：

您可於主畫面中選擇安全性等級，選擇後將會套用至所有使用預設配置的服務。

  1. 選擇合適的安全性等級。
  2. 按一下確定。

修改 TLS / SSL 安全性等級至特定服務：

您可根據服務所需之安全性，調整其安全性等級。

  1. 按一下客製化設定以顯示所有服務及其對應的安全性等級。
  2. 按一下欲修改之服務目前使用的安全性等級。
  3. 從下拉式選單中選擇合適的安全性等級。
  4. 按一下確定。

注意：

  • 不在服務清單中的連線將套用系統預設的安全性等級。
