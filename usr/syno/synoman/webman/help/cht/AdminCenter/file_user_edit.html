編輯使用者

選取您要編輯的使用者帳號後（可在搜尋欄位中輸入關鍵字尋找），連按兩下，或按編輯按鈕修改帳號資料。

    注意：
  1. 如果您停用某個帳號，該帳號將停止運作。
  2. 空間配額設定值須為整數，其範圍大於或等於 0 ，而且須小於硬碟／儲存空間的容量大小。空間配額設定值若為 0，則表示有效配額將由群組配額決定。
  3. 如果伺服器有一個以上的儲存空間 / 共用資料夾，您可以為使用者在每個儲存空間 / 共用資料夾設定空間配額。
  4. 系統管理者必須在「通知設定」頁面設定正確的電子郵件資訊，系統才會在新增或修改使用者個人資料後，發信通知該使用者。使用者在收到 email 通知帳號密碼後，可自行登入修改個人資料。
  5. 設定使用者速度限制時，若使用者已套用其群組的速度限制，您會在結果下方看到數字。
  6. 使用者的速度限制不會包含在其所屬群組的速度限制中。

注意：

  • EDS14 與 DDSM 皆不支援空間配額的編輯。
  • 唯有具備下列套件架構的 VDSM 及機種可透過共用資料夾來編輯群組的使用空間配額：Apollo Lake、Avoton、Braswell、Broadwell、Bromolow、Cedarview 及 Grantley。若要了解您的機種支援哪個套件架構，請參考此文章。
