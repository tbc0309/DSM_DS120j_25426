檔案索引

為共用資料夾或內部的子資料夾建立索引後，搜尋檔案時即可快速找到內部檔案及檔案內容。

注意：

掛載至 Synology NAS 上的遠端資料夾、雲端資料夾或 .iso 資料夾皆無法建立索引。

若要索引資料夾或檔案內容：

索引資料夾時，File Station 將會依照名稱為該資料夾及子資料夾 / 檔案建立索引，讓您可透過檔案名稱搜尋需要的項目。如果再依檔案類型進行索引，File Station 則會依照名稱及內容來索引對應的檔案；如此一來，即可透過檔案名稱及檔案內容搜尋需要的項目。

  1. 請前往控制台 > 索引服務 > 檔案索引，然後按一下索引資料夾清單。
  2. 在彈出視窗中，按一下新增 > 選擇，便可顯示能夠建立索引的資料夾。
  3. 選擇想要索引的資料夾，按一下選擇。若要新增資料夾並建立索引，按一下建立資料夾並繼續。
  4. 按一下想要索引的資料夾，然後按一下選擇。
  5. 若要為特定類型的檔案內容建立索引，請前往檔案類型，選擇照片、音樂、影片或文件。如果某些檔案類型 (例如音樂) 未被選取，僅可依照檔案名稱搜尋該類型的檔案 (例如音樂檔案)。
  6. 按確定以完成索引。

注意：

  • 僅可選取至多 1000 個資料夾進行索引；選取資料夾中的子資料夾則不計入數量內。
  • 如果在 File Station 選擇啟動檔案內容索引，便可在索引資料夾內搜尋檔案內容。
  • 索引資料夾所在的儲存空間必須有 10 MB 以上的可用空間。否則將會暫停索引。若要重新接續索引，請擴增儲存空間並按一下重建索引。
  • 如果同時為資料夾及其子資料夾建立索引，系統僅會針對子資料夾進行索引，以節省系統資源。
  • 資料夾建立完整索引，再透過 SMB 或 AFP 通訊協定掛載至 Mac 電腦上後，即可利用 Finder 來搜尋檔案內容。

若要刪除索引資料夾：

  1. 前往控制台 > 索引服務 > 檔案索引 > 索引資料夾清單，選擇想要刪除的索引資料夾。
  2. 按一下刪除，便可立即停止索引該資料夾。

為檔案重建索引

在一般情況下，您不需要為檔案重建索引。然而，如果資料夾內檔案符合搜尋條件，但未出現在檔案搜尋結果裡，可能須重新索引檔案來重建索引結構。

若要為共用資料夾中的檔案重建索引：

  1. 前往控制台 > 索引服務 > 檔案索引 > 索引資料夾清單。
  2. 選擇清單中的共用資料夾來重建索引。
  3. 按一下重建索引。

注意：

File Station 不支援在遠端資料夾或外接儲存裝置 (例如 USB 硬碟 / SD 卡) 上建立檔案索引。

透過桌面管理檔案索引

檔案索引開始後，可在工作列上找到索引進度 (浮動的放大鏡圖示)。由於檔案索引需要較多 CPU 資源，此項目能讓您節約使用 CPU 資源。

  • 若要暫停 / 重新繼續檔案索引，按一下播放按鍵。
  • 若要延後檔案索引至適當時間，請從下拉式選單中選取適合的時間。
  • 若要檢視正在索引的資料夾，按一下檢視索引資料夾清單。

注意：

  • 如果暫停索引動作，僅有按一下桌面工作列的播放鍵才能重新繼續。
