工作管理員*

您可於工作管理員頁籤中檢視各個服務和程序的 CPU 及記憶體使用率。

服務

按一下服務頁籤來監控各服務的各項數值，包含 CPU 使用率、CPU Time、記憶體使用率以及硬碟的讀取 / 寫入狀態。

程序

按一下程序頁籤來檢視各個程序的 CPU 及記憶體使用率，系統會根據 CPU 負載由高至低排列。

程序狀態包含執行中、睡眠中及已停止。程序狀態對應至 Linux 則為 running、stopped/tracing、sleeping。

睡眠中可後接英文字母以表示詳細資訊，如「D」代表 disk sleep，「Z」代表 zombie，「X」代表 dead。

共用記憶體欄位顯示該程序正在使用，且可與其他程序共用的實體記憶體數量。專用記憶體欄位顯示該程序正在使用，且無法供其他程序使用的實體記憶體數量。

____
*僅適用於特定機型。
