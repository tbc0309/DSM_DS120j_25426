設定

iSNS

iSNS (Internet Storage Name Service) 提供了中央管理的機制，您可以向 iSNS 伺服器註冊所有的 iSCSI Target。

若要向 iSNS 伺服器註冊 iSCSI Target：

  1. 按一下 iSNS並勾選啟用 iSNS。
  2. 輸入 iSNS 伺服器的 IP 位址。按一下確定來套用設定。

I/O 佇列深度

預設值為 16，您可以修改所有 Target 的 I/O 佇列深度。

低容量寫入

若儲存空間可用容量低於 1 GB，儲存空間上的 LUN 將受到保護以避免檔案系統毀損，使用者將無法寫入資料至受保護的 LUN。低容量寫入讓使用者在儲存空間容量用盡時仍能持續寫入資料至 LUN；然而，建議您在解決空間不足問題後停用此選項。

注意：

  • 即使啟用此功能，仍請避免持續將資料寫入 LUN。

空間回收模式

下列為空間回收的三種模式，藉由調整速度，您可以控制空間回收對於 LUN 效能的影響。

  • I/O 優先：I/O 優先模式能在系統回收舊版 (Legacy) 進階 LUN 所釋放出的空間時，將其對於效能的影響降至最低。回收速度在 I/O 進行時會較低，當接受到連續的空間回收指令時，回收過程將被延後，直到不再繼續收到任何空間回收指令為止。
  • 平衡：此為預設模式，會在 I/O 速度以及回收速度間取得平衡。
  • 回收優先：在此模式下，I/O 對空間回收速度的影響較小，系統將在不影響 I/O 的情況下全速進行空間回收。
