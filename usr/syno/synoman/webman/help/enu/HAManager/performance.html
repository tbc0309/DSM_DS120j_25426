Performance

On the Performance page, you can monitor resource usage on both active and passive servers side by side. The two servers can be easily distinguished by the server name and the respective roles (active or passive). When the passive server is powered off, resource usage on the passive server will not be displayed.

CPU

You can check the overall CPU usage by the system, users, and I/O queue. After you open the page, the CPU load may be higher because the system is loading the page while gathering system usage information at the same time.

Memory

Overall usage of the physical memory will be displayed. If memory usage remains high, it may be because the system stores frequently-used data in the cache to access data quickly, without having to access the disk. When overall memory is insufficient, the system will release cache memory.

Network

The network traffic graph displays data transfer rate in KB per second, with different colors:

  • Blue represents upload rate.
  • Green represents download rate.
  • When upload and download rates overlap, the overlapping area will be dark green.

Disk

The graph displays the overall disk usage.
