Copy a User

You can copy the settings of an existing user to a new user. This will allow you to quickly create a new user without having to go configure the same settings again.

  1. Select the user that you want to copy.
  2. Click Create > Copy user. The window of the User Creation Wizard will be launched.
  3. The settings of the selected user will be applied to the new user by default (excluding personal information such as username and email).
