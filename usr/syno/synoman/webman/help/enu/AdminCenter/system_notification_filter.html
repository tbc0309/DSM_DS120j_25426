Advanced

You can select the types of events that will trigger the system to send notification messages as well as the medium (e-mail, SMS, and mobile devices) through which you will receive messages.

Each of the events has its own default notification message, but you are also provided with the option to customize your own notification content by editing the notification message. The default message can be restored if you decide to discard your customized settings later.

To search for notification events:

  1. Type a keyword into the search field in the upper right corner.
  2. Events related to the entered keyword will be displayed.

To select notification mediums for notification events:

  1. Tick the checkboxes for each individual event to select the notification mediums you prefer.
  2. Click Save to apply the settings.

To edit notification messages:

  1. Select an event from the event list and then click Edit Message.
  2. In the window that appears, edit the subject and the body content to customize the notification message.
  3. You can also edit the following variables. The updated variables will apply to all notification messages:
    • %COMPANY_NAME%: The signature that appears at the end of a notification message. The default value is Synology DiskStation, Synology RackStation, Synology FlashStation or Synology Unified Controller, depending on your model.
    • %HTTP_URL%: A hyperlink to your Synology NAS. If not specified, the system will list all possible links in the notification message.
  4. Click Save to apply the changes.
  5. To restore the notification message to the default:
    1. Select an event from the event list and then click Edit Message.
    2. In the window that appears, click Restore defaults and then Yes on the dialog prompt for confirmation.

Note:

  • It is recommended that you reserve the string segments containing '%' in both the subject and the body content. They will be replaced with actual system information when a notification is sent.
