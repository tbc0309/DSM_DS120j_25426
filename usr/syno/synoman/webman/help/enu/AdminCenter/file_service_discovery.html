Advanced

In the Advanced tab, you can enable the File Fast Clone function to save storage space while making file clones. Moreover, you can configure the settings to allow users to find this Synology NAS on the network via the Bonjour and SSDP (Simple Service Discovery Protocol) services.

To perform file fast clone on your Synology NAS:

With the file fast clone function, physical data blocks will be copied only upon modification of the cloned files, and this will help to save your storage space.

  1. Under the File Fast Clone section, tick the Enable file fast clone checkbox.
  2. Click Apply to save the settings.

To make your Synology NAS discoverable via Bonjour service:

The Bonjour service allows users to find the Synology NAS, and perform printing and backup tasks from the Mac computer.

  1. Under the Bonjour section, tick the Enable Bonjour service discovery.
  2. You can select Bonjour-related options to suit your needs:
    • Enable Bonjour Printer Broadcast: Select to allow Mac computers to find the USB printer connected to your Synology NAS for document printing.
    • Enable Bonjour Time Machine broadcast via SMB: Select to allow Mac Time Machine to find your Synology NAS and back up files to DSM shared folders via SMB.
    • Enable Bonjour Time Machine broadcast via AFP: Select to allow Mac Time Machine to find your Synology NAS and back up files to DSM shared folders via AFP.
  3. If you have selected any option related to Time Machine, click Set Time Machine Folders to specify destination folders for Time Machine backups.
  4. Click Apply to save the settings.

Note:

  • If you disable the SMB/AFP service (at Control Panel > File Services > SMB/AFP/NFS), the associated Bonjour option will be automatically disabled.
  • When Enable Bonjour Time Machine broadcast via SMB is selected, the following options will be enabled (at Control Panel > File Services > SMB/AFP/NFS > Advanced Settings):
    • Enable SMB 3 (for Max protocol for Windows file service)
    • Opportunistic Locking
    • Enable SMB2 lease
    • Enable SMB durable handles (Cross-protocol file locking will be disabled)

To make your Synology NAS discoverable via SSDP:

The SSDP service allows users to find the Synology NAS via Windows network discovery.

  1. Under the SSDP section, tick the Enable Windows network discovery to allow access via web browsers checkbox.
  2. Click Apply to save the settings.

To make your Synology NAS discoverable via WS-Discovery:

The WS-Discovery service allows users to find SMB service on the Synology NAS via Windows network discovery.

  1. Under the WS-Discovery section, tick the Enable Windows network discovery to allow file access via SMB checkbox.
  2. Click Apply to save the settings.
