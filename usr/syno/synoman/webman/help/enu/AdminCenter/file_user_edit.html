Edit a User

Find the user account (by entering the user name in the Search field), double click on it or click Edit, and then you can edit its account data.

Note:
  1. If you disable an account, the account will stop functioning.
  2. Quota should be an integer greater than or equal to 0, and less than the disk/volume capacity. A quota value of 0 means Effective quota will be determined by Group Limit.
  3. If there is more than one volume/shared folder on the server, you can set a quota for the user on every volume/shared folder.
  4. Email information on the "Notification" page must be set correctly in order for the system to notify the user by email after the account information has been created or modified.
  5. When setting speed limit for a user, you will see numbers under Result if there is already a group speed limit applied to the user.
  6. The speed limit assigned to a user will not be included in the speed limit of the group that the user belongs to.

Note:

  • Editing quota is not supported by EDS14 or DDSM.
  • Only VDSM and models with the following package architectures support editing a group's usage quota through shared folder: Apollo Lake, Avoton, Braswell, Broadwell, Bromolow, Cedarview, and Grantley. To see which package architecture your model supports, please refer to this article.
