Media Indexing

Media Indexing can automatically scan multimedia files stored on your Synology NAS. These files will be compiled into the media library for indexing, and allow browsing and playback by DMA devices and Synology's multimedia applications (e.g. Media Server, Video Station, Audio Station, Photo Station, and iTunes Server).

Managing Indexed Folders

To index an existing folder or create an indexed folder:

  1. Go to Control Panel > Indexing Service > Media Indexing.
  2. Click Indexed Folder > Create > Select, and do either below:
    • To index an existing folder: Select the desired folder on the list, and click Select.
    • To create an indexed folder: Select the parent folder on the list. Click Create folder to specify the folder name, and click OK. Then select the created folder and click Select.
  3. Select the file types (i.e. Photo, Music, and Video) for indexing. Only files of the indexed types allow access from DMA devices and Synology's multimedia applications.
  4. Click OK to begin indexing.

Note:

  • Default shared folders (e.g. music) created for Synology's multimedia applications are automatically indexed, while other folders can only be manually indexed.
  • Remote folders mounted using File Station cannot be added as indexed folders.
  • External storage devices must be re-indexed every time your Synology NAS restarts.
  • You can add up to 100 indexed folders on your Synology NAS.

To edit an indexed folder:

  1. Go to Control Panel > Indexing Service > Media Indexing > Indexed Folder.
  2. In the pop-up window, select the desired folder.
  3. Click Edit, and make changes to the settings.
  4. Click OK and then click Save.

To remove folder(s) from the list of indexed folders:

Removed folders will be de-indexed. The contained multimedia files will not be accessible to DMA devices and Synology's multimedia applications.

  1. Go to Control Panel > Indexing Service > Media Indexing > Indexed Folder.
  2. In the pop-up window, select the unwanted folder. To select multiple unwanted folders, press and hold Shift or Ctrl during selection.
  3. Click Delete.
  4. Click Yes and then click Save.

Note:

  • Removed folders will not be deleted from your Synology NAS.

To re-index multimedia files:

If the index structure is lost, re-indexing the indexed folders helps reconstruct the structure.

  1. Go to Control Panel > Indexing Service > Media Indexing.
  2. Click Re-index to index again all the previously indexed folders.

Note:

  • Multimedia files added to indexed folders via the NFS or SCP protocol will not be scanned automatically. To index such files, click Re-index.

Converting Indexed Multimedia Files

To set thumbnail quality:

Thumbnails will be automatically generated for uploaded photos/videos for use with Media Server or Photo Station. You can set the thumbnail quality here.

  1. Go to Control Panel > Indexing Service > Media Indexing > Thumbnail Settings.
  2. Choose a quality from the drop-down menu.
  3. Click Apply.

Note:

  • Creating high-quality thumbnails will take more time.
  • To upload a large number of photos/videos to your Photo Station Library, we recommend using Photo Station Uploader (a Synology desktop utility available at Download Center) for rapid thumbnail generation.

To enable video conversion for mobile devices:

Videos in the photo shared folder (dedicated to Photo Station) can be played on your mobile device if they are well converted.

  1. Go to Control Panel > Indexing Service > Media Indexing > Video Settings.
  2. Select Enable video conversion for mobile devices.
  3. Click Apply, and you will be able to see videos on your mobile devices on-the-go.

Note:

  • Only the videos uploaded to the photo shared folder of Photo Station can be converted into the format supported by mobile devices.
  • MP4 files will not be converted.

To view photo/video conversion progress:

  1. When photos or videos are being uploaded to Synology NAS, click the Conversion Progress icon appearing at the top-right corner of taskbar.
  2. In the pop-up window, do either below:
    • In the Conversion Progress section, monitor the progress and status of thumbnail creation performed in the background.
    • To pause the conversion, follow the steps below:
      1. Choose Delay for [number] hour(s) from the drop-down menu and click Pause to pause the progress.
      2. Conversion progress will resume after the specified delay time. Alternatively, you can click Continue to resume the process immediately.
      Note: Since conversion progress demands higher CPU resources, pausing the process can save system resources for any other ongoing tasks.
  3. When the conversion process is complete, the icon at the taskbar will disappear.
