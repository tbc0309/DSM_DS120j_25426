Power Schedule

At Control Panel > Hardware & Power > Power Schedule, you can set the system to start up or shut down automatically at a specified time.

To add a new startup/shutdown schedule:

  1. Click Create button to open the Add Schedule dialog.
  2. Choose Startup or Shutdown.
  3. Set the Date and Time for startup or shutdown.
  4. Click OK.
  5. Click Save.

The new schedule will be added to the schedule list.

To view schedule summary:

Click Summary to browse the weekly schedule for startup/shutdown.

Note:

  • You can specify up to 100 schedules for one Synology NAS.
  • In the following cases, the system may fail to shut down automatically:
    • Storage Manager is performing tasks.
    • The system is encrypting share folders.
    • The system is performing Secure Erase on hard drives.
    • DSM is being upgraded.
    • The system is repairing RAID volumes.
    • Package Center is performing tasks.
    • The system is upgrading databases.
    • The system is performing data backup tasks.

Note:

  • Power schedules will not be applied if there's any temporary power interruptions to your Synology NAS. To ensure continuous power supply for your Synology NAS, we highly recommend using SNMP or USB UPS.
  • Only Synology NAS models with package architecture such as Denverton, Grantley, Broadwell (including Broadwellnk, Broadwellntb, and Broadwellntbap), Bromolow, Apollolake, and Purley will still implement the startup schedule after experiencing temporary interruption of power supply. Please refer to this article to learn more about the package architecture of your Synology NAS.
  • EDS14 does not support automatic startup/shutdown. For information on how to schedule automatic DC output on/off on EDS14, please refer to this help article.
