Overview

The Overview page in iSCSI Manager is an overview dashboard for you to quickly monitor the three most important elements of the iSCSI service:

  • Target
  • LUN
  • Events: Unread information, warning, and error events will be shown here.
    • A maximum of 99 unread events can be shown.
    • Warning and error events will be shown first.
    • To view all events, please go to the Log page.
