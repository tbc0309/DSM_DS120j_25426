Create a Volume

Because a volume is constructed on a storage pool, you need to create a storage pool before creating a volume. However, if quick creation mode is supported on your model, you can create a volume and a storage pool simultaneously and Volume Creation Wizard will automatically optimize storage capacity and performance without worrying about technical aspects. Quick creation mode is only available on models that support SHR (Synology Hybrid RAID). For more information, please visit Synology website.

To create a volume with models that support quick creation mode:

You can create both a volume and a storage pool at the same time. When creating a new volume, you can choose one of the following methods:

  • Quick: This is the best option for beginners. Choose this mode if you want to automatically optimize storage capacity and performance, without worrying about technical aspects, such as RAID types.
  • Custom: For users who want to configure the advanced options of volume creation, such as RAID types and capacity allocation. (Refer to here for how to configure the storage pool.)

Follow the steps below to create a volume in the fast creation mode.

  1. Open Storage Manager.
  2. Go to Volume and click Create.
  3. You'll see the volume creation wizard. Select Quick or Custom depending on your needs. Then follow the steps of the wizard to finish creating the volume.

To create a volume with other models:

  1. Make sure your Synology NAS has an existing storage pool with allocatable space.
  2. Click the Create button.
  3. Choose a storage pool on which the new volume will be created.
  4. Set the allocation size for the volume.
  5. Click Apply to confirm the settings.

Note:

  • If you select Better performance as the storage pool type when creating a storage pool, you cannot set the allocation size of volumes created on this storage pool.
  • The maximum volume size varies depending on models. Refer to model specifications for details.
  • The minimum size allocated on a volume must be at least 10 GB.
  • A Btrfs volume will reserve 5% of space for metadata.
