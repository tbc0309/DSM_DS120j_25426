Convert a Single-Volume Storage Pool to a Multiple-Volume Storage Pool

You can add drives to a storage pool and convert the storage pool from one that supports only a single volume to one that supports multiple volumes. The original data on the single volume will not be affected.

To convert a single-volume storage pool to a multiple-volume storage pool:

  1. Launch Storage Manager.
  2. Go to Storage Pool.
  3. Click Action > Convert a Single-Volume Storage Pool to a Multiple-Volume Storage Pool.

Note:

  • This feature is supported on SHR storage pools only. SHR (Synology Hybrid RAID) is available on specific models only. For more information, please visit Synology website.
  • The number of added drives must increase the storage pool size; otherwise, you will not be able to perform the conversion.
  • We recommend using drives in the Synology Products Compatibility List to prevent any issues. Using incompatible drives may lower the system performance or even cause data loss on your Synology NAS.
