Expand the Volume Size

The size of a volume can be changed if the volume is created on a storage pool that supports multiple volumes. However, if a volume is created on a storage pool that supports only a single volume, you can still expand the volume size by adding drives to the storage pool to expand the storage pool capacity first.

For a multiple-volume storage pool:

Please follow the steps below to expand the size of a volume:

  1. Go to the Volume page and select the volume you want to expand.
    Note: You can add drives to the corresponding storage pool to expand the storage pool capacity first if needed. Refer to Add Drives to Expand the Storage Pool Capacity for details.
  2. Click Configure from the Action drop-down menu.
  3. Go to the General tab and enter the new volume size in the Modify allocated size (GB) field, or click the Max button to automatically fill in the maximum allocatable size.
  4. Click OK to finish.

For a single-volume storage pool:

Please follow the steps below to expand the size of a volume:

  1. Add drives to a storage pool to expand the storage pool capacity.
    Note: Refer to Add Drives to Expand the Storage Pool Capacity for details.
  2. The size of the corresponding volume will be expanded automatically once the storage pool capacity has been expanded.
