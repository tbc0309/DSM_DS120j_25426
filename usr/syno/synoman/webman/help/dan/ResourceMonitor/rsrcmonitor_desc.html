Ressourceovervågning

Ressourceovervågning sætter dig i stand til at overvåge CPU-brug, hukommelsesbrug, diskudnyttelse og netværks-flow. Du kan vælge at overvåge i realtid eller se de foregående data.

Bemærk:

  • Foregående data vises kun, hvis Aktiver brugshistorik er markeret på fanen Indstillinger.
  • Ved Docker DSM angiver CPU og hukommelse forbrugsoplysninger om Docker Host.
