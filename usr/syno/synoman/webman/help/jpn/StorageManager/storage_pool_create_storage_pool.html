新しいストレージ プールを作成

重要：

  • 開始する前に、ストレージ ドライブが作成される先のドライブ上に重要なデータがないことを確認してください。増設中に既存のデータはすべて消去されます。
  • HDD/SSD ページを開き、各ドライブの状態が [初期化済み] または [未初期化] となっているか確認してください。
  • 当社は、Synology 製品互換性リストのドライブを使用して問題を防止することを推奨しています。互換性のないドライブを使用すると、システム のパフォーマンスが低下したり、あるいは Synology NAS でデータの損失が生じることさえあります。

ストレージ プールを作成する方法：

  1. [ストレージ マネージャ] を開きます。
  2. [ストレージ プール] に行き、[作成]ボタンをクリックします。
  3. ストレージ プール タイプを選択します。
    • より良いパフォーマンス：このタイプのストレージ プールは単一のボリュームしかサポートしませんが、より良いパフォーマンスを提供します。
    • より高い柔軟性：このタイプのストレージ プールは複数のボリューム、Synology Hybrid RAID (SHR)、および RAID Group 機能をサポートします。
  4. ストレージ プールのプロパティを構成：
    • ストレージ プールの説明：このストレージ プールの説明。
    • RAID タイプ：RAID タイプによって、提供するデータ保護のレベルが異なります。詳しい情報に関しては、全般を参照してください。
    • 1つの RAID あたりのドライブの最小数：選択された RAID タイプに必要なドライブの最小数です。
    • 1つの RAID あたりのドライブの最大数1つの RAID あたりのドライブの数が増えると、RAID の不具合の可能性が増します。RAID Groupを使用して単一の RAID におけるドライブの上限数を調整し、データ セキュリティとストレージ プール キャパシティの間でバランスを取ることをお勧めします
    • RAID Groupの機能：RAID Group の機能が選択された RAID タイプでサポートされているかどうか。単一の RAID にはドライブ数に制限がありますが、RAID Group 機能によりストレージ ブロックが複数の RAIDによって構成できるようになり、容量とドライブの冗長性が増加します。RAID Group は RAID 5、RAID 6、および RAID F1 によってサポートされています。
  5. ウィザードの手順に従ってストレージ プールの構築を完了してください。

注：

  • SHR (Synology Hybrid RAID) は特定のモデルでのみ利用可能です。詳しくは、Synology website をご覧ください。
  • RAID および RAID Group あたりの最大ドライブ数の機能は特定のモデルのみで利用可能です。詳しくは、Synology website をご覧ください。
  • 1つの RAID あたりのドライブの最大数は選択した RAID タイプが RAID Group 機能をサポートしている場合にのみ変更できます。
  • ストレージプールの RAID タイプが RAID 5、RAID 6、RAID F1、SHR (3 ドライブ超) または SHR-2 である場合、システムはそのストレージプールに RAID 再同期 を実行し、アクセス速度を高め、データ回復力を上げます。
