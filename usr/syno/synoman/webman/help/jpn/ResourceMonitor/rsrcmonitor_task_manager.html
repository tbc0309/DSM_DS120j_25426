タスク マネージャ*

[タスク マネージャー] タブでは、サービスおよびプロセスの CPU およびメモリの使用率を表示することができます。

サービス

[サービス] タブをクリックして、CPU 使用率、メモリ使用率、ディスクの書き込み/読み込みステータスを含む、各サービスのさまざまな値を監視します。

処理

[プロセス] タブをクリックすると、各処理の CPU とメモリ使用量を CPU 負荷の少ない順に表示します。

プロセスの状態には、作動中、スリープ中、停止があります。 Linux でのプロセスの状態は、作動中、停止/トレーシング、スリープ中があります。

スリープ中は、次の文字でその種類が示されます。「D」はディスク スリープ、「Z」はゾンビ、「X」は死亡などです。

共有メモリーは処理で使用されている物理的メモリー (他の処理で共有できるもの) を表示します。 プライベート メモリーは処理で使用されている物理的メモリー (他の処理で使用できないもの) を表示します。

____
* 特定のモデルでのみ使用できます。
