ファイルインデックス処理

共有フォルダやそのサブフォルダにインデックスを付けると、ファイルを検索しながら含まれるファイルやコンテンツを簡単に検索できます。

注意：

Synology NAS にマウントされているリモート フォルダ、クラウド フォルダ、.iso フォルダにはインデックスを付けることができません。

フォルダーとファイル コンテンツをインデックス化する：

フォルダのインデックス化を行う際、File Station がフォルダやサブフォルダ/ファイルを名前でインデックス化を行います。これにより、ファイル名で必要なアイテムを検索できるようになります。ファイルの種類で更にインデックス化を行うと、File Station は、関連したファイルを、名前だけでなく、コンテンツでインデックス化します。こうすることで、ファイル名やファイルのコンテンツで、必要なアイテムの検索を行えるようになります。

  1. [コントロールパネル] > [インデックス サービス] > [ファイルインデックス処理] を選択し、[インデックス付きフォルダ リスト] をクリックします。
  2. 開いた画面で [作成] > [選択] をクリックすると、インデックス化の準備ができているフォルダが表示されます。
  3. フォルダを選択し、[選択] をクリックします。インデックス化するために新しいフォルダを作成するには、[フォルダの作成] をクリックします。
  4. インデックス化したいフォルダをクリックし、[選択] をクリックします。
  5. 特定のファイルタイプをインデックス化するには、[ファイル タイプ] に移動し、[画像]、[オーディオ]、[動画]、[ドキュメント] を選択します。一部のファイル タイプ（例：オーディオ）が選択されていない場合は、ファイル名でのみ種類（例：オーディオ ファイル）を検索できます。
  6. [OK] をクリックして、インデックスを完了します。

注意：

  • インデックス化には、最大 1,000 個のフォルダまで選択できます。選択されたフォルダのサブフォルダは、これに含まれません。
  • File Station で [ファイルコンテンツ検索を有効にする] を選択すると、インデックス化したフォルダ内のファイル コンテンツを検索することができます。
  • インデックス化したフォルダをホストするボリュームには、10 MB 以上の空きスペースが必要です。これ以下の場合は、インデックスが一時停止してしまいます。インデックスを再開するには、ボリュームを拡張して [再インデックス] をクリックします。
  • インデックス化したフォルダとそのサブフォルダがある場合は、システムのリソースを節約するためにサブフォルダだけがインデックス化されます。
  • フォルダが完全にインデックス化され、SMB または AFP プロトコルによって Mac コンピューターにマウントした後、Finder のスポットライトを使ってファイル コンテンツを検索することができます。

インデックス化したフォルダを削除する：

  1. [コントロール パネル] > [インデックス サービス] > [ファイルインデックス処理] > [インデックス付きフォルダ リスト] を選択し、インデックス化したフォルダを選択します。
  2. [削除] をクリックすると、フォルダのインデックス化が停止します。

ファイルに再インデックスする

ファイルの再作成は通常の状況では必要ありません。但し、フォルダのファイルが特定の検索条件に一致しているが、検索中に紛失エラーが表示された場合は、インデックス構造を再構築するために再インデックスを実行する必要があります。

共有フォルダのファイルを再作成する

  1. [コントロール パネル] > [インデックス サービス] > [ファイルインデックス処理] > [インデックスを付けたフォルダー リスト] の順に選択します。
  2. 再インデックスするために、リストされたフォルダを選択します。
  3. [再インデックス] をクリックします。

注意：

File Station はリモート フォルダまたは外部ストレージ デバイス（USB ドライブ/SD カードなど）上でのファイル インデックス化に対応していません。

デスクトップでファイルのインデックス化を管理する

インデックス化が開始されると、タスクバーに 「インデックス処理」 (浮動拡大鏡アイコン) と表示されます。インデックス化にはより多くの CPU リソースが必要となるため、これを実行することで CPU リソースを節約することができます。

  • インデックス化を一時停止/再開するには、再生ボタンをクリックしてください。
  • 適切な時間までインデックス化を遅らせるには、ドロップダウン メニューから適切なオプションを選択してください。
  • インデックス化しているフォルダを確認するには、[インデックス付きフォルダ リストを表示] をクリックします。

注意：

  • インデックス化を一時停止した場合は、デスクトップのタスクバーから再生ボタンをクリックすると再開できます。
