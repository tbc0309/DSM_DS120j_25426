ユーザーの作成

Synology NAS で複数のユーザー アカウントを作成、管理できます。

ユーザーを作成する：

  1. [コントロール パネル] > [ユーザー] の順に選択します。
  2. [作成] ボタンをクリックした後で、[ユーザーの作成] をクリックします。 これを実行すると、ユーザー作成ウィザードが起動します。
  3. [ユーザー情報] ページで次のオプションを指定してください。
    • 名前：ユーザーの名前を入力します。 これは必須項目です。
    • 説明：ユーザーに関する簡単な説明を入力します。
    • 電子メール：ユーザーの Ｅ メール アドレスを入力します。 パスワード リセット メッセージなどのシステム通知は、ここで指定したアドレスに送信されます。
    • 新規ユーザーに通知メールを送る：このオプションを有効にすると、システムがユーザーに対して新しいアカウントについて知らせるためにメッセージを送信します。 このオプションを有効にするには、[メイン メニュー] > [コントロール パネル] > [通知] > [電子メール] を選択して、システムの電子メール通知を有効にする必要があります。 デフォルトにより、通知メッセージにはユーザーのパスワードは表示されません。 通知メッセージにパスワードも記載して欲しい場合は、[通知メールにユーザー パスワードを表示する] にチェックマークを付けます。
    • アカウントパスワードの変更をユーザーに許可しない：このオプションを有効にすると、ユーザーは DSM にログインするためのパスワードを変更することができなくなります。
  4. [グループ] > [作成] > [グループ作成ウィザード] ページでは、新しいユーザーが属するグループを指定します（例：users、administrators、その他）。
  5. [共有フォルダに権限を割り当てる] ページで、新しいユーザーのアクセス権を変更することによって、このユーザーがアクセスできる共有フォルダを選択します。 権限の優先順位は、 アクセスなし > 読み取り/書き込み > 読み取り専用です。
    • プレビュー：現在の設定とユーザーが属するグループの属性に基づいて、ユーザーの実際の共有フォルダに対するアクセス権が表示されます。
    • グループ権限：ユーザーが属するグループの属性に基づいて、共有フォルダに対するアクセス権が表示されます。
  6. [ユーザーデータ容量の割り当て設定] ページで、新しいユーザーが使用できるストレージ スペースをコントロールできます。 ユーザーが使用できる容量を制限するには、各ボリューム/共有フォルダ について [容量割り当てを有効にする]の下のボックスにチェックマークを付けます。 次に、[割り当て容量] 欄に使用できる最大量を入力します（例：10 GB）。

    注意：

    • 各共有フォルダへの割り当てを有効にするオプションは、VDSM および以下のパッケージ アーキテクチャのあるモデルのみでサポートされます。 Apollo Lake、Avoton、Braswell、Broadwell、Bromolow、Cedarview、および Grantley。 お使いのモデルがどのパッケージ アーキテクチャをサポートするかについては、この記事をご覧ください。
    • ユーザーを作成するオプションは EDS14 には適用されません。
    • DDSM はユーザーの割り当て設定をサポートしません。
  7. [アプリケーション権限の割り当て] ページで、ユーザーがアクセスできるサービスをコントロールできます。
  8. [グループの速度制限設定] ページでは、各サービス毎に速度制限を有効にして（例：File Station、FTP、その他）、ユーザーがファイルを転送する時に使用する帯域量を制限することができます。 それぞれのサービスに対して、次のいずれかのオプションを選択できます。
    • なし： ユーザーが属するグループと同じ速度制限の設定が適用されます。
    • 固定：常に一定の速度制限が適用されます。 左側の欄にアップロードとダウンロードの速度制限を設定します。
    • 可変：スケジュールに基づいて、ユーザーに 2 つの速度制限を指定して適用することができます。 速度制限の設定やスケジュールを変更するには、[カスタマイズ] ボタンをクリックします。
  9. 設定を確定したら、[適用] をクリックして完了してください。

ユーザー名の制限：

  • ユーザー名は大文字と小文字を区別しません。また、以下の記号を除き、1～64 のユニコード文字で構成できます：
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • 最初の文字にマイナス記号またはスペースは使用できません。また、最後の文字にスペースは使用できません。
  • 次のユーザー名は、システム用に予約されています： mailer-daemon、postmaster。
  • ユーザーの説明は、最大 64 文字のユニコードが表示可能です。

パスワード情報：

  • パスワードは大文字と小文字が区別され、文字、数字、記号、スペースを含め最高 127 文字まで使用できます。

割り当て制限：

  • 各ユーザーに割り当てられる最大量は、約 4 TB (4095 GB) です。

    • 注意：

    • この機能は EDS14 や DDSM でサポートされていません。
