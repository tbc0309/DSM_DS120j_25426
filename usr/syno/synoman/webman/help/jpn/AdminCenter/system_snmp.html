SNMP

SNMP サービスは、ネットワーク管理ソフトウェアを使って Synology NAS を監視します。[コントロール パネル] > [端末& SNMP] > [SNMP] を選択してください。現在、SNMPv1、SNMPv2c、SNMPv3 プロトコルに対応しています。

SNMP サービスを有効にする

[SNMP サービスを有効にする] を選択し、次の手順に従ってください。

SNMPv1、SNMPv2c サービスを有効にする：

  1. [SNMPv1、SNMPv2c サービス] にチェックマークを付けます。
  2. [コミュニティ] 名を入力します。
  3. [適用] をクリックします。

SNMPv3 サービスを有効にする：

  1. [SNMPv3 サービス] にチェックマークを付けます。
  2. ユーザー名 と パスワード を入力します。使用する[プロトコル]を選択します。
  3. [適用] をクリックします。

SNMP プライバシーを有効にする方法：

  1. [SNMP プライバシーを有効にする] にチェックマークを付けます。
  2. 使用する[プロトコル]を選択します。次にパスワードを入力します。
  3. [適用] をクリックします。

さらに、名前、場所、連絡先を入力すると、Synology NAS の識別に役立ちます。このフィールドはオプションです。

注：

  • SNMP サービスは、IPv4 と IPv6 に対応しています。
  • SNMP サービスは、SNMPv1、SNMPv2c、SNMPv3 に同時に対応します。

SNMPv1、SNMPv2c コミュニティの制限：

コミュニティ名は、1 個から 64 個までの半角英数字に限られます。次の文字は使用できません。
"" ' \ およびスペース。

SNMPv3 ユーザー名の制限：

ユーザー名は、1 個から 64 個までの半角英数字に限られます。次の文字は使用できません。
"" ' \ およびスペース。

SNMPv3 パスワードの制限：

パスワードは半角英数字、記号、スペースを使用した 8～127 文字まで登録可能です。英字の大文字 小文字も区別されるので、気を付けてください。次の文字は使用できません。
"" ' \ およびスペース。

SNMP プライバシー パスワードの制限：

パスワードは半角英数字、記号、スペースを使用した 8～127 文字まで登録可能です。英字の大文字 小文字も区別されるので、気を付けてください。次の文字は使用できません。
"" ' \ およびスペース。

Synology MIB ファイルのダウンロード

Synology はネットワーク管理システムを使って、システム、ディスク、RAID ボリュームの状態など、Synology NAS の状態を監視できる機能を提供します。

注：

  • RAID は EDS14 などのディスクレス モデルでは使用できません。.詳しくは、Synology ウェブサイト をご覧ください。

詳しい情報や、関連ファイルをダウンロードしたい方は、Synology ウェブサイト をご覧ください。

注：

  • ネットワーク管理システムは、監視デバイスの定義に使用される MIB ファイルに依存します。詳しくは、ここをご覧ください。
