ユーザー

柔軟なユーザー管理オプションを使用することで、家族や仕事の各メンバーにユーザー アカウントを作成することができます。 ユーザーを作成すると、共有フォルダへのアクセス権限、ストレージ/共有フォルダの割り当て量、帯域制限など、各ユーザーの権限を定義し、管理することが可能になります。

    注意：
  1. 共有フォルダ管理は、Bromolow、Grantley、Avoton、Braswell、Broadwell、Kvmx64、Cedarview、Appololake、Denverton の各プラットフォームにのみ適用されます。 この機能をサポートするモデルについては、この記事をご覧ください。
  2. ストレージ/共有フォルダ割り当て管理は Docker DSM には適用されません。
