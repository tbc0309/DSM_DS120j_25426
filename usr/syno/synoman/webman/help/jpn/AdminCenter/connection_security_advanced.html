詳細

次の詳細設定は、DSM ブラウズ セッションで [コントロール パネル] > [セキュリティ] > [詳細] を選択すると変更できます。

HTTP 圧縮

HTTP 圧縮を Synology NASに対して有効化できます。 ネットワーク トラフィックの帯域幅を減らすと、ウェブ ページの読み込み速度を上げることができます。

HTTP 圧縮を有効化/無効化する方法：

  • [HTTP 圧縮]を[コントロール パネル] > [セキュリティ] > [詳細] でチェック/チェック解除します。

TLS / SSL プロファイル レベル

TLS / SSL プロファイルで、HTTPSの暗号化された接続のセキュリティ レベルを選ぶことができます。 それは以下の3レベルを提供します:

  • 最新の互換性 - 最新のブラウザとモバイル デバイスのみと互換性があります。
  • 標準の互換性 - 最新の互換性と古い下位互換性の中間です。これはシステムのデフォルト値です。
  • 古い下位互換性 - このオプションは安全ではありませんが、ほとんどのブラウザとモバイル デバイスをサポートします。

TLS / SSL プロファイル レベルをすべてのサービスに対して:構成する方法：

TLS /SSL プロファイル レベルはメイン ページで選択できます。これは[デフォルト プロファイル]を使用するすべてのサービスに適用されます。

  1. 適切な TLS / SSL プロファイル レベルを選択します。
  2. [OK]をクリックしてください。

TLS / SSL プロファイル レベルを特定のサービスに対して:構成する方法：

サービスのセキュリティ要件に基づいて、サービスのプロファイル レベルを変えることができます。

  1. [カスタム設定]をクリックして、すべてのサービスおよびそれに対応するプロファイル レベルを表示します。
  2. ターゲットのサービスの現在のプロファイル レベルをクリックします。
  3. ドロップダウン メニューから適切なプロファイル レベルを選択します。
  4. [OK]をクリックしてください。

注：

  • [システム デフォルト]プロファイル レベルはサービス リストにない接続に適用されます。
