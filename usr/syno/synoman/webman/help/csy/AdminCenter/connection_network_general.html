Obecné

Obecné volby sítě, například výchozí brána, server DNS, proxy a tunelování IPv6, se nacházejí v části Ovládací panel > Síť > Obecné.

Přejmenování zařízení Synology NAS

Chcete-li svoje zařízení Synology NAS přejmenovat, zadejte nový název serveru a klikněte na možnost Použít.

V názvu serveru se rozlišují velká a malá písmena a může se skládat z 1 až 15 znaků – písmen, čísel, podtržítek a symbolů mínus. První znak názvu musí být písmeno.

Určení výchozí brány

Stačí zadat IP adresu pro výchozí bránu a uložit nastavení kliknutím na možnost Použít.

Určení IP adresy služby DNS Server

IP adresu serveru DNS můžete nakonfigurovat ručně. V takovém případě musíte zadat IP adresu upřednostňovaného serveru DNS. Můžete také zadat alternativní IP adresu serveru DNS jako zálohu pro případ výskytu problémů s upřednostňovaným serverem DNS. Po nakonfigurování IP adres serveru DNS podle svých požadavků uložte nastavení kliknutím na možnost Použít.

Rozšířená nastavení:

  • Pokud je cílová IP adresa totožná s místní adresou nakonfigurovanou na příchozím rozhraní, odpovědět na požadavek ARP: ARP neodpovídá v síťovém protokolu na požadavek z totožné IP adresy. Tuto možnost povolte v případě, že chcete odpovídat na požadavek ARP, a přesvědčte se, že je tabulka ARP kompletní.
  • Povolit více bran: Tuto možnost povolte v případě, že chcete v rozhraní podporovat více než jednu bránu.
  • Povolit detekci konfliktů IP adres: Tuto možnost povolte v případě, že chcete automaticky zjišťovat, jestli se v síti nachází stejná IP adresa.

Poznámka:

Pokud jsou všechna síťová rozhraní konfigurována automaticky nebo pokud používáte připojení PPPoE, pole Server DNS a Výchozí brána se nastaví automaticky a nelze je změnit.

Připojení prostřednictvím služby Proxy Server

Chcete-li na zařízení Synology NAS povolit připojení prostřednictvím serveru proxy, zaškrtněte možnost Připojit prostřednictvím serveru proxy, zadejte hodnoty Adresa a Port a poté klikněte na možnost Použít.

Poznámka:

  • Kliknutím na možnost Rozšířená nastavení provedete konfiguraci serveru proxy HTTP, proxy HTTPS a údajů pro ověření.
  • Podle výchozího nastavení jsou nastavení serveru proxy HTTPS stejná jako nastavení serveru proxy HTTP.
  • Pokud zaškrtnete možnost Připojit prostřednictvím serveru proxy, funkce Aktualizace firmwaru DSM, Centrum balíčků a další balíčky se budou vždy připojovat k internetu prostřednictvím serveru proxy.
  • Pokud zaškrtnete možnost Záložní server proxy pro místní adresy, tak se přesvědčte, že server DNS používaný zařízením Synology NAS dokáže vyřešit dotazy na zařízení v místní síti.

Povolování tunelového propojení IPv6

Pokud protokol IPv6 zcela nahrazuje protokol IPv4, je potřeba, aby tunel IPv6 umožnil hostiteli s adresou IPv6 dosáhnout na služby IPv4. Tímto způsobem umožníte stávajícím sítím IPv4 nést provoz IPv6.
Pokud si nejste jistí, jak nakonfigurovat protokol IPv6 nebo vyřešit problémy s připojením, obraťte se na poskytovatele internetových služeb.

Povolení tunelového propojení IPv6:

  1. Zaškrtnutím políčka Povolit tunel zapněte podporu tunelového propojení.
  2. Do pole Adresa serveru zadejte název hostitele nebo IP adresu poskytovatele tunelového propojení, například anonymous.freenet6.net.
  3. Vyberte jednu z možností Připojit anonymně nebo Připojit se stávajícím účtem.

Poznámka:

Vaši externí adresu IPv6 může poskytovatel služeb automaticky aktualizovat. Zařízení Synology NAS však neodstraní starou adresu IPv6 okamžitě.
