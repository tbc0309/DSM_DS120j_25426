Kopírovat uživatele

Nastavení stávajícího uživatele je možné zkopírovat do nového uživatele. Můžete tak rychle vytvořit nového uživatele, aniž byste museli znovu konfigurovat stejná nastavení.

  1. Vyberte uživatele, kterého chcete zkopírovat.
  2. Klikněte na možnost Vytvořit > Kopírovat uživatele. Otevře se okno Průvodce vytvořením uživatele.
  3. Nastavení vybraného uživatele se automaticky použijí na nového uživatele (kromě osobních informací, například uživatelského jména a e-mailové adresy).
