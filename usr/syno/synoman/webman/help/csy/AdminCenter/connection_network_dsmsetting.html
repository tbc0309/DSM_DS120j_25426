Nastavení systému DSM

V části Ovládací panel > Síť > Nastavení DSM je možné změnit číslo portu, který klienti používají při přístupu k zařízení Synology NAS.

Změna výchozího čísla portu:

Připojení HTTP a HTTPS jsou v systému DSM povolená vždy. Výchozí port HTTP je 5000 a výchozí port HTTPS je 5001.

  1. Do pole HTTP nebo HTTPS zadejte číslo portu.
  2. Klikněte na možnost Použít.

Přesměrování připojení HTTP na připojení HTTPS:

Když se klienti pokouší přistupovat k zařízení Synology NAS přes port HTTP, můžete je přesměrovat na port HTTPS.

  1. Vyberte možnost Automaticky přesměrovat připojení HTTP na HTTPS (kromě služeb Web Station a Photo Station).
  2. Klikněte na možnost Použít.

Porty používané pro připojení HTTPS:

  • DSM: Výchozím číslem portu je 5001, takže do systému DSM je možné se bezpečně připojit zadáním následující adresy: https://název-hostitele-serveru:5001/
  • Služby Web Station a Photo Station: Výchozím číslem portu je 443, takže ke službám Web Station a Photo Station je možné se bezpečně připojit zadáním následující adresy: https://název-hostitele-serveru/

Povolení HTTP/2:

Povolením HTTP/2 můžete zrychlit připojení HTTPS.

  1. Zaškrtněte možnost Povolit HTTP/2.
  2. Klikněte na možnost Použít.

Přizpůsobení maximálního počtu souběžných připojení HTTP:

Povolením funkce Přizpůsobit maximální počet souběžných připojení HTTP můžete na zařízení Synology NAS zvýšit počet připojení HTTP.

Poznámka:

  • Vztahuje se pouze na platformy Bromolow, Cedarview, Avoton, Grantley, Broadwell, Dockerx64, Broadwellnk, Apollolake a Braswell. Chcete-li zjistit, jestli váš model tuto funkci podporuje, přečtěte si tento článek.
  • Čím vyšší je maximální počet souběžných připojení HTTP, tím více systémové paměti vyžaduje.

Přizpůsobení záhlaví „Server“ v odpovědích HTTP:

  1. Zaškrtněte možnost Povolit v odpovědích HTTP záhlaví Server.
  2. Zadejte záhlaví pro webový server (např. nginx).
  3. Klikněte na možnost Použít.

Zrušením zaškrtnutí tohoto políčka můžete záhlaví „Server“ zakázat.

Doména

V části Doména je možné zadat vlastní název domény pro přístup k zařízení Synology NAS.

Povolení vlastní domény:

  1. Zaškrtněte políčko Povolit vlastní doménu.
  2. Zadejte použitý název domény pro své zařízení Synology NAS.
  3. Pokud zaškrtnete také políčko Povolit HSTS, vynutíte v této doméně šifrovaná připojení HTTPS.
  4. Klikněte na možnost Použít.

Poznámka:

  • Každý název domény je možné na zařízení Synology NAS použít pouze pro jednu samostatnou aplikaci.
  • Jestliže název domény nepřekročí délku 15 znaků a splňuje pravidla pro názvy NetBIOS, systém název domény (např. MojeDSM) zaregistruje a bude vysílat automaticky. Uživatelé systému Windows se k ploše systému DSM mohou ve stejné místní síti dostat pomocí podobné adresy URL: http://MyDSM nebo https://MyDSM
  • Když k aplikacím přistupujete prostřednictvím adres URL, které používají názvy domén, budete procházet přes standardní port HTTP (80) nebo standardní port HTTPS (443).
