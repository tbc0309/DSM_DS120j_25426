Uživatel

Pomocí flexibilních možností správy uživatelů je možné vytvářet uživatelské účty pro jednotlivé členy rodiny nebo firmy. Vytváření uživatelů umožňuje definování a správu oprávnění pro jednotlivé osoby, jako jsou například přístupová oprávnění sdílených složek, kvóty kapacity úložiště/sdílené složky nebo omezení šířky pásma.

    Poznámka:
  1. Správa sdílené složky funguje pouze na platformách Bromolow, Grantley, Avoton, Braswell, Broadwell, Kvmx64, Cedarview, Appololake a Denverton. Chcete-li zjistit, jestli váš model podporuje tuto funkci, přečtěte si tento článek.
  2. Správa kvóty sdílené složky nebo úložiště se nevztahuje na systém Docker DSM.
