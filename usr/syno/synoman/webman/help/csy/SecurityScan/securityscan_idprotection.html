Analýza přihlášení

Analýza přihlášení umožňuje správcům systému kontrolovat informace o zabezpečení aktuálního systému a účtů.

Analýzu přihlášení můžete zakázat a pravidla upravit v části Security Advisor > Rozšířené > Přizpůsobit kontrolní seznam.

Nové přihlašovací aktivity

Nástroj Security Advisor analyzuje přihlašovací údaje každého uživatele, včetně uživatele-agenta HTTP, IP adresy a geografického umístění. V případě zjištění nové přihlašovací aktivity odešle nástroj Security Advisor upozornění správci systému a příslušnému uživateli.

Pokud se potvrdí, že je určitý záznam přihlášení nenormální, důrazně doporučujeme použít silnější heslo, povolit dvoufázové ověření nebo dokonce omezit povolené IP adresy a zabezpečení účtu tak zvýšit.

Hádání hesla (útoky hrubou silou)

Nástroj Security Advisor analyzuje neúspěšné pokusy o přihlášení z různých IP adres a protokolů. Pokud se někdo pokusí o přihlášení hádáním hesel a příliš často neuspěje, odešle nástroj Security Advisor upozornění správci a příslušnému uživateli. Správce systému může zvýšit zabezpečení účtu povolením funkce Automatický blok nebo Ochrana účtu (v části Ovládací panel > Zabezpečení > Účet).
